概要

第6章では、代表的な連続型分布について解説する。 連続一様分布、正規分布、指数分布、ガンマ分布、ベータ分布、コーシー分布、対数正規分布、2変量正規分布、多変量正規分布、混合正規分布を扱う。 また、正規分布からの無作為標本に基づく統計量の標本分布に現れるカイ二乗分布、$t$ 分布、$F$ 分布（非心分布を含む）も扱う。

キーワード：連続一様分布、正規分布、指数分布、ガンマ分布、ベータ分布、コーシー分布、対数正規分布、2変量正規分布、多変量正規分布、混合正規分布、カイ二乗分布、$t$ 分布、$F$ 分布（非心分布を含む）

重要概念

連続一様分布 $U(a, b)$

  • $a < b$ に対し、確率密度関数：$f(x) = \dfrac{1}{b - a} \quad (a \leq x \leq b)$
  • $E[X] = \dfrac{a + b}{2}$、$V[X] = \dfrac{(b - a)^2}{12}$
  • モーメント母関数：$M(t) = \dfrac{e^{bt} - e^{at}}{(b - a)t}$

正規分布 $N(\mu, \sigma^2)$

  • 確率密度関数：$f(x) = \dfrac{1}{\sqrt{2\pi}\,\sigma} \exp\!\left(-\dfrac{(x - \mu)^2}{2\sigma^2}\right)$
  • $\mu = 0,\; \sigma = 1$ のとき $N(0, 1)$ を標準正規分布という。確率密度関数を $\phi(z)$、累積分布関数を $\Phi(z)$ で表す
  • $E[X] = \mu$、$V[X] = \sigma^2$
  • モーメント母関数：$M(t) = \exp\!\left(\mu t + \dfrac{\sigma^2 t^2}{2}\right)$
  • $X \sim N(\mu, \sigma^2)$ のとき $Z = (X - \mu)/\sigma \sim N(0, 1)$（標準化）
  • 再生性：$X_1 \sim N(\mu_1, \sigma_1^2),\; X_2 \sim N(\mu_2, \sigma_2^2)$ が独立 $\Rightarrow$ $X_1 + X_2 \sim N(\mu_1 + \mu_2,\; \sigma_1^2 + \sigma_2^2)$

指数分布 $\mathrm{Exp}(\lambda)$

  • $\lambda > 0$ に対し、確率密度関数：$f(x) = \lambda e^{-\lambda x},\; x > 0$
  • 累積分布関数：$F(x) = 1 - e^{-\lambda x},\; x > 0$
  • $E[X] = \dfrac{1}{\lambda}$、$V[X] = \dfrac{1}{\lambda^2}$
  • モーメント母関数：$M(t) = \dfrac{\lambda}{\lambda - t},\; t < \lambda$
  • 無記憶性：$P(X \geq t_1 + t_2 \mid X \geq t_1) = P(X \geq t_2)$

ガンマ分布 $\mathrm{Ga}(a, b)$

  • $a > 0,\; b > 0$ に対し、確率密度関数：$f(x) = \dfrac{1}{\Gamma(a)\, b^a}\, x^{a-1}\, e^{-x/b},\; x > 0$
  • $\Gamma(a) = \displaystyle\int_0^\infty x^{a-1} e^{-x}\, dx$（ガンマ関数）
  • $a = 1$ のとき $\mathrm{Ga}(1, b) = \mathrm{Exp}(1/b)$
  • $E[X] = ab$、$V[X] = ab^2$
  • モーメント母関数：$M(t) = (1 - bt)^{-a},\; t < 1/b$
  • 再生性：$X_1 \sim \mathrm{Ga}(a_1, b),\; X_2 \sim \mathrm{Ga}(a_2, b)$ が独立 $\Rightarrow$ $X_1 + X_2 \sim \mathrm{Ga}(a_1 + a_2, b)$

ベータ分布 $\mathrm{Be}(a, b)$

  • $a > 0,\; b > 0$ に対し、確率密度関数：$f(x) = \dfrac{1}{B(a, b)}\, x^{a-1}(1 - x)^{b-1},\; 0 < x < 1$
  • $B(a, b) = \displaystyle\int_0^1 x^{a-1}(1-x)^{b-1}\, dx$（ベータ関数）
  • $E[X] = \dfrac{a}{a + b}$、$V[X] = \dfrac{ab}{(a+b)^2(a+b+1)}$
  • ガンマ分布からの導出：$X_1 \sim \mathrm{Ga}(a_1, b),\; X_2 \sim \mathrm{Ga}(a_2, b)$ が独立のとき $\dfrac{X_1}{X_1 + X_2} \sim \mathrm{Be}(a_1, a_2)$

コーシー分布

  • 確率密度関数：$f(x) = \dfrac{1}{\pi(1 + x^2)}$
  • 裾が重い分布であり、平均やより高次のモーメントが存在しない
  • 一般に位置母数 $\mu$ と尺度母数 $\sigma > 0$ を導入し $\dfrac{1}{\pi\sigma\!\left(1 + \left(\frac{x - \mu}{\sigma}\right)^2\right)}$ とする

対数正規分布 $\Lambda(\mu, \sigma^2)$

  • $Y \sim N(\mu, \sigma^2)$ のとき $X = e^Y > 0$ が従う分布
  • 確率密度関数：$f(x) = \dfrac{1}{\sqrt{2\pi}\,\sigma\, x} \exp\!\left(-\dfrac{(\log x - \mu)^2}{2\sigma^2}\right),\; x > 0$
  • $X \sim \Lambda(\mu, \sigma^2)$ のとき $\log X \sim N(\mu, \sigma^2)$
  • $E[X] = \exp\!\left(\mu + \dfrac{\sigma^2}{2}\right)$、$V[X] = \exp(2\mu + \sigma^2)(e^{\sigma^2} - 1)$

2変量正規分布 $N_2(\boldsymbol{\mu}, \Sigma)$

  • 平均ベクトル $\boldsymbol{\mu} = (\mu_1, \mu_2)^\top$、分散共分散行列 $\Sigma = \begin{pmatrix} \sigma_1^2 & \rho\sigma_1\sigma_2 \\ \rho\sigma_1\sigma_2 & \sigma_2^2 \end{pmatrix}$
  • $\rho$ は $X_1$ と $X_2$ の相関係数。$\rho = 0$ のとき $X_1$ と $X_2$ は独立
  • 周辺分布：$X_1 \sim N(\mu_1, \sigma_1^2)$、$X_2 \sim N(\mu_2, \sigma_2^2)$
  • 条件付き分布 $X_2 \mid X_1 = x_1$：
    • 期待値：$\mu_2 + \rho\dfrac{\sigma_2}{\sigma_1}(x_1 - \mu_1)$
    • 分散：$\sigma_2^2(1 - \rho^2)$
  • 1次結合：任意の $\mathbf{c} = (c_1, c_2)^\top$ に対し $\mathbf{c}^\top \mathbf{X} \sim N(\mathbf{c}^\top \boldsymbol{\mu},\; \mathbf{c}^\top \Sigma \mathbf{c})$
  • モーメント母関数：$M(\mathbf{t}) = \exp\!\left(\boldsymbol{\mu}^\top \mathbf{t} + \dfrac{1}{2}\mathbf{t}^\top \Sigma \mathbf{t}\right)$

多変量正規分布 $N_p(\boldsymbol{\mu}, \Sigma)$

  • 2変量の一般化。$p$ 次ベクトル $\boldsymbol{\mu}$ と $p \times p$ 正定値行列 $\Sigma$ に対する分布
  • 確率密度関数：$f(\mathbf{x}) = \dfrac{1}{(2\pi)^{p/2} (\det\Sigma)^{1/2}} \exp\!\left(-\dfrac{1}{2}(\mathbf{x} - \boldsymbol{\mu})^\top \Sigma^{-1}(\mathbf{x} - \boldsymbol{\mu})\right)$
  • モーメント母関数は2変量の場合と同じ形

混合正規分布

  • $K$ 個の正規分布 $N(\mu_j, \sigma_j^2)$ の確率密度関数 $f_j(x)$ と混合比率 $p_1, \ldots, p_K$（$p_j > 0$、$\sum p_j = 1$）に対し
  • 確率密度関数：$f(x) = p_1 f_1(x) + \cdots + p_K f_K(x)$
  • $K = 2$、$p_1 = p_2 = 1/2$、$\sigma_1 = \sigma_2 = \sigma$ のとき、二峰性の条件は $|\mu_1 - \mu_2|/\sigma > 2$

標本分布

カイ二乗分布 $\chi^2(n)$

  • $Z_i \sim N(0, 1)$（$i = 1, \ldots, n$）が互いに独立のとき $Y = Z_1^2 + \cdots + Z_n^2 \sim \chi^2(n)$
  • ガンマ分布 $\mathrm{Ga}(n/2, 2)$ と一致
  • $E[Y] = n$、$V[Y] = 2n$
  • モーメント母関数：$M(t) = (1 - 2t)^{-n/2},\; t < 1/2$
  • 再生性：$Y_1 \sim \chi^2(n_1),\; Y_2 \sim \chi^2(n_2)$ が独立 $\Rightarrow$ $Y_1 + Y_2 \sim \chi^2(n_1 + n_2)$
  • 正規母集団からの標本分散：$X_1, \ldots, X_n \sim N(\mu, \sigma^2)$ のとき
    • $\bar{X}$ と $S^2$ は独立
    • $\sqrt{n}(\bar{X} - \mu)/\sigma \sim N(0, 1)$
    • $(n-1)S^2/\sigma^2 = \displaystyle\sum_{i=1}^{n}\left(\frac{X_i - \bar{X}}{\sigma}\right)^2 \sim \chi^2(n-1)$

非心カイ二乗分布 $\chi^2(n, \lambda)$

  • $Z_i \sim N(\mu_i, 1)$ のとき $Y = Z_1^2 + \cdots + Z_n^2 \sim \chi^2(n, \lambda)$（$\lambda = \mu_1^2 + \cdots + \mu_n^2$）
  • $\chi^2(n, 0) = \chi^2(n)$
  • $E[Y] = n + \lambda$、$V[Y] = 2n + 4\lambda$
  • 再生性：$Y_1 \sim \chi^2(n_1, \lambda_1),\; Y_2 \sim \chi^2(n_2, \lambda_2)$ が独立 $\Rightarrow$ $Y_1 + Y_2 \sim \chi^2(n_1 + n_2,\; \lambda_1 + \lambda_2)$

$t$ 分布 $t(n)$

  • $Z \sim N(0, 1),\; Y \sim \chi^2(n)$ が独立のとき $T = \dfrac{Z}{\sqrt{Y/n}} \sim t(n)$
  • 確率密度関数：$f(t) = \dfrac{\Gamma((n+1)/2)}{\sqrt{n\pi}\,\Gamma(n/2)} \left(1 + \dfrac{t^2}{n}\right)^{-(n+1)/2}$
  • $n = 1$ のときコーシー分布に一致
  • $n \to \infty$ のとき標準正規分布 $N(0, 1)$ に収束
  • $E[T] = 0 \;(n > 1)$、$V[T] = \dfrac{n}{n-2} \;(n > 2)$
  • $n = 1$ のとき平均は存在せず、$n = 1, 2$ のとき分散は存在しない

非心 $t$ 分布 $t(n, \lambda)$

  • $t$ 分布の定義で $Z \sim N(\lambda, 1)$ とした場合。$t(n, 0) = t(n)$
  • $t$ 統計量：$t = \dfrac{\bar{X} - \mu_0}{S/\sqrt{n}}$
    • $\mu = \mu_0$ のとき $t \sim t(n-1)$
    • $\mu \neq \mu_0$ のとき $t \sim t(n-1,\; \sqrt{n}\,\mu/\sigma)$（非心 $t$ 分布）
  • $t$ 分布は検定統計量の帰無分布に、非心 $t$ 分布は検出力に現れる

$F$ 分布 $F(n_1, n_2)$

  • $Y_1 \sim \chi^2(n_1),\; Y_2 \sim \chi^2(n_2)$ が独立のとき $X = \dfrac{Y_1/n_1}{Y_2/n_2} \sim F(n_1, n_2)$
  • $E[X] = \dfrac{n_2}{n_2 - 2} \;(n_2 > 2)$、$V[X] = \dfrac{2n_2^2(n_1 + n_2 - 2)}{n_1(n_2 - 2)^2(n_2 - 4)} \;(n_2 > 4)$
  • 2標本の標本分散の比：$\dfrac{S_1^2/\sigma_1^2}{S_2^2/\sigma_2^2} \sim F(n_1 - 1, n_2 - 1)$

非心 $F$ 分布 $F(n_1, n_2, \lambda)$

  • $F$ 分布の定義で $Y_1 \sim \chi^2(n_1, \lambda)$ とした場合。$F(n_1, n_2, 0) = F(n_1, n_2)$
  • $T \sim t(n, \lambda)$ のとき $T^2 \sim F(1, n, \lambda^2)$
  • 回帰モデルや分散分析モデルにおける $F$ 検定の検出力の計算に現れる

例題・演習

問6.1：正規分布の偏差値と四分位範囲

受験者1000人のテストの得点が $N(65, 10^2)$ に従う。A君は85点、B君は60点。

(1) A君およびB君の偏差値はいくらか。

標準化得点：A君 $(85 - 65)/10 = 2$、B君 $(60 - 65)/10 = -0.5$

$$\text{A君の偏差値} = 50 + 10 \times 2 = \mathbf{70}$$ $$\text{B君の偏差値} = 50 + 10 \times (-0.5) = \mathbf{45}$$

(2) A君の得点とB君の得点の間に入る受験者の人数はおおよそ何人か。

標準化得点が $-0.5$ から $2$ に入る割合は

$$P(-0.5 < Z \leq 2) = \Phi(2) - \Phi(-0.5) = (1 - 0.0228) - 0.3085 = 0.6687$$

$1000 \times 0.6687 = 668.7$ より、おおよそ 669人。

(3) 全受験者の得点の箱ひげ図を描いた場合、四分位範囲はいくらか。

$N(0, 1)$ の下側25%点は $-0.675$ なので

$$x_{0.25} = 65 - 0.675 \times 10, \quad x_{0.75} = 65 + 0.675 \times 10$$
$$\text{四分位範囲} = x_{0.75} - x_{0.25} = 2 \times 0.675 \times 10 = 13.5 \approx \mathbf{14\text{点}}$$

(4) 65点以上の受験者のみを集めた場合、彼らの得点の平均値はおおよそいくらか。

$Z = (X - 65)/10 \sim N(0,1)$ とする。$Z \geq 0$ のもとでの条件付き分布を考えると

$$E[Z \mid Z \geq 0] = \int_0^\infty z \cdot 2\phi(z)\, dz = \frac{2}{\sqrt{2\pi}} = \sqrt{\frac{2}{\pi}} \approx 0.798$$
$$E[X \mid X \geq 65] = 65 + 10 \times 0.798 = 72.98 \approx \mathbf{73\text{点}}$$

問6.2：2変量正規分布の相関と条件付き期待値

英語能力試験の Listening（平均305, 標準偏差80）、Reading（平均250, 標準偏差90）、Total（平均555, 標準偏差150）。

(1) Listening と Reading の相関係数はいくらか。

$V[\text{Total}] = V[L] + V[R] + 2\,\mathrm{Cov}(L, R)$ より

$$\mathrm{Cov}(L, R) = \frac{150^2 - 80^2 - 90^2}{2} = \frac{22500 - 6400 - 8100}{2} = 4000$$
$$\rho = \frac{4000}{80 \times 90} = 0.5556 \approx \mathbf{0.56}$$

(2) Listening が335点の人たちの Reading の条件付き期待値の推定値はいくらか。

$$E[R \mid L = 335] = 250 + 0.556 \times \frac{90}{80} \times (335 - 305) = 250 + 0.556 \times 1.125 \times 30 = 268.8 \approx \mathbf{269}$$

問6.3：指数分布と生存関数

患者の死亡までの時間 $T \geq 0$ に対し $S(t) = P(T > t) = \exp(-\lambda t)$ とした。

(1) 確率密度関数 $f(t)$ を求めよ。

$$f(t) = \lambda \exp(-\lambda t), \quad t \geq 0$$

(2) 平均と上側25%点を求めよ。

平均は $E[T] = \mathbf{1/\lambda}$。上側25%点は $S(t) = 1/4$ を解いて

$$e^{-\lambda t} = \frac{1}{4} \;\Longrightarrow\; t = \frac{\log 4}{\lambda}$$

(3) 10人の標本平均が3.0年のとき、上側25%点の推定値はいくらか。

$E[T] = 1/\lambda$ を $\bar{T} = 3.0$ で推定する。上側25%点の推定値は

$$\frac{\log 4}{\lambda} = \log 4 \times \frac{1}{\lambda} \approx 1.4 \times 3.0 = \mathbf{4.2\text{年}}$$

問6.4：混合正規分布と合格率

受講生500名（文系300名 $N(65, 4^2)$、理系200名 $N(85, 3^2)$）の混合正規分布。

(1) 文系のAさん（67点）と理系のBさん（82点）の偏差値はいくらか。

$$\text{Aさん：} 50 + 10 \times \frac{67 - 65}{4} = \mathbf{55}$$ $$\text{Bさん：} 50 + 10 \times \frac{82 - 85}{3} = \mathbf{40}$$

(2) 60点以上を合格としたとき、合格率はおよそ何%か。

混合正規分布の累積分布関数は $P(X \leq x) = \dfrac{3}{5}\,\Phi\!\left(\dfrac{x - 65}{4}\right) + \dfrac{2}{5}\,\Phi\!\left(\dfrac{x - 85}{3}\right)$

$$P(X \geq 60) = 1 - \frac{3}{5}\,\Phi(-1.25) - \frac{2}{5}\,\Phi(-8.33)$$
$$= 1 - \frac{3}{5} \times 0.1056 - \frac{2}{5} \times 0.0000 = 1 - 0.0634 = 0.9366$$

よって合格率はおよそ 94%。

メモ

  • 正規分布・ガンマ分布・カイ二乗分布はいずれも再生性（独立な同種分布の和が同種分布）をもつ
  • 指数分布は連続分布における無記憶性をもつ唯一の分布。離散分布では幾何分布が対応
  • カイ二乗分布 $\chi^2(n)$ はガンマ分布 $\mathrm{Ga}(n/2, 2)$ の特殊ケース
  • $t$ 分布は $n = 1$ でコーシー分布、$n \to \infty$ で標準正規分布。正規分布より裾が重い
  • $T \sim t(n, \lambda)$ ならば $T^2 \sim F(1, n, \lambda^2)$ が成り立ち、非心カイ二乗・非心 $t$・非心 $F$ の三者は定義から自然に関連する
  • 対数正規分布の $k$ 次モーメントは、正規分布のモーメント母関数の $t = k$ における値に等しい
  • 2変量正規分布で $\rho = 0$ ならば独立。一般の分布では「無相関 $\neq$ 独立」だが正規分布では一致する
  • 混合正規分布は必ずしも多峰性を示すとは限らない。二峰性の条件 $|\mu_1 - \mu_2|/\sigma > 2$（$K = 2$、等比率・等分散の場合）に注意
