第23章「判別分析」のキーポイント

判別分析は，新しいデータがどのグループ（クラス）に属するかを判定する手法です．

手法概要特徴
フィッシャーの線形判別分析 グループ間分散をグループ内分散で割った比（分離度）を最大化する方向を求める 線形境界のみ．解釈しやすい
正準判別分析 3群以上に対応した線形判別の拡張．射影軸 $LD_1, LD_2, \ldots$ を求める 低次元への可視化に便利
2次判別分析（QDA） 各クラスの正規分布を仮定し，事後確率の対数比を判別関数とする 2次の判別境界．LDAより柔軟
ソフトマージン SVM マージン最大化でサポートベクターを用いた判別境界を学習する カーネルで非線形境界も対応可

問23.1 — 問題文（線形判別 vs ガウシアンカーネル SVM）

問題

2群（クラス）の2次元データを，フィッシャーの線形判別分析と，ガウシアンカーネルを用いたソフトマージン SVM によって判別する． それぞれの方法で得られた判別境界を図23.3 (a) と (b) に示す． 図に示したものと同じ分布に従うテストデータを判別したときの判別の正答率は，線形判別分析では 0.845，SVM では 0.815 であった．

(a) 線形判別分析による判別境界
(b) SVM による判別境界
  1. 通常，図23.3 のようなデータに対してよりよい判別精度が期待できるのはフィッシャーの線形判別分析とガウシアンカーネルを用いたソフトマージン SVM のどちらであるか，理由とともに答えよ．
  2. 問題に示した正答率や図23.3 (a), (b) のような結果になった理由と，ガウシアンカーネルを用いたソフトマージン SVM の精度を高くするための方法を議論せよ．

問23.1 [1] — どちらの手法が高精度か

解答・解説
2つの手法の特徴をおさらいする
手法判別境界の形向いているデータ
線形判別分析（LDA） 直線（2次元），平面（3次元以上） 2群が直線で（ほぼ）分離できる場合
ガウシアンカーネル SVM 曲線・複雑な非線形境界 2群が直線で分離できない場合

図23.3 のデータを見て判断する

データの構造を読む

図23.3 のデータを見ると，○群と＋群が入り混じっており，直線1本で2群をきれいに分離することができません． 2群の境界は直線ではなく，曲線的な分離が必要なことがデータから明らかです．

答え：ガウシアンカーネルを用いたソフトマージン SVM

2群が線形な判別直線（平面）で分離できないことが明らかなので，非線形判別曲面を当てはめることが可能なガウシアンカーネルを用いた SVM がよりよい判別精度を実現できると考えられる．

ガウシアンカーネルを使うと，元の特徴空間では直線で分けられないデータでも， 高次元の特徴空間に写像することで線形分離が可能になります． これが非線形な判別境界（図23.3 (b) の曲線状の境界）を作れる理由です．

問23.1 [2] — 実際の結果の理由と改善策

解答・解説
問題の状況を整理する

本来ならSVMのほうが高精度なはずなのに，実際は：

  • 線形判別分析：正答率 0.845（より高い）
  • ガウシアンカーネル SVM：正答率 0.815（より低い）

なぜこのような逆転が起きたのでしょうか？

① 結果の理由：過学習（オーバーフィッティング）

過学習とは？

過学習（overfitting）とは，モデルが学習データに対して「適合しすぎ」てしまい， 未知のテストデータに対する精度が低下する現象です．

図23.3 (b) のように，SVMの判別境界が学習データに沿って複雑な形になっている場合， 学習データのノイズや外れ値まで覚えてしまっている可能性があります． そのため，テストデータには対応できず精度が落ちます．

② 理由のまとめ

なぜ SVM の精度が低くなったか

線形判別分析による判別のほうが精度が高くなっていることから， ガウシアンカーネルのパラメータやソフトマージンパラメータが適切に設定されていないため， 学習データに過学習してしまい，テストデータに対する予測精度が低くなってしまっていると考えられる．

具体的には，ガウシアンカーネルの幅パラメータ $\gamma$（バンド幅）が大きすぎる場合や， ソフトマージンのペナルティパラメータ $C$ が大きすぎる場合に過学習が起きやすくなります．

③ SVM の精度を高くするための方法

改善策：クロスバリデーションによるパラメータ選択

ガウシアンカーネルを用いたソフトマージン SVM の性能を向上させるためには， たとえばクロスバリデーション（交差検証）を用いて， カーネルのパラメータ（ガウシアンカーネルの幅 $\gamma$）や ソフトマージンのパラメータ（$C$）を適切な値に設定する方法が考えられる．

パラメータ値が大きい場合値が小さい場合
$\gamma$（カーネル幅） 局所的な境界（過学習リスク高） 滑らかな境界（汎化性能高）
$C$（マージンペナルティ） 誤分類を許さない（過学習リスク） 誤分類を許容する（汎化性能高）

クロスバリデーションでは，学習データをいくつかの折（fold）に分割し， 1つを検証用・残りを訓練用として精度を評価し，その平均が最高になるパラメータの組み合わせを選びます（グリッドサーチ等）．

問23.2 — 問題文（正準判別分析によるアルファベット分類）

問題

さまざまなフォントで表示された文字の画像にノイズを加えたものから，16種類の画像特徴量を用いて， 記された文字が英語アルファベット E, F, Q のどれであるかを正準判別分析によって分類する． 簡単のため，16種類の特徴量から5種類を選び，個々のサンプルを5次元ベクトルとして扱う． 各群のデータはすべて50点である．あらかじめ各群で共分散行列を単位行列に白色化して， 正準判別分析により射影軸 $LD_1, LD_2$ を求め，$(LD_1, LD_2)$ 軸に射影したデータをプロットした（図23.4）．

$LD_1, LD_2$ に対応する射影軸はそれぞれ

$$w_1 = \begin{pmatrix}-0.1\\-0.1\\0.2\\0.4\\0.9\end{pmatrix}, \quad w_2 = \begin{pmatrix}0.0\\0.2\\-0.2\\0.9\\-0.3\end{pmatrix}$$

であった．また，クラス E, F, Q それぞれに属する5次元データの平均はそれぞれ

$$\bar{x}_1 = \begin{pmatrix}1.2\\1.4\\0.3\\6.0\\4.6\end{pmatrix},\quad \bar{x}_2 = \begin{pmatrix}1.4\\1.4\\0.3\\0.7\\2.1\end{pmatrix},\quad \bar{x}_3 = \begin{pmatrix}0.8\\0.3\\1.6\\3.6\\9.2\end{pmatrix}$$

であった．ここで，アルファベット L を表すサンプル $x_L = (2.5,\ -1.1,\ 0.0,\ -0.8,\ 1.0)^T$ を軸 $LD_1, LD_2$ に射影したものを図23.4に◇で示した．

新しい観測サンプルを，$LD_1, LD_2$ 平面における重心が最も近いような群に分類するとすると， この新しく観測した L に対応するサンプルは E, F, Q のどの群に分類されるか．

問23.2 — 解答（最近傍重心法による分類）

解答・解説
解法の方針：3ステップ
  1. 各クラス（E, F, Q）の重心 $\bar{x}_1, \bar{x}_2, \bar{x}_3$ を $LD_1, LD_2$ 平面に射影する
  2. サンプル $x_L$ を $LD_1, LD_2$ 平面に射影する
  3. 射影した $x_L$ から各重心までのユークリッド距離を計算し，最も近い群に分類する

ステップ1：各クラスの重心を射影する

射影先の座標は $( w_1^T \bar{x}_k,\ w_2^T \bar{x}_k )$（内積を計算するだけです）．

クラス E（$\bar{x}_1$）の射影：

$$w_1^T \bar{x}_1 = (-0.1)(1.2)+(-0.1)(1.4)+(0.2)(0.3)+(0.4)(6.0)+(0.9)(4.6)$$ $$= -0.12 - 0.14 + 0.06 + 2.40 + 4.14 = 6.34$$
$$w_2^T \bar{x}_1 = (0.0)(1.2)+(0.2)(1.4)+(-0.2)(0.3)+(0.9)(6.0)+(-0.3)(4.6)$$ $$= 0 + 0.28 - 0.06 + 5.40 - 1.38 = 4.24$$

→ クラス E の重心の射影点：$(6.34,\ 4.24)$

クラス F（$\bar{x}_2$）の射影：

$$w_1^T \bar{x}_2 = (-0.1)(1.4)+(-0.1)(1.4)+(0.2)(0.3)+(0.4)(0.7)+(0.9)(2.1)$$ $$= -0.14 - 0.14 + 0.06 + 0.28 + 1.89 = 1.95$$
$$w_2^T \bar{x}_2 = (0.0)(1.4)+(0.2)(1.4)+(-0.2)(0.3)+(0.9)(0.7)+(-0.3)(2.1)$$ $$= 0 + 0.28 - 0.06 + 0.63 - 0.63 = 0.22$$

→ クラス F の重心の射影点：$(1.95,\ 0.22)$

クラス Q（$\bar{x}_3$）の射影：

$$w_1^T \bar{x}_3 = (-0.1)(0.8)+(-0.1)(0.3)+(0.2)(1.6)+(0.4)(3.6)+(0.9)(9.2)$$ $$= -0.08 - 0.03 + 0.32 + 1.44 + 8.28 = 9.93$$
$$w_2^T \bar{x}_3 = (0.0)(0.8)+(0.2)(0.3)+(-0.2)(1.6)+(0.9)(3.6)+(-0.3)(9.2)$$ $$= 0 + 0.06 - 0.32 + 3.24 - 2.76 = 0.22$$

→ クラス Q の重心の射影点：$(9.93,\ 0.22)$

ステップ2：サンプル $x_L$ を射影する

$$w_1^T x_L = (-0.1)(2.5)+(-0.1)(-1.1)+(0.2)(0.0)+(0.4)(-0.8)+(0.9)(1.0)$$ $$= -0.25 + 0.11 + 0 - 0.32 + 0.90 = 0.44$$
$$w_2^T x_L = (0.0)(2.5)+(0.2)(-1.1)+(-0.2)(0.0)+(0.9)(-0.8)+(-0.3)(1.0)$$ $$= 0 - 0.22 + 0 - 0.72 - 0.30 = -1.24$$

→ $x_L$ の射影点：$(0.44,\ -1.24)$

ステップ3：各重心までの距離を計算する

$$d_E = \sqrt{(6.34-0.44)^2+(4.24-(-1.24))^2} = \sqrt{5.90^2+5.48^2} = \sqrt{34.81+30.03} = \sqrt{64.84} \approx 8.05$$
$$d_F = \sqrt{(1.95-0.44)^2+(0.22-(-1.24))^2} = \sqrt{1.51^2+1.46^2} = \sqrt{2.28+2.13} = \sqrt{4.41} \approx 2.10$$
$$d_Q = \sqrt{(9.93-0.44)^2+(0.22-(-1.24))^2} = \sqrt{9.49^2+1.46^2} = \sqrt{90.06+2.13} = \sqrt{92.19} \approx 9.60$$
クラス射影点距離
E$(6.34,\ 4.24)$$\approx 8.05$
F$(1.95,\ 0.22)$$\approx \mathbf{2.10}$（最小）
Q$(9.93,\ 0.22)$$\approx 9.60$
結論：群 F に分類される

$x_L$ の射影点 $(0.44,\ -1.24)$ から最も近い重心は F の重心 $(1.95,\ 0.22)$（距離 $\approx 2.10$）であるため， このサンプルは群 F に分類される．

問23.3 — 問題文（2次判別分析）

問題

それぞれ平均が $\mu_1 = (2,2)^T$，$\mu_2 = (0,0)^T$，分散共分散行列が

$$\Sigma_1 = \begin{pmatrix}0.8 & 0\\0 & 0.5\end{pmatrix},\quad \Sigma_2 = \begin{pmatrix}0.4 & 0\\0 & 0.25\end{pmatrix}$$

で与えられる2次元正規分布に従う2クラスの2次元データを考える．クラス1のサンプルサイズは100，クラス2のサンプルサイズは200とする．

  1. 2次判別分析を用いて，入力 $x$ からクラスラベルを予測するとする．クラスラベル $y$ で条件付けた $x$ の分布の対数比 $$f_q(x) = \log\frac{P(y=1|x)}{P(y=-1|x)}$$ で2クラス判別のための判別関数を構成する．$P(x|y=1)$ および $P(x|y=-1)$ をそれぞれ正規分布 $N(\mu_1,\Sigma_1)$，$N(\mu_2,\Sigma_2)$ の確率密度関数に基づいて表されるとしたとき，この判別器は $x$ の2次関数 $$f(x) = x^T A x + b^T x + c, \quad A\in\mathbb{R}^{2\times 2},\ b\in\mathbb{R}^2$$ として表すことができる．$y=1,-1$ それぞれの事前分布を $\pi_1,\pi_2$ として，行列 $A$，ベクトル $b$ および実数 $c$ を，$\Sigma_j,\mu_j,\pi_j,\ j=1,2$ を用いて表せ．
  2. 正規分布の母数と事前分布 $\pi_j$ は観測データを用いた標本平均などで置き換えて実際の判別関数を構成する．$\mu_j$ はクラス $j$ の標本平均，$\Sigma_j$ はクラス $j$ の標本分散共分散行列，クラス $j$ のサンプルサイズを $n_j$ として $\pi_j$ を $n_j/(n_1+n_2)$ で置き換える． 簡単のため，標本平均，標本分散共分散行列がすべて母集団のものと一致するとき，点 $(1,1)$ と点 $(0,1)$ はそれぞれクラス1，クラス2のどちらに分類されるか，判別関数の正負を計算して答えよ．

問23.3 — わかりやすく解説（一から丁寧に）

そもそも「判別分析」って何をしたいの？

目的はシンプルです。「新しいデータが届いたとき、それがクラス1とクラス2のどちらに属するかを判定したい」ということです。

この問題のイメージ：

  • クラス1：中心が $(2,2)$ あたりにまとまっているデータ群
  • クラス2：中心が $(0,0)$ あたりにまとまっているデータ群

2Dの散布図を想像すると、右上に固まったグループと左下に固まったグループがあります。 そこへ未知の点 $(1,1)$ が来たとき「どっちのグループ？」と判定するのがタスクです。

なぜ「2次」判別なの？「線形」じゃダメ？

線形判別（LDA）は「境界が直線」です。2つのグループが直線で綺麗に分かれるなら十分ですが、 この問題では2つのクラスの分散（広がり方）が違います。

クラス1クラス2
平均 $\mu$$(2,2)^T$$(0,0)^T$
分散 $\Sigma$$\begin{pmatrix}0.8&0\\0&0.5\end{pmatrix}$（広め）$\begin{pmatrix}0.4&0\\0&0.25\end{pmatrix}$（狭め）
サンプル数100200

$\Sigma_2$ は $\Sigma_1$ のちょうど半分の分散です。つまりクラス2はクラス1より中心付近に密集しています。 分散が違うと境界は「曲線（楕円）」になります。これを扱えるのが2次判別分析（QDA）です。

クラス1（青・大きな楕円）とクラス2（赤・小さな楕円）の分布イメージ

問[1]の核心：「確率の比」で判断する

ある点 $x$ が来たとき、次の値の正負で判断します：

$$f_q(x) = \log\frac{P(\text{クラス1} | x)}{P(\text{クラス2} | x)}$$
$f_q(x)$ の値意味判定
$> 0$クラス1の確率の方が高いクラス1
$= 0$どちらとも言えない判別境界
$< 0$クラス2の確率の方が高いクラス2

「クラス1らしさ ÷ クラス2らしさ」の対数、と直感的に捉えると分かりやすいです。

ベイズの定理→正規分布を代入→展開する流れ

ステップ①：ベイズの定理を使う

$P(x)$ が分子・分母で共通なので打ち消されて：

$$f_q(x) = \log\frac{P(x|\text{クラス1}) \cdot \pi_1}{P(x|\text{クラス2}) \cdot \pi_2}$$

ステップ②：正規分布の式を代入する

正規分布の密度関数を代入して対数を取ると、$(2\pi)$ の部分が共通で消えます。

$$f_q(x) = \log\frac{\pi_1}{\pi_2} + \log\frac{|\Sigma_1|^{-1/2}}{|\Sigma_2|^{-1/2}} - \frac{1}{2}(x-\mu_1)^T\Sigma_1^{-1}(x-\mu_1) + \frac{1}{2}(x-\mu_2)^T\Sigma_2^{-1}(x-\mu_2)$$

ステップ③：$(x-\mu)^T\Sigma^{-1}(x-\mu)$ を展開する

普通の $(x-a)^2 = x^2 - 2ax + a^2$ と同じ構造です：

$$(x-\mu)^T\Sigma^{-1}(x-\mu) = \underbrace{x^T\Sigma^{-1}x}_{\text{2次}} - \underbrace{2\mu^T\Sigma^{-1}x}_{\text{1次}} + \underbrace{\mu^T\Sigma^{-1}\mu}_{\text{定数}}$$

ステップ④：$x$ の次数ごとにまとめると

$$f_q(x) = \underbrace{x^T \cdot \frac{1}{2}(-\Sigma_1^{-1}+\Sigma_2^{-1}) \cdot x}_{x^T A x} + \underbrace{(\Sigma_1^{-1}\mu_1 - \Sigma_2^{-1}\mu_2)^T x}_{b^T x} + \underbrace{\text{定数}}_{c}$$

ポイント： $A$ にクラスごとの $\Sigma^{-1}$ の差が入るので「2次項」が残ります。 もし $\Sigma_1 = \Sigma_2$ なら $A=0$ になって線形判別（LDA）に退化します。

問[2]の計算の流れを直感的に理解する

具体的な数値を代入して $A$、$b$、$c$ を求め、最後に点を代入するだけです。 対角行列の逆行列は「各対角成分を逆数にするだけ」なので：

$$\Sigma_1^{-1} = \begin{pmatrix}1/0.8&0\\0&1/0.5\end{pmatrix} = \begin{pmatrix}1.25&0\\0&2\end{pmatrix}, \quad \Sigma_2^{-1} = \begin{pmatrix}1/0.4&0\\0&1/0.25\end{pmatrix} = \begin{pmatrix}2.5&0\\0&4\end{pmatrix}$$

最終的に得られる判別関数は：

$$f(x_1, x_2) = 0.625x_1^2 + x_2^2 + 2.5x_1 + 4x_2 - 7.886$$

この式に点を代入して正負を見るだけです：

点$f$ の計算符号判定
$(1,1)$ $0.625 + 1 + 2.5 + 4 - 7.886 = 0.239$ $> 0$ クラス1
$(0,1)$ $0 + 1 + 0 + 4 - 7.886 = -2.886$ $< 0$ クラス2
直感的に納得できる？
  • 点 $(1,1)$：2つの中心の中間あたりにありますが、クラス1は分散が大きく広く広がっているため $(1,1)$ もクラス1の分布に含まれやすい → クラス1
  • 点 $(0,1)$：クラス2の中心 $(0,0)$ に近く、クラス2は分散が小さく密集しているため $(0,1)$ はクラス2の分布に入りやすい。 さらにクラス2のサンプル数が2倍多い（$\pi_2 = 2/3$）という事前確率の影響もある → クラス2
問23.3 全体の流れのまとめ
ステップやることポイント
1確率の対数比 $f_q(x)$ を設定正なら→クラス1、負なら→クラス2
2ベイズの定理で $P(x)$ を消す$P(x|\text{クラス})\times\pi$ の比に変換
3正規分布の式を代入して対数を取る$(2\pi)^{p/2}$ が消えて指数部分が残る
42次形式を $x$ の2次・1次・定数に整理$f = x^TAx + b^Tx + c$ の形に
5具体的な数値を代入して $A,b,c$ を計算対角行列の逆行列は各成分の逆数
6判定したい点を $f$ に代入正ならクラス1、負ならクラス2

問23.3 [1] — 2次判別関数の導出

解答・解説
解法の方針：ベイズの定理 + 正規分布の密度関数を展開する

ベイズの定理より，

$$P(y=1|x) = \frac{P(x|y=1)\pi_1}{P(x)}$$

したがって対数比は：

$$f_q(x) = \log\frac{P(y=1|x)}{P(y=-1|x)} = \log\frac{P(x|y=1)\pi_1}{P(x|y=-1)\pi_2}$$

$P(x|y=k)$ が正規分布 $N(\mu_k,\Sigma_k)$ の密度関数（以下 $\phi_k(x)$ と書く）であるとき，

$$\phi_k(x) = \frac{1}{(2\pi)^{p/2}|\Sigma_k|^{1/2}} \exp\!\left(-\frac{1}{2}(x-\mu_k)^T\Sigma_k^{-1}(x-\mu_k)\right)$$

Step 1：対数比を展開する

$$f_q(x) = \log\frac{\phi_1(x)\pi_1}{\phi_2(x)\pi_2}$$
$$= \log\frac{\pi_1}{\pi_2} + \log\frac{|\Sigma_1|^{-1/2}}{|\Sigma_2|^{-1/2}} - \frac{1}{2}(x-\mu_1)^T\Sigma_1^{-1}(x-\mu_1) + \frac{1}{2}(x-\mu_2)^T\Sigma_2^{-1}(x-\mu_2)$$
指数部分を整理するコツ

2次形式の展開：$(x-\mu)^T\Sigma^{-1}(x-\mu) = x^T\Sigma^{-1}x - 2\mu^T\Sigma^{-1}x + \mu^T\Sigma^{-1}\mu$

Step 2：2次・1次・定数項に整理する

クラス1に関する2次形式の展開：

$$-\frac{1}{2}(x-\mu_1)^T\Sigma_1^{-1}(x-\mu_1) = -\frac{1}{2}x^T\Sigma_1^{-1}x + \mu_1^T\Sigma_1^{-1}x - \frac{1}{2}\mu_1^T\Sigma_1^{-1}\mu_1$$

クラス2に関する2次形式の展開：

$$+\frac{1}{2}(x-\mu_2)^T\Sigma_2^{-1}(x-\mu_2) = +\frac{1}{2}x^T\Sigma_2^{-1}x - \mu_2^T\Sigma_2^{-1}x + \frac{1}{2}\mu_2^T\Sigma_2^{-1}\mu_2$$

合算すると：

$$f_q(x) = x^T\underbrace{\frac{1}{2}(-\Sigma_1^{-1}+\Sigma_2^{-1})}_{=A}x + \underbrace{(\mu_1^T\Sigma_1^{-1} - \mu_2^T\Sigma_2^{-1})}_{=b^T}x + \underbrace{\frac{1}{2}(\mu_2^T\Sigma_2^{-1}\mu_2 - \mu_1^T\Sigma_1^{-1}\mu_1) + \log\frac{|\Sigma_2|^{1/2}}{|\Sigma_1|^{1/2}} + \log\frac{\pi_1}{\pi_2}}_{=c}$$
結果
$$A = \frac{1}{2}(-\Sigma_1^{-1}+\Sigma_2^{-1})$$
$$b = \Sigma_1^{-1}\mu_1 - \Sigma_2^{-1}\mu_2$$
$$c = \frac{1}{2}(\mu_2^T\Sigma_2^{-1}\mu_2 - \mu_1^T\Sigma_1^{-1}\mu_1) + \log\frac{|\Sigma_2|^{1/2}}{|\Sigma_1|^{1/2}} + \log\frac{\pi_1}{\pi_2}$$

$f_q(x) > 0$ ならクラス1（$y=1$），$f_q(x) < 0$ ならクラス2（$y=-1$）に分類されます．

線形判別（LDA）との違い

もし $\Sigma_1 = \Sigma_2$（両クラスの分散共分散行列が等しい）ならば，$A = (-\Sigma^{-1}+\Sigma^{-1})/2 = 0$ となり， 2次項が消えて線形判別関数（LDA）になります． クラス間の分散が異なる場合に2次判別分析が必要となります．

問23.3 [2] — 判別関数への代入と分類

解答・解説

Step 1：各値を確認する

事前分布 $\pi_j$ の計算

$n_1 = 100$，$n_2 = 200$，$n_1 + n_2 = 300$ なので：

$$\pi_1 = \frac{100}{300} = \frac{1}{3}, \quad \pi_2 = \frac{200}{300} = \frac{2}{3}$$

Step 2：$A$ を計算する

$\Sigma_1 = \text{diag}(0.8, 0.5)$ なので $\Sigma_1^{-1} = \text{diag}(1/0.8,\ 1/0.5) = \text{diag}(1.25,\ 2)$

$\Sigma_2 = \text{diag}(0.4, 0.25)$ なので $\Sigma_2^{-1} = \text{diag}(1/0.4,\ 1/0.25) = \text{diag}(2.5,\ 4)$

$$A = \frac{1}{2}(-\Sigma_1^{-1}+\Sigma_2^{-1}) = \frac{1}{2}\left(-\begin{pmatrix}1.25&0\\0&2\end{pmatrix}+\begin{pmatrix}2.5&0\\0&4\end{pmatrix}\right) = \frac{1}{2}\begin{pmatrix}1.25&0\\0&2\end{pmatrix} = \begin{pmatrix}0.625&0\\0&1\end{pmatrix}$$

Step 3：$b$ を計算する

$\mu_1 = (2,2)^T$，$\mu_2 = (0,0)^T$ なので：

$$b = \Sigma_1^{-1}\mu_1 - \Sigma_2^{-1}\mu_2 = \begin{pmatrix}1.25&0\\0&2\end{pmatrix}\begin{pmatrix}2\\2\end{pmatrix} - \begin{pmatrix}2.5&0\\0&4\end{pmatrix}\begin{pmatrix}0\\0\end{pmatrix} = \begin{pmatrix}2.5\\4\end{pmatrix}$$

Step 4：$c$ を計算する

各項を順に求めます：

$$\mu_1^T\Sigma_1^{-1}\mu_1 = (2,2)\begin{pmatrix}1.25&0\\0&2\end{pmatrix}\begin{pmatrix}2\\2\end{pmatrix} = 2\times 1.25\times 2 + 2\times 2\times 2 = 5 + 8 = 13$$
$$\mu_2^T\Sigma_2^{-1}\mu_2 = (0,0)\begin{pmatrix}2.5&0\\0&4\end{pmatrix}\begin{pmatrix}0\\0\end{pmatrix} = 0$$
\frac{1}{2}(\mu_2^T\Sigma_2^{-1}\mu_2 - \mu_1^T\Sigma_1^{-1}\mu_1) = \frac{1}{2}(0 - 13) = -6.5

次に $\log\frac{|\Sigma_2|^{1/2}}{|\Sigma_1|^{1/2}} = \frac{1}{2}\log\frac{|\Sigma_2|}{|\Sigma_1|}$ を計算します：

$$\frac{|\Sigma_2|}{|\Sigma_1|} = \left|\Sigma_2\Sigma_1^{-1}\right| = \left|\begin{pmatrix}0.4&0\\0&0.25\end{pmatrix}\begin{pmatrix}1.25&0\\0&2\end{pmatrix}\right| = \left|\begin{pmatrix}0.5&0\\0&0.5\end{pmatrix}\right| = 0.5\times 0.5 = 0.25$$
$$\log\frac{|\Sigma_2|^{1/2}}{|\Sigma_1|^{1/2}} = \frac{1}{2}\log 0.25 = \frac{1}{2}(-\log 4) = -\frac{\log 4}{2} \approx -0.693$$

最後に：

$$\log\frac{\pi_1}{\pi_2} = \log\frac{1/3}{2/3} = \log\frac{1}{2} = -\log 2 \approx -0.693$$

すべて合算：

$$c = -6.5 + (-0.693) + (-0.693) \approx -7.886 \approx -8$$

Step 5：判別関数のまとめ

$$f(x) = x^T\begin{pmatrix}0.625&0\\0&1\end{pmatrix}x + \begin{pmatrix}2.5\\4\end{pmatrix}^T x - 8$$

Step 6：点 $(1,1)$ と $(0,1)$ に代入する

点 $(1,1)$ の場合
$$f\!\begin{pmatrix}1\\1\end{pmatrix} = (1,1)\begin{pmatrix}0.625&0\\0&1\end{pmatrix}\begin{pmatrix}1\\1\end{pmatrix} + (2.5,4)\begin{pmatrix}1\\1\end{pmatrix} - 8$$
$$= (0.625\times 1 + 1\times 1) + (2.5 + 4) - 8 = 1.625 + 6.5 - 8 = 0.125 > 0$$

$f > 0$ なのでクラス1に分類される．

点 $(0,1)$ の場合
$$f\!\begin{pmatrix}0\\1\end{pmatrix} = (0,1)\begin{pmatrix}0.625&0\\0&1\end{pmatrix}\begin{pmatrix}0\\1\end{pmatrix} + (2.5,4)\begin{pmatrix}0\\1\end{pmatrix} - 8$$
$$= (0 + 1\times 1) + (0 + 4) - 8 = 1 + 4 - 8 = -3 < 0$$

$f < 0$ なのでクラス2に分類される．

結論

点 $(1,1)$ はクラス1，点 $(0,1)$ はクラス2に分類される．

直感的な確認

クラス1の中心は $\mu_1 = (2,2)$，クラス2の中心は $\mu_2 = (0,0)$ です． 点 $(1,1)$ はちょうど2つの中心の中間あたりにありますが， クラス1の分散がクラス2より大きいため，$(1,1)$ はクラス1の広い分布の中に含まれやすく，クラス1に分類されます．

点 $(0,1)$ はクラス2の中心 $(0,0)$ に近いですが，$x_2$ 方向の成分だけが1あります． クラス2は分散が小さく中心 $(0,0)$ に密集しているので，$(0,1)$ はクラス2の分布に属しやすく，クラス2に分類されます．
