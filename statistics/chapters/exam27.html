第27章「時系列解析」のキーポイント

■ AR過程（自己回帰過程）

現在の値が過去の値の線形結合と誤差項で表されるモデルをAR（AutoRegressive）過程といいます．

$$\text{AR}(p): \quad Y_t = \phi_0 + \phi_1 Y_{t-1} + \phi_2 Y_{t-2} + \cdots + \phi_p Y_{t-p} + U_t$$
パラメータ意味AR(1)での具体的な影響
$\phi_1 \in (-1,1)$定常AR(1)過去の値に引き寄せられる
$0 < \phi_1 < 1$正の自己相関なめらかな変動（惰性が続く）
$-1 < \phi_1 < 0$負の自己相関上下に激しく振動する
$\phi_1 = 1$単位根（非定常）ランダムウォーク，分散が時間とともに増大

■ MA過程（移動平均過程）

$$\text{MA}(q): \quad Y_t = \mu + U_t + \theta_1 U_{t-1} + \theta_2 U_{t-2} + \cdots + \theta_q U_{t-q}$$

MA($q$)過程の自己共分散 $\gamma_h$ は$|h| > q$ で0になります（有限ラグ後に切断）．

■ ACF・PACF によるモデル識別

モデル標本自己相関（ACF）標本偏自己相関（PACF）
AR($p$)次数とともに減衰（無限に続く）ラグ$p$以降で0になる（突然切断）
MA($q$)ラグ$q$以降で0になる（突然切断）次数とともに減衰（無限に続く）
ARMA減衰（無限に続く）減衰（無限に続く）

■ DW比（Durbin-Watson 統計量）

$$DW = \frac{\displaystyle\sum_{t=2}^{T}(e_t - e_{t-1})^2}{\displaystyle\sum_{t=1}^{T}e_t^2} \approx 2(1 - \hat{\rho}_1)$$

ここで $\hat{\rho}_1$ は残差の1次自己相関係数の推定値です． 正の自己相関（$\hat{\rho}_1 > 0$）があるとDW比は0に近く， 自己相関なし（$\hat{\rho}_1 \approx 0$）のとき DW $\approx 2$， 負の自己相関（$\hat{\rho}_1 < 0$）があると4に近くなります．

■ AIC（赤池情報量基準）

$$\text{AIC} = -2\log\hat{L} + 2k$$

$\hat{L}$ は最大化された尤度，$k$ はパラメータ数です． AICが最も小さいモデルが最良とみなされます（小さいほど良いモデル）．

に対して，パラメータ $\phi_1$ を $-0.8,\ 0,\ 0.7,\ 1$ と変えて時系列データをシミュレーションした． それぞれのシミュレーション結果のグラフ（時点 $t$ を横軸，$Y_t$ を縦軸）が図27.1の (a)〜(d) である．

図27.1　AR(1)過程のシミュレーション（各グラフは概略）

  1. グラフ (a)〜(d) はそれぞれ $\phi_1 = -0.8,\ 0,\ 0.7,\ 1$ のうちどれに対応するか答えよ．
  2. 式 (27.1) の $Y_t$ を被説明変数，$Y_{t-1}$ を説明変数とする回帰モデルを考え， 誤差項が AR(1) 過程に従うとした場合，OLS（最小二乗法）推定量の性質について述べよ． また，残差の正の自己相関が存在するとき DW 比はどのような値をとるか述べよ．

問27.1 [1] — AR(1)のグラフ識別

解答・解説
AR(1)過程のパラメータと時系列の見た目

AR(1)過程 $Y_t = \phi_0 + \phi_1 Y_{t-1} + U_t$ において， パラメータ $\phi_1$ の値によって時系列の性質が大きく異なります．

各 $\phi_1$ 値の特徴と識別ポイント

$\phi_1 = 1$（単位根・非定常過程）→ グラフ (c)

$\phi_1 = 1$ のとき，モデルは $Y_t = 2(1-1) + 1 \cdot Y_{t-1} + U_t = Y_{t-1} + U_t$ となり， これはランダムウォーク（+ドリフトなし）です．

特徴：分散が時間とともに際限なく大きくなる（非定常）． 平均への引き戻しが起こらないため，一方向に漂い続ける（トレンドが見られる）傾向があります． グラフ (c) は時間が進むにつれて下降し続けており，単位根の特徴に一致します．

$\phi_1 = 0.7$（正の自己相関・定常）→ グラフ (a)

$|\phi_1| < 1$ なので定常過程です．$\phi_1 > 0$ のとき，正の自己相関があります． 「今の値が大きければ次の値も大きい傾向がある」ため，なめらかに上下する波のような形になります．

特徴：隣り合う時点の値が似ており，変化が緩やかです． グラフ (a) はなめらかな波形を示しており，$\phi_1 = 0.7$ の特徴と一致します．

$\phi_1 = 0$（ホワイトノイズ）→ グラフ (b)

$\phi_1 = 0$ のとき，$Y_t = 2 + U_t$ となり，$\{Y_t\}$ は互いに独立な確率変数の列（ホワイトノイズ）になります．

特徴：隣り合う時点の値に相関がなく，上下にばらばらに変動します． グラフ (b) は連続性のないランダムな変動を示しており，$\phi_1 = 0$ の特徴と一致します．

$\phi_1 = -0.8$（負の自己相関・定常）→ グラフ (d)

$|\phi_1| < 1$ なので定常過程です．$\phi_1 < 0$ のとき，負の自己相関があります． 「今の値が大きければ次の値は小さくなる傾向がある」ため，上下に激しく振動するジグザグなパターンになります．

特徴：隣り合う時点の値が反対の方向に動き，変化が激しいです． グラフ (d) は1時点ごとに上下を繰り返す激しいジグザグを示しており，$\phi_1 = -0.8$ の特徴と一致します．

結論：グラフとパラメータの対応
グラフ$\phi_1$特徴
(a)$\phi_1 = 0.7$正の自己相関，なめらかな波
(b)$\phi_1 = 0$自己相関なし，ホワイトノイズ
(c)$\phi_1 = 1$単位根（非定常），一方向に漂う
(d)$\phi_1 = -0.8$負の自己相関，ジグザグな激しい変動
補足：定常性と自己相関の直感的理解

AR(1)過程 $Y_t = \phi_0 + \phi_1 Y_{t-1} + U_t$ において：

  • 定常条件：$|\phi_1| < 1$ — 過去の影響が指数的に減衰するため，平均・分散・自己共分散が時間に依存しない定常状態が保たれます．
  • $\phi_1$ が0に近いほどホワイトノイズに近く，1に近いほど変動が持続します（なめらか）．
  • $\phi_1 = 1$（単位根）では，ショック（誤差 $U_t$）の影響が永久に残り続けるため，分散が時間とともに増大し定常でなくなります．

問27.1 [2] — OLS推定量の性質と DW 比

解答・解説
問題の設定

回帰モデル $Y_t = \beta_0 + \beta_1 Y_{t-1} + \varepsilon_t$ において， 誤差項 $\varepsilon_t$ が AR(1) 過程 $\varepsilon_t = \rho\,\varepsilon_{t-1} + e_t$（$|\rho| < 1$，$e_t \sim \text{i.i.d.}(0,\sigma^2)$） に従う場合を考えます．

OLS推定量の性質

不偏性（Unbiasedness）：OLS は不偏推定量か？

OLS（最小二乗法）推定量 $\hat{\beta}$ は，誤差項に自己相関があっても， 誤差項の期待値が0（$E[\varepsilon_t] = 0$）であれば不偏推定量です：

$$E[\hat{\beta}] = \beta$$

ただし，この不偏性は厳密には有限サンプルでは成立せず，大標本（漸近的）な意味での一致性に言い換えることが多いです．

有効性（Efficiency）：OLS は最も分散が小さいか？

有効推定量ではありません（BLUE = 最良線形不偏推定量ではない）．

ガウス・マルコフの定理によれば，「誤差項が互いに独立で等分散」という条件のもとで， OLS が線形不偏推定量クラスで最も分散が小さい（BLUE）ことが保証されます． しかし，誤差項に自己相関（$\varepsilon_t = \rho\,\varepsilon_{t-1} + e_t$）がある場合，この条件が破れるため， OLS は不偏だが BLUE ではなく，GLS（一般化最小二乗法）の方がより小さい分散を持ちます．

分散の過小推定と t 検定・F 検定への影響

正の自己相関（$\rho > 0$）がある場合，OLS の標準誤差が過小推定されます． その結果，t統計量が実際より大きく見え，有意でない係数を有意と判断する 偽の有意性（見せかけの有意性）が生じる危険があります．

正の自己相関と DW 比

DW 比の値域と自己相関の関係

DW 比の近似式 $DW \approx 2(1 - \hat{\rho}_1)$ から，

残差の自己相関$\hat{\rho}_1$ の値DW 比の値
正の自己相関$\hat{\rho}_1 \approx +1$$DW \approx 0$（0に近い）
自己相関なし$\hat{\rho}_1 \approx 0$$DW \approx 2$（2に近い）
負の自己相関$\hat{\rho}_1 \approx -1$$DW \approx 4$（4に近い）

したがって，残差に正の自己相関が存在するとき，DW 比は 0 に近い値をとります．

結論

① OLS 推定量の性質： 誤差項が AR(1) 過程に従う場合，OLS 推定量は（漸近的に）不偏推定量ではあるが， ガウス・マルコフの定理の条件（独立等分散）が成立しないため， 線形不偏推定量クラスで最小分散（BLUE）ではない（有効でない）． GLS が OLS より効率的な推定量となる．

② DW 比： 残差に正の自己相関が存在するとき，DW 比は0 に近い値をとる．

問27.2 — MA(2)過程の自己共分散の計算

問題

MA(2)過程

$$Y_t = U_t + \theta_1\,U_{t-1} + \theta_2\,U_{t-2}, \qquad \{U_t\} \overset{\text{i.i.d.}}{\sim} (0,\,\sigma^2) \tag{27.2}$$

について，ラグ $h$（$h = 0, 1, 2$ および $h > 2$）の自己共分散 $\gamma_h = \mathrm{Cov}(Y_t,\,Y_{t-h})$ を求めよ．

解答・解説
計算の基本方針

MA(2)は平均0なので $\gamma_h = E[Y_t\,Y_{t-h}]$ と書けます． $\{U_t\}$ が i.i.d.$(0, \sigma^2)$ であることから：

  • $E[U_t^2] = \sigma^2$
  • $E[U_t\,U_s] = 0 \quad (t \neq s)$

$Y_t = U_t + \theta_1 U_{t-1} + \theta_2 U_{t-2}$ を代入し展開して，添字が一致する項だけが $\sigma^2$ として残ります．

$\gamma_0$（分散）の計算

$\gamma_0 = E[Y_t^2]$

$$\gamma_0 = E\!\left[(U_t + \theta_1 U_{t-1} + \theta_2 U_{t-2})^2\right]$$

展開すると：

$$= E[U_t^2] + \theta_1^2\,E[U_{t-1}^2] + \theta_2^2\,E[U_{t-2}^2] + 2\theta_1\underbrace{E[U_t U_{t-1}]}_{=0} + 2\theta_2\underbrace{E[U_t U_{t-2}]}_{=0} + 2\theta_1\theta_2\underbrace{E[U_{t-1}U_{t-2}]}_{=0}$$

クロス項はすべて0なので：

$$\boxed{\gamma_0 = (1 + \theta_1^2 + \theta_2^2)\,\sigma^2}$$

$\gamma_1$（ラグ1の自己共分散）の計算

$\gamma_1 = E[Y_t\,Y_{t-1}]$

$$\gamma_1 = E\!\left[(U_t + \theta_1 U_{t-1} + \theta_2 U_{t-2})(U_{t-1} + \theta_1 U_{t-2} + \theta_2 U_{t-3})\right]$$

この積を展開したとき，添字が一致する（非ゼロになる）組み合わせだけを拾います：

  • $\theta_1\,E[U_{t-1}^2] = \theta_1\,\sigma^2$（$U_{t-1}$ 同士）
  • $\theta_2 \cdot \theta_1\,E[U_{t-2}^2] = \theta_1\theta_2\,\sigma^2$（$U_{t-2}$ 同士）
  • その他の組み合わせはすべて添字が異なり0
$$\boxed{\gamma_1 = (\theta_1 + \theta_1\theta_2)\,\sigma^2 = \theta_1(1+\theta_2)\,\sigma^2}$$

$\gamma_2$（ラグ2の自己共分散）の計算

$\gamma_2 = E[Y_t\,Y_{t-2}]$

$$\gamma_2 = E\!\left[(U_t + \theta_1 U_{t-1} + \theta_2 U_{t-2})(U_{t-2} + \theta_1 U_{t-3} + \theta_2 U_{t-4})\right]$$

添字が一致する組み合わせは $U_{t-2}$ 同士のみ：

  • $\theta_2\,E[U_{t-2}^2] = \theta_2\,\sigma^2$（$U_{t-2}$ 同士）
$$\boxed{\gamma_2 = \theta_2\,\sigma^2}$$

$\gamma_h$（$h > 2$）の計算

$\gamma_h = 0 \quad (h > 2)$

$h > 2$ のとき，$Y_t = U_t + \theta_1 U_{t-1} + \theta_2 U_{t-2}$ と $Y_{t-h} = U_{t-h} + \theta_1 U_{t-h-1} + \theta_2 U_{t-h-2}$ では， 使っている $U$ の添字が完全に重ならないため，

$$\boxed{\gamma_h = 0 \quad (h > 2)}$$

これは MA($q$) 過程の一般的な性質であり，「ラグが $q$ を超えると自己共分散は0」（$q$ 次での切断）となります．

結論：MA(2)の自己共分散まとめ
ラグ $h$$\gamma_h$
$h = 0$$(1+\theta_1^2+\theta_2^2)\,\sigma^2$
$h = 1$$\theta_1(1+\theta_2)\,\sigma^2$
$h = 2$$\theta_2\,\sigma^2$
$h > 2$$0$

※ 対称性 $\gamma_{-h} = \gamma_h$ より，ラグが負の場合も同様です．

補足：自己相関係数との関係

自己相関係数 $\rho_h = \gamma_h / \gamma_0$ は：

$$\rho_1 = \frac{\theta_1(1+\theta_2)}{1+\theta_1^2+\theta_2^2}, \quad \rho_2 = \frac{\theta_2}{1+\theta_1^2+\theta_2^2}, \quad \rho_h = 0 \;(h > 2)$$

MA($q$) 過程の標本 ACF（自己相関関数）はラグ $q$ 以降で完全に0になるため， ACF のパターンから MA 過程の次数を識別することができます．

問27.3 — 問題文（ACF・PACF によるモデル選択と DW 比）

問題

ある回帰モデルの残差に対して標本自己相関（ACF）と標本偏自己相関（PACF）を計算したところ， 図27.7のような結果が得られた（破線は有意水準の閾値を示す）．

この結果において，標本自己相関係数は $\hat{\rho}_1 = 0.691$，$\hat{\rho}_2 = 0.211$ であった．

  1. ACF と PACF のパターンから，残差にはどのような自己回帰モデルが適切か判断し，理由とともに答えよ．
  2. 上記の $\hat{\rho}_1 = 0.691$ の値を使い，DW 比の近似値を求めよ． なお，DW 比の近似式として $DW \approx 2(1 - \hat{\rho}_1)$ を用いること．

問27.3 [1] — ACF・PACF からのモデル識別

解答・解説
ACF と PACF の読み方（識別ルール）
モデルACF（自己相関関数）PACF（偏自己相関関数）
AR($p$) 指数的・振動的に減衰（無限に続く） ラグ $p$ で突然切断（$p+1$ 次以降は0）
MA($q$) ラグ $q$ で突然切断（$q+1$ 次以降は0） 指数的・振動的に減衰（無限に続く）

図27.7 の ACF・PACF パターンを分析する

ACF のパターン確認

ラグ $h=1, 2$ では有意水準の閾値（破線）を超えており，ラグ3以降は閾値内に収まっています． しかし，ラグ1→2と値が減衰しており，指数的に減衰するパターンと解釈できます． ACF がラグとともに徐々に減衰（滑らかに0に近づく）するのはAR過程の特徴です．

PACF のパターン確認（決定的な判断材料）

PACF はラグ $h=1$ と $h=2$ で有意（閾値を超える）， ラグ $h=3$ 以降はほぼ0（閾値内に収まる）．

これは AR($p$) 過程の PACF の典型的パターンです： 「ラグ $p$ まで有意で，ラグ $p+1$ 以降は突然0になる（切断される）」．

今回は ラグ2で切断されているので，$p = 2$ と判断されます．

結論

適切なモデル：AR(2)過程

理由：

  • ACF がラグとともに徐々に減衰している（AR過程の特徴）
  • PACF がラグ2まで有意でラグ3以降は0になっている（次数 $p=2$ での切断 = AR(2)の特徴）
補足：PACF の定義と直感的意味

偏自己相関（Partial Autocorrelation）とは， ラグ $h$ の相関を「中間のラグ（1〜$h-1$）の影響を取り除いた」純粋な相関です．

AR($p$) 過程では，ラグ $p+1$ 以降の PACF は理論上完全に0になります（回帰に余計な変数を加えても係数が0になる）． このため，PACF の「切断点」を見ることで AR の次数を識別できます．

問27.3 [1] わかりやすく解説 — ACF・PACF からのモデル識別

超わかりやすく解説
まず「ACF（自己相関関数）」って何？

ACF とは「今日の値と $h$ 日前の値が，どれくらい似ているか」を数値化したものです．

たとえば気温を例にとると：

  • $h=1$（1日前）の ACF が高い → 昨日暑ければ今日も暑い傾向がある
  • $h=7$（7日前）の ACF が高い → 1週間前と今週の気温が似ている（季節性）

⚠️ ACF の弱点：「間接的な影響」も含まれてしまう！

$Y_t$ と $Y_{t-2}$ の間に相関がある場合，それが「$Y_{t-2}$ が $Y_t$ に直接影響している」のか， それとも「$Y_{t-2} \to Y_{t-1} \to Y_t$ という連鎖の結果」なのかを区別できません．

「PACF（偏自己相関関数）」って何？　── 中間の影響を取り除いた純粋な相関

PACF は「ラグ 1〜$h-1$ の影響を取り除いた上で，$h$ 期前との純粋な相関」です．

わかりやすく言うと： 「$Y_{t-2}$ が $Y_t$ に影響しているように見えても，それが単に『$Y_{t-1}$ を経由しているだけ』なら， PACF(h=2) はほぼ0になる．直接的な影響がある場合だけ大きな値になる．」

識別ルール（これを丸暗記するより仕組みを理解しよう！）

AR($p$) 過程

$Y_t = \phi_1 Y_{t-1} + \cdots + \phi_p Y_{t-p} + U_t$

「過去 $p$ 期分が直接影響する」モデル

PACF：ラグ $p$ で突然ゼロに（切断）
→ 直接影響はラグ $p$ まで，それ以降はゼロ

ACF：ゆっくり減衰（なかなかゼロにならない）
→ 間接影響が連鎖するため，遠い過去まで尾を引く

MA($q$) 過程

$Y_t = U_t + \theta_1 U_{t-1} + \cdots + \theta_q U_{t-q}$

「過去 $q$ 期分の誤差が影響する」モデル

ACF：ラグ $q$ で突然ゼロに（切断）
→ 共通する誤差項がなくなるため相関がゼロに

PACF：ゆっくり減衰（なかなかゼロにならない）
→ ACFがゆっくり減衰するAR(∞)と等価だから

覚え方のコツ

「切断（カットオフ）しているほうが，そのモデルの次数」

  • PACF がラグ $p$ で切断 → AR($p$)
  • ACF がラグ $q$ で切断 → MA($q$)
  • 両方ともゆっくり減衰 → ARMA（混合モデル）
なぜ AR($p$) の PACF はラグ $p$ で切断されるの？

AR($p$) モデルを回帰式として見てみましょう：

$$Y_t = \phi_1 Y_{t-1} + \phi_2 Y_{t-2} + \cdots + \phi_p Y_{t-p} + U_t$$

AR($p$) の「直接の原因」はラグ $p$ までしかありません． ラグ $p+1$ 以降は，すでに $Y_{t-1}, \ldots, Y_{t-p}$ に含まれる間接的な情報しかなく， それらをコントロール（偏）したら「直接の影響」はゼロになります．

具体的に AR(2) で考えると：

AR(2) の PACF がラグ3以降ゼロになる理由

  • $Y_t$ に直接影響しているのは $Y_{t-1}$ と $Y_{t-2}$ だけ（モデルの定義より）
  • $Y_{t-3}$ が $Y_t$ に相関があるように見えるのは，$Y_{t-3} \to Y_{t-2} \to Y_t$ や $Y_{t-3} \to Y_{t-1} \to Y_t$ という連鎖の結果（間接影響）
  • $Y_{t-1}, Y_{t-2}$ の影響をコントロール（偏）すると，$Y_{t-3}$ は $Y_t$ と無関係になる
  • よって PACF($h=3$) = 0
この問題（図27.7）に当てはめてみよう！

図27.7 の ACF・PACF を見ると：

ラグ1ラグ2ラグ3以降パターン
ACF 有意（大）
$\hat{\rho}_1 = 0.691$		 有意（小）
$\hat{\rho}_2 = 0.211$		 ほぼゼロ ラグが増えるにつれて徐々に減衰
PACF 有意（大） 有意 突然ゼロ ラグ2まで有意 → ラグ3で切断

STEP 1：PACF を見る（最重要！）

PACF がラグ3以降で突然ゼロ → 「PACF がラグ2で切断」
→ AR($p$) 型，次数は $p = 2$

STEP 2：ACF で確認する

ACF は $0.691 \to 0.211 \to \approx 0$ と徐々に減衰している → AR 過程の ACF パターンと一致（MA 過程なら突然ゼロになるはず）

STEP 3：結論

→ AR(2) 過程が適切

残差に AR(2) 過程の自己相関が残っているので， 回帰モデルに $Y_{t-1}$ と $Y_{t-2}$ を説明変数として追加するか， ARMA モデルで明示的に扱う必要があります．

よくある混乱ポイント

Q：この問題の ACF もラグ3以降ゼロに見えるから MA(2) じゃないの？

A：「ゆっくり減衰」と「突然ゼロ」を区別しましょう！

  • $0.691 \to 0.211 \to \approx 0$ という変化は，比率（$0.211/0.691 \approx 0.31$）から見ると指数的に減衰しています
  • MA(2) の ACF は「ラグ3でピタッとゼロ」ですが，AR(2) では「まだ少し残っている（はずだが測定誤差でゼロに見える）」という違いがあります
  • 最終判断は PACF で行います：PACF がラグ2で突然ゼロ → AR(2)，ACF がラグ2で突然ゼロ → MA(2)

問27.3 [2] — DW 比の近似値の計算

解答・解説
DW 比の近似式

DW 比は残差 $e_t$ の1次自己相関係数 $\hat{\rho}_1$ と次の近似関係にあります：

$$DW \approx 2(1 - \hat{\rho}_1)$$

※ これは，DW 統計量の定義式 $DW = \sum(e_t - e_{t-1})^2 / \sum e_t^2$ を展開すると， 分子 $\approx 2\sum e_t^2 - 2\sum e_t e_{t-1}$， よって $DW \approx 2 - 2\hat{\rho}_1 = 2(1-\hat{\rho}_1)$ と近似できることによります．

計算

$\hat{\rho}_1 = 0.691$ を代入する

$$DW \approx 2(1 - \hat{\rho}_1) = 2(1 - 0.691) = 2 \times 0.309 = 0.618$$
結論
$$DW \approx 2(1 - 0.691) = \boldsymbol{0.618}$$
この値の意味

$DW \approx 0.618$ は2よりかなり小さい値です（理想値は2）． DW 統計量の一般的な判断基準では：

  • $DW \approx 2$：自己相関なし（理想）
  • $DW \lesssim 1.5$：正の自己相関が疑われる（棄却域に入る可能性）
  • $DW \approx 0.618$ は正の自己相関が存在することを強く示唆しており， モデルに問題がある（AR過程での補正が必要）ことを意味します．

問27.4 — AIC によるモデル選択

問題

AR($p$) 過程について $p = 0, 1, 2, 3, 4$ の各モデルの AIC（赤池情報量基準）を計算したところ， 下表の結果が得られた．AIC に基づいて最適なモデルを選べ．

モデルAIC
AR(0)$-1.171$
AR(1)$-4.720$
AR(2)$-4.764$
AR(3)$-4.771$
AR(4)$-4.766$
解答・解説
AIC（赤池情報量基準）の選択基準

AIC は値が小さいほど良いモデル（適合度が高くパラメータ数のペナルティを考慮したバランスが良い）です．

$$\text{AIC} = -2\log\hat{L} + 2k$$

第1項 $-2\log\hat{L}$ はモデルの当てはまり（小さいほど良い）， 第2項 $2k$ はパラメータ数 $k$ に対するペナルティ（多すぎる過適合を防ぐ）です．

各モデルの AIC を比較する

AIC の値を昇順に確認する

モデルAIC最小か？
AR(0)$-1.171$—（最大）
AR(1)$-4.720$—
AR(2)$-4.764$—
AR(3)$-4.771$✓ 最小
AR(4)$-4.766$—（AR(3)より大きい）

5つのモデルの中で AIC が最小（$= -4.771$）なのは AR(3) です．

AR(3) から AR(4) にすると AIC が増える理由

AR(3) から AR(4) にすると，パラメータが1つ増えるため，ペナルティ項 $2k$ が $+2$ 増加します． 一方，$-2\log\hat{L}$（対数尤度の改善）がその $+2$ を上回らないため， AR(4) の AIC は AR(3) より大きくなっています．

つまり，ラグ4の係数を加えてもモデルの当てはまりがほとんど改善せず， 余分なパラメータによるペナルティの方が大きいということを意味します（過適合）．

結論

AIC が最小（$-4.771$）となる AR(3) が最適なモデルとして選択される．

補足：AIC の理解を深める

AIC はモデルの「真の分布」と「推定した分布」の乖離を測るカルバック・ライブラー情報量の 推定量に基づいています（赤池弘次, 1974）．

AR($p$) 過程のパラメータ数（切片＋$p$個の係数＋分散 = $p+2$）のペナルティを考慮しているため， 不必要に複雑なモデル（過適合）を避け，予測精度が高いモデルを選べます．

なお，AIC と似た基準に BIC（ベイズ情報量基準） があります． BIC は $2k$ の代わりに $k\log T$ をペナルティとして使うため， サンプルサイズ $T$ が大きいほどより簡単なモデルを選ぶ傾向があります．

基準ペナルティ特徴
AIC$2k$サンプルサイズに依存しない
BIC$k\log T$大標本では罰則が強くなる（よりシンプルなモデルを好む）
