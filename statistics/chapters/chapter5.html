概要

第5章では、代表的な離散型分布を体系的に学ぶ。 離散一様分布、ベルヌーイ分布、二項分布、超幾何分布、ポアソン分布、幾何分布、負の二項分布、多項分布を扱い、 それぞれの確率関数・期待値・分散・確率母関数と、分布間の関係（再生性・極限関係）を整理する。

キーワード：離散一様分布、ベルヌーイ分布、二項分布、超幾何分布、ポアソン分布、幾何分布、負の二項分布、多項分布

重要概念

離散一様分布

  • $X$ が $1, 2, \ldots, K$ を等確率 $1/K$ でとる分布
  • $E[X] = \dfrac{K+1}{2}$、$V[X] = \dfrac{K^2 - 1}{12}$
  • 確率母関数：$G(s) = \dfrac{s(1 - s^K)}{K(1 - s)}$

ベルヌーイ分布 $\mathrm{Bin}(1, p)$

  • 成功（$X=1$）確率 $p$、失敗（$X=0$）確率 $q = 1-p$ の分布
  • 確率関数：$P(X = x) = p^x q^{1-x},\; x = 0, 1$
  • $E[X] = p$、$V[X] = pq$
  • 確率母関数：$G(s) = ps + q$

二項分布 $\mathrm{Bin}(n, p)$

  • 独立なベルヌーイ試行（成功確率 $p$）を $n$ 回行ったときの成功回数 $Y$ の分布
  • 確率関数：$P(Y = y) = \binom{n}{y} p^y q^{n-y},\; y = 0, 1, \ldots, n$
  • $E[Y] = np$、$V[Y] = npq$
  • 確率母関数：$G(s) = (ps + q)^n$
  • 再生性：$Y_1 \sim \mathrm{Bin}(n_1, p),\; Y_2 \sim \mathrm{Bin}(n_2, p)$ が独立 $\Rightarrow$ $Y_1 + Y_2 \sim \mathrm{Bin}(n_1 + n_2, p)$

超幾何分布 $\mathrm{HG}(N, M, n)$

  • $N$ 個の玉（赤 $M$ 個、白 $N-M$ 個）から非復元抽出で $n$ 個を取り出したとき、赤玉の個数 $Y$ の分布
  • 確率関数：$P(Y = y) = \dfrac{\binom{M}{y}\binom{N-M}{n-y}}{\binom{N}{n}}$
  • $E[Y] = n\dfrac{M}{N}$、$V[Y] = n\dfrac{M}{N}\dfrac{N-M}{N}\dfrac{N-n}{N-1}$
  • $\dfrac{N-n}{N-1}$ を有限母集団修正という。$n$ と $M/N$ を固定し $N \to \infty$ とすると二項分布に収束

ポアソン分布 $\mathrm{Po}(\lambda)$

  • 確率関数：$P(Y = y) = \dfrac{\lambda^y e^{-\lambda}}{y!},\; y = 0, 1, 2, \ldots$
  • $E[Y] = \lambda$、$V[Y] = \lambda$（期待値と分散が等しい）
  • 確率母関数：$G(s) = e^{\lambda(s-1)}$
  • 二項分布 $\mathrm{Bin}(n, p)$ で $np = \lambda$ に固定して $n \to \infty$ とした極限
  • 再生性：$Y_1 \sim \mathrm{Po}(\lambda_1),\; Y_2 \sim \mathrm{Po}(\lambda_2)$ が独立 $\Rightarrow$ $Y_1 + Y_2 \sim \mathrm{Po}(\lambda_1 + \lambda_2)$

幾何分布 $\mathrm{Geo}(p)$

  • 独立なベルヌーイ試行ではじめて成功するまでの失敗回数 $X$ の分布
  • 確率関数：$P(X = x) = pq^x,\; x = 0, 1, 2, \ldots$（幾何数列）
  • $E[X] = \dfrac{q}{p}$、$V[X] = \dfrac{q}{p^2}$
  • 確率母関数：$G(s) = \dfrac{p}{1 - qs},\; |s| < 1/q$
  • 無記憶性：$P(X \geq t_1 + t_2 \mid X \geq t_1) = P(X \geq t_2)$

負の二項分布 $\mathrm{NB}(r, p)$

  • $r$ 回目の成功が起こるまでの失敗回数 $Y$ の分布。$\mathrm{NB}(1, p) = \mathrm{Geo}(p)$
  • 確率関数：$P(Y = y) = \binom{y+r-1}{y} p^r q^y,\; y = 0, 1, 2, \ldots$
  • $E[Y] = \dfrac{rq}{p}$、$V[Y] = \dfrac{rq}{p^2}$
  • 確率母関数：$G(s) = \left(\dfrac{p}{1 - qs}\right)^r$
  • 互いに独立な $X_1, \ldots, X_r \sim \mathrm{Geo}(p)$ の和 $X_1 + \cdots + X_r \sim \mathrm{NB}(r, p)$
  • 再生性：$Y_1 \sim \mathrm{NB}(r_1, p),\; Y_2 \sim \mathrm{NB}(r_2, p)$ が独立 $\Rightarrow$ $Y_1 + Y_2 \sim \mathrm{NB}(r_1 + r_2, p)$

多項分布 $\mathrm{Mn}(n;\, p_1, \ldots, p_K)$

  • $K$ 個の結果が起こる試行を独立に $n$ 回行い、各結果の出現回数 $(Y^{(1)}, \ldots, Y^{(K)})$ の分布
  • $K = 2$ のとき二項分布に帰着
  • 確率関数：$P(Y^{(1)} = y^{(1)}, \ldots) = \dfrac{n!}{y^{(1)}! \cdots y^{(K)}!}\, p_1^{y^{(1)}} \cdots p_K^{y^{(K)}}$
  • $E[Y^{(j)}] = np_j$、$V[Y^{(j)}] = np_j(1 - p_j)$
  • $\mathrm{Cov}(Y^{(i)}, Y^{(j)}) = -np_i p_j \quad (i \neq j)$
  • 確率母関数：$G(s_1, \ldots, s_K) = (p_1 s_1 + \cdots + p_K s_K)^n$

公式・定理

離散型分布の一覧

  • 離散一様分布：$E = (K+1)/2$、$V = (K^2-1)/12$
  • ベルヌーイ：$E = p$、$V = pq$
  • 二項：$E = np$、$V = npq$、再生性あり
  • 超幾何：$E = nM/N$、$V = npq \cdot (N-n)/(N-1)$（$p = M/N$）
  • ポアソン：$E = V = \lambda$、再生性あり
  • 幾何：$E = q/p$、$V = q/p^2$、無記憶性
  • 負の二項：$E = rq/p$、$V = rq/p^2$、再生性あり
  • 多項：$E[Y^{(j)}] = np_j$、$V[Y^{(j)}] = np_j(1-p_j)$、$\mathrm{Cov} = -np_ip_j$

分布間の関係

  • ベルヌーイ $\xrightarrow{n\text{個の和}}$ 二項
  • 二項 $\xrightarrow{np=\lambda,\; n\to\infty}$ ポアソン
  • 超幾何 $\xrightarrow{N\to\infty}$ 二項
  • 幾何 $\xrightarrow{r\text{個の和}}$ 負の二項
  • 多項 $\xrightarrow{K=2}$ 二項

例題・演習

例1：超幾何分布によるA型識別力の検証

ゼミ生10人のうちA型は5人いる。学生が性格からA型5人を当てようとし、結果を分割表にまとめた。A型を4人正しく当てた場合、識別力があると主張できるか。

解：識別力がまったくない場合、正解数 $Y$ は超幾何分布 $\mathrm{HG}(10, 5, 5)$ に従う。$Y \geq 4$ の確率は

$$P(Y=4) + P(Y=5) = \frac{\binom{5}{4}\binom{5}{1}}{\binom{10}{5}} + \frac{\binom{5}{5}\binom{5}{0}}{\binom{10}{5}} = \frac{25}{252} + \frac{1}{252} = 0.103$$

まったく識別力がなくても10.3%は起こるので、A型を見分けられると主張するには無理がある。この考え方がフィッシャーの正確検定（第28章）の基礎となる。

問5.1：ウイルス発見確率

$n$ 株のウイルスを検査し、各株の発見率は $p$ で独立とする。

(1) 少なくとも1株ウイルスAがみつかる確率 $\beta$ を求める式を示せ。

$$\beta = 1 - (1 - p)^n$$

$Y \sim \mathrm{Bin}(n, p)$ より $\beta = P(Y \geq 1) = 1 - P(Y = 0) = 1 - (1-p)^n$。

(2) $p = 1/5000,\; \beta = 0.98$ のとき $n$ を求めよ。

$(1 - 1/5000)^n = 0.02$ の両辺の対数をとり、$\log(1 - 1/5000) \approx -1/5000$ を用いると

$$n \approx \frac{-3.9}{-1/5000} = \mathbf{19500}$$

問5.2：超幾何分布の確率関数

79人から25人を無作為に選ぶとき、男性就業者（40人）の人数 $X$ の確率関数を求めよ。

解：$X \sim \mathrm{HG}(79, 40, 25)$ より

$$P(X = x) = \frac{\binom{40}{x}\binom{39}{25-x}}{\binom{79}{25}}, \quad 0 \leq x \leq 25$$

問5.3：非復元抽出の期待値・分散

9人中3人が関東出身。非復元抽出で4人を選び、$i$ 番目が関東出身なら $X_i = 1$、そうでなければ $X_i = 0$ とする。

(1) $E[X_i]$ を求めよ。

$$E[X_i] = P(X_i = 1) = \frac{3}{9} = \mathbf{\frac{1}{3}}$$

(2) $i \neq j$ に対し $E[X_i X_j]$ を求めよ。

$$E[X_i X_j] = P(X_i = 1 \text{ かつ } X_j = 1) = \frac{3}{9} \times \frac{2}{8} = \mathbf{\frac{1}{12}}$$

(3) 標本平均 $\bar{X} = \frac{1}{4}\sum_{i=1}^{4} X_i$ の分散を求めよ。

$V[X_i] = 1/3 - (1/3)^2 = 2/9$、$\mathrm{Cov}[X_i, X_j] = 1/12 - (1/3)^2 = -1/36$ より

$$V\!\left[\sum X_i\right] = 4 \times \frac{2}{9} + 4 \cdot 3 \times \left(-\frac{1}{36}\right) = \frac{8}{9} - \frac{1}{3} = \frac{5}{9}$$
$$V[\bar{X}] = \frac{1}{16} \times \frac{5}{9} = \mathbf{\frac{5}{144}}$$

問5.4：ポアソン分布の再生性と条件付き分布

チーム $T_1$ の得点 $X \sim \mathrm{Po}(1.5)$、$T_2$ の得点 $Y \sim \mathrm{Po}(3)$ が独立とする。

(1) $X + Y$ の分布は？

再生性より $X + Y \sim \mathrm{Po}(4.5)$。平均 $\mathbf{4.5}$、分散 $\mathbf{4.5}$。

(2) $X + Y = 5$ のもとでの $X$ の分布は？

$$P(X = x \mid X + Y = 5) = \binom{5}{x}\left(\frac{1.5}{4.5}\right)^x\left(\frac{3}{4.5}\right)^{5-x} = \binom{5}{x}\left(\frac{1}{3}\right)^x\left(\frac{2}{3}\right)^{5-x}$$

よって $X \mid (X+Y=5) \sim \mathrm{Bin}(5, 1/3)$。平均は $\mathbf{5/3}$。

問5.5：カード収集問題（クーポンコレクター問題）

4種類のカードが等確率で付いてくるお菓子を買う。

(1) 4種類すべてを揃えるまでの購入回数の期待値を求めよ。

$k$ 種類揃ったあと $k+1$ 種類目を引く確率は $p_k = (4-k)/4$。期待値は $1/p_k = 4/(4-k)$ 回。

$$\sum_{k=0}^{3} \frac{4}{4-k} = \frac{4}{4} + \frac{4}{3} + \frac{4}{2} + \frac{4}{1} = 1 + \frac{4}{3} + 2 + 4 = \mathbf{\frac{25}{3}} \approx 8.33$$

(2) 4種類揃えた後に新1種類追加された場合と、最初から5種類の場合の期待値の差 $z_1 - z_2$ を求めよ。

$z_1$：4種類揃えた後に5種類目を引く期待値は $5/(5-4) = 5$ 回。$z_1 = 25/3 + 5 = 40/3$。

$z_2$：最初から5種類の場合 $\displaystyle\sum_{k=0}^{4} \frac{5}{5-k} = 1 + \frac{5}{4} + \frac{5}{3} + \frac{5}{2} + 5 = \frac{137}{12}$。

$$z_1 - z_2 = \frac{40}{3} - \frac{137}{12} = \mathbf{\frac{23}{12}} \approx 1.92$$

メモ

  • 二項分布・ポアソン分布・負の二項分布はいずれも再生性（独立な同種分布の和が同種分布）をもつ
  • 超幾何分布は非復元抽出、二項分布は復元抽出に対応。$N$ が十分大きければ両者は近似する
  • ポアソン分布は「稀な事象の回数」のモデルとして広く使われる。期待値＝分散が特徴
  • 幾何分布の無記憶性は離散分布では幾何分布のみがもつ性質。連続分布では指数分布が対応
  • 問5.4(2)のように、独立なポアソン変数の合計を条件としたとき、各変数は二項分布に従う（ポアソン分布と二項分布の関係）
  • 問5.5はクーポンコレクター問題。幾何分布の期待値 $1/p$ を各段階で合計する方法で解ける
  • 多項分布は二項分布の多次元への一般化。共分散が負になる点に注意
