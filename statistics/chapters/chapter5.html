1. 離散型確率分布とは何か

確率変数 $X$ が取りうる値が、$0, 1, 2, \ldots$ のように飛び飛び（離散的）である場合、 その確率の分布を離散型確率分布と呼びます。

確率関数（probability mass function; PMF）とは

$X$ が値 $x$ をとる確率を $P(X = x)$ と書き、これを確率関数といいます。 確率関数は次の2条件を満たします。

  1. すべての $x$ に対して $P(X = x) \geq 0$
  2. すべての確率の合計が 1：$\displaystyle\sum_{\text{すべての } x} P(X = x) = 1$

期待値・分散・確率母関数の復習

各分布を学ぶ前に、共通で使う道具を整理しておきます。

期待値（平均）
$$E[X] = \sum_x x \cdot P(X = x)$$

直感：確率を重みとして、値 $x$ を「加重平均」したもの。 サイコロを何度も振ったときの目の平均値がイメージしやすいです。

分散
$$V[X] = E\!\left[(X - E[X])^2\right] = E[X^2] - (E[X])^2$$

直感：$X$ が平均からどれだけ「散らばっている」かを示します。 $V[X]$ が大きいほど値がばらつきやすい。

確率母関数（PGF）
$$G(s) = E[s^X] = \sum_{x=0}^{\infty} s^x \cdot P(X = x)$$

確率母関数は分布全体を一つの関数に"収納"したものです。 $s=1$ を代入すると必ず $G(1)=1$ になります（確率の合計が1だから）。 さらに微分すると期待値・分散を取り出せる便利な道具です。

活用方法（重要）：$G'(1) = E[X]$、$G''(1) = E[X(X-1)]$ を利用して

$$E[X] = G'(1), \qquad V[X] = G''(1) + G'(1) - \{G'(1)\}^2$$

そして再生性の証明に非常に有効です（独立な変数の和のPGFは、各PGFの積になる）。

2. 離散一様分布

どんな分布か？（直感）

$1, 2, \ldots, K$ のどの値も同じ確率で起こる分布です。 例：1〜6の目が等確率のサイコロ、くじびきで10枚から1枚引く、など。

定義

$X$ が $K$ 個の値 $\{1, 2, \ldots, K\}$ を等確率でとるとき

$$P(X = x) = \frac{1}{K}, \quad x = 1, 2, \ldots, K$$

期待値の導出

定義に従って計算します。

$$E[X] = \sum_{x=1}^{K} x \cdot \frac{1}{K} = \frac{1}{K} \sum_{x=1}^{K} x = \frac{1}{K} \cdot \frac{K(K+1)}{2} = \frac{K+1}{2}$$

ここで $\displaystyle\sum_{x=1}^{K} x = \dfrac{K(K+1)}{2}$（等差数列の公式）を使いました。 直感：$1$ から $K$ までの真ん中の値が平均になることは自然ですね。

分散の導出

$V[X] = E[X^2] - (E[X])^2$ を使います。まず $E[X^2]$ を求めます。

$$E[X^2] = \sum_{x=1}^{K} x^2 \cdot \frac{1}{K} = \frac{1}{K} \cdot \frac{K(K+1)(2K+1)}{6} = \frac{(K+1)(2K+1)}{6}$$

ここで $\displaystyle\sum_{x=1}^{K} x^2 = \dfrac{K(K+1)(2K+1)}{6}$（平方和の公式）を使いました。よって

$$V[X] = \frac{(K+1)(2K+1)}{6} - \left(\frac{K+1}{2}\right)^2 = \frac{(K+1)(2K+1)}{6} - \frac{(K+1)^2}{4}$$

通分（分母を $12$ にする）して整理します。

$$= \frac{2(K+1)(2K+1)}{12} - \frac{3(K+1)^2}{12} = \frac{(K+1)\bigl[2(2K+1) - 3(K+1)\bigr]}{12}$$
$$= \frac{(K+1)(4K + 2 - 3K - 3)}{12} = \frac{(K+1)(K-1)}{12} = \boxed{\frac{K^2 - 1}{12}}$$

3. ベルヌーイ分布 $\mathrm{Bin}(1, p)$

直感的イメージ

コインを1回投げる試行：表（成功）が確率 $p$、裏（失敗）が確率 $q = 1-p$ で起こる。 これがベルヌーイ試行で、その結果の分布をベルヌーイ分布といいます。 成功を $X=1$、失敗を $X=0$ と数値化します。

定義

$p$: 成功確率、$q = 1 - p$: 失敗確率

$$P(X = x) = p^x q^{1-x}, \quad x = 0, 1$$

この式のポイント：$x=1$ のとき $P(X=1) = p^1 q^0 = p$、 $x=0$ のとき $P(X=0) = p^0 q^1 = q$ と自動的になります。 1つの式で2つの場合をスッキリ表現しています。

期待値・分散の導出

$$E[X] = 0 \cdot q + 1 \cdot p = p$$
$$E[X^2] = 0^2 \cdot q + 1^2 \cdot p = p$$
$$V[X] = E[X^2] - (E[X])^2 = p - p^2 = p(1-p) = pq$$

確率母関数の導出

$$G(s) = E[s^X] = s^0 \cdot q + s^1 \cdot p = q + ps = ps + q$$
💡 分散 $V[X] = pq$ の直感

$p = 0$ や $p = 1$（結果が確定している）のとき $V[X] = 0$（散らばりなし）。 $p = 0.5$（最も不確か）のとき $V[X] = 0.25$ で最大。 不確かさが大きいほど分散も大きくなる、という自然な性質です。

4. 二項分布 $\mathrm{Bin}(n, p)$

直感的イメージ

コインを $n$ 回投げる（ = ベルヌーイ試行を $n$ 回繰り返す）。 各回は独立で成功確率は常に $p$。 このとき成功した回数 $Y$ が従う分布が二項分布です。

例：打率 3割のバッターが10打席立つとき、何本ヒットを打つか → $Y \sim \mathrm{Bin}(10, 0.3)$

確率関数の導出

$n$ 回中ちょうど $y$ 回成功する確率を求めます。

  1. 「どの $y$ 回が成功か」の選び方は $\dbinom{n}{y}$ 通り。
  2. 特定の $y$ 回が成功し、残り $n-y$ 回が失敗する確率は $p^y \cdot q^{n-y}$（独立なので掛け算）。
  3. 以上を掛け合わせて $$P(Y = y) = \binom{n}{y} p^y q^{n-y}, \quad y = 0, 1, \ldots, n$$

確認：$\displaystyle\sum_{y=0}^{n} \binom{n}{y} p^y q^{n-y} = (p+q)^n = 1^n = 1$ ✓ （二項定理を使いました）

期待値の導出

「$n$ 回のベルヌーイ試行の和」と見れば、期待値の線形性から

$$Y = X_1 + X_2 + \cdots + X_n \quad (X_i \sim \mathrm{Bin}(1,p) : \text{独立})$$ $$E[Y] = E[X_1] + \cdots + E[X_n] = \underbrace{p + p + \cdots + p}_{n\text{個}} = np$$

分散の導出

各 $X_i$ は独立なので分散も加算できます。

$$V[Y] = V[X_1] + \cdots + V[X_n] = npq$$

確率母関数の導出

独立な和のPGFは各PGFの積になるという性質を使います。

$$G_Y(s) = \prod_{i=1}^{n} G_{X_i}(s) = \prod_{i=1}^{n}(ps+q) = (ps+q)^n$$

再生性の証明

再生性（重要定理）

$Y_1 \sim \mathrm{Bin}(n_1, p)$、$Y_2 \sim \mathrm{Bin}(n_2, p)$ が独立ならば

$$Y_1 + Y_2 \sim \mathrm{Bin}(n_1 + n_2, p)$$

証明： PGFの積の性質を使います。

$$G_{Y_1+Y_2}(s) = G_{Y_1}(s) \cdot G_{Y_2}(s) = (ps+q)^{n_1} \cdot (ps+q)^{n_2} = (ps+q)^{n_1+n_2}$$

これは $\mathrm{Bin}(n_1+n_2, p)$ のPGFに一致します。PGFが一致すれば分布も一致するので証明完了。

💡 再生性の直感

コインを $n_1$ 回投げた成功数と、別のコインを $n_2$ 回投げた成功数を足すと、 合計 $n_1+n_2$ 回投げた成功数と同じ分布になる。当然の結果を数学的に確認したものです。

5. 超幾何分布 $\mathrm{HG}(N, M, n)$

直感的イメージ

箱の中に玉が $N$ 個あり、赤が $M$ 個、白が $N-M$ 個入っています。 この中から戻さずに $n$ 個取り出すとき、赤玉の個数 $Y$ が従うのが超幾何分布。

二項分布との違い

二項分布は「戻して抽出」（各回独立）。 超幾何分布は「戻さずに抽出」（各回が独立でない）。 この違いが分散に影響し、「有限母集団修正」が現れます。

確率関数の導出

$N$ 個から $n$ 個を選ぶ全体の選び方：$\dbinom{N}{n}$

赤を $y$ 個、白を $n-y$ 個選ぶ選び方：$\dbinom{M}{y} \times \dbinom{N-M}{n-y}$

$$P(Y = y) = \frac{\dbinom{M}{y}\dbinom{N-M}{n-y}}{\dbinom{N}{n}}, \quad \max(0, n-N+M) \leq y \leq \min(n, M)$$

期待値の導出

各試行の結果を $X_i = 1$（$i$ 番目が赤）または $0$ と定義します。 このとき $Y = X_1 + X_2 + \cdots + X_n$ と表せます。

$$E[X_i] = P(X_i = 1) = \frac{M}{N}$$

なぜ $M/N$ か：$N$ 個の玉から1個取り出したとき赤である確率は $M/N$（これは位置に依存しない）。

$$E[Y] = \sum_{i=1}^{n} E[X_i] = n \cdot \frac{M}{N}$$

分散の導出

$p = M/N$ とおきます。まず各 $X_i$ の分散：

$$V[X_i] = p(1-p) = \frac{M}{N} \cdot \frac{N-M}{N}$$

$i \neq j$ のとき、共分散を求めます。

$$E[X_i X_j] = P(X_i = 1 \text{ かつ } X_j = 1) = \frac{M}{N} \cdot \frac{M-1}{N-1}$$

（1個目が赤の確率 $M/N$、その後2個目も赤の確率 $(M-1)/(N-1)$）

$$\mathrm{Cov}(X_i, X_j) = E[X_i X_j] - E[X_i]E[X_j] = \frac{M(M-1)}{N(N-1)} - \frac{M^2}{N^2} = \frac{M(M-1)N^2 - M^2 N(N-1)}{N^2 \cdot N(N-1)}$$
$$= \frac{MN\bigl[(M-1)N - M(N-1)\bigr]}{N^3(N-1)} = \frac{MN(MN - N - MN + M)}{N^3(N-1)} = \frac{MN(M-N)}{N^3(N-1)} = -\frac{M(N-M)}{N^2(N-1)}$$

$V[Y]$ を求めます（$n$ 個の変数の和の分散公式）：

$$V[Y] = \sum_{i=1}^{n} V[X_i] + 2\sum_{i < j} \mathrm{Cov}(X_i, X_j)$$ $$= n \cdot \frac{M(N-M)}{N^2} + n(n-1) \cdot \left(-\frac{M(N-M)}{N^2(N-1)}\right)$$
$$= \frac{nM(N-M)}{N^2} \left(1 - \frac{n-1}{N-1}\right) = \frac{nM(N-M)}{N^2} \cdot \frac{N-n}{N-1}$$

$p = M/N$、$q = (N-M)/N$ を代入すると最終的に

$$\boxed{V[Y] = n \cdot \frac{M}{N} \cdot \frac{N-M}{N} \cdot \frac{N-n}{N-1} = npq \cdot \frac{N-n}{N-1}}$$
有限母集団修正 $\dfrac{N-n}{N-1}$ とは

$\dfrac{N-n}{N-1}$ を有限母集団修正と呼びます。

  • $n = 1$ のとき：$\dfrac{N-1}{N-1} = 1 \Rightarrow$ 二項分布の分散 $pq$ と一致
  • $n = N$ のとき：$\dfrac{N-N}{N-1} = 0 \Rightarrow$ 全部取り出せばばらつきゼロ（当然）
  • $N \to \infty$ のとき：$\dfrac{N-n}{N-1} \to 1 \Rightarrow$ 二項分布の分散 $npq$ に収束

直感：母集団が小さいほど「1個取り出すと残りが変わる」影響が大きい。 母集団が無限大なら戻す・戻さないの違いがなくなり、二項分布に近づく。

6. ポアソン分布 $\mathrm{Po}(\lambda)$

直感的イメージ

「ある時間・空間の中で、稀な事象が何回起こるか」を表す分布。例：

  • 1日に来店する客数（客が来る頻度が低い専門店）
  • 1ページあたりの誤字の数
  • 1時間あたりの放射線崩壊数

$\lambda$ は単位時間（空間）あたりの平均発生件数です。

確率関数
$$P(Y = y) = \frac{\lambda^y e^{-\lambda}}{y!}, \quad y = 0, 1, 2, \ldots$$

二項分布からポアソン分布を導く（極限）

「$n$ 回中成功確率 $p = \lambda/n$ の試行」を考え、$n \to \infty$ とします。 これが「まれで回数が多い」状況の数学的極限です。

$$P(Y = y) = \binom{n}{y}\left(\frac{\lambda}{n}\right)^y\left(1 - \frac{\lambda}{n}\right)^{n-y}$$

各因子を $n \to \infty$ で評価します。

$$\binom{n}{y}\frac{1}{n^y} = \frac{n(n-1)\cdots(n-y+1)}{y! \cdot n^y} = \frac{1}{y!}\cdot 1 \cdot \frac{n-1}{n} \cdots \frac{n-y+1}{n} \to \frac{1}{y!}$$
$$\left(1 - \frac{\lambda}{n}\right)^{n-y} = \left(1 - \frac{\lambda}{n}\right)^n \cdot \left(1 - \frac{\lambda}{n}\right)^{-y} \to e^{-\lambda} \cdot 1 = e^{-\lambda}$$

（$\left(1 - \lambda/n\right)^n \to e^{-\lambda}$ はネイピア数の定義から）

したがって

$$P(Y = y) \to \frac{\lambda^y e^{-\lambda}}{y!}$$

確率の合計が1になることの確認

$$\sum_{y=0}^{\infty} \frac{\lambda^y e^{-\lambda}}{y!} = e^{-\lambda} \sum_{y=0}^{\infty} \frac{\lambda^y}{y!} = e^{-\lambda} \cdot e^{\lambda} = 1 \checkmark$$

（$e^\lambda = \sum_{y=0}^\infty \lambda^y / y!$ : 指数関数のマクローリン展開）

期待値・分散の導出

$$E[Y] = \sum_{y=0}^{\infty} y \cdot \frac{\lambda^y e^{-\lambda}}{y!} = e^{-\lambda} \sum_{y=1}^{\infty} \frac{\lambda^y}{(y-1)!}$$

$k = y-1$ と置くと

$$= e^{-\lambda} \cdot \lambda \sum_{k=0}^{\infty} \frac{\lambda^k}{k!} = e^{-\lambda} \cdot \lambda \cdot e^{\lambda} = \lambda$$

同様の計算（$E[Y(Y-1)]$ を経由）で $V[Y] = \lambda$ も導けます。

期待値 = 分散（ $E[Y] = V[Y] = \lambda$ ）

ポアソン分布の最大の特徴は期待値と分散がどちらも $\lambda$ で等しいことです。 実際のデータで分散が平均よりずっと大きくなる（過分散）場合は、 ポアソンでなく負の二項分布などの使用を検討します。

確率母関数

$$G(s) = E[s^Y] = \sum_{y=0}^{\infty} s^y \cdot \frac{\lambda^y e^{-\lambda}}{y!} = e^{-\lambda} \sum_{y=0}^{\infty} \frac{(\lambda s)^y}{y!} = e^{-\lambda} \cdot e^{\lambda s} = e^{\lambda(s-1)}$$

再生性の証明

再生性

$Y_1 \sim \mathrm{Po}(\lambda_1)$ と $Y_2 \sim \mathrm{Po}(\lambda_2)$ が独立ならば $Y_1 + Y_2 \sim \mathrm{Po}(\lambda_1 + \lambda_2)$

PGFで確認：

$$G_{Y_1+Y_2}(s) = e^{\lambda_1(s-1)} \cdot e^{\lambda_2(s-1)} = e^{(\lambda_1+\lambda_2)(s-1)}$$

これは $\mathrm{Po}(\lambda_1 + \lambda_2)$ のPGFと一致します。

7. 幾何分布 $\mathrm{Geo}(p)$

直感的イメージ

ベルヌーイ試行を繰り返して、はじめて成功するまでの失敗回数 $X$ の分布。 例：宝くじで初めて当たるまでに外れる枚数、バグのないプログラムを書くまでの試行回数 − 1。

確率関数の導出

$X = x$ とは「$x$ 回失敗してから1回成功」すること。 各試行は独立なので

$$P(X = x) = \underbrace{q \cdot q \cdots q}_{x\text{回失敗}} \cdot p = q^x p, \quad x = 0, 1, 2, \ldots$$

確認：$\displaystyle\sum_{x=0}^{\infty} pq^x = p \cdot \frac{1}{1-q} = p \cdot \frac{1}{p} = 1$ ✓（等比級数の公式）

期待値の導出

$$E[X] = \sum_{x=0}^{\infty} x \cdot pq^x = p \sum_{x=1}^{\infty} x q^x$$

$\displaystyle\sum_{x=1}^{\infty} x q^x = \dfrac{q}{(1-q)^2} = \dfrac{q}{p^2}$（等比級数を微分）を用いると

$$E[X] = p \cdot \frac{q}{p^2} = \frac{q}{p}$$

分散の導出

$\displaystyle\sum_{x=0}^{\infty} x(x-1)q^x = \dfrac{2q^2}{p^3}$ を利用します。

$$E[X(X-1)] = p \cdot \frac{2q^2}{p^3} = \frac{2q^2}{p^2}$$
$$E[X^2] = E[X(X-1)] + E[X] = \frac{2q^2}{p^2} + \frac{q}{p}$$
$$V[X] = E[X^2] - (E[X])^2 = \frac{2q^2}{p^2} + \frac{q}{p} - \frac{q^2}{p^2} = \frac{q^2}{p^2} + \frac{q}{p} = \frac{q(q+p)}{p^2} = \frac{q}{p^2}$$

無記憶性の証明

無記憶性（幾何分布の最大の特徴）
$$P(X \geq t_1 + t_2 \mid X \geq t_1) = P(X \geq t_2)$$

直感：「すでに $t_1$ 回失敗した」という情報は、 「あと何回失敗するか」の確率分布に一切影響しない。 つまり「過去を忘れる」分布です。

証明：

$$P(X \geq t) = \sum_{x=t}^{\infty} pq^x = pq^t \cdot \frac{1}{1-q} = q^t$$
$$P(X \geq t_1 + t_2 \mid X \geq t_1) = \frac{P(X \geq t_1 + t_2)}{P(X \geq t_1)} = \frac{q^{t_1+t_2}}{q^{t_1}} = q^{t_2} = P(X \geq t_2) \checkmark$$

8. 負の二項分布 $\mathrm{NB}(r, p)$

直感的イメージ

幾何分布（1回目の成功まで）を拡張して、 $r$ 回目の成功が起こるまでの失敗回数 $Y$ の分布です。 $r = 1$ のとき $\mathrm{NB}(1, p) = \mathrm{Geo}(p)$ になります。

確率関数の導出

$Y = y$ とは「$y+r-1$ 回目までに $r-1$ 回成功・$y$ 回失敗し、 $y+r$ 回目に成功」することです。

$$P(Y = y) = \underbrace{\binom{y+r-1}{y}}_{\text{最後以外の並べ方}} \cdot \underbrace{p^{r-1} q^y}_{\text{最後の前}} \cdot \underbrace{p}_{\text{最後は成功}} = \binom{y+r-1}{y} p^r q^y$$

期待値・分散・PGF

「$r$ 個の独立な幾何変数 $X_1, \ldots, X_r \sim \mathrm{Geo}(p)$ の和」と見るのが最速です。

$$E[Y] = r \cdot E[X_i] = r \cdot \frac{q}{p} = \frac{rq}{p}$$
$$V[Y] = r \cdot V[X_i] = r \cdot \frac{q}{p^2} = \frac{rq}{p^2}$$
$$G(s) = \left(\frac{p}{1-qs}\right)^r \quad (|s| < 1/q)$$

再生性

再生性

$Y_1 \sim \mathrm{NB}(r_1, p)$、$Y_2 \sim \mathrm{NB}(r_2, p)$ が独立なら $Y_1 + Y_2 \sim \mathrm{NB}(r_1+r_2, p)$。

$$G_{Y_1+Y_2}(s) = \left(\frac{p}{1-qs}\right)^{r_1} \cdot \left(\frac{p}{1-qs}\right)^{r_2} = \left(\frac{p}{1-qs}\right)^{r_1+r_2}$$

9. 多項分布 $\mathrm{Mn}(n;\, p_1, \ldots, p_K)$

直感的イメージ

サイコロを $n$ 回振って目 $k$（$k = 1, \ldots, 6$）が出た回数 $Y^{(k)}$ の同時分布、 というイメージです。二項分布は「成功か失敗か」の2択でしたが、 多項分布は $K$ 択に拡張したものです。

定義・確率関数

結果が $K$ 通りある試行を $n$ 回独立に行い、 結果 $k$ が起こる確率を $p_k$（$\sum p_k = 1$）とします。 各結果の出現回数を $Y^{(k)}$（$\sum Y^{(k)} = n$）とすると

$$P\!\left(Y^{(1)}=y^{(1)}, \ldots, Y^{(K)}=y^{(K)}\right) = \frac{n!}{y^{(1)}!\cdots y^{(K)}!}\, p_1^{y^{(1)}} \cdots p_K^{y^{(K)}}$$

$\dfrac{n!}{y^{(1)}!\cdots y^{(K)}!}$ は多項係数で、 「$n$ 回の試行をどの回に何の結果を割り当てるか」の場合の数。

期待値・分散・共分散

$Y^{(j)}$ だけに着目すると（他の結果をまとめて「失敗」とみなす）、 $Y^{(j)} \sim \mathrm{Bin}(n, p_j)$ なので

$$E[Y^{(j)}] = np_j, \quad V[Y^{(j)}] = np_j(1-p_j)$$

$i \neq j$ の共分散は負になります。直感：$Y^{(i)}$ が大きくなれば $n$ という合計の制約上 $Y^{(j)}$ は小さくなる。

$$\mathrm{Cov}(Y^{(i)}, Y^{(j)}) = -np_ip_j \quad (i \neq j)$$

10. 分布間の関係まとめ

一覧表

分布 $E[X]$ $V[X]$ 再生性 特記
離散一様 $\mathrm{Unif}(K)$ $(K+1)/2$ $(K^2-1)/12$ ─ ─
ベルヌーイ $\mathrm{Bin}(1,p)$ $p$ $pq$ ─ ─
二項 $\mathrm{Bin}(n,p)$ $np$ $npq$ ✓ ─
超幾何 $\mathrm{HG}(N,M,n)$ $nM/N$ $npq \cdot \tfrac{N-n}{N-1}$ ─ 有限母集団修正
ポアソン $\mathrm{Po}(\lambda)$ $\lambda$ $\lambda$ ✓ $E = V$
幾何 $\mathrm{Geo}(p)$ $q/p$ $q/p^2$ ─ 無記憶性
負の二項 $\mathrm{NB}(r,p)$ $rq/p$ $rq/p^2$ ✓ ─
多項 $\mathrm{Mn}(n;\mathbf{p})$ $np_j$ $np_j(1-p_j)$ ─ $\mathrm{Cov}=-np_ip_j$

分布の包含関係

  • ベルヌーイ $\xrightarrow{n \text{ 個の独立和}}$ 二項
  • 幾何 $\xrightarrow{r \text{ 個の独立和}}$ 負の二項
  • 多項（$K=2$）$\longleftrightarrow$ 二項

極限による近似

  • 二項 $\xrightarrow{np=\lambda,\; n\to\infty}$ ポアソン（まれな事象の極限）
  • 超幾何 $\xrightarrow{N\to\infty}$ 二項（非復元→復元の近似）
  • ポアソン分布の条件付き分布は二項分布になる（問5.4参照）

11. 例題・演習（オリジナル問題）

【例題1】二項分布の確率計算

問題

ある地域の新生児において、男児が生まれる確率は 0.52 とする。 この地域で無作為に選んだ 7 名の新生児のうち、男児がちょうど 3 人である確率を求めよ。 さらに、男児が 3 人以上である確率も求めよ。

解答

$Y$ を 7 人中の男児数とすると $Y \sim \mathrm{Bin}(7, 0.52)$、$q = 1 - 0.52 = 0.48$。

（1）$P(Y = 3)$ を求める。

$$P(Y = 3) = \binom{7}{3}(0.52)^3(0.48)^4$$

各項を計算します。

$$\binom{7}{3} = \frac{7!}{3!\,4!} = \frac{7 \times 6 \times 5}{3 \times 2 \times 1} = 35$$
$$(0.52)^3 = 0.140608, \qquad (0.48)^4 = 0.053010\ldots$$
$$P(Y = 3) = 35 \times 0.140608 \times 0.053010 \approx \mathbf{0.2609}$$

（2）$P(Y \geq 3)$ を求める。

余事象を使います：

$$P(Y \geq 3) = 1 - P(Y \leq 2) = 1 - \bigl[P(Y=0) + P(Y=1) + P(Y=2)\bigr]$$
$$P(Y=0) = (0.48)^7 \approx 0.003836$$
$$P(Y=1) = \binom{7}{1}(0.52)^1(0.48)^6 = 7 \times 0.52 \times 0.007971 \approx 0.028982$$
$$P(Y=2) = \binom{7}{2}(0.52)^2(0.48)^5 = 21 \times 0.2704 \times 0.025480 \approx 0.096895$$
$$P(Y \geq 3) \approx 1 - (0.003836 + 0.028982 + 0.096895) \approx \mathbf{0.870}$$

【例題2】超幾何分布と二項分布の分散の比較

問題

社員 20 人のうち女性が 8 人いる。この中から 5 人を無作為に選ぶとき、 女性の人数 $X$ の期待値と分散を求めよ。 また、同じ比率のまま社員が無限に多いとした場合（復元抽出と同様）の分散と比較し、 有限母集団修正の働きを確かめよ。

解答

$X \sim \mathrm{HG}(20, 8, 5)$。$p = M/N = 8/20 = 0.4$、$q = 0.6$。

（1）期待値

$$E[X] = n \cdot \frac{M}{N} = 5 \times 0.4 = \mathbf{2}$$

（2）超幾何分布の分散

$$V[X] = npq \cdot \frac{N-n}{N-1} = 5 \times 0.4 \times 0.6 \times \frac{20-5}{20-1} = 1.2 \times \frac{15}{19} \approx \mathbf{0.947}$$

（3）二項分布（復元抽出）の分散

$$V_{\text{Bin}} = npq = 5 \times 0.4 \times 0.6 = \mathbf{1.2}$$

比較：超幾何分布の分散 0.947 < 二項分布の分散 1.2。 有限母集団修正 $\dfrac{N-n}{N-1} = \dfrac{15}{19} \approx 0.789$ が分散を縮める効果を持っています。 非復元抽出では「取り出したものは戻らない」ため、結果の多様性が制限され、ばらつきが小さくなります。

【例題3】ポアソン分布の累積確率

問題

ある交差点では1時間あたり平均 2.5 件の交通事故が発生するとする。 事故発生件数はポアソン分布に従うと仮定して、次の確率を求めよ。

  1. 1時間に事故が1件も起きない確率
  2. 1時間に事故が 3 件以上起きる確率
解答

$Y \sim \mathrm{Po}(2.5)$。$e^{-2.5} \approx 0.08208$。

（1）$P(Y = 0)$

$$P(Y = 0) = \frac{2.5^0 e^{-2.5}}{0!} = e^{-2.5} \approx \mathbf{0.082}$$

（2）$P(Y \geq 3)$

$$P(Y \geq 3) = 1 - \bigl[P(Y=0) + P(Y=1) + P(Y=2)\bigr]$$
$$P(Y=1) = \frac{2.5^1 e^{-2.5}}{1!} = 2.5 \times 0.08208 \approx 0.2052$$
$$P(Y=2) = \frac{2.5^2 e^{-2.5}}{2!} = \frac{6.25 \times 0.08208}{2} \approx 0.2565$$
$$P(Y \geq 3) \approx 1 - (0.0821 + 0.2052 + 0.2565) = 1 - 0.5438 \approx \mathbf{0.456}$$

つまり、この交差点では1時間中に約 45.6% の確率で 3 件以上事故が起きることになります。

【例題4】幾何分布の無記憶性の応用

問題

確率 $p = 0.2$ で成功するゲームを繰り返す。

  1. はじめて成功するまでの失敗回数 $X$ の期待値と分散を求めよ。
  2. すでに 3 回失敗している。あと 2 回以上失敗する確率を求めよ。
解答

$X \sim \mathrm{Geo}(0.2)$、$q = 0.8$。

（1）期待値・分散

$$E[X] = \frac{q}{p} = \frac{0.8}{0.2} = \mathbf{4}（\text{回}）$$
$$V[X] = \frac{q}{p^2} = \frac{0.8}{0.04} = \mathbf{20}$$

平均して 4 回失敗してから初めて成功するということです。

（2）無記憶性の適用

無記憶性より、すでに 3 回失敗した事実は今後の確率に影響しません。

$$P(X \geq 3+2 \mid X \geq 3) = P(X \geq 2) = q^2 = (0.8)^2 = \mathbf{0.64}$$

「あと 2 回以上失敗する確率」は、最初から何も起きていない場合に 「2 回以上失敗する確率」と全く同じです。

【例題5】負の二項分布の期待値の応用

問題

あるスマートフォンゲームでは、レアアイテムが出る確率は $p = 0.05$ である。 このゲームを独立に何度もプレイして 3 個レアアイテムを集めたい。 3 個集まるまでに外れる（レアが出ない）回数 $Y$ の期待値と分散を求めよ。 また、3 個集めるまでの総試行回数（当たりも含む）の期待値も求めよ。

解答

$Y \sim \mathrm{NB}(3, 0.05)$、$q = 0.95$。

（1）失敗回数 $Y$ の期待値・分散

$$E[Y] = \frac{rq}{p} = \frac{3 \times 0.95}{0.05} = \frac{2.85}{0.05} = \mathbf{57}（\text{回}）$$
$$V[Y] = \frac{rq}{p^2} = \frac{3 \times 0.95}{0.0025} = \frac{2.85}{0.0025} = \mathbf{1140}$$

（2）総試行回数 $Y + r = Y + 3$ の期待値

$$E[Y + 3] = E[Y] + 3 = 57 + 3 = \mathbf{60}（\text{回}）$$

直感：「1 回で当たる確率が 0.05 なら、平均 $1/p = 20$ 回で 1 個」→ 3 個集めるには平均 $3/0.05 = 60$ 回必要。$E[Y+r] = r/p$ という形で覚えると便利です。

💡 確認：$E[Y+r] = r/p$
$$E[Y + r] = \frac{rq}{p} + r = r\left(\frac{q}{p} + 1\right) = r \cdot \frac{q+p}{p} = \frac{r}{p}$$

$q + p = 1$ を使っています。式変形の確認が大切です。

【例題6】多項分布の共分散

問題

あるアンケートで「賛成・反対・どちらでもない」の 3 択を 50 人に聞く。 それぞれの選択肢の割合は $p_1 = 0.5$、$p_2 = 0.3$、$p_3 = 0.2$ とする。 $Y^{(1)}$（賛成人数）と $Y^{(2)}$（反対人数）について、 期待値・分散・共分散・相関係数を求めよ。

解答

$(Y^{(1)}, Y^{(2)}, Y^{(3)}) \sim \mathrm{Mn}(50;\, 0.5, 0.3, 0.2)$。

（1）期待値

$$E[Y^{(1)}] = np_1 = 50 \times 0.5 = 25, \qquad E[Y^{(2)}] = np_2 = 50 \times 0.3 = 15$$

（2）分散

$$V[Y^{(1)}] = np_1(1-p_1) = 50 \times 0.5 \times 0.5 = 12.5$$
$$V[Y^{(2)}] = np_2(1-p_2) = 50 \times 0.3 \times 0.7 = 10.5$$

（3）共分散

$$\mathrm{Cov}(Y^{(1)}, Y^{(2)}) = -np_1 p_2 = -50 \times 0.5 \times 0.3 = -7.5$$

負の値：賛成が多いほど反対（やどちらでもない）は少なくなる傾向。 合計が $n = 50$ という制約から自然に生まれる負の相関です。

（4）相関係数

$$\rho = \frac{\mathrm{Cov}(Y^{(1)}, Y^{(2)})}{\sqrt{V[Y^{(1)}] \cdot V[Y^{(2)}]}} = \frac{-7.5}{\sqrt{12.5 \times 10.5}} = \frac{-7.5}{\sqrt{131.25}} = \frac{-7.5}{11.456} \approx \mathbf{-0.655}$$
💡 一般的な公式：多項分布の相関係数
$$\rho(Y^{(i)}, Y^{(j)}) = \frac{-np_ip_j}{\sqrt{np_i(1-p_i) \cdot np_j(1-p_j)}} = -\sqrt{\frac{p_ip_j}{(1-p_i)(1-p_j)}}$$

相関係数が $-\sqrt{\cdot}$ の形になるのは多項分布の特徴です。
