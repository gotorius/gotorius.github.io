概要

本章では、統計学におけるシミュレーション手法を学ぶ。乱数の生成方法（逆関数法、採択棄却法）、モンテカルロ法による積分の近似とその精度向上法、そして統計量の標準誤差やバイアスを推定するジャックナイフ法・ブートストラップ法について扱う。

重要概念

乱数（Random Number）

  • 物理乱数（physical random number）：熱雑音や原子核分裂など物理現象を利用。周期や再現性がない。専用ハードウェアが必要
  • 擬似乱数（pseudo random number）：漸化式に基づく。計算機と合った速度で生成可能。一様性・独立性・高次元での稠密性が必要
  • メルセンヌツイスター：非常に長い周期と高次元での稠密性をもつ擬似乱数生成法

逆関数法（Inverse Transformation Method）

$F_X(x)$ の逆関数 $F_X^{-1}$ が陽に得られる場合：

  1. $U(0,1)$ から一様乱数 $u$ をサンプリング
  2. $x = F_X^{-1}(u)$ とする

指数分布、コーシー分布、レイリー分布などに適用可能。$F_X^{-1}$ が陽に表現できなくてもニュートン法等で近似可能。

採択棄却法（Acceptance-Rejection Method）

目標密度 $f(x) = c^{-1}h(x)$ に対し、$Mg(x) \ge h(x)$ を満たす密度 $g(x)$ と定数 $M > 0$ を用いる：

  1. $g(x)$ から乱数 $x$、$U(0,1)$ から $u$ をサンプリング
  2. $u \le h(x)/(Mg(x))$ なら $x$ を出力、そうでなければ1に戻る

採択確率は $1/M$。$M$ が小さいほど効率がよい。

モンテカルロ法の精度向上

  • 主部の分離：$g(x)$ のマクローリン展開等で主部 $l(x)$ を取り出し、$g(x) - l(x) + \int l\,dx$ の分散を低減
  • 負の相関を利用：$g(X)$ と $g(1-X)$ の負の相関を利用。$\text{se}(\hat{I}_2) = \sqrt{(1+\rho)/2}\;\text{se}(\hat{I}_1)$
  • 加重サンプリング
  • 層別サンプリング

ジャックナイフ法

$n$ 個のデータから $i$ 番目を除いた $n-1$ 個で統計量を計算する方法。標準誤差の推定に利用。

ブートストラップ法（Bootstrap Method）

経験分布関数（各 $x_i$ に確率 $1/n$ を与えた分布）からのリサンプリングを $B$ 回繰り返し、統計量の分布を推定する方法。

  • ノンパラメトリックブートストラップ：経験分布から復元抽出
  • パラメトリックブートストラップ：推定パラメータを代入した確率分布からリサンプリング

公式・定理

逆関数法の原理

$$P(F_X^{-1}(U) \le x) = P(U \le F_X(x)) = F_X(x) \tag{32.1}$$

単純モンテカルロ法

$\theta = \int g(x)f(x)\,dx = E[g(X)]$ を推定。$X_1, \ldots, X_m \sim f$ として：

$$\hat{\theta} = \frac{1}{m}\sum_{i=1}^{m} g(X_i) \tag{32.2}$$

分散と標準誤差：

$$V[\hat{\theta}] = \frac{\sigma^2}{m}, \quad \sigma^2 = E[(g(X) - \theta)^2], \quad \text{se}(\hat{\theta}) = \frac{\hat{\sigma}}{\sqrt{m}}$$

$\hat{\sigma}^2 = \frac{1}{m}\sum_{i=1}^{m}(g(X_i) - \hat{\theta})^2$ で推定。中心極限定理より $(\hat{\theta} - \theta)/\text{se}(\hat{\theta}) \to N(0,1)$。

モンテカルロ法による $\pi$ の近似

$(U_i, V_i) \sim U(0,1)^2$ を $N$ 組生成し、$U_i^2 + V_i^2 < 1$ の個数 $M$ として $\hat{\pi} = 4M/N$。

$$\text{SD}[4M/N] = \sqrt{\frac{\pi(4-\pi)}{N}} \le 0.01 \implies N \ge \frac{\pi(4-\pi)}{0.0001} \approx 27000$$

ジャックナイフ標準誤差

$\hat{\theta}_{(i)} = T_{n-1}(x_{(-i)})$（$i$ 番目を除いた統計量）、$\bar{\hat{\theta}}_{(\cdot)} = \frac{1}{n}\sum \hat{\theta}_{(i)}$ として：

$$\text{se}_{\mathrm{jack}} = \sqrt{\frac{n-1}{n}\sum_{i=1}^{n}\left(\hat{\theta}_{(i)} - \bar{\hat{\theta}}_{(\cdot)}\right)^2}$$

ブートストラップ標準誤差

$B$ 回のリサンプリングで $\hat{\theta}^{*(b)} = T_n(x^{*(b)})$, $b = 1, \ldots, B$ として：

$$\text{se}_B = \sqrt{\frac{1}{B-1}\sum_{b=1}^{B}\left(\hat{\theta}^{*(b)} - \bar{\hat{\theta}}^*\right)^2}, \quad \bar{\hat{\theta}}^* = \frac{1}{B}\sum_{b=1}^{B}\hat{\theta}^{*(b)}$$

標準誤差の推定では $B$ は 50〜200 程度で十分。

バイアスのブートストラップ推定量

$$\widehat{\text{bias}}(\hat{\theta}) = \bar{\hat{\theta}}^* - \hat{\theta}$$

例題・演習

例1：モンテカルロ法で $\pi$ を近似

$\hat{\pi} = 4M/N$ の標準偏差を 0.01 以下にするには $N \ge \pi(4-\pi)/0.0001 \approx 27000$ 組。

例2：逆関数法（指数分布）

平均 $1/2$ の指数分布。$F_X(x) = 1 - e^{-2x}$, $F_X^{-1}(u) = -\frac{1}{2}\log(1-u)$。

$U \sim U(0,1)$ から $u$ をサンプリングし、$x = -\frac{1}{2}\log u$ とする（$1-U \sim U(0,1)$ を利用）。

例3：採択棄却法（$\mathrm{Be}(2,2)$）

$f(x) = 6x(1-x)$、$g(x) = 1$（一様分布）。$M_1 = 1$, $M_2 = 1/4$。

$M_1$ の採択確率 $1/6$、$M_2$ の採択確率 $2/3$。$M_2$ の方が $M_1/M_2 = 4$ 倍効率がよい。

例5：主部の分離による精度向上

$I = \int_0^1 e^x\,dx = e - 1 = 1.718$。$g_1(x) = e^x$、$g_2(x) = e^x - (x+1) + 3/2$（主部 $l(x) = x + 1$ を分離）。

$\sigma_1^2 = 0.242$, $\sigma_2^2 = 0.0044$ → $\text{se}(\hat{I}_1) = 0.00492$, $\text{se}(\hat{I}_2) = 0.00210$（$m = 10^4$）

例6：負の相関を利用する方法

$g_2(x) = (e^x + e^{1-x})/2$ として $\hat{I}_2 = \frac{1}{m}\sum g_2(X_i)$。2m個の乱数で比較。

$\rho = \text{Cor}[e^X, e^{1-X}] = -0.234/0.242 = -0.967$

$\text{se}(\hat{I}_2) = \sqrt{(1+\rho)/2}\;\text{se}(\hat{I}_1) = 0.033 \times \text{se}(\hat{I}_1)$

例7-8：ブートストラップとジャックナイフ

東京都の月別交通事故死者数（$n=12$）。標本平均 $11.083$、標本分散 $13.538$。

(1) バイアス修正後の推定値：$11.083 - 0.021 = 11.062$

(2) $\text{se}(\bar{x}) = \sqrt{13.538/12} = 1.062$, $\text{se}_B/\text{se}(\bar{x}) = 1.010/1.062 = 0.951$

ジャックナイフ推定量：$\sqrt{11 \times 0.10256} = 1.062$（ブートストラップとほぼ同じ）

問32.1：$\pi$ 近似の2通りの方法

$U \sim U(0,1)$ として $\sqrt{1-U^2}$ の標本平均で $\pi/4$ を推定。

(1) 単純法：$\text{Var}[\sqrt{1-U^2}] = \pi/4 - (\pi/4)^2 \approx 0.05$ → $n \ge 16 \times 0.05 / 0.0001 = 8000$

(2) 主部分離法：$h(x) = \sqrt{1-x^2} - (1-x) + 1/2$ を利用。分散 $\approx 0.0144$ → $n \ge 16 \times 0.0144 / 0.0001 = 2304$

方法(2)は方法(1)の約 $8000/2304 \approx 3.75$ 倍効率がよい

メモ

  • 乱数生成の基本：逆関数法（$F^{-1}(U)$）と採択棄却法（$Mg(x) \ge h(x)$、$M$ が小さいほど効率的）
  • モンテカルロ法の精度は $\text{se} = \sigma/\sqrt{m}$ で $m$ に依存。精度向上法：主部分離、負の相関、加重/層別サンプリング
  • $\pi$ の近似：$N \ge 27000$ で SD $\le 0.01$（$4M/N$ 法）
  • ジャックナイフ法：1個ずつ除いた統計量の変動から標準誤差を推定。$\text{se}_{\text{jack}} = \sqrt{(n-1)/n \cdot \sum(\hat{\theta}_{(i)} - \bar{\hat{\theta}})^2}$
  • ブートストラップ法：経験分布からの復元抽出を $B$ 回繰り返し。標準誤差は $B = 50$〜$200$ で十分。バイアス推定は $\bar{\hat{\theta}}^* - \hat{\theta}$
  • 例7-8の結果：ブートストラップ（$\text{se}_B = 1.010$）とジャックナイフ（$\text{se}_{\text{jack}} = 1.062$）はほぼ同じ結果
