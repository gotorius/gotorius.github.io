概要

実際のデータ解析では、アンケートの回答拒否や測定器具の不具合などにより、データが不完全な形でしか得られないことがよくある。本章では、そのような不完全データ（incomplete data）の扱い方を学ぶ。

データ行列 $X = \{x_{ij}\}$（$n \times p$）の各要素がすべて得られているとき完全（complete）、いくつかの要素が得られていないとき不完全（incomplete）という。不完全データを何らかの方策で擬似的に完全な形にしたものを擬似完全（pseudocomplete）データという。

本章では、欠測メカニズムの分類 → 削除法と補完法 → 1変量正規分布（打ち切り・トランケーション）→ 2変量正規分布（MAR）→ EMアルゴリズム の順に議論する。

重要概念

欠測メカニズム（Missing Mechanism）

  • MCAR（Missing Completely At Random）：欠測は、欠測データおよび観測データの両方に依存せず、ランダムに生じる
  • MAR（Missing At Random）：欠測は、観測データに依存して生じるが、欠測データの値にはよらない
  • MNAR（Missing Not At Random）：欠測は、欠測データそのものの値に依存して生じる

MCARでは各統計量の値は完全データの場合とほぼ同じ。MARでは平均が過小評価、標準偏差が小さくなるが、回帰パラメータはほとんど変わらない。MNARでは平均・標準偏差・回帰パラメータすべてが影響を受ける。

削除法（Deletion Method）

  • CC（Complete Case）解析：1箇所でも欠測のある個体を個体ごと削除し、すべての変量が観測されている個体のみを用いる（case deletion）
  • AC（Available Case）解析：当該変量の使えるデータは全部使う方法。各変量の平均や相関係数の計算では、関連する変量が観測されていれば利用する

変量数が多い場合、CC解析では解析に使えるデータがかなり少なくなる可能性がある。AC解析では変量ごとに個体数が異なるため、分散共分散行列の正定値性が崩れることもありうる。

補完法（Imputation Method）

  • 平均値代入：当該変量の観測データのみから求めた平均値を代入。平均の過小評価は解消されず、標準偏差がさらに過小評価になる
  • 回帰代入：回帰式によって欠測部分を予測して代入。平均の過小評価は解消されるが、$y$ 方向のばらつきが考慮されないため標準偏差は過小評価、相関係数は過大評価
  • HotDeck法：欠測のある個体と類似の個体を同じデータセットから探し出し、その観測データを欠測部分に代入
  • 多重代入法（multiple imputation）：偏りの除去のために提案された方法

打ち切り（Censoring）とトランケーション（Truncation）

  • 打ち切り：$n$ 個のデータを観測する計画で、$c$ 以下の $m$ 個は観測され、$c$ を超えた $n - m$ 個は値が不明だが個数は既知
  • トランケーション：$c$ 以下の $m$ 個のデータは観測されたが、$c$ を超えたデータの個数が不明

両者の違いはパラメータ推定に想像以上に大きな影響を与える。

欠測メカニズムと対処法のまとめ

メカニズム推奨される対処注意点
MCARCC解析が妥当（補完は不要）補完によりむしろ悪影響の恐れ
MAR回帰代入 or EM法CC・平均値代入は偏り、回帰代入は標準偏差過小・相関過大
MNAR個別対応が必要単純な補完法では偏りが生じる

公式・定理

1変量正規分布における推測

母集団分布 $N(\mu, \sigma^2)$ で、$c$ 以下のみ値が得られる場合の条件付き密度関数：

$$f_c(x) = \begin{cases} \dfrac{1}{\Phi(a)} \cdot \dfrac{1}{\sqrt{2\pi}\sigma} \exp\!\left(-\dfrac{(x-\mu)^2}{2\sigma^2}\right) & (x \le c) \\[6pt] 0 & (x > c) \end{cases}$$

ここで $a = (c - \mu)/\sigma$、$\Phi(\cdot)$ は標準正規分布の累積分布関数。

$Z = (X - \mu)/\sigma \sim N(0,1)$ について：

$$E[Z \mid Z \le a] = -\frac{\phi(a)}{\Phi(a)}, \quad V[Z \mid Z \le a] = 1 - \frac{a\,\phi(a)}{\Phi(a)} - \left(\frac{\phi(a)}{\Phi(a)}\right)^2$$

一般の $\mu, \sigma$ では：

$$E[X \mid X \le c] = \mu - \frac{\phi(a)}{\Phi(a)}\,\sigma, \quad V[X \mid X \le c] = \sigma^2\!\left(1 - \frac{a\,\phi(a)}{\Phi(a)} - \left(\frac{\phi(a)}{\Phi(a)}\right)^2\right)$$

打ち切りの場合のEMアルゴリズム（式29.2）

分散 $\sigma^2$ 既知、母平均 $\mu$ を推定。観測データの標本平均 $\bar{x}$、$a^{(t)} = (c - \hat{\mu}^{(t)})/\sigma$ として：

$$\hat{\mu}^{(t+1)} = \frac{1}{n}\left[m\bar{x} + (n-m)\left(\hat{\mu}^{(t)} + \frac{\phi(a^{(t)})}{1 - \Phi(a^{(t)})}\,\sigma\right)\right]$$

トランケーションの場合の反復（式29.3）

観測データの標本平均 $\bar{x}$、$a^{(t)} = (c - \hat{\mu}^{(t)})/\sigma$ として：

$$\hat{\mu}^{(t+1)} = \bar{x} + \frac{\phi(a^{(t)})}{\Phi(a^{(t)})}\,\sigma$$

2変量正規分布における推測

$(X, Y) \sim N_2(\mu_X, \mu_Y, \sigma_X^2, \sigma_Y^2, \sigma_{XY})$ で $X$ はすべて観測、$Y$ のみ欠測が生じる場合。

$X$ の周辺パラメータの最尤推定値（$n$ 個の全データ）：

$$\hat{\mu}_X = \frac{1}{n}\sum_{i=1}^{n} x_i, \quad \hat{\sigma}_X^2 = \frac{1}{n}\sum_{i=1}^{n}(x_i - \hat{\mu}_X)^2 \tag{29.5}$$

両変量が観測された $m$ 組に関する統計量：

$$\bar{x}' = \frac{1}{m}\sum_{i=1}^{m}x_i, \quad s_X'^2 = \frac{1}{m}\sum_{i=1}^{m}(x_i - \bar{x}')^2, \quad \bar{y} = \frac{1}{m}\sum_{i=1}^{m}y_i, \quad s_Y^2 = \frac{1}{m}\sum_{i=1}^{m}(y_i - \bar{y})^2$$
$$s_{XY} = \frac{1}{m}\sum_{i=1}^{m}(x_i - \bar{x}')(y_i - \bar{y}), \quad r = \frac{s_{XY}}{s_X' \cdot s_Y} \tag{29.6}$$

回帰パラメータの最尤推定値：

$$\hat{a} = \bar{y} - \hat{b}\,\bar{x}', \quad \hat{b} = \frac{s_{XY}}{s_X'^2}, \quad \hat{\tau}^2 = s_Y^2 - \frac{s_{XY}^2}{s_X'^2} \tag{29.7}$$

パラメータの関係式 $\mu_Y = a + b\mu_X$, $\sigma_Y^2 = \tau^2 + b^2\sigma_X^2$, $\sigma_{XY} = b\sigma_X^2$ より、MAR下の最尤推定値：

$$\hat{\mu}_Y = \hat{a} + \hat{b}\,\hat{\mu}_X, \quad \hat{\sigma}_Y^2 = \hat{\tau}^2 + \hat{b}^2\,\hat{\sigma}_X^2, \quad \hat{\sigma}_{XY} = \hat{b}\,\hat{\sigma}_X^2 \tag{29.8}$$

EMアルゴリズム（一般形）

データが $X$（欠測）と $Y=y$（観測）に分かれ、$(X,Y)$ の同時密度を $f(x,y;\theta)$ とする。

  • E-step：$\theta$ の現在値 $\theta^*$ のもとで、完全データの対数尤度の条件付き期待値を計算 $$Q(\theta \mid \theta^*) = \int \log f(x, y; \theta) \cdot f_{X|Y}(x; \theta^*)\,dx$$ しばしば欠測値 $X$ を $\theta^*$ のもとでの条件付き期待値で補完することに対応
  • M-step：$Q(\theta \mid \theta^*)$ を $\theta$ について最大化（補完されたデータを完全データとみなして最尤推定）

E-stepとM-stepを交互に繰り返すと尤度が非減少となり、収束すれば尤度関数の極大値に到達する。

  • 打ち切り（式29.2）：初期値 $\hat{\mu}^{(0)} = 46.25$ → 3回の反復で $\hat{\mu} = 50.05$ に収束

例2：2変量正規分布における欠測（MAR）

表29.1の $x$ とMARの列の $y$ を2変量データとして扱い、各パラメータの最尤推定値を求める。

$X$ の周辺パラメータ（式29.5）：$\hat{\mu}_X = 50.0$, $\hat{\sigma}_X^2 = 100.6$

両変量観測の7組の統計量（式29.6）：

$\bar{x}' = 45.0$, $s_X'^2 = 59.71$, $\bar{y} = 47.14$, $s_Y^2 = 62.41$, $r = 0.64$

回帰パラメータ（式29.7）：$\hat{a} = 17.65$, $\hat{b} = 0.66$, $\hat{\tau}^2 = 36.75$

最尤推定値（式29.8）：

  • $\hat{\mu}_Y = 17.65 + 0.66 \times 50.0 = 50.65$
  • $\hat{\sigma}_Y^2 = 36.75 + (0.66)^2 \times 100.6 = 80.57$
  • $\hat{\sigma}_{XY} = 0.66 \times 100.6 = 65.94$
  • $\hat{\rho} = 65.94 / \sqrt{100.6 \times 80.57} = 0.73$

問29.1：スキルテストと欠測メカニズム

65人中15人が基準 $c$ を超えて受講免除。2回のテストの点数は $\rho > 0$ の2変量正規分布に従う。$y$ の欠測メカニズムはMAR（$x$ の値に依存）。

(1) TAのAさん：2回とも受験した50人のみで $\bar{y}_A$ と $r_A$ を計算 → 答：⑤

$\rho > 0$ より、免除された学生の2回目の点数は概ね高い → $\bar{y}_A$ は $\mu_Y$ を過小評価。分布の端の観測値が失われるため、$r_A$ は $\rho$ を過小評価。

(2) TAのBさん：50人から回帰直線 $\hat{y} = a + bx$ を求め、免除15人の予測値を計算して65組のデータから $\bar{y}_B$ と $r_B$ を計算 → 答：②

MARのもとで回帰直線は全員受験した場合とほぼ一致 → 予測値を含む $\bar{y}_B$ は $\mu_Y$ を偏りなく推定。ただし回帰予測値は直線まわりのばらつきを伴わないため、$r_B$ は $\rho$ を過大評価。

メモ

  • MCARならCC解析で十分（補完はむしろ悪影響）。MARなら回帰代入 or EM法が妥当。MNARは個別対応が必要
  • 打ち切りとトランケーションの違い：欠測の個数情報があるかないかで推定精度が大きく異なる
  • 回帰代入では平均の偏りは解消されるが、標準偏差の過小評価と相関係数の過大評価が残る → 多重代入法で改善可能
  • EMアルゴリズムはE-step（欠測値を条件付き期待値で補完）とM-step（完全データとみなして最尤推定）の反復。尤度は非減少
  • 2変量正規MARの推定では、$X$ の周辺パラメータは全 $n$ 個から、回帰パラメータは完全な $m$ 組から推定し、式(29.8)で変換
  • 「失われたデータは戻ってこない」— 事後の解析より事前に欠測を防ぐ手立てを講じることが最も重要
