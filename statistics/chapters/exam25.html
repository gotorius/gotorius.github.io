第25章「因子分析・グラフィカルモデル」のキーポイント

■ 因子分析（Factor Analysis）

複数の観測変数の背後に存在する共通因子（潜在変数）を推定し，変数間の共変動を少数の因子で説明する手法です．

$$x_{ij} = a_j f_i + d_j u_{ij} \qquad \text{（1因子モデル）}$$
記号名称意味
$f_i$因子スコア個体 $i$ の共通因子の値（推定値）
$a_j$因子負荷量共通因子と変数 $j$ の関係を表す係数
$a_j^2$共通性変数 $j$ の変動のうち共通因子で説明できる割合
$d_j^2$独自性変数 $j$ の変動のうち共通因子で説明できない割合
$$a_j^2 + d_j^2 = 1 \quad \text{（標準化されているとき）}$$

$m$ 因子モデルでは，変数 $j$ の共通性は $\displaystyle\sum_{k=1}^m a_{jk}^2$ になります（$a_{jk}$ は第 $k$ 因子への負荷量）．

■ グラフィカルモデルと構造方程式

構造方程式（パス解析）は変数間の因果関係を有向グラフで表現し，各矢印にパス係数を対応させたモデルです． モラルグラフ（道徳グラフ）は有向グラフを無向グラフに変換したもので，条件付き独立性の読み取りに用います．

モラルグラフの作り方（2ステップ）
  1. 共通の子ノードを持つ親ノード同士を辺で結ぶ（「結婚させる」）
  2. すべての有向辺を無向辺に変換する

問25.1 — 問題文（コーヒー豆の因子分析）

問題

コーヒー豆の種類によるコーヒーの味の違いを可視化するために，100人の消費者を対象に， AからJまでの10種類のコーヒーの味について「苦み」「酸味」「すっきり感」「コク」「香りのよさ」の 5項目についてアンケート調査を行った．100人のデータからコーヒー豆ごとに各項目の平均を求め， その10行5列の集計結果に2因子モデルを適用して5項目間の関連を考察することにした． 推定には最尤法，回転にはバリマックス回転を用いた． 各因子に対する因子負荷量と共通性は次の表のようになった．

この結果から，第1因子を「酸味 or 苦味」，第2因子を「味の濃さ」と名付けることにした． 下図はコーヒー豆の因子得点を横軸を第1因子，縦軸を第2因子としてプロットしたものである．

因子負荷量と共通性

項目 第1因子 第2因子 共通性
苦み $-0.92$ $\text{ア}$ $0.8545$
酸味 $0.83$ $-0.08$ $0.6953$
すっきり感 $-0.09$ $-0.84$ $0.7137$
コク $0.02$ $0.90$ $0.8104$
香りのよさ $-0.35$ $0.45$ $0.3250$

図25.1　コーヒー豆の因子得点プロット（概略）

  1. $\boxed{\text{ア}}$ の因子負荷量の値の絶対値を求めよ．
  2. 「酸味は強くないが苦味が強いコクのある味わいのコーヒーがほしい」という人がいた場合，AからJのどのコーヒーを勧めるのが最も適切と思うか答えよ．

問25.1 [1] — ア の絶対値を求める

解答・解説
共通性の公式（2因子モデルの場合）

$m$ 因子モデルでは，変数 $j$ の共通性（communality）は 各因子への因子負荷量の2乗和で与えられます：

$$\text{（共通性）} = \sum_{k=1}^{m} a_{jk}^2 = a_{j1}^2 + a_{j2}^2 + \cdots + a_{jm}^2$$

今回は $m = 2$ の2因子モデルなので，「苦み」の共通性は：

$$\text{（苦みの共通性）} = a_{\text{苦},1}^2 + a_{\text{苦},2}^2$$ $$0.8545 = (-0.92)^2 + |\text{ア}|^2$$

計算手順

Step 1：$(-0.92)^2$ を計算する

$(-0.92)^2 = 0.8464$

Step 2：$|\text{ア}|^2$ を求める

$|\text{ア}|^2 = 0.8545 - 0.8464 = 0.0081$

Step 3：$|\text{ア}|$ を計算する

$|\text{ア}| = \sqrt{0.0081} = 0.09$

結論
$$|\text{ア}| = \sqrt{0.8545 - 0.92^2} = \sqrt{0.8545 - 0.8464} = \sqrt{0.0081} = \boldsymbol{0.09}$$
補足：共通性・独自性・因子負荷量の関係

標準化された変数では $\text{V}[x_j] = 1$ であり，因子モデルの仮定から $\text{V}[x_j] = a_{j1}^2 + a_{j2}^2 + d_j^2 = 1$ が成立します． したがって共通性＋独自性＝1となります． 苦みの独自性は $1 - 0.8545 = 0.1455$ です．

※ ア の符号は絶対値では判断できません．問題は「絶対値を求めよ」なので 0.09 が正解です． （文脈から ア > 0，すなわち苦みは第2因子「味の濃さ」にも正の寄与をすると解釈されることが多いです）

問25.1 [2] — 因子得点プロットから最適なコーヒーを選ぶ

解答・解説
因子の解釈を整理する

因子負荷量の表から，各因子の意味を確認します：

因子名前正の方向負の方向
第1因子（横軸） 酸味 or 苦味 酸味が強い（$+0.83$） 苦みが強い（$-0.92$）
第2因子（縦軸） 味の濃さ コクがある（$+0.90$） すっきりしている（$-0.84$）

希望するコーヒーの条件を因子得点で表す

「酸味は強くない」

第1因子の正の方向が「酸味が強い」なので，酸味が強くないということは第1因子のスコアが小さい（左方向）

「苦みが強い」

第1因子の負の方向が「苦みが強い」なので，苦みが強いということは第1因子のスコアが負に大きい（さらに左方向）

「コクのある味わい」

第2因子の正の方向が「コクがある」なので，第2因子のスコアが大きい（上方向）

因子スコアプロットで探す位置

条件をまとめると，「第1因子スコアが最も小さく（最も左），第2因子スコアが最も大きい（最も上）」コーヒーを選べばよいことになります． プロット図を確認すると，この条件に最も合致するのは左上に位置する J です．

結論

因子1のスコアが最も小さく（苦みが強く酸味が弱い），因子2のスコアが最も大きい（コクがある）コーヒーは J である．よって，コーヒー J を勧めるのが最も適切である．

各コーヒーの特徴（プロットの読み方）
位置コーヒー味の特徴
左上J★苦みが強く酸味が弱い，コクがある → 今回の答え
右上A酸味が強い，コクもある
左下H苦みが強い，すっきりしている（コクが薄い）
右下B酸味がある，すっきりしている
中央付近D, E, Iバランス型

問25.2 — 問題文（構造方程式・パス係数・モラルグラフ）

問題

変数 $X, Y, Z, W$ の関係を下図のグラフが定義する構造方程式を考える：

$$Y = aX + bW + u, \qquad Z = cX + dY + v \tag{25.3}$$

ここで，$X, Y, Z, W$ は標準化されているとし，$X$ と $W$ は独立と仮定する． $u, v$ は誤差項で，$u$ は $X, W$ と，$v$ は $X, Y$ と独立であるとする． $X$ と $Y$ の共分散を $\rho_{xy}$，$X$ と $Z$ の共分散を $\rho_{xz}$， $Y$ と $Z$ の共分散を $\rho_{yz}$，$Y$ と $W$ の共分散を $\rho_{yw}$， $Z$ と $W$ の共分散を $\rho_{zw}$ とする．

  1. パス係数 $a, b, c, d$ を $\rho_{xy}, \rho_{xz}, \rho_{yz}, \rho_{yw}, \rho_{zw}$ を用いて表せ．
  2. $\rho_{xz} = c + ad$ となっていることを示せ．
  3. 上の有向グラフのモラルグラフを描け．

問25.2 [1] — パス係数 a, b, c, d を求める

解答・解説
基本方針：両辺に変数を掛けて期待値を取る

構造方程式の両辺に各変数を掛けて期待値（平均）をとることで， パス係数と共分散の関係式が導出できます． 変数が標準化（平均0，分散1）されているので，$\mathrm{E}[X^2] = \mathrm{Var}(X) = 1$ などが使えます． また，$X \perp W$ より $\mathrm{E}[XW] = 0$，誤差項の独立性より $\mathrm{E}[uX] = \mathrm{E}[uW] = \mathrm{E}[vX] = \mathrm{E}[vY] = 0$ となります．

■ a と b の導出（第1式 $Y = aX + bW + u$ から）

第1式の両辺に X を掛けて期待値を取る
$$\mathrm{E}[YX] = a\,\mathrm{E}[X^2] + b\,\mathrm{E}[WX] + \mathrm{E}[uX]$$ $$\rho_{xy} = a \cdot 1 + b \cdot 0 + 0$$ $$\therefore\quad \boxed{a = \rho_{xy}}$$

・$\mathrm{E}[X^2]=1$（標準化），$\mathrm{E}[WX]=0$（$X \perp W$），$\mathrm{E}[uX]=0$（$u \perp X$）

第1式の両辺に W を掛けて期待値を取る
$$\mathrm{E}[YW] = a\,\mathrm{E}[XW] + b\,\mathrm{E}[W^2] + \mathrm{E}[uW]$$ $$\rho_{yw} = a \cdot 0 + b \cdot 1 + 0$$ $$\therefore\quad \boxed{b = \rho_{yw}}$$

・$\mathrm{E}[W^2]=1$（標準化），$\mathrm{E}[XW]=0$（$X \perp W$），$\mathrm{E}[uW]=0$（$u \perp W$）

■ c と d の導出（第2式 $Z = cX + dY + v$ から）

第2式の両辺に X を掛けて期待値を取る
$$\mathrm{E}[ZX] = c\,\mathrm{E}[X^2] + d\,\mathrm{E}[YX] + \mathrm{E}[vX]$$ $$\rho_{xz} = c \cdot 1 + d \cdot \rho_{xy} + 0$$ $$\therefore\quad \rho_{xz} = c + d\rho_{xy} \qquad \cdots\,(*) $$
第2式の両辺に Y を掛けて期待値を取る
$$\mathrm{E}[ZY] = c\,\mathrm{E}[XY] + d\,\mathrm{E}[Y^2] + \mathrm{E}[vY]$$ $$\rho_{yz} = c\rho_{xy} + d \cdot 1 + 0$$ $$\therefore\quad \rho_{yz} = c\rho_{xy} + d \qquad \cdots\,(**)$$
連立方程式 (*), (**) を解く

$(*)$ より $c = \rho_{xz} - d\rho_{xy}$ を $(**)$ に代入する：

$$\rho_{yz} = (\rho_{xz} - d\rho_{xy})\rho_{xy} + d = \rho_{xz}\rho_{xy} + d(1 - \rho_{xy}^2)$$ $$d(1 - \rho_{xy}^2) = \rho_{yz} - \rho_{xy}\rho_{xz}$$ $$\therefore\quad \boxed{d = \frac{\rho_{yz} - \rho_{xy}\rho_{xz}}{1 - \rho_{xy}^2}}$$

これを $(*)$ に代入して $c$ を求める：

$$c = \rho_{xz} - \rho_{xy} \cdot \frac{\rho_{yz} - \rho_{xy}\rho_{xz}}{1 - \rho_{xy}^2} = \frac{\rho_{xz}(1-\rho_{xy}^2) - \rho_{xy}\rho_{yz} + \rho_{xy}^2\rho_{xz}}{1 - \rho_{xy}^2}$$ $$\therefore\quad \boxed{c = \frac{\rho_{xz} - \rho_{xy}\rho_{yz}}{1 - \rho_{xy}^2}}$$
結論：パス係数のまとめ
パス係数対応する矢印値
$a$$X \to Y$$a = \rho_{xy}$
$b$$W \to Y$$b = \rho_{yw}$
$c$$X \to Z$（直接効果）$\displaystyle c = \frac{\rho_{xz} - \rho_{xy}\rho_{yz}}{1 - \rho_{xy}^2}$
$d$$Y \to Z$$\displaystyle d = \frac{\rho_{yz} - \rho_{xy}\rho_{xz}}{1 - \rho_{xy}^2}$

問25.2 [2] — $\rho_{xz} = c + ad$ を示す

解答・解説
直接効果と間接効果

パス図において，$X$ が $Z$ に与える影響には2種類の経路があります：

  • 直接効果（direct effect）：$X \xrightarrow{c} Z$ の1本の矢印
  • 間接効果（indirect effect）：$X \xrightarrow{a} Y \xrightarrow{d} Z$ の2本の矢印の経路（パス係数の積 $ad$）

$\rho_{xz} = c + ad$ という等式は， 「$X$ と $Z$ の相関は，直接効果と間接効果の和で表せる」ことを意味します．

証明

[1] の結果から，

$$c + ad = \frac{\rho_{xz} - \rho_{xy}\rho_{yz}}{1 - \rho_{xy}^2} + \rho_{xy} \cdot \frac{\rho_{yz} - \rho_{xy}\rho_{xz}}{1 - \rho_{xy}^2}$$

分子をまとめると，

$$\begin{align} \text{分子} &= \rho_{xz} - \rho_{xy}\rho_{yz} + \rho_{xy}\rho_{yz} - \rho_{xy}^2\rho_{xz}\\ &= \rho_{xz} - \rho_{xy}^2\rho_{xz}\\ &= \rho_{xz}(1 - \rho_{xy}^2) \end{align}$$

したがって，

$$c + ad = \frac{\rho_{xz}(1 - \rho_{xy}^2)}{1 - \rho_{xy}^2} = \rho_{xz} \qquad \blacksquare$$
直感的な理解：経路を追って相関を計算する

標準化変数の場合，共分散＝相関係数であることを使って，構造方程式から直接導くこともできます：

$$\rho_{xz} = \mathrm{Cov}(X, Z) = \mathrm{Cov}(X,\ cX + dY + v)$$ $$= c\,\mathrm{Var}(X) + d\,\mathrm{Cov}(X,Y) + \mathrm{Cov}(X,v)$$ $$= c \cdot 1 + d \cdot \rho_{xy} + 0 = c + d\rho_{xy}$$

一方，$a = \rho_{xy}$ を代入すると $c + d\rho_{xy} = c + da = c + ad$ となり，$\rho_{xz} = c + ad$ が示されます．

※ これは $(*)$ 式そのものであり，実は Step で既に示されていた内容です．問[1]の導出の中ですでに $\rho_{xz} = c + d\rho_{xy} = c + ad$ が使われています．

問25.2 [3] — モラルグラフを描く

解答・解説
モラルグラフ（道徳グラフ）の作り方

モラルグラフとは，有向グラフ（DAG）を無向グラフに変換したものです． グラフィカルモデルにおいて条件付き独立性を読み取るために用いられます． 　

手順1：「結婚」— 共通の子ノードを持つ親ノード同士を無向辺で結ぶ

各ノードの親ノード（そのノードへの矢印の出発点）を確認し，2人以上の親を持つノードがあれば，その親たちを辺で結びます．

手順2：「モラル化」— すべての有向辺を無向辺に変換する

矢印の向きを消去し，単純な辺（無向辺）にします．

元のパス図の辺と親ノードを確認する

ノード親ノード（矢印の始点）処理
$X$なし—
$W$なし—
$Y$ $X$ と $W$（2人の親） $X$ と $W$ を辺で結ぶ ★
$Z$ $X$ と $Y$（2人の親） $X$ と $Y$ はすでに辺あり（$X \to Y$）→ 追加不要

モラルグラフの辺一覧

  • $X - Y$（もとの有向辺 $X \to Y$ を無向化）
  • $W - Y$（もとの有向辺 $W \to Y$ を無向化）
  • $X - Z$（もとの有向辺 $X \to Z$ を無向化）
  • $Y - Z$（もとの有向辺 $Y \to Z$ を無向化）
  • $X - W$（手順1で追加：$Y$ の親 $X$ と $W$ を結ぶ）
結論：モラルグラフ

モラルグラフは辺 $\{X\text{-}Y,\ W\text{-}Y,\ X\text{-}Z,\ Y\text{-}Z,\ X\text{-}W\}$ を持つ無向グラフ（図25.3）である．

モラルグラフから条件付き独立性を読む

モラルグラフにおいて，ノード $A$ と $B$ の間のすべてのパスが集合 $S$ によって遮断されるとき， $A$ と $B$ は $S$ が与えられたもとで条件付き独立になります（マルコフ性）．

今回のモラルグラフでは，$W$ と $Z$ を結ぶ直接の辺がありません． $W - Y - Z$ というパスが存在しますが，$Y$（または $\{X, Y\}$）を条件付ければ， $W$ と $Z$ のパスが遮断されます．すなわち，$X, Y$ を与えた下で $W \perp Z$ であることがわかります．
