はじめに：「回帰モデルは仮定の上に成り立っている」

回帰分析では、「誤差項 $\varepsilon_i$ は独立で、分散が一定で、正規分布に従う」という3つの仮定をおいています。 しかし現実のデータがこの仮定を満たしているかどうかは、計算するだけでは確認できません。

たとえ話で理解しよう

たとえば「体重を身長で予測する」回帰直線を引いたとします。この直線からのズレ（=残差）が、 背が低い人でも高い人でもほぼ同じくらいならOK。でも、背が高い人ほどズレが大きい場合は 「等分散の仮定が怪しい」とわかります。

回帰診断とは、このような「仮定のひずみ」をグラフや数値で検出する技術です。

第17章では以下の4つの診断ツールを学びます：

  • ①残差プロット：等分散・独立性のチェック
  • ②正規Q-Qプロット：正規性のチェック
  • ③てこ比（leverage）・Cookの距離：影響力の強い観測値の検出
  • ④ダービン・ワトソン比：残差の時系列的な相関のチェック

キーワード：残差プロット、てこ比、Cookの距離、ダービン・ワトソン比、外れ値

1. 残差プロット

「残差」とは何か？

回帰直線で $y$ の予測値を $\hat{y}_i$ とすると、実際の値 $y_i$ との差が残差です：

$$e_i = y_i - \hat{y}_i$$

残差は「モデルで説明しきれなかった部分」。誤差項 $\varepsilon_i$ の推定値とも言えます。 もし回帰モデルが正しければ、残差はランダムな「ノイズ」のはずです。

残差プロット（Residuals vs Fitted）の読み方

横軸に予測値 $\hat{y}_i$、縦軸に残差 $e_i$を打点した散布図です。 「良い」プロットと「悪い」プロットの見分け方は以下の通りです：

仮定が満たされている場合（良い状態）

  • 残差がゼロの水平線の周りにランダムに散らばっている
  • 残差のばらつきが予測値によらずほぼ一定（等分散）
  • 特定のパターン（曲線・上昇トレンドなど）が見られない

仮定が怪しい場合（問題ありのパターン）

  • 漏斗型（ファンネル型）：予測値が大きいほど残差のばらつきが大きくなる → 等分散性の違反。対数変換などが有効。
  • 曲線状パターン：残差が系統的に増えたり減ったりする → 線形性の違反。説明変数の2乗項などを追加。
  • 飛び離れた点（外れ値）：他の点から大きく外れた残差がある → その観測値を詳しく調べる。

チェックすべき仮定のまとめ

  • 等分散性（homoscedasticity）：残差の分散 $\text{Var}[\varepsilon_i] = \sigma^2$ が全データで同じ
  • 独立性：残差 $e_1, e_2, \ldots, e_n$ が互いに独立（時系列では特に要確認）
  • 正規性：残差が正規分布に従う（小標本では重要）
  • 線形性：$y$ と $x$ の関係が線形で表せる

2. 正規Q-Qプロット・Scale-Locationプロット

正規Q-Qプロット（Normal Q-Q plot）

Q-Q は「Quantile-Quantile（分位点-分位点）」の略です。 残差が正規分布に従っているかを視覚的に確認するためのプロットです。

作り方の手順：

  1. 標準化残差 $r_i = e_i / (\hat{\sigma}\sqrt{1-h_{ii}})$ を小さい順に並べる
  2. $n$ 個のデータに対して、正規分布の理論的な $k$ 番目の分位点（$z_{(k)}$）を計算する
  3. 横軸に理論分位点 $z_{(k)}$、縦軸に標準化残差の順序統計量を打点する

Q-Qプロットの見方

  • 直線（45度線）上に乗っている：正規性 OK → 回帰の推論が信頼できる
  • 両端が上下に外れる（S字型）：裾が正規分布より重い（尖度が高い）
  • 片側が大きく外れる：歪みあり、変換（対数など）を検討

正規性の仮定が成立しないと、t検定・F検定の結果が信頼できなくなります。 ただし、標本サイズが大きい場合は中心極限定理により多少ずれていても問題は小さくなります。

Scale-Location プロット（等分散性の確認）

横軸に予測値 $\hat{y}_i$、縦軸に $\sqrt{|r_i|}$（標準化残差の絶対値の平方根）を打点したものです。 横軸に沿って水平にランダムに散らばっていれば等分散。 右に行くほど上がっていれば「予測値が大きいほど残差のばらつきが大きい」ことを示します。

残差プロットでは正負の両側を見るのに対し、Scale-Locationプロットは絶対値で見るため、 等分散性の検出がより明確です。

3. てこ比（leverage）

「てこ比」のイメージ

「てこ（lever）」の原理を思い出してください。支点から遠い場所に力を加えると、より大きな回転モーメントが生まれます。 回帰分析でも同様に、説明変数の空間で平均から遠い観測値ほど、回帰直線を大きく動かす力を持ちます。 その「てこの効き具合」を数値化したものがてこ比（leverage）です。

ハット行列（Hat Matrix）$H$

最小二乗法の推定値 $\hat{y} = X(X^TX)^{-1}X^Ty$ を見ると、$\hat{y}$ は $y$ に行列をかけたものです。 この行列 $H = X(X^TX)^{-1}X^T$ をハット行列といいます（$y$ に「ハット（^）」をかぶせるから）。

ハット行列の$(i,i)$ 要素（$i$ 行 $i$ 列の対角要素）が$i$ 番目のてこ比 $h_{ii}$です：

$$h_{ii} = x_i^T (X^T X)^{-1} x_i$$

$x_i$ は $i$ 番目の観測の説明変数ベクトルです。

てこ比の性質と目安

  • 必ず $0 \leq h_{ii} \leq 1$ の範囲に収まる
  • 全ての $h_{ii}$ の合計は $\sum_{i=1}^n h_{ii} = p+1$（説明変数の数＋切片）
  • 平均値は $(p+1)/n$ → これの2倍以上なら「てこ比が大きい」と判断する目安
  • $h_{ii}$ が大きい観測は、説明変数の値が他の観測と大きく異なる「外れた」点

てこ比が大きいことは「悪い」こと？

てこ比が大きい＝「回帰直線をよく引っ張る力がある」だけで、必ずしも悪いわけではありません。 問題になるのは、てこ比が大きくてかつ残差が大きい場合です（→ Cookの距離で評価）。

残差の分散とてこ比の関係

残差の分散は $\text{Var}[e_i] = \sigma^2(1 - h_{ii})$ で表せます。

つまり、てこ比 $h_{ii}$ が大きい観測ほど残差の分散が小さくなります。 言い換えれば、「影響力が大きい点ほど回帰直線が自分に合わせてくれるため、残差が小さく見えやすい」のです。 この性質から、てこ比の大きな点の外れ値は残差プロットで見逃しやすいため注意が必要です。

4. Cookの距離

「この1点を除いたら、どのくらい変わる？」

Cookの距離（Cook's distance）は、ある1つの観測値を除いたとき、回帰係数の推定値がどのくらい変化するかを測る指標です。 「この観測が回帰モデル全体に与える影響力」の総合スコアと思ってください。

Cookの距離の計算式

$$D_i = \frac{(\hat{\beta} - \hat{\beta}_{(-i)})^T X^TX (\hat{\beta} - \hat{\beta}_{(-i)})}{(p+1)\hat{\sigma}^2}$$

$\hat{\beta}_{(-i)}$ は $i$ 番目のデータを除いて推定したときの係数ベクトルです。 直接計算しなくても、残差とてこ比を使って次のように求められます：

$$D_i = \frac{e_i^2}{(p+1)\hat{\sigma}^2} \cdot \frac{h_{ii}}{(1 - h_{ii})^2} = \frac{r_i^2}{p+1} \cdot \frac{h_{ii}}{1-h_{ii}}$$

つまり Cookの距離は「残差の大きさ（$r_i^2$）」と「てこ比（$h_{ii}$）」の両方を掛け合わせた指標です。 残差が大きくてもてこ比が小さければ影響は小さい。てこ比が大きくても残差が小さければ問題ない。 両方が大きいときに初めて「影響力が強い」と判断されます。

判断の目安

  • $D_i > 0.5$：要注意（影響力が強い可能性）
  • $D_i > 1$：明らかに影響力が大きい（外れ値の候補として調査すべき）

外れ値の除外は慎重に

Cookの距離が大きいからといって、そのデータをすぐに「除外」してはいけません。 まず「なぜその値が外れているのか」の理由を調べることが重要です。 入力ミスならば除外・修正すべきですが、本当にそういう値が存在するなら重要な情報かもしれません。

5. 自己相関とダービン・ワトソン（DW）比

「自己相関」とは何か？

時系列データ（株価・気温・月次売上など）を回帰分析する場合、 「今日の残差が大きければ、明日の残差も大きくなりやすい」という系列相関（自己相関）が生じることがあります。

たとえば「物価を金利で回帰した残差」は、景気の波の影響で連続して正の値をとることがあります。 このような自己相関があると、標準誤差の推定が歪み、F検定・t検定の結果が信頼できなくなります。

ダービン・ワトソン比（DW）の計算式

残差 $e_1, e_2, \ldots, e_n$（データの並び順通り）について：

$$DW = \frac{\sum_{i=2}^{n}(e_i - e_{i-1})^2}{\sum_{i=1}^{n} e_i^2}$$

分子は「隣り合う残差の差の2乗和」、分母は「残差の2乗和」です。 隣り合う残差が似た値（正の自己相関）なら差が小さく → DWが小さくなります。

DW比の読み方

DW値の範囲意味対処
$DW \approx 2$自己相関なし（理想的）そのままOK
$DW < 1.0$（目安）正の自己相関：残差が連続して同じ符号ラグ変数の追加、一般化最小二乗法
$DW > 3.0$（目安）負の自己相関：残差が交互に符号変化モデルの見直し

正確な棄却域は DW表（標本数 $n$ と説明変数の数 $p$ から決まる上限・下限値）を参照します。 ただし、DW比はデータが時間順・空間順など何らかの順序で並んでいる場合にのみ意味を持ちます。 年齢順・アルファベット順にデータがソートされているだけの場合に使っても意味がありません。

6. 公式まとめ

標準化残差（Standardized Residual）

$$r_i = \frac{e_i}{\hat{\sigma}\sqrt{1 - h_{ii}}}$$

$|r_i| > 2$ なら外れ値の可能性（正規分布なら95%のデータが $\pm 2$ 以内）。 てこ比 $h_{ii}$ が大きいほど分母が小さくなり、標準化残差が大きく見えることに注意。

残差の分散

$$\text{Var}[e_i] = \sigma^2 (1 - h_{ii})$$

てこ比の大きい観測は残差の分散が小さく抑えられる → 外れ値が見えにくい。

Cookの距離（簡略版）

$$D_i = \frac{r_i^2}{p+1} \cdot \frac{h_{ii}}{1 - h_{ii}}$$

Cookの距離は「残差の大きさ」と「leverageの大きさ」の両方を考慮した総合指標。

ダービン・ワトソン比とAR(1)の関係

AR(1)過程 $e_i = \rho e_{i-1} + u_i$（$|\rho| < 1$）において、DW $\approx 2(1-\rho)$ が成立します。

  • $\rho = 0$（自己相関なし）のとき DW $\approx 2$
  • $\rho = 1$（強い正の自己相関）のとき DW $\approx 0$

7. 例題・演習

問17.1：道府県別乗用車所有率の回帰診断

問題

47都道府県のデータで乗用車所有率 $y$ を人口密度の正の平方根 $\sqrt{x}$ で単回帰した。Rで生成した4つの回帰診断図（図17.5）について次の問いに答えよ。

  1. 残差プロット（Residuals vs Fitted）から読み取れることを述べよ。
  2. 正規Q-Qプロットから正規性について述べよ。
  3. Residuals vs Leverageプロットから、影響力の強い都道府県について述べよ。
  4. 東京・大阪・神奈川を除いた44道府県で同じモデルを再分析した結果が図17.6である。図17.5と比較してわかることを述べよ。
解答

(1) 残差プロットの読み取り：

残差が予測値に対してほぼランダムに散らばっていれば等分散・線形性は概ね満足。 ただし、東京・大阪・神奈川など大都市圏の残差が他と大きく離れていれば、 これらが回帰直線を大きく引っ張っている（leverage が高い）可能性を示す。

(2) 正規Q-Qプロットによる正規性の評価：

標準化残差が理論分位点の45度線の近くに並んでいれば正規性 OK。 両端が大きく外れていれば、特定の都道府県（東京など）が分布の裾を重くしている可能性がある。

(3) てこ比・Cookの距離による影響力の評価：

東京・大阪・神奈川は人口密度が他の道府県と突出して高く、説明変数 $\sqrt{x}$ が非常に大きい。 そのため てこ比 $h_{ii}$ が大きく、Residuals vs Leverage プロットで右端に現れる。 Cookの距離が0.5を超えているなら「影響力が強い」と判断できる。

(4) 44道府県での再分析（図17.6）との比較：

  • 大都市圏3都府県を除くと、残差プロットとScale-Locationプロットから残差の散らばりに規則性がなくなり、等分散性が満たされると考えられる
  • 正規Q-Qプロットにはあまり変化がみられない
  • てこ比も大きな値をもつ観測値がなくなり、Cookの距離が0.5を超えるものがなくなる
  • ただし自由度調整済み決定係数は大きく変化する

参考として47都道府県と44道府県で得られた回帰式と（）内に自由度調整済み決定係数を以下に示す（図17.7）。回帰式はほとんど変化がないが、自由度調整済み決定係数が大きく異なる。

47都道府県：$\hat{y} = 0.74332 - 0.05208\sqrt{x}$ （調整済み $R^{*2} = 0.7011$）

44道府県：$\hat{y} = 0.74285 - 0.05191\sqrt{x}$ （調整済み $R^{*2} = 0.3653$）

(DW比について：)都道府県データは時系列ではないため、この場合DW比は通常使わない。もしデータが何らかの順序（人口密度順など）でソートされていない限り意味をなさない。

8. まとめ・ポイント

4つの診断プロットと何を見るか

プロット横軸縦軸チェックする仮定
Residuals vs Fitted予測値 $\hat{y}$残差 $e$等分散性・線形性
Normal Q-Q理論分位点標準化残差正規性
Scale-Location予測値 $\hat{y}$$\sqrt{|r_i|}$等分散性（確認）
Residuals vs Leverageてこ比 $h_{ii}$標準化残差 $r_i$影響力の強い点
  • 回帰分析は「モデルを当てはめて終わり」ではなく、診断プロットで仮定の妥当性を必ず確認する
  • Cookの距離は「残差の大きさ」と「てこ比」の両方を考慮した総合的な影響力指標
  • てこ比が大きい点は必ずしも問題ではない（説明変数空間の端にあるだけ）
  • DW比はデータが時系列的な順序で並んでいるときだけ意味をもつ
  • 外れ値の除外は安易に行わず、その理由（入力ミス・特殊事情など）を明確にしてから判断する
  • 標準化残差 $|r_i| > 2$ が外れ値の目安（正規分布なら全体の約5%が範囲外）
