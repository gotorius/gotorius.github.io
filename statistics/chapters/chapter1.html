概要

第1章では確率論の基礎となる事象と確率の概念を扱う。 事象の和・積・余事象と確率の関係、包除原理、条件付き確率と独立性、 ベイズの定理、確率変数の期待値と分散まで、統計学全体の土台となる重要な内容が凝縮されている。

キーワード：確率の計算、統計的独立、条件付き確率、ベイズの定理、包除原理、確率関数、確率密度関数、期待値、分散

重要概念

事象と確率

  • 事象 $A$ の確率を $P(A)$ と表す
  • 余事象 $\bar{A}$：「$A$ が起きないこと」。$P(\bar{A}) = 1 - P(A)$
  • 積事象 $A \cap B$：「$A, B$ の両方が起きること」
  • 和事象 $A \cup B$：「$A, B$ の少なくとも一方が起きること」
  • 空事象 $\emptyset$ の確率は $P(\emptyset) = 0$、全事象 $\Omega$ の確率は $P(\Omega) = 1$
  • $A$ と $B$ が排反（$A \cap B = \emptyset$）なら $P(A \cup B) = P(A) + P(B)$

条件付き確率と独立性

  • 条件付き確率：$A$ が起きたもとで $B$ が起きる確率 $$P(B \mid A) = \frac{P(A \cap B)}{P(A)} \quad (P(A) > 0)$$
  • 分母を払うと 乗法定理：$P(A \cap B) = P(A) \cdot P(B \mid A)$
  • 独立：$P(A \cap B) = P(A) \cdot P(B)$ が成り立つこと。このとき $P(B \mid A) = P(B)$
  • 条件付き独立：$C$ を与えたもとで $P(A \cap B \mid C) = P(A \mid C) \cdot P(B \mid C)$ が成立

確率変数

  • 確率変数：ランダムに変動する変数。大文字 $X$ で表し、とりうる値を小文字 $x$ で表す
  • 確率関数（離散型）：$p(x) = P(X = x)$
  • 確率密度関数（連続型）：$$f(x) = \lim_{\varepsilon \to 0} \frac{P(x \leq X \leq x + \varepsilon)}{\varepsilon}$$

公式・定理

包除原理

一般の（排反でない）場合：

$$P(A \cup B) = P(A) + P(B) - P(A \cap B)$$

度数や相対度数についても同様に使える。

ベイズの定理

$$P(A \mid B) = \frac{P(B \mid A) \, P(A)}{P(B \mid A) \, P(A) + P(B \mid \bar{A}) \, P(\bar{A})}$$

$P(A)$ を事前確率、$P(A \mid B)$ を事後確率と呼ぶ。結果 $B$ が観測されたとき、原因 $A$ の確率を逆算する。

ベイズの定理（一般形）

全事象が排反な $A_1, \dots, A_K$ の和で表されるとき：

$$P(A_i \mid B) = \frac{P(B \mid A_i) \, P(A_i)}{\displaystyle\sum_{j=1}^{K} P(B \mid A_j) \, P(A_j)}$$

期待値と分散

離散型：

$$\mu = E[X] = \sum_x x \, p(x)$$
$$\sigma^2 = V[X] = E[(X - \mu)^2] = \sum_x (x - \mu)^2 \, p(x)$$

展開公式：

$$V[X] = E[X^2] - \mu^2$$

連続型：

$$E[X] = \int_{-\infty}^{\infty} x \, f(x) \, dx, \qquad V[X] = \int_{-\infty}^{\infty} (x - \mu)^2 \, f(x) \, dx$$

例題・演習

例1：包除原理（度数の応用）

50名のクラスで、番組Aを見た生徒20名、番組Bを見た生徒15名、両方を見た生徒5名。どちらも見なかった生徒は？

解：包除原理より $20 + 15 - 5 = 30$ 名が少なくとも一方を視聴。補集合をとり $50 - 30 = \mathbf{20}$ 名。

問1.1：全体の合格率とベイズの定理

女子比率 0.4、女子合格率 0.5、男子合格率 0.4 の大学。

(1) 全体の合格率：

$$0.5 \times 0.4 + 0.4 \times 0.6 = \mathbf{0.44}$$

(2) 合格者のうち女子の確率：ベイズの定理より

$$P(W \mid S) = \frac{P(S \mid W) \, P(W)}{P(S)} = \frac{0.5 \times 0.4}{0.44} = \mathbf{\frac{5}{11}}$$

問1.2：サイコロの期待値と分散

1, 2, 3 の数字が書かれた正六面体。$P(X=1) = P(X=2)$ かつ $E[X] > 2$。

条件より $P(X=1) = P(X=2) = \frac{1}{6}$、$P(X=3) = \frac{4}{6}$

(1) 期待値と分散：

$$\mu = \frac{1}{6} \cdot 1 + \frac{1}{6} \cdot 2 + \frac{4}{6} \cdot 3 = \frac{5}{2}$$
$$V[X] = E[X^2] - \mu^2 = \left(\frac{1+4+36}{6}\right) - \frac{25}{4} = \frac{41}{6} - \frac{25}{4} = \frac{7}{12}$$

(2) $P(Y = 3)$：2回投げて大きいほうを $Y$ とする。余事象（$X_1 \leq 2$ かつ $X_2 \leq 2$）を考えると

$$P(X_1 \leq 2) = \frac{2}{6} = \frac{1}{3} \implies P(Y = 3) = 1 - \left(\frac{1}{3}\right)^2 = \mathbf{\frac{8}{9}}$$

問1.3：検査とベイズの定理

有病率 1%。検査1：感度 99%、偽陽性率 2%。検査2：感度 90%、偽陽性率 10%。

(1) 検査1で陽性のとき、罹患確率：

$$P(X \mid Y_1) = \frac{0.99 \times 0.01}{0.99 \times 0.01 + 0.02 \times 0.99} = \frac{0.0099}{0.0297} = \mathbf{\frac{1}{3}}$$

(2) さらに検査2でも陽性のとき：(1)の事後確率を事前確率として再度ベイズの定理を適用

$$P(X \mid Y_1, Y_2) = \frac{0.9 \times \frac{1}{3}}{0.9 \times \frac{1}{3} + 0.1 \times \frac{2}{3}} = \frac{\frac{0.9}{3}}{\frac{0.9 + 0.2}{3}} = \mathbf{\frac{9}{11}}$$

メモ

  • ベイズの定理は「原因→結果」の確率から「結果→原因」の確率を求める逆推論のツール
  • 問1.3のように、事前確率が低い場合は検査1回だけでは陽性的中率が低くなる（$1/3$ に留まる）。検査を重ねることで確度が上がる好例
  • 分散の計算は $V[X] = E[X^2] - (E[X])^2$ の展開公式を使うと速い
  • 包除原理は度数（人数）に対しても直接適用できる
