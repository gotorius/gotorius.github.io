統計的仮説検定の基礎――「背理法の確率版」

統計的仮説検定は、日常生活でもよく使われる論理的手続きです。 「この新薬は効果がない」という仮説を立て、その仮説のもとでは想いがけない結果が観察されたなら、仮説を疑います。 これは数学の背理法に似た構造をもちます。

ただし統計的検定では「絶対矛盾」ではなく「極めて稀なことが起きた」という確率的判断に基づきます。 そのため誤りの可能性を定量的に管理することが重要になります。

1. 検定の枠組み

帰無仮説と対立仮説

帰無仮説・対立仮説の設定
  • 帰無仮説 $H_0$（null hypothesis）：否定しようとしている仮説。「効果がない」「差がない」「現状と変わらない」という主張が典型例。
  • 対立仮説 $H_1$（alternative hypothesis）：証明したい主張。「効果がある」「上昇した」「差がある」などが典型例。

※ 検定は「$H_0$ を棄却できるか否か」の判断。「$H_1$ が正しい」を証明するわけではない。

検定の論理構造

  1. $H_0$ を仮定する（背理法の「仮定」に対応）
  2. $H_0$ のもとで分布が既知の検定統計量を定める（例：$Z_0 = \frac{\bar{X} - \mu_0}{\sigma/\sqrt{n}}$ は $N(0,1)$ に従う）
  3. 観測データから検定統計量を計算する
  4. 「$H_0$ が真なら極めて稀」な範囲（棄却域）に入っているか判断する
  5. 棄却域に入ったら $H_0$ を棄却（「矛盾」に対応）

対立仮説の種類と棄却域

例：平均 $\mu$ の検定（分散既知）。$H_0: \mu = \mu_0$、有意水準 $\alpha$

対立仮説 棄却域 呼称
$H_1: \mu > \mu_0$$Z_0 \geq z_{\alpha/2}$右片側検定
$H_1: \mu < \mu_0$$Z_0 \leq -z_{\alpha/2}$左片側検定
$H_1: \mu \neq \mu_0$$|Z_0| \geq z_{\alpha/2}$両側検定
💡 片側・両側の使い分け

「上がった（下がった）かどうか」を検証したいなら片側検定。 「変化があったかどうか（方向不明）」なら両側検定。 データを見てから対立仮説を選んではいけない（検定前に決める）。

2. P値と棄却域

P値の定義

P値（p-value）とは、 「$H_0$ が真であるもとで、観測されたものと同程度かそれ以上に稀な結果が得られる確率」です。

右片側の場合：$\text{P値} = P(Z \geq z_0)$（$z_0$ は検定統計量の実現値）
両側の場合：$\text{P値} = 2 \cdot P(Z \geq |z_0|)$

判断規則：P値 $< \alpha$（有意水準）ならば $H_0$ を棄却、有意差あり。

💡 P値の誤解と正しい解釈
  • ❌「P値 = $H_0$ が真である確率」は誤り。P値は仮定のもとでデータの稀さを表す。
  • ❌「P値が大きい = $H_0$ が正しい」は誤り。「棄却できなかった」に過ぎない。
  • ✅「P値 $= 0.03$ は、$H_0$ が真ならこれ以上極端な結果が3%の確率でしか起きない」
  • ✅ P値は有意水準より先に決めず、データを見てから初めて計算される値

有意水準と棄却限界値の関係

有意水準 $\alpha$ を事前に設定し、棄却限界値（臨界値）を決めます。たとえば：

  • 有意水準 $\alpha = 5\%$（両側） → 棄却限界値 $z_{0.025} = 1.96$
  • 有意水準 $\alpha = 1\%$（両側） → 棄却限界値 $z_{0.005} = 2.576$
  • 有意水準 $\alpha = 5\%$（片側） → 棄却限界値 $z_{0.05} = 1.645$

3. 検定の過誤と検出力

2種類の誤り
$H_0$ を棄却しない $H_0$ を棄却する
$H_0$ が真 正しい判断 第一種の過誤（$\alpha$）
$H_1$ が真 第二種の過誤（$\beta$） 正しい判断（検出力 $1-\beta$）
  • 第一種の過誤（type I error）：$H_0$ が真なのに棄却する誤り。確率 $= \alpha$（有意水準で制御）
  • 第二種の過誤（type II error）：$H_1$ が真なのに棄却しない誤り。確率 $= \beta$
  • 検出力（power）：$1 - \beta$。$H_1$ が真のとき正しく棄却できる確率

$\alpha$ と $\beta$ のトレードオフ

$n$ を固定したまま $\alpha$ を小さくする（棄却限界値を厳しくする）と、$\beta$ は大きくなります（检出力が下がります）。 逆に $\alpha$ を大きくすると検出力は上がりますが、無駄な棄却が増えます。

唯一の解決策はサンプルサイズを増やすこと：$n$ を大きくすると、$\alpha$ を一定に保ったまま $\beta$ を小さくできます（= 2つの正規分布が離れる）。

4. サンプルサイズ設計

「この研究で検出したい効果の大きさを確実に見つけるには、何人の標本が必要か？」 という問いに答えるのがサンプルサイズ設計です。

サンプルサイズの決定式（平均値検定の場合）

サンプルサイズの公式

有意水準 $\alpha$（両側）、検出力 $1-\beta$ を確保するために必要な $n$：

$$n = \frac{(z_{\alpha/2} + z_\beta)^2}{\Delta^2}, \quad \Delta = \frac{|\mu_1 - \mu_0|}{\sigma}$$

$\Delta$ はエフェクトサイズ（effect size）と呼ばれ、「平均の差が標準偏差の何倍か」を表します。

よく使う数値：$z_{0.025} = 1.96$（$\alpha = 5\%$ 両側）、$z_{0.20} = 0.842$（検出力80%）、$z_{0.10} = 1.282$（検出力90%）

💡 サンプルサイズ設計の直感

$H_0$ のもとでの検定統計量の分布（$N(0,1)$）と $H_1$ のもとでの分布（$N(\Delta\sqrt{n}, 1)$）の2つの正規分布が「離れるほど」検出力は高まります。 エフェクトサイズ $\Delta$ が小さいほど（検出したい差が小さいほど）、分離するために大きな $n$ が必要です。

5. 抜取検査

工場から出荷されるロット（製品の束）の良否を判定する際、全数検査が困難な場合に 抜取検査（sampling inspection）を行います。

計数値抜取検査の設定
  1. ロットの大きさ $N$（合計製品数）を特定する
  2. 抜取標本の大きさ $n$（検査個数）を定める
  3. 合格判定個数 $c$ を定める：不良品が $k \leq c$ なら合格、$k \geq c+1$ なら不合格
生産者危険と消費者危険

不良率が $p$ のとき、$n$ 個中の不良個数は $\mathrm{Bin}(n, p)$ に従います。

$$\text{生産者危険（FP）} = P(X \geq c+1 \mid p = p_0) = \sum_{k=c+1}^{n}\binom{n}{k}p_0^k(1-p_0)^{n-k}$$
$$\text{消費者危険（FN）} = P(X \leq c \mid p = p_1) = \sum_{k=0}^{c}\binom{n}{k}p_1^k(1-p_1)^{n-k}$$
  • 生産者危険（FP）：本来合格のロット（$p = p_0$）が不合格と判定される確率 → 生産者が損をする
  • 消費者危険（FN）：本来不合格のロット（$p = p_1$）が合格と判定される確率 → 消費者が不良品を受け取る

6. 例題・演習

問10.1：支持率調査の検出力とサンプルサイズ設計

問題

ある政党の先月の支持率は 0.45 であった。 今月、「支持率が先月より5ポイント上がって 0.50 になった」という主張を検証したい。 全国の有権者から無作為に $n$ 人の有効回答を得るとし、帰無仮説 $H_0: p_0 = 0.45$ 対 対立仮説 $H_1: p_1 = 0.50$ の両側検定を有意水準 5% で行う。

  1. $n = 600$ のとき、この検定の検出力（$H_1$ が真のときに正しく棄却できる確率）を求めよ。
  2. 検出力を 80% 以上とするために必要な最小の標本サイズを求めよ。
解答

(1) $n = 600$ のときの検出力：

$H_0$ のもとでの検定統計量 $Z_0 = \dfrac{\hat{p} - 0.45}{\sqrt{0.45 \times 0.55/n}}$ は $N(0,1)$ に従う。

$H_1: p = 0.50$ のもとでの $Z_0$ の期待値を求め、$z_\beta$ を計算する：

$$z_\beta = \frac{0.50 - 0.45}{\sqrt{0.45 \times 0.55/600}} - z_{\alpha/2} = \frac{0.05}{\sqrt{0.000413}} - 1.96 = \frac{0.05}{0.02031} - 1.96 = 2.461 - 1.96 = 0.501$$

$z_\beta = 0.501$ → 標準正規分布表より $\beta = P(Z \leq -0.501) \approx 0.308$

検出力 $= 1 - \beta = 1 - 0.308 = \mathbf{0.692}$（約 69.2%）

(2) 検出力 80% に必要な標本サイズ：

検出力 80% とは $\beta = 0.20$ → $z_{0.20} = 0.842$

$$n = \frac{(z_{\alpha/2} + z_\beta)^2}{\Delta^2} \times p_0(1-p_0) = \frac{(1.96 + 0.842)^2}{(0.05)^2} \times 0.45 \times 0.55$$
$$= \frac{(2.802)^2}{0.0025} \times 0.2475 = \frac{7.851}{0.0025} \times 0.2475 = 3140.4 \times 0.2475 \approx 777$$

最低 777人 の標本が必要です。

問10.2：製品の分散比較――F検定と信頼区間のセット

問題

あるメーカーは既製品Aに対して新製品Bを開発した。開発目標は「重さのばらつき（分散）を小さくすること」。 既製品A、新製品Bの重さはそれぞれ独立に正規分布 $N(\mu_A, \sigma_A^2)$、$N(\mu_B, \sigma_B^2)$ に従うとする。 各製品を $n = 16$ 個ずつ無作為に測定した結果、偏差平方和はそれぞれ $T_A = 300$、$T_B = 100$ であった。

  1. 新製品Bの母分散 $\sigma_B^2$ の95%信頼区間を求めよ（自由度15の $\chi^2$ : 上側2.5%点 = 27.49、下側2.5%点 = 6.26）。
  2. 「$\sigma_A^2 > \sigma_B^2$（新製品Bの方がばらつきが小さい）」という主張を有意水準5%の片側検定で確認せよ。
    帰無仮説 $H_0: \sigma_A^2 = \sigma_B^2$、対立仮説 $H_1: \sigma_A^2 > \sigma_B^2$。
    自由度 (15, 15) の $F$ 分布の上側5%点 = 2.40。
解答

(1) $\sigma_B^2$ の95%信頼区間：

偏差平方和 $T_B = 100$、自由度 $\phi = 15$:

$$\left(\frac{T_B}{\chi^2_{0.025}(15)},\; \frac{T_B}{\chi^2_{0.975}(15)}\right) = \left(\frac{100}{27.49},\; \frac{100}{6.26}\right) = \mathbf{(3.64,\; 15.97)}$$

(2) 片側 $F$ 検定：

検定統計量 $F = T_A/T_B$（分子・分母が $\sigma_A^2 = \sigma_B^2$ のもとでは不偏分散の比に等しい）。 自由度は 分子 $= n_A - 1 = 15$、分母 $= n_B - 1 = 15$。

$$F = \frac{T_A/(n_A-1)}{T_B/(n_B-1)} = \frac{300/15}{100/15} = \frac{20}{6.67} = 3.0$$

$F = 3.0 > F_{0.05}(15, 15) = 2.40$ → 帰無仮説を棄却できる。新製品Bのばらつきの方が統計的に有意に小さい。

問10.3：抜取検査の生産者危険・消費者危険の計算

問題

ある検査方式では、箱の中から5個の商品を無作為に抜き取り、 その中に不良品が2個以上あれば箱全体を不合格とする。 不良率 $p$ のとき5個中不良品 $r$ 個が出る確率を二項分布で計算した表（一部）を使う。

不良個数 $r$ $p=0.1$ $p=0.2$ $p=0.3$ $p=0.4$ $p=0.5$
$r=0$0.590.330.170.080.03
$r=1$0.330.410.360.260.16
$r=2$0.070.200.310.350.31
$r=3$0.010.050.130.230.31
$r=4$0.000.010.030.080.16
$r=5$0.000.000.000.010.03
  1. $p = 0.3$ のとき、$r = 0, 1, 2$ の確率を検算し、表の太字部分を確認せよ（二項分布の公式を使うこと）。
  2. 「不良率が0.1以下なら合格ロット、そうでなければ不合格」とする。 不良率 $p = 0.1$ のときの生産者危険（合格ロットが不合格と判定される確率）と、 不良率 $p = 0.2$ のときの消費者危険（不合格ロットが合格と判定される確率）をそれぞれ求めよ。
解答

(1) $p=0.3$ のときの確認：

$$P(X=0 \mid p=0.3) = \binom{5}{0}(0.3)^0(0.7)^5 = 0.168 \approx \mathbf{0.17}$$ $$P(X=1 \mid p=0.3) = \binom{5}{1}(0.3)^1(0.7)^4 = 5 \times 0.3 \times 0.2401 = 0.360 \approx \mathbf{0.36}$$ $$P(X=2 \mid p=0.3) = \binom{5}{2}(0.3)^2(0.7)^3 = 10 \times 0.09 \times 0.343 = 0.309 \approx \mathbf{0.31}$$

表の値と一致する。

(2) 生産者危険と消費者危険：

合格条件：不良品 $\leq 1$ 個（$c=1$）。よって不良品 $\geq 2$ 個で不合格。

$$\text{生産者危険} = P(X \geq 2 \mid p=0.1) = 0.07 + 0.01 + 0.00 + 0.00 = \mathbf{0.08}$$
$$\text{消費者危険} = P(X \leq 1 \mid p=0.2) = 0.33 + 0.41 = \mathbf{0.74}$$

消費者危険が0.74と非常に高い。$n=5$ という小さい標本では不良率0.2のロットを高確率（74%）で誤って合格させてしまう。

7. まとめ・ポイント

  • 仮説検定は確率的背理法：$H_0$ のもとで「稀なことが起きた」なら $H_0$ を疑う。数学的な矛盾とは異なり確率的判断のため誤りがある。
  • P値は事後計算・有意水準は事前設定：P値 $< \alpha$ なら棄却。P値が $H_0$ の「正しさの確率」を意味するわけではない。
  • 第一種・第二種の過誤はトレードオフ：$\alpha$ を下げると検出力 $1-\beta$ も下がる。解決はサンプルサイズを増やすこと。
  • エフェクトサイズと $n$：検出したい差が小さいほど（エフェクトサイズが小さいほど）、大きな標本サイズが必要。
  • サンプルサイズ公式：$n = {(z_{\alpha/2}+z_\beta)^2}/{\Delta^2} \times \sigma^2$。$\alpha=5\%$ 両側・検出力80%では $z_{\alpha/2}+z_\beta = 1.96+0.842 = 2.80$。
  • 抜取検査の消費者危険は高くなりやすい：標本サイズが小さいと不合格ロットを通してしまうリスクが高い。$n$ と $c$ の設計が重要。
