概要

第10章では統計的仮説検定の基礎と検定法の導出を扱う。 数学的背理法との類似から検定の考え方を理解し、帰無仮説・対立仮説の設定、 有意水準・棄却域・P-値の概念、検定の過誤（第一種・第二種）、検出力、 サンプルサイズ設計、抜取検査を学ぶ。

キーワード：帰無仮説、対立仮説、有意水準、棄却限界値、P-値、検出力、抜取検査

重要概念

統計的仮説検定の考え方

  • 数学的背理法の応用：「矛盾」の代わりに「極めて稀にしか起こらないことが観測された」ことを根拠に仮説を否定する
  • 帰無仮説 $H_0$：否定したい仮説（例：$\mu = \mu_0$）
  • 対立仮説 $H_1$：検証したい仮説（例：$\mu \neq \mu_0$）
  • 確率的事象に基づく判断のため、誤りが生じる可能性がある

検定法の構成要素

  • 検定統計量：$H_0$ のもとで分布が既知の統計量（例：$Z_0 = \dfrac{\bar{X} - \mu_0}{\sqrt{\sigma^2/n}}$）
  • 有意水準 $\alpha$：$H_0$ のもとで「稀」と判断する確率の閾値（例：5%）
  • 棄却限界値（臨界値）：棄却域と受容域の境界値
  • 棄却域：$H_0$ を棄却する検定統計量の範囲
  • 両側検定：$H_1: \mu \neq \mu_0$ → $|Z_0| \geq 1.96$
  • 片側検定：$H_1: \mu > \mu_0$ → $Z_0 \geq 1.96$

P-値

  • $H_0$ のもとで、観測データよりも稀なデータが生じる確率
  • 片側P-値：$P(Z \geq z_0)$
  • 両側P-値：$2 \cdot P(Z \geq |z_0|)$
  • P-値 $< \alpha$ なら $H_0$ を棄却

検定の過誤

  • 第一種の過誤（type I error）：$H_0$ が真なのに棄却する。確率 $\alpha$ で制御
  • 第二種の過誤（type II error）：$H_1$ が真なのに棄却できない。確率 $\beta$
  • 検出力（power）：$1 - \beta$。$H_1$ が真のとき正しく棄却する確率
  • $\alpha$ と $\beta$ はトレードオフの関係

抜取検査

  • 検査ロットから抽出した標本の不良個数 $k$ に基づくロットの合否判定
  • 生産者危険（FP）：本来合格のロットが不合格と判定される確率
  • 消費者危険（FN）：本来不合格のロットが合格と判定される確率

公式・定理

サンプルサイズ設計

片側有意水準 $\alpha/2$、検出力 $1 - \beta$ を確保するための等式：

$$z_{\alpha/2} + z_\beta = \frac{\mu_1 - \mu_0}{\sqrt{\sigma^2/n}}$$

$n$ について解くと：

$$n = \frac{(z_{\alpha/2} + z_\beta)^2}{\Delta^2}$$

ここで $\Delta = (\mu_1 - \mu_0)/\sigma$ はエフェクトサイズ。エフェクトサイズが小さいほど必要な標本サイズが大きくなる。

抜取検査の確率

不良率 $p$ で標本サイズ $n$、合格判定個数 $c$ のとき

$$\text{生産者危険} = \sum_{k=c+1}^{n} \binom{n}{k} p_0^k (1-p_0)^{n-k}$$
$$\text{消費者危険} = \sum_{k=0}^{c} \binom{n}{k} p_1^k (1-p_1)^{n-k}$$

例題・演習

問10.1：支持率検定の検出力とサンプルサイズ

$H_0: p_0 = 0.45$ vs $H_1: p_1 = 0.50$。有意水準5%の両側検定。

(1) $n = 600$ のときの検出力：

$$z_\beta = \frac{0.50 - 0.45}{\sqrt{0.45 \times 0.55 / 600}} - 1.96 = 0.5018$$

正規分布表より $\beta = 0.3085$、検出力 $= 1 - \beta = \mathbf{0.6915}$

(2) 検出力80%に必要な標本サイズ：

$$n = \frac{(1.96 + 0.84)^2}{(0.50 - 0.45)^2} \times 0.45 \times 0.55 = 776.1 \implies n = \mathbf{777}$$

問10.2：分散の信頼区間とF検定

既製品A、新製品B を16個ずつ測定。$T_A = 300$、$T_B = 100$。

(1) $\sigma_B^2$ の95%信頼区間：自由度15のカイ二乗分布の分位点より

$$\left(\frac{100}{27.49},\; \frac{100}{6.26}\right) = \mathbf{(3.64,\; 15.97)}$$

(2) $F = T_A/T_B = 3.0$。自由度 $(15, 15)$ の $F$ 分布の上側5%点と比較し、帰無仮説は棄却できる。

問10.3：抜取検査

5個抜き取り、不良品2個以上で不合格。

(1) $p = 0.3$ のとき：$P(X=0|0.3) = 0.17$、$P(X=1|0.3) = 0.36$、$P(X=2|0.3) = 0.31$

(2) $p = 0.1$ の生産者危険 $= 0.07 + 0.01 + 0.00 + 0.00 = \mathbf{0.08}$

$p = 0.2$ の消費者危険 $= 0.33 + 0.41 = \mathbf{0.74}$

メモ

  • 統計的仮説検定は「帰無仮説のもとで稀なことが起きた」ことをもって仮説を否定する手法であり、数学的に矛盾を示すわけではない
  • 有意水準は第一種の過誤を制御するが、第二種の過誤（検出力不足）には直接対処できない
  • P-値は「データが観測された後に計算される値」であり、事前に設定される有意水準とは異なる概念
  • サンプルサイズ設計ではエフェクトサイズが重要：検出したい差が小さいほど多くの標本が必要
  • 抜取検査の消費者危険は $p = 0.2$ のとき0.74と高く、標本サイズ5は検出力が不十分
