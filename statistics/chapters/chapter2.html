概要

第2章では、確率変数の分布を「道具」を使って体系的に分析する方法を学びます。 具体的には次の4つのテーマを扱います。

  1. 累積分布関数（CDF）・生存関数・ハザード関数：確率変数がある値以下になる確率を関数として捉える
  2. 同時分布・周辺分布・条件付き分布：2つ以上の確率変数の「関係」を記述する
  3. 確率母関数（PGF）・モーメント母関数（MGF）：確率分布を1つの関数として要約し、期待値・分散を機械的に求める
  4. 母関数の応用：独立な確率変数の和の分布を効率よく求める

この章のキモ：母関数を使うと、複雑な計算を「微分するだけ」に変換できます。 また「独立な変数の和 → 母関数の積」という性質により、和の分布を直接計算せずに特定できます。

キーワード：累積分布関数、生存関数、ハザード関数、同時確率密度関数、周辺確率密度関数、条件付き確率密度関数、モーメント母関数（積率母関数）、確率母関数

1．累積分布関数・生存関数・ハザード関数

累積分布関数（CDF）とは

確率変数 $X$ の 累積分布関数（cumulative distribution function, CDF） $F(x)$ は

$$F(x) = P(X \leq x)$$

と定義されます。「$X$ が値 $x$ 以下になる確率」を表します。

直感的には、数直線を左端から $x$ まで「塗りつぶした」部分の確率の合計です。

離散型の場合

確率変数 $X$ が整数値などの離散的な値をとる場合、CDF は確率関数 $p(x_i) = P(X = x_i)$ の累積和です：

$$F(x) = \sum_{x_i \leq x} p(x_i)$$

逆に、確率関数は CDF の「跳び」から取り戻せます： $$p(x) = F(x) - F(x-)$$ ここで $F(x-) = \lim_{x' \uparrow x} F(x')$ は、$x$ に左から近づいたときの極限値（左極限）です。 離散型では CDF は階段関数になるため、各 $x_i$ でジャンプし、そのジャンプの大きさが $p(x_i)$ です。

連続型の場合

確率変数 $X$ が連続的な値をとる場合、確率密度関数（pdf）$f(x)$ を使って CDF を積分で書けます：

$$F(x) = \int_{-\infty}^{x} f(t)\,dt$$

これを微分すると確率密度関数が戻ってきます（微分積分学の基本定理）： $$f(x) = F'(x)$$

イメージ：$f(x)$ はグラフの「高さ」、$F(x)$ は「グラフの左側の面積の合計」です。 $F(x)$ を微分すると高さ $f(x)$ が復元されます。

生存関数（Survival Function）

$X$ が（機械や生物の）寿命を表す確率変数の場合、 「時刻 $x$ にまだ生きている（故障していない）確率」を 生存関数 $S(x)$ と呼びます：

$$S(x) = P(X > x) = 1 - F(x)$$

CDF が「$x$ 以前に死亡した確率」なら、生存関数は「$x$ より後まで生き延びた確率」です。

ハザード関数（Hazard Function）

ハザード関数 $h(x)$ は「時刻 $x$ に生存している人が、その直後の短時間 $\Delta x$ 内に死亡する率」です。 式で書くと

$$h(x) = \frac{f(x)}{S(x)} = \frac{f(x)}{1 - F(x)}$$

【式の読み方】分子 $f(x)$ は「ちょうど時刻 $x$ に死亡する密度」、 分母 $S(x) = 1 - F(x)$ は「時刻 $x$ まで生き延びてきた確率」です。 ハザード関数はこの比なので、「生き残っている人のうち何割が直後に死ぬか」を表しています。

また、$S(x)$ を使うと次のように書けます：

$$h(x) = -\frac{d}{dx} \log S(x)$$

【導出】$S(x) = 1 - F(x)$ を微分すると $S'(x) = -f(x)$。したがって

$$-\frac{d}{dx}\log S(x) = -\frac{S'(x)}{S(x)} = \frac{f(x)}{S(x)} = h(x)$$

ハザード関数の詳細は第32章（生存時間解析）で改めて学びます。

2．同時分布・周辺分布・条件付き分布

なぜ2変数が必要か

現実のデータでは複数の変数（例：身長と体重、気温と売上）が同時に観測されます。 2変数の「同時のふるまい」を記述するために 同時分布 を導入します。

離散型：同時確率関数・周辺確率関数・条件付き確率関数

$X$, $Y$ が2つの離散確率変数のとき、「$X = x$ かつ $Y = y$ となる確率」を 同時確率関数（joint probability function）と呼び

$$p(x, y) = P(X = x,\; Y = y)$$

と書きます。

周辺確率関数（marginal probability function）は、一方の変数に関する不確かさを「消去」して得られます。 $Y$ についてすべての可能な値で足し合わせることで $X$ だけの確率関数が得られます：

$$p_X(x) = \sum_{y} p(x, y)$$

イメージ：同時確率関数を表（行：$x$、列：$y$）として並べたとき、 各行の合計が $p_X(x)$（行和）、各列の合計が $p_Y(y)$（列和）です。 表の「周辺（欄外）」に書くことから「周辺分布」と呼ばれます。

条件付き確率関数（conditional probability function）は、$X = x$ が与えられたもとでの $Y$ の分布です。 条件付き確率の定義 $P(A|B) = P(A \cap B)/P(B)$ をそのまま適用します：

$$p_{Y|X}(y \mid x) = \frac{p(x, y)}{p_X(x)} \quad (\text{ただし } p_X(x) > 0)$$

覚え方：「条件付き ＝ 同時 ÷ 周辺」です。

連続型：同時確率密度関数・周辺・条件付き

$X$, $Y$ が連続確率変数のとき、累積分布関数 $F(x, y) = P(X \leq x, Y \leq y)$ に対して 同時確率密度関数（joint probability density function）を次のように定義します：

$$f(x, y) = \frac{\partial^2}{\partial x \,\partial y} F(x, y)$$

1次元でCDFを微分すると密度関数が得られたように、2次元では偏微分を2回行います。

$f(x,y)$ を積分すると確率が得られます： $$P(a \leq X \leq b,\; c \leq Y \leq d) = \int_a^b \int_c^d f(x, y)\,dy\,dx$$

また、全範囲での積分は1です：$\displaystyle\iint_{-\infty}^{\infty} f(x,y)\,dx\,dy = 1$。

周辺確率密度関数は、もう一方の変数について積分で消去します：

$$f_X(x) = \int_{-\infty}^{\infty} f(x, y)\,dy, \qquad f_Y(y) = \int_{-\infty}^{\infty} f(x, y)\,dx$$

条件付き確率密度関数は離散型と同じ「同時 ÷ 周辺」の形です：

$$f_{Y|X}(y \mid x) = \frac{f(x, y)}{f_X(x)}$$

条件付き独立

$Z = z$ を与えたもとで $X$ と $Y$ が独立のとき（条件付き独立）、条件付き同時密度が積に分解されます：

$$f_{X,Y|Z}(x, y \mid z) = f_{X|Z}(x \mid z) \cdot f_{Y|Z}(y \mid z)$$

例：$Z$ = 学年を与えると、「身長 $X$」と「体重 $Y$」が条件付き独立になるかもしれません。 つまり、同じ学年内では身長と体重が独立とみなせる状況です。

3．母関数（確率母関数・モーメント母関数）

なぜ母関数が必要か

期待値や分散を定義の式から直接計算するのは大変なことがあります。 また、独立な確率変数の和の分布を直接求めるには「畳み込み」と呼ばれる積分が必要で面倒です。 母関数（generating function）を使うとこれらの問題を「微分するだけ」に変換でき、計算が劇的に楽になります。

確率母関数（PGF：Probability Generating Function）

整数値（$0, 1, 2, \ldots$）をとる確率変数 $X$ の確率母関数を次で定義します：

$$G(s) = E[s^X] = \sum_{x=0}^{\infty} s^x \, p(x)$$

記号の意味： $s$ は実数で、$1$ を含む適当な範囲の値をとります。 $p(x) = P(X = x)$ は確率関数です。 $G(s)$ は各確率を「$s^x$ の係数」として詰め込んだ多項式（または冪級数）です。

直感：$G(s)$ は確率分布全体の情報を1つの関数にエンコードしたものです。 微分することで各モーメントが取り出せます。

PGFから期待値・分散を求める

$G(s) = \sum_x s^x p(x)$ を $s$ で微分します：

$$G'(s) = \sum_x x \, s^{x-1} p(x) = \frac{E[s^{X-1} \cdot X]}{1}$$

$s = 1$ を代入すると：

$$G'(1) = \sum_x x \cdot p(x) = E[X]$$

もう一度微分して $s = 1$ を代入すると：

$$G''(1) = \sum_x x(x-1) \cdot p(x) = E[X(X-1)] = E[X^2] - E[X]$$

したがって $E[X^2] = G''(1) + G'(1)$。分散は

$$V[X] = E[X^2] - (E[X])^2 = G''(1) + G'(1) - (G'(1))^2$$

モーメント母関数（MGF：Moment Generating Function）

連続確率変数（とより一般的に）に使われる母関数が モーメント母関数 です。 確率母関数の $s$ を $e^\theta$ に置き換えたもの、と理解できます：

$$m(\theta) = E[e^{\theta X}] = G(e^\theta)$$

連続型では $m(\theta) = \int_{-\infty}^{\infty} e^{\theta x} f(x)\,dx$ と積分で書けます。 MGF は $\theta = 0$ 付近で収束することを前提とします。

MGFから原点まわりのモーメントを求める

$m(\theta) = E[e^{\theta X}]$ を $\theta$ で微分します。 $e^{\theta X}$ を $\theta$ で微分すると $X e^{\theta X}$ になるので：

$$m'(\theta) = E[X e^{\theta X}]$$

$\theta = 0$ を代入すると：

$$m'(0) = E[X \cdot e^0] = E[X]$$

同様に $k$ 回微分して $\theta = 0$ を代入すると：

$$m^{(k)}(0) = E[X^k]$$

つまり MGFを $\theta = 0$ で $k$ 回微分すれば $k$ 次モーメント $E[X^k]$ が得られます。 これが「モーメント（積率）母関数」という名前の由来です。

特性関数（Characteristic Function）

MGFは常に収束するとは限りません（例：コーシー分布のMGFは存在しない）。 そこで $\theta$ を純虚数 $it$（$t$ は実数、$i$ は虚数単位）に置き換えた 特性関数

$$\varphi(t) = E[e^{itX}] = m(it)$$

は、$|e^{itX}| = 1$ なのでどんな分布に対しても常に存在します。 モーメントが存在しない分布でも扱えるため、理論的に最も一般的な道具です。

母関数の2大重要性質

母関数が便利な理由は次の2つの性質に集約されます。

性質1（分布との1対1対応）：異なる確率分布は異なる母関数を持つ。 逆に言えば、母関数が同じなら分布は同じ。
性質2（独立な和の母関数は積）：$X_1, X_2$ が独立なら $$m_{X_1+X_2}(\theta) = m_{X_1}(\theta) \cdot m_{X_2}(\theta)$$

性質2の導出：$X_1, X_2$ が独立のとき $E[f(X_1)g(X_2)] = E[f(X_1)] \cdot E[g(X_2)]$が成り立つので（独立性の性質）

$$m_{X_1+X_2}(\theta) = E[e^{\theta(X_1+X_2)}] = E[e^{\theta X_1} \cdot e^{\theta X_2}] = E[e^{\theta X_1}] \cdot E[e^{\theta X_2}] = m_{X_1}(\theta) \cdot m_{X_2}(\theta)$$

この性質を繰り返し使うと：$X_1, \ldots, X_n$ が i.i.d. で共通のMGFを $m(\theta)$ とすれば $$m_{X_1 + \cdots + X_n}(\theta) = m(\theta)^n$$

公式まとめ

確率母関数と期待値・分散

$$G(s) = E[s^X], \quad G'(1) = E[X], \quad G''(1) = E[X(X-1)]$$ $$V[X] = G''(1) + G'(1) - \bigl(G'(1)\bigr)^2$$

モーメント母関数と各次モーメント

$$m(\theta) = E[e^{\theta X}], \quad m^{(k)}(0) = E[X^k]$$ $$E[X] = m'(0), \quad V[X] = m''(0) - \bigl(m'(0)\bigr)^2$$

応用例：ベルヌーイ分布から二項分布へ

コイン投げ1回：$P(X=1)=p,\; P(X=0)=1-p$（ベルヌーイ分布）の確率母関数は

$$G_{\text{Be}}(s) = (1-p)s^0 + p \cdot s^1 = (1-p) + ps = 1 + p(s-1)$$

独立に $n$ 回繰り返した合計 $X_1 + \cdots + X_n$（二項分布）の確率母関数は、性質2より

$$G_{\text{Bin}}(s) = \bigl(1 + p(s-1)\bigr)^n$$

これを展開すると二項定理より $P(\text{合計} = k) = \binom{n}{k}p^k(1-p)^{n-k}$ が確認できます。

例題・演習

【例題1】単位円内の周辺分布と条件付き分布

$xy$ 平面の単位円内部 $\{(x,y) \mid x^2 + y^2 \leq 1\}$ から点を一様に抽出する。 同時確率密度関数を次のように定める：

$$f(x, y) = \begin{cases} \dfrac{1}{\pi} & (x^2 + y^2 \leq 1) \\ 0 & (\text{それ以外}) \end{cases}$$

$X$ の周辺確率密度関数 $f_X(x)$ と、$X = x$ を与えたときの $Y$ の条件付き確率密度関数 $f_{Y|X}(y \mid x)$ を求めよ。

【解答】

Step 1：$y$ の積分範囲を求める。
$-1 \leq x \leq 1$ のとき、$f(x,y) > 0$ となる $y$ の範囲は $x^2 + y^2 \leq 1$ から $y^2 \leq 1 - x^2$、すなわち $-\sqrt{1-x^2} \leq y \leq \sqrt{1-x^2}$ です。

Step 2：周辺確率密度関数を計算する。

$$f_X(x) = \int_{-\sqrt{1-x^2}}^{\sqrt{1-x^2}} \frac{1}{\pi}\,dy = \frac{1}{\pi} \cdot 2\sqrt{1-x^2} = \frac{2\sqrt{1-x^2}}{\pi}$$

積分の計算：定数 $\dfrac{1}{\pi}$ を積分すると $\dfrac{1}{\pi} \times [\text{幅}]$ です。 幅は $\sqrt{1-x^2} - (-\sqrt{1-x^2}) = 2\sqrt{1-x^2}$ です。

Step 3：条件付き確率密度関数を計算する。

$$f_{Y|X}(y \mid x) = \frac{f(x,y)}{f_X(x)} = \frac{1/\pi}{2\sqrt{1-x^2}/\pi} = \frac{1}{2\sqrt{1-x^2}}$$

この式は $y$ に依存しない定数なので、$-\sqrt{1-x^2} \leq y \leq \sqrt{1-x^2}$ の区間上の 一様分布 になっています。 区間の幅 $2\sqrt{1-x^2}$ の逆数が確率密度となっています（一様分布の密度 = 1/幅）。

【問2.1】三角形状の同時密度関数

領域 $\{(x, y) \mid x \geq 0,\; y \geq 0,\; x + y \leq 1\}$（三角形）上の確率密度関数を $f(x, y) = c$ （定数）とおく。

(1) 基準化定数 $c$ を求めよ。

(2) $X$ の周辺確率密度関数 $f_X(x)$ を求めよ（$0 \leq x \leq 1$）。

(3) $X = x$ が与えられたときの $Y$ の条件付き確率密度関数 $f_{Y|X}(y \mid x)$ を求めよ。

【解答】

(1) 基準化定数の計算

三角形の面積は $\dfrac{1}{2} \times 1 \times 1 = \dfrac{1}{2}$ なので、確率の合計が1になるためには

$$\iint_{\text{三角形}} c\,dx\,dy = c \times \frac{1}{2} = 1 \implies \boxed{c = 2}$$

(2) 周辺確率密度関数の計算

$x$ が固定されたとき、$y$ の範囲は $0 \leq y \leq 1 - x$（三角形の条件 $x + y \leq 1$ より）です。

$$f_X(x) = \int_0^{1-x} 2\,dy = 2(1-x) \qquad (0 \leq x \leq 1)$$

確認：$\int_0^1 2(1-x)\,dx = 2\bigl[x - \tfrac{x^2}{2}\bigr]_0^1 = 2 \times \tfrac{1}{2} = 1$ ✓

(3) 条件付き確率密度関数の計算

$$f_{Y|X}(y \mid x) = \frac{f(x,y)}{f_X(x)} = \frac{2}{2(1-x)} = \frac{1}{1-x} \qquad (0 \leq y \leq 1-x)$$

これは区間 $[0, 1-x]$ 上の一様分布 $\mathrm{Uniform}(0,\; 1-x)$ の確率密度になっています。 直感的にも、三角形上の一様分布から $X=x$ の縦線を引いたとき、 $Y$ は $[0, 1-x]$ 上で一様に分布します。

【問2.2】ポアソン分布の確率母関数・期待値・分散

確率変数 $X$ がパラメータ $\lambda > 0$ のポアソン分布に従うとする。 すなわち $x = 0, 1, 2, \ldots$ について $$p(x) = \frac{e^{-\lambda} \lambda^x}{x!}$$ $X$ の確率母関数 $G(s)$ を求め、そこから期待値と分散を求めよ。

【解答】

Step 1：確率母関数を求める。

$$G(s) = E[s^X] = \sum_{x=0}^{\infty} s^x \cdot \frac{e^{-\lambda}\lambda^x}{x!} = e^{-\lambda} \sum_{x=0}^{\infty} \frac{(s\lambda)^x}{x!}$$

$\displaystyle\sum_{x=0}^{\infty} \frac{u^x}{x!} = e^u$（指数関数のマクローリン展開）を使うと：

$$G(s) = e^{-\lambda} \cdot e^{s\lambda} = e^{\lambda(s - 1)}$$

Step 2：期待値を求める。

$G(s) = e^{\lambda(s-1)}$ を $s$ で微分します。$G'(s) = \lambda e^{\lambda(s-1)}$。

$$E[X] = G'(1) = \lambda e^{\lambda \cdot 0} = \lambda$$

Step 3：分散を求める。

$G'(s) = \lambda e^{\lambda(s-1)}$ をもう一度微分します：$G''(s) = \lambda^2 e^{\lambda(s-1)}$。

$$G''(1) = \lambda^2$$
$$V[X] = G''(1) + G'(1) - \bigl(G'(1)\bigr)^2 = \lambda^2 + \lambda - \lambda^2 = \lambda$$

ポアソン分布では期待値と分散がともに $\lambda$ に一致するという有名な性質が確認できました。

【問2.3】指数分布のモーメント母関数・期待値・分散

確率変数 $X$ がパラメータ $\lambda > 0$ の指数分布に従うとする。 確率密度関数は $f(x) = \lambda e^{-\lambda x}$（$x \geq 0$）である。 $X$ のモーメント母関数 $m(\theta)$ を求め、そこから期待値と分散を求めよ。

【解答】

Step 1：MGFを求める。（$\theta < \lambda$ の場合に収束します）

$$m(\theta) = E[e^{\theta X}] = \int_0^{\infty} e^{\theta x} \cdot \lambda e^{-\lambda x}\,dx = \lambda \int_0^{\infty} e^{-(\lambda - \theta)x}\,dx$$

指数関数の積分：$\displaystyle\int_0^{\infty} e^{-ax}\,dx = \frac{1}{a}$（$a > 0$ のとき）を使います。 ここで $a = \lambda - \theta > 0$（$\theta < \lambda$ の仮定から保証）：

$$m(\theta) = \lambda \cdot \frac{1}{\lambda - \theta} = \frac{\lambda}{\lambda - \theta}$$

Step 2：期待値を求める。

商の微分法則 $\left(\dfrac{u}{v}\right)' = \dfrac{u'v - uv'}{v^2}$ を使います。$u = \lambda$（定数）, $v = \lambda - \theta$：

$$m'(\theta) = \frac{0 \cdot (\lambda - \theta) - \lambda \cdot (-1)}{(\lambda - \theta)^2} = \frac{\lambda}{(\lambda - \theta)^2}$$
$$E[X] = m'(0) = \frac{\lambda}{\lambda^2} = \frac{1}{\lambda}$$

Step 3：$E[X^2]$ を求める。

$$m''(\theta) = \frac{2\lambda}{(\lambda - \theta)^3}$$
$$E[X^2] = m''(0) = \frac{2\lambda}{\lambda^3} = \frac{2}{\lambda^2}$$

Step 4：分散を求める。

$$V[X] = E[X^2] - (E[X])^2 = \frac{2}{\lambda^2} - \left(\frac{1}{\lambda}\right)^2 = \frac{2}{\lambda^2} - \frac{1}{\lambda^2} = \frac{1}{\lambda^2}$$

指数分布の期待値は $1/\lambda$、分散は $1/\lambda^2$ です。

まとめ・ポイント

  • CDF の役割：$F(x) = P(X \leq x)$ は離散・連続共通の定義。連続型では $f(x) = F'(x)$、離散型では $p(x) = F(x) - F(x-)$（左極限との差）で密度・確率関数を復元できる
  • 周辺・条件付き：「同時 → 片側を積分するだけ（周辺）」「同時 ÷ 周辺 = 条件付き」。この2式で多変数の分布解析が完結する
  • PGF の使い方：$G(s)$ を求め $s=1$ で微分 → 期待値・分散が機械的に求まる。整数値確率変数向け
  • MGF の使い方：$m(\theta)$ を求め $\theta=0$ で微分 → $k$ 次モーメントが求まる。$m^{(k)}(0) = E[X^k]$
  • 独立な和の分布：母関数（PGF または MGF）を積にするだけ。畳み込みの積分は不要。これが母関数の最大の利点
  • 特性関数：$\varphi(t) = E[e^{itX}]$。MGF が存在しない分布（コーシー分布など裾が重い分布）でも常に存在する。理論解析で重要
  • ハザード関数：生存時間解析（第32章）で再登場。「生き残っている者が直後に死ぬ率」を表す
