概要

第23章では判別分析を扱う。データを既知のクラス（群）に分類するための統計的手法として、 フィッシャーの線形判別関数、2次判別分析、正準判別分析を学ぶ。さらに機械学習分野での代表的分類手法である サポートベクターマシン（SVM）、および分類性能の評価指標（混同行列・ROC曲線・AUC）を理解する。

キーワード：フィッシャーの線形判別関数、マハラノビス距離、2次判別、正準判別分析、SVM、カーネル法、混同行列、ROC曲線、AUC

重要概念

フィッシャーの線形判別関数（2群）

2つの群の重心間距離を群内分散に対して最大にする線形結合 $z = \boldsymbol{w}^T\boldsymbol{x}$ を求める。

$$\boldsymbol{w} = S_W^{-1}(\bar{\boldsymbol{x}}_1 - \bar{\boldsymbol{x}}_2)$$
  • $S_W$：群内共分散行列（pooled covariance matrix）
  • 判別の基準：$z$ の値と閾値 $z_0 = \frac{1}{2}\boldsymbol{w}^T(\bar{\boldsymbol{x}}_1 + \bar{\boldsymbol{x}}_2)$ を比較

マハラノビスの平方距離

$$D^2 = (\boldsymbol{x} - \boldsymbol{\mu})^T \Sigma^{-1}(\boldsymbol{x} - \boldsymbol{\mu})$$

各群の重心からのマハラノビス距離が近い方に分類する方法。

2次判別分析

各群の共分散行列が異なる場合（$\Sigma_1 \neq \Sigma_2$）に用いる。判別境界が2次曲面となる。

$$f(\boldsymbol{x}) = \boldsymbol{x}^T A\boldsymbol{x} + \boldsymbol{b}^T\boldsymbol{x} + c$$
  • $A = \frac{1}{2}(\Sigma_2^{-1} - \Sigma_1^{-1})$
  • $\boldsymbol{b} = \Sigma_1^{-1}\boldsymbol{\mu}_1 - \Sigma_2^{-1}\boldsymbol{\mu}_2$
  • $c = -\frac{1}{2}\boldsymbol{\mu}_1^T\Sigma_1^{-1}\boldsymbol{\mu}_1 + \frac{1}{2}\boldsymbol{\mu}_2^T\Sigma_2^{-1}\boldsymbol{\mu}_2 - \frac{1}{2}\ln\frac{|\Sigma_1|}{|\Sigma_2|}$
  • $f(\boldsymbol{x}) > 0 \Rightarrow$ クラス1、$f(\boldsymbol{x}) < 0 \Rightarrow$ クラス2

正準判別分析（多群）

$g$ 群の場合、群間分散 $S_B$ と群内分散 $S_W$ の比を最大化する：

$$S_W^{-1}S_B\boldsymbol{u} = \lambda\boldsymbol{u}$$

最大 $\min(p, g-1)$ 個の正準判別関数が得られる。

サポートベクターマシン（SVM）

マージン（境界面と最近接データ点の距離）を最大化するように分離超平面を決定する。

$$\max_{\boldsymbol{w},b}\; \frac{2}{\|\boldsymbol{w}\|} \quad \text{s.t.}\quad y_i(\boldsymbol{w}^T\boldsymbol{x}_i + b) \geq 1\; (\forall i)$$
  • サポートベクター：マージン境界上のデータ点
  • カーネル法：線形分離不可能なデータを高次元空間に写像して分離。代表的カーネル：
    • 多項式カーネル：$K(\boldsymbol{x},\boldsymbol{y}) = (\boldsymbol{x}^T\boldsymbol{y} + c)^d$
    • RBFカーネル（ガウスカーネル）：$K(\boldsymbol{x},\boldsymbol{y}) = \exp(-\gamma\|\boldsymbol{x}-\boldsymbol{y}\|^2)$
  • ソフトマージンSVM：スラック変数 $\xi_i \geq 0$ を導入し、誤分類をある程度許容
$$\min_{\boldsymbol{w},b,\boldsymbol{\xi}}\; \frac{1}{2}\|\boldsymbol{w}\|^2 + C\sum_{i=1}^{n}\xi_i \quad \text{s.t.}\quad y_i(\boldsymbol{w}^T\boldsymbol{x}_i + b) \geq 1 - \xi_i$$

$C$ が大きいと過学習、小さいと汎化性能向上。

混同行列と評価指標

予測：陽性予測：陰性
実際：陽性TP（真陽性）FN（偽陰性）
実際：陰性FP（偽陽性）TN（真陰性）
  • 正解率（Accuracy）：$(TP+TN)/(TP+FP+TN+FN)$
  • 適合率（Precision）：$TP/(TP+FP)$
  • 感度（Sensitivity）= 再現率（Recall）：$TP/(TP+FN)$
  • 特異度（Specificity）：$TN/(TN+FP)$
  • F値（F-measure）：$2 \times \frac{\text{Precision} \times \text{Recall}}{\text{Precision} + \text{Recall}}$

ROC曲線とAUC

横軸に偽陽性率（1−特異度）、縦軸に感度をプロット。閾値を連続的に変化させて曲線を描く。

  • AUC（Area Under the Curve）：ROC曲線下の面積。1に近いほど判別性能が高い
  • AUC = 0.5 → ランダムと同等、AUC = 1.0 → 完全分類

公式・定理

フィッシャーの線形判別係数

$$\boldsymbol{w} = S_W^{-1}(\bar{\boldsymbol{x}}_1 - \bar{\boldsymbol{x}}_2)$$

マハラノビスの平方距離

$$D^2 = (\boldsymbol{x} - \boldsymbol{\mu})^T \Sigma^{-1}(\boldsymbol{x} - \boldsymbol{\mu})$$

2次判別関数

$$f(\boldsymbol{x}) = \boldsymbol{x}^T A\boldsymbol{x} + \boldsymbol{b}^T\boldsymbol{x} + c$$

$A = \frac{1}{2}(\Sigma_2^{-1} - \Sigma_1^{-1}), \quad \boldsymbol{b} = \Sigma_1^{-1}\boldsymbol{\mu}_1 - \Sigma_2^{-1}\boldsymbol{\mu}_2$

正準判別の固有値問題

$$S_W^{-1}S_B\boldsymbol{u} = \lambda\boldsymbol{u}$$

SVMの目的関数（ソフトマージン）

$$\min \frac{1}{2}\|\boldsymbol{w}\|^2 + C\sum_{i}\xi_i \quad \text{s.t.}\quad y_i(\boldsymbol{w}^T\boldsymbol{x}_i + b) \geq 1-\xi_i,\;\xi_i \geq 0$$

例題・演習

問23.1：SVMと線形判別の比較

結果：あるデータセットでSVMの正答率が線形判別より低い場面があった。

原因：SVMは高次元空間で複雑な境界を学習するため、学習データへの過学習（overfitting）が起きやすい。

対策：

  • 正則化パラメータ $C$ の適切なチューニング（小さくすると汎化性能向上）
  • カーネルの選択やパラメータ（例：RBFの $\gamma$）の調整
  • 交差検証によるモデル選択

線形判別はモデルが単純で過学習しにくいが、非線形な構造を捉えられない。

問23.2：アルファベットの正準判別分析

アルファベットE, F, Qの3群を8つの特徴量で正準判別分析。

正準判別関数：3群なので最大 $\min(8, 3-1) = 2$ 個の判別関数

  • 第1正準判別関数：EとQを主に分離
  • 第2正準判別関数：EとFを主に分離

新しい文字Lの分類：

正準判別得点を計算し、各群の重心からのマハラノビス距離が最小の群に分類 → 群Fに分類された。

（LはFに形状が最も近い：横棒がある直線的な文字）

問23.3：2次判別関数の構成

2変量、2クラスのデータ：

  • クラス1：$\boldsymbol{\mu}_1 = (2, 3)^T$、$\Sigma_1 = \begin{pmatrix} 4 & 0 \\ 0 & 1 \end{pmatrix}$
  • クラス2：$\boldsymbol{\mu}_2 = (0, -1)^T$、$\Sigma_2 = \begin{pmatrix} 1 & 0 \\ 0 & 1 \end{pmatrix}$

2次判別関数の係数：

$\Sigma_1^{-1} = \begin{pmatrix} 0.25 & 0 \\ 0 & 1 \end{pmatrix}$, $\Sigma_2^{-1} = \begin{pmatrix} 1 & 0 \\ 0 & 1 \end{pmatrix}$

$A = \frac{1}{2}(\Sigma_2^{-1} - \Sigma_1^{-1}) = \begin{pmatrix} 0.375 & 0 \\ 0 & 0 \end{pmatrix}$

$\boldsymbol{b} = \Sigma_1^{-1}\boldsymbol{\mu}_1 - \Sigma_2^{-1}\boldsymbol{\mu}_2 = (0.5, 3)^T - (0, -1)^T = (0.5, 4)^T$

$c = -\frac{1}{2}(2, 3)\begin{pmatrix}0.25&0\\0&1\end{pmatrix}\begin{pmatrix}2\\3\end{pmatrix} + \frac{1}{2}(0,-1)\begin{pmatrix}1&0\\0&1\end{pmatrix}\begin{pmatrix}0\\-1\end{pmatrix} - \frac{1}{2}\ln\frac{|\Sigma_1|}{|\Sigma_2|}$

$= -\frac{1}{2}(1+9) + \frac{1}{2}(1) - \frac{1}{2}\ln 4 = -5 + 0.5 - 0.693 \approx -5.19$

判別結果：

  • $\boldsymbol{x} = (1, 1)^T$：$f = 0.375 + 0.5 + 4 - 5.19 = -0.315 < 0$ → クラス2
  • $\boldsymbol{x} = (3, 2)^T$：$f = 0.375 \times 9 + 0.5 \times 3 + 4 \times 2 - 5.19 = 3.19 > 0$ → クラス1

メモ

  • フィッシャーの判別は分布の仮定不要（群間/群内分散比の最大化のみ）
  • 2次判別は共分散構造が群で異なる場合に有効だが、パラメータ数が多く推定精度が落ちやすい
  • SVMは高次元・小標本に強いが、パラメータ $C$ やカーネルパラメータのチューニングが重要
  • ROC-AUCは不均衡データでも安定した評価指標（正解率はクラス比に依存）
  • 正準判別分析は主成分分析の分類版と考えられる（分散最大化 → 群間/群内分散比最大化）
  • 混同行列の各指標の意味を正確に把握しておくこと（特に感度・特異度・適合率・再現率の違い）
