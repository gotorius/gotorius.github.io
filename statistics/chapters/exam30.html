第30章「モデル選択」のキーポイント
  • 情報量規準（Information Criterion）：尤度（当てはまりのよさ）とパラメータ数（モデルの簡潔さ）のトレードオフをバランスさせてモデルを選ぶための規準。
  • AIC（赤池情報量規準）：$\text{AIC} = -2\log L + 2k$。$k$ はパラメータ数，$L$ は最大尤度。AICを最小にするモデルを選ぶ。大標本近似に基づき導出。
  • BIC（ベイズ情報量規準）：$\text{BIC} = -2\log L + k\log n$。$n$ が大きいとき AIC よりペナルティが大きく，より簡潔なモデルを選ぶ傾向がある。$n \to \infty$ で真のモデルを確率1で選択する一致性をもつ。
  • 重回帰分析における最大尤度：$L = \left(\dfrac{2\pi S_e}{n}\right)^{-n/2}\!\exp\!\left(-\dfrac{n}{2}\right)$，ここで $S_e$ は残差平方和。
  • 重回帰分析でのAIC・BIC：説明変数を $k$ 個選んだときの残差平方和を $S_e^{(k)}$ とすると，
    $\text{AIC}(k) = n\!\left(\log S_e^{(k)} + \log\dfrac{2\pi}{n} + 1\right) + 2(k+2)$，
    $\text{BIC}(k) = n\!\left(\log S_e^{(k)} + \log\dfrac{2\pi}{n} + 1\right) + (k+2)\log n$。
  • 過学習（Overfitting）：パラメータ数が多いモデルほど学習データへの当てはまりはよくなるが，将来データへの予測精度はかえって低下する現象。情報量規準はこれを防ぐための正則化の役割を果たす。
  • クロスバリデーション：データを $K$ 分割し，テスト用と学習用を入れ替えながらモデルの予測精度を評価する方法。Leave-one-out 法は $K=n$ の特殊ケース。

問30.1　最大尤度の式 (30.2) の導出

問題

重回帰モデル $\boldsymbol{y} = X\boldsymbol{\beta} + \boldsymbol{\varepsilon}$， $\boldsymbol{\varepsilon} \sim N_n(\boldsymbol{0},\, \sigma^2 I_n)$ において， 次の式 (30.2) を示せ：

$$L = \left(\frac{2\pi S_e}{n}\right)^{-n/2} \exp\!\left(-\frac{n}{2}\right) \tag{30.2}$$

ただし，$S_e$ は残差平方和 $S_e = \boldsymbol{y}^\top(I_n - X(X^\top X)^{-1}X^\top)\boldsymbol{y}$ である．

ヒント

$\boldsymbol{\beta}$ および $\sigma^2$ の最尤推定量は， $\hat{\boldsymbol{\beta}} = (X^\top X)^{-1}X^\top\boldsymbol{y}$， $\hat{\sigma}^2 = \dfrac{S_e}{n}$ と表される．

解答・解説
準備：正規分布の尤度関数の形

$\boldsymbol{y} \sim N_n(X\boldsymbol{\beta},\, \sigma^2 I_n)$ より， $\boldsymbol{\beta}$ と $\sigma^2$ の尤度関数は次のように書けます：

$$L(\boldsymbol{\beta}, \sigma^2) = (2\pi)^{-n/2}\,|\sigma^2 I_n|^{-1/2} \exp\!\left(-\frac{1}{2}(\boldsymbol{y} - X\boldsymbol{\beta})^\top (\sigma^2 I_n)^{-1}(\boldsymbol{y} - X\boldsymbol{\beta})\right)$$

$|\sigma^2 I_n| = (\sigma^2)^n$ を代入して整理すると：

$$L(\boldsymbol{\beta}, \sigma^2) = (2\pi)^{-n/2}(\sigma^2)^{-n/2} \exp\!\left(-\frac{1}{2\sigma^2}\|\boldsymbol{y} - X\boldsymbol{\beta}\|^2\right)$$

最大尤度 $L$ は，$\boldsymbol{\beta}$ と $\sigma^2$ を最尤推定量に置き換えることで得られます．

Step 1：$\hat{\boldsymbol{\beta}}$ を代入する

残差平方和 $S_e$ の定義

最尤推定量 $\hat{\boldsymbol{\beta}} = (X^\top X)^{-1}X^\top\boldsymbol{y}$ を代入すると， 残差 $\boldsymbol{y} - X\hat{\boldsymbol{\beta}}$ の二乗ノルムが残差平方和 $S_e$ に一致します：

$$\|\boldsymbol{y} - X\hat{\boldsymbol{\beta}}\|^2 = (\boldsymbol{y} - X\hat{\boldsymbol{\beta}})^\top(\boldsymbol{y} - X\hat{\boldsymbol{\beta}}) = S_e$$

（これは $\hat{\boldsymbol{\beta}}$ が残差平方和を最小化する最小二乗推定量であることからも自然にわかります．）

$\hat{\boldsymbol{\beta}}$ を代入した後の尤度

$\hat{\boldsymbol{\beta}}$ を代入すると，尤度は $\sigma^2$ のみの関数になります：

$$L(\sigma^2) = (2\pi)^{-n/2}(\sigma^2)^{-n/2} \exp\!\left(-\frac{S_e}{2\sigma^2}\right)$$

Step 2：$\hat{\sigma}^2 = S_e/n$ を代入する

最尤推定量 $\hat{\sigma}^2$ の代入

$\hat{\sigma}^2 = S_e/n$ を代入します：

$$L = (2\pi)^{-n/2}\left(\frac{S_e}{n}\right)^{-n/2} \exp\!\left(-\frac{S_e}{2\cdot S_e/n}\right)$$

指数の部分を整理します：

$$\frac{S_e}{2 \cdot S_e/n} = \frac{S_e \cdot n}{2 S_e} = \frac{n}{2}$$

Step 3：式 (30.2) の形にまとめる

$(2\pi)^{-n/2}$ と $(S_e/n)^{-n/2}$ を合わせる

$(2\pi)^{-n/2} \cdot (S_e/n)^{-n/2} = \left(2\pi \cdot \frac{S_e}{n}\right)^{-n/2} = \left(\frac{2\pi S_e}{n}\right)^{-n/2}$ なので：

$$\boxed{L = \left(\frac{2\pi S_e}{n}\right)^{-n/2} \exp\!\left(-\frac{n}{2}\right)}$$

これが式 (30.2) です．$\blacksquare$

式 (30.2) から AIC の式を導く

式 (30.2) の対数をとって $-2$ 倍すると， AIC の $-2\log L$ の部分が得られます：

$$-2\log L = -2\left[-\frac{n}{2}\log\frac{2\pi S_e}{n} - \frac{n}{2}\right] = n\log\frac{2\pi S_e}{n} + n$$

$\log\dfrac{2\pi S_e}{n} = \log S_e + \log\dfrac{2\pi}{n}$ と分解すると：

$$-2\log L = n\left(\log S_e + \log\frac{2\pi}{n} + 1\right)$$

重回帰モデルのパラメータ数は，説明変数 $p$ 個・定数項 $\beta_0$・分散 $\sigma^2$ の 合計 $k = p + 2$ 個なので，AIC の式は次のようになります：

$$\text{AIC}(p) = n\left(\log S_e^{(p)} + \log\frac{2\pi}{n} + 1\right) + 2(p+2)$$

解答・解説

Step 1：AIC・BIC の式を確認する

重回帰分析における AIC(k) と BIC(k) の公式

問30.1 の結果より，説明変数を $k$ 個選んだときの残差平方和を $S_e^{(k)}$ とおくと：

$$\text{AIC}(k) = n\left(\log S_e^{(k)} + \log\frac{2\pi}{n} + 1\right) + 2(k+2)$$ $$\text{BIC}(k) = n\left(\log S_e^{(k)} + \log\frac{2\pi}{n} + 1\right) + (k+2)\log n$$

ここで $k$ は説明変数の個数です（定数項 $\beta_0$ や分散 $\sigma^2$ は別にカウントされており，パラメータ総数は $k+2$）．

Step 2：共通部分を先に計算する

$\log(2\pi/n)$ を計算する

$\log\dfrac{2\pi}{n} = \log 2\pi - \log n = 1.838 - \log 20$

$\log 20 = 2.996$ より：

$$\log\frac{2\pi}{n} = 1.838 - 2.996 = -1.158$$

したがって，式の共通部分は：

$$n\!\left(\log S_e^{(k)} + \log\frac{2\pi}{n} + 1\right) = 20\bigl(\log S_e^{(k)} - 1.158 + 1\bigr) = 20\bigl(\log S_e^{(k)} - 0.158\bigr)$$

Step 6：結果の解釈とAIC・BICの違い

なぜ AIC と BIC で選ばれる変数が異なるのか？

AIC は $2(k+2)$，BIC は $(k+2)\log n$ をペナルティとして用います． $n=20$ では $\log 20 = 2.996 \approx 3.0$ なので， 変数を1個増やすたびのペナルティが AIC（$+2$）より BIC（$+2.996$）のほうが大きいです．

そのため，BIC はより少ない変数のモデルを選ぼうとする傾向があります．

第30章全体のまとめ

第30章「モデル選択」の主要な流れ：

  1. 情報量規準の必要性：変数を増やすほど残差平方和は必ず減少するが，過学習が生じる。尤度の大きさとパラメータ数のバランスをとる情報量規準でモデルを選ぶ。
  2. AIC = $-2\log L + 2k$：重回帰での $L$ は式 (30.2) で与えられる。$\text{AIC}(k) = n(\log S_e^{(k)} + \log(2\pi/n) + 1) + 2(k+2)$ を最小化する変数の組合せを選ぶ。
  3. BIC = $-2\log L + k\log n$：$n \geq 8$ で AIC より大きいペナルティを課す。$n\to\infty$ で一致性をもつ（真のモデルを確率1で選択）。
  4. AIC と F 検定の関係：$n$ が大きいとき，AIC は F 統計量の棄却限界値を 2 とした変数選択に対応する。
  5. クロスバリデーション：データを $K$ 分割してテストデータでモデルの予測精度を評価する。Leave-one-out 法（$K=n$）は特別な場合。計算量は大きくなるが，過学習の検出に有効。
