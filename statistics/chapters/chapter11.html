正規分布に関する検定――平均・分散を厳密に比較する

第10章で学んだ仮説検定の枠組みを、母集団分布が正規分布であるという仮定のもとで具体的に展開します。 正規分布の場合、有限標本でも厳密に（近似ではなく）分布が導出できることが最大の利点です。

正規母集団の検定問題は「群の数（1標本 or 2標本）」と「分散が既知か未知か」で場合分けされます。

🔑 この章の構成

  1. 1標本平均の検定――$\sigma^2$ 既知 →$Z$ 検定、$\sigma^2$ 未知 → $t$ 検定
  2. 2標本平均の差の検定――$\sigma^2$ 既知・等分散未知・非等分散
  3. 1標本・2標本の分散の検定――$\chi^2$ 検定・$F$ 検定

1. 1標本平均の検定（母分散 $\sigma^2$ が既知）

$X_1, \ldots, X_n \overset{\text{i.i.d.}}{\sim} N(\mu, \sigma^2)$（$\sigma^2$ は既知）。 帰無仮説 $H_0: \mu = \mu_0$ のもとで検定統計量を構成します。

$Z$ 検定（分散既知）

$H_0: \mu = \mu_0$ のもとで

$$Z = \frac{\bar{X} - \mu_0}{\sigma/\sqrt{n}} \sim N(0,1)$$
対立仮説 棄却域 P値
$H_1: \mu > \mu_0$$Z \geq z_{\alpha/2}$$P(Z \geq z_0)$
$H_1: \mu < \mu_0$$Z \leq -z_{\alpha/2}$$P(Z \leq z_0)$
$H_1: \mu \neq \mu_0$$|Z| \geq z_{\alpha/2}$$2P(Z \geq |z_0|)$

2. 1標本平均の検定（分散未知）――$t$ 検定

$\sigma^2$ が未知の場合、$\sigma$ の代わりに標本標準偏差 $S = \sqrt{\frac{1}{n-1}\sum(X_i-\bar{X})^2}$ を使います。

1標本 $t$ 検定

$H_0: \mu = \mu_0$ のもとで

$$T = \frac{\bar{X} - \mu_0}{S/\sqrt{n}} \sim t(n-1)$$
対立仮説 棄却域（有意水準 $\alpha$）
$H_1: \mu > \mu_0$$T \geq t_{\alpha/2}(n-1)$
$H_1: \mu < \mu_0$$T \leq -t_{\alpha/2}(n-1)$
$H_1: \mu \neq \mu_0$$|T| \geq t_{\alpha/2}(n-1)$

$t_{\alpha/2}(n-1)$：自由度 $n-1$ の $t$ 分布の上側 $\alpha/2$ 点。

💡 なぜ自由度が $n-1$ か

$S^2 = \frac{1}{n-1}\sum(X_i - \bar{X})^2$ の計算で $\bar{X}$ を推定に使ったため、 $n$ 個の偏差 $X_i - \bar{X}$ は $\sum(X_i - \bar{X}) = 0$ という制約により 1自由度が消費されます。残りの自由度は $n-1$ です。

3. 2標本平均の検定（分散既知）

群A（$n_A$ 個、$N(\mu_A, \sigma_A^2)$）と群B（$n_B$ 個、$N(\mu_B, \sigma_B^2)$）、 $\sigma_A^2$、$\sigma_B^2$ は既知。差 $\delta = \mu_A - \mu_B$ を検定します。

2標本 $Z$ 検定（分散既知）

$H_0: \delta = \delta_0$ のもとで

$$Z = \frac{(\bar{X}_A - \bar{X}_B) - \delta_0}{\sqrt{\sigma_A^2/n_A + \sigma_B^2/n_B}} \sim N(0,1)$$

特に $\sigma_A = \sigma_B = \sigma$ の等分散の場合：$\sqrt{\sigma^2(1/n_A + 1/n_B)}$ が分母。

対応のある2標本

「同じ被験者に2回測定（例：治療前・治療後）」のように対応がある場合は、 差 $D_i = X_{Ai} - X_{Bi}$ を計算して1標本問題に帰着させます。

$$T = \frac{\bar{D} - \delta_0}{S_D/\sqrt{n}} \sim t(n-1) \quad \text{（$\sigma^2$ 未知の場合）}$$

4. 2標本平均の検定（分散未知）――2標本 $t$ 検定

$\sigma_A^2$、$\sigma_B^2$ が未知の場合。等分散 $\sigma_A^2 = \sigma_B^2 = \sigma^2$ を仮定するかどうかで場合分けされます。

等分散の2標本 $t$ 検定（プールした分散推定量）

$\sigma_A^2 = \sigma_B^2 = \sigma^2$ を仮定（等分散仮定）。プールした標本分散：

$$S_p^2 = \frac{(n_A-1)S_A^2 + (n_B-1)S_B^2}{n_A + n_B - 2}$$

$H_0: \delta = \delta_0$ のもとで：

$$T = \frac{(\bar{X}_A - \bar{X}_B) - \delta_0}{S_p\sqrt{1/n_A + 1/n_B}} \sim t(n_A + n_B - 2)$$
💡 ベーレンス・フィッシャー問題（等分散でない場合）

$\sigma_A^2 \neq \sigma_B^2$（非等分散）の場合、厳密な $t$ 分布に従う統計量は存在しません（ベーレンス・フィッシャー問題）。 実用的な近似としてウェルチの $t$ 検定があり、自由度をサターウェイト法で近似します：

$$T_W = \frac{(\bar{X}_A - \bar{X}_B) - \delta_0}{\sqrt{S_A^2/n_A + S_B^2/n_B}}, \quad \hat{\nu} = \frac{(S_A^2/n_A + S_B^2/n_B)^2}{\frac{(S_A^2/n_A)^2}{n_A-1} + \frac{(S_B^2/n_B)^2}{n_B-1}}$$

$T_W$ は近似的に自由度 $\hat{\nu}$ の $t$ 分布に従います。

5. 分散の検定

1標本の分散の検定（$\chi^2$ 検定）

$\mu$ は未知。$H_0: \sigma^2 = \sigma_0^2$ に対し、

カイ二乗検定（分散の検定）
$$V = \frac{(n-1)S^2}{\sigma_0^2} \sim \chi^2(n-1)$$
対立仮説 棄却域
$H_1: \sigma^2 > \sigma_0^2$$V \geq \chi^2_{\alpha/2}(n-1)$
$H_1: \sigma^2 < \sigma_0^2$$V \leq \chi^2_{1-\alpha/2}(n-1)$
$H_1: \sigma^2 \neq \sigma_0^2$$V \geq \chi^2_{\alpha/2}$ または $V \leq \chi^2_{1-\alpha/2}$

2標本の分散の比の検定（$F$ 検定）

$\mu_A$、$\mu_B$ は未知。$H_0: \sigma_A^2 = \sigma_B^2$ に対し、

$F$ 検定（2標本分散比の検定）
$$F = \frac{S_B^2}{S_A^2} = \frac{\sum(X_{Bi}-\bar{X}_B)^2/(n_B-1)}{\sum(X_{Ai}-\bar{X}_A)^2/(n_A-1)} \sim F(n_B-1,\; n_A-1)$$
  • $H_1: \sigma_A^2 < \sigma_B^2$：$F \geq F_{\alpha/2}(n_B-1, n_A-1)$ で棄却
  • $H_1: \sigma_A^2 > \sigma_B^2$：$F \leq F_{1-\alpha/2}(n_B-1, n_A-1)$ で棄却

（$F_{1-\alpha/2}(m,n) = 1/F_{\alpha/2}(n,m)$ の関係を利用）

6. 検定統計量一覧

検定 統計量 帰無分布
1標本平均（$\sigma^2$ 既知）$Z = \dfrac{\bar{X}-\mu_0}{\sigma/\sqrt{n}}$$N(0,1)$
1標本平均（$\sigma^2$ 未知）$T = \dfrac{\bar{X}-\mu_0}{S/\sqrt{n}}$$t(n-1)$
2標本平均（$\sigma^2$ 既知）$Z = \dfrac{(\bar{X}_A-\bar{X}_B)-\delta_0}{\sqrt{\sigma_A^2/n_A+\sigma_B^2/n_B}}$$N(0,1)$
2標本平均（等分散未知）$T = \dfrac{(\bar{X}_A-\bar{X}_B)-\delta_0}{S_p\sqrt{1/n_A+1/n_B}}$$t(n_A+n_B-2)$
1標本分散$V = \dfrac{(n-1)S^2}{\sigma_0^2}$$\chi^2(n-1)$
2標本分散比$F = S_B^2/S_A^2$$F(n_B-1,n_A-1)$

7. 例題・演習

問11.1：ポテトの重量に関する1標本 $t$ 検定

問題

あるファストフードチェーンの大阪店で販売される Mサイズのポテトは、 仕様上「平均120g以上」とされている。マネージャーが「実際はもっと多い」と考えて検証することにした。 無作為に10個抽出して重さを測定したところ、標本平均 $\bar{x} = 125$g、標本標準偏差 $s = 10.0$g であった。 帰無仮説 $H_0: \mu = 120$ に対して対立仮説 $H_1: \mu > 120$ の右片側 $t$ 検定を有意水準 2.5% で行え。 （自由度9の $t$ 分布の上側2.5%点：$t_{0.025}(9) = 2.262$）

解答

(1) 検定統計量の計算：

$$T = \frac{\bar{x} - \mu_0}{s/\sqrt{n}} = \frac{125 - 120}{10/\sqrt{10}} = \frac{5}{3.162} \approx \mathbf{1.581}$$

(2) 棄却限界値と判定：

右片側検定で有意水準 2.5% → 棄却域：$T \geq t_{0.025}(9) = 2.262$

$T = 1.581 < 2.262$ → 帰無仮説は棄却されない。

「$n=10$ の標本ではポテトが120gより重いとは統計的に言えない。」

(3) 最小標本サイズ（棄却されるために）：

検定統計量 $T = \frac{5}{10/\sqrt{n}} = \frac{\sqrt{n}}{2}$ が $t_{0.025}(n-1)$ 以上になる $n$ を求める。 $t_{0.025}$ は $n$ が大きいほど小さくなるので、試行する：

  • $n=17$：$T = \sqrt{17}/2 = 2.06$、$t_{0.025}(16) = 2.12$ → まだ棄却されない
  • $n=18$：$T = \sqrt{18}/2 = 2.12$、$t_{0.025}(17) = 2.11$ → $T > t_{0.025}$、棄却される

→ 最小は $n = \mathbf{18}$。

問11.2：東京と大阪のポテト重量の比較――2標本 $t$ 検定

問題

同じチェーンの東京店と大阪店で、それぞれ独立に Mサイズポテトを10個ずつ抽出して重さを測定した。 結果：大阪店 $\bar{x}_A = 125$g、$s_A = 10.0$g（$n_A = 10$）； 東京店 $\bar{x}_B = 115$g、$s_B = 8.0$g（$n_B = 10$）。 等分散を仮定した2標本 $t$ 検定で、2店の平均重量に差があるかを有意水準5%で両側検定せよ。 （自由度18の $t$ 分布の上側2.5%点：$t_{0.025}(18) = 2.101$）

解答

(1) プールした標本分散：

$$S_p^2 = \frac{(n_A-1)s_A^2 + (n_B-1)s_B^2}{n_A+n_B-2} = \frac{9 \times 100 + 9 \times 64}{18} = \frac{900 + 576}{18} = \frac{1476}{18} = \mathbf{82}$$

(2) 検定統計量：

$$T = \frac{(\bar{x}_A - \bar{x}_B) - 0}{S_p\sqrt{1/n_A + 1/n_B}} = \frac{125 - 115}{\sqrt{82}\sqrt{1/10 + 1/10}} = \frac{10}{\sqrt{82 \times 0.2}} = \frac{10}{\sqrt{16.4}} = \frac{10}{4.050} \approx \mathbf{2.469}$$

(3) 棄却の判定：

両側検定で自由度 $n_A + n_B - 2 = 18$ → 棄却域 $|T| \geq t_{0.025}(18) = 2.101$

$|T| = 2.469 > 2.101$ → 帰無仮説を棄却する。

「大阪店と東京店のポテトの平均重量には統計的に有意な差がある（95%水準）。」

問11.3（オリジナル）：新工程の品質ばらつき検定

問題

従来工程の製品の重さのばらつきは $\sigma_0^2 = 9$ (g²) とわかっている。 新しい工程を導入したところ、25個の製品を検査した結果 $(n-1)s^2 = 168$ (g²) だった。 新工程の分散は従来工程と異なるかを有意水準5%の両側検定で確認せよ。 （自由度24の $\chi^2$ の上側2.5%点 = 39.36、下側2.5%点 = 12.40）

解答

検定統計量：

$$V = \frac{(n-1)S^2}{\sigma_0^2} = \frac{168}{9} = \mathbf{18.67}$$

棄却域：$V \geq 39.36$ または $V \leq 12.40$（両側、有意水準5%）

$12.40 < 18.67 < 39.36$ → 帰無仮説は棄却されない。

「この標本では新工程の分散が従来と異なるとは言えない。」

8. まとめ・ポイント

  • $\sigma^2$ 既知 → $Z$ 検定（$N(0,1)$）、未知 → $t$ 検定（$t(n-1)$）：$\sigma$ の代わりに $S$ を使ったことの「不確かさ増加」が $t$ 分布の裾の重さとして現れる。
  • 2標本 $t$ 検定では等分散の仮定に注意：仮定が満たされない場合はウェルチの $t$ 検定（自由度をサターウェイト近似）を使う。
  • 対応のある2標本は1標本問題に帰着：差 $D_i = X_{Ai} - X_{Bi}$ を使って $t$ 検定を行う。
  • 分散の検定は $\chi^2$ 分布、分散比の検定は $F$ 分布：どちらも非対称分布なので棄却域の定め方に注意。
  • $F$ 検定の統計量は $S_B^2/S_A^2$：自由度の順番を間違えないこと（分子の自由度が第1引数）。
  • サンプルサイズと検出力：問11.1のように、同じ効果でも $n$ が小さいと検出できないことがある。十分な $n$ を事前設計することが重要。
