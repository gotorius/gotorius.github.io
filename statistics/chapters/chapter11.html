概要

第11章では、母集団分布が正規分布であると仮定したうえでの平均の検定と分散の検定を体系的に扱う。 1標本問題（分散既知・未知）の検定、2標本問題（分散既知・未知）の検定、 1標本・2標本の分散の検定について、検定統計量・棄却域・P-値を学ぶ。

キーワード：平均の検定、分散の検定、$t$ 分布、$\chi^2$ 分布、$F$ 分布

重要概念

1標本の平均の検定（分散が既知）

$X_1, \ldots, X_n \overset{i.i.d.}{\sim} N(\mu, \sigma^2)$、$\sigma^2$ は既知。

$H_0: \mu = \mu_0$ のもとで、検定統計量

$$Z = \frac{\bar{X} - \mu_0}{\sigma / \sqrt{n}} \sim N(0,1)$$
  • $H_1: \mu > \mu_0$（右側）：$Z \geq z_{\alpha/2}$ で棄却
  • $H_1: \mu < \mu_0$（左側）：$Z \leq -z_{\alpha/2}$ で棄却
  • $H_1: \mu \neq \mu_0$（両側）：$|Z| \geq z_{\alpha/2}$ で棄却

1標本の平均の検定（分散が未知）― $t$ 検定

$\sigma^2$ は未知。標本分散 $S^2 = \dfrac{1}{n-1}\sum_{i=1}^{n}(X_i - \bar{X})^2$ で推定。

$H_0: \mu = \mu_0$ のもとで、検定統計量

$$T = \frac{\bar{X} - \mu_0}{S / \sqrt{n}} \sim t(n-1)$$
  • $H_1: \mu > \mu_0$：$T \geq t_{\alpha/2}(n-1)$ で棄却
  • $H_1: \mu < \mu_0$：$T \leq -t_{\alpha/2}(n-1)$ で棄却
  • $H_1: \mu \neq \mu_0$：$|T| \geq t_{\alpha/2}(n-1)$ で棄却

2標本の平均の検定（分散が既知）

群A：$X_{Ai} \sim N(\mu_A, \sigma_A^2)$（$n_A$ 個）、群B：$X_{Bi} \sim N(\mu_B, \sigma_B^2)$（$n_B$ 個）。$\delta = \mu_A - \mu_B$ とする。

$H_0: \delta = \delta_0$ のもとで、検定統計量

$$Z = \frac{(\bar{X}_A - \bar{X}_B) - \delta_0}{\sqrt{\sigma_A^2/n_A + \sigma_B^2/n_B}} \sim N(0,1)$$

$\sigma_A = \sigma_B = \sigma$ の場合：

$$Z = \frac{(\bar{X}_A - \bar{X}_B) - \delta_0}{\sigma\sqrt{1/n_A + 1/n_B}}$$

2標本の平均の検定（分散が未知）― 2標本 $t$ 検定

共通の母分散 $\sigma_A^2 = \sigma_B^2 = \sigma^2$ を仮定。プールした標本分散：

$$S_p^2 = \frac{(n_A - 1)S_A^2 + (n_B - 1)S_B^2}{n_A + n_B - 2}$$

$H_0: \delta = \delta_0$ のもとで、検定統計量

$$T = \frac{(\bar{X}_A - \bar{X}_B) - \delta_0}{S_p\sqrt{1/n_A + 1/n_B}} \sim t(n_A + n_B - 2)$$
  • $H_1: \delta > \delta_0$：$T \geq t_{\alpha/2}(n_A + n_B - 2)$ で棄却
  • $H_1: \delta < \delta_0$：$T \leq -t_{\alpha/2}(n_A + n_B - 2)$ で棄却
  • $H_1: \delta \neq \delta_0$：$|T| \geq t_{\alpha/2}(n_A + n_B - 2)$ で棄却

※ 2群の分散が異なる場合はベーレンス・フィッシャー問題が生じる。

1標本の分散の検定

$\mu$ は未知。$H_0: \sigma^2 = \sigma_0^2$ のもとで、検定統計量

$$V = \frac{(n-1)S^2}{\sigma_0^2} \sim \chi^2(n-1)$$
  • $H_1: \sigma^2 > \sigma_0^2$：$V \geq \chi^2_{\alpha/2}(n-1)$ で棄却
  • $H_1: \sigma^2 < \sigma_0^2$：$V \leq \chi^2_{1-\alpha/2}(n-1)$ で棄却
  • $H_1: \sigma^2 \neq \sigma_0^2$：$V \geq \chi^2_{\alpha/2}(n-1)$ または $V \leq \chi^2_{1-\alpha/2}(n-1)$ で棄却

2標本の分散の検定（$F$ 検定）

$\mu_A, \mu_B$ は未知。$H_0: \sigma_A^2 = \sigma_B^2$ のもとで、検定統計量

$$F = \frac{S_B^2}{S_A^2} = \frac{\sum(X_{Bi} - \bar{X}_B)^2/(n_B - 1)}{\sum(X_{Ai} - \bar{X}_A)^2/(n_A - 1)} \sim F(n_B - 1, n_A - 1)$$
  • $H_1: \sigma_A^2 < \sigma_B^2$：$F \geq F_{\alpha/2}(n_B-1, n_A-1)$ で棄却
  • $H_1: \sigma_A^2 > \sigma_B^2$：$F \leq F_{1-\alpha/2}(n_B-1, n_A-1)$ で棄却
  • $H_1: \sigma_A^2 \neq \sigma_B^2$：上側または下側で棄却

公式・定理

検定統計量一覧

検定 統計量 帰無分布
1標本平均（$\sigma^2$ 既知） $Z = \dfrac{\bar{X}-\mu_0}{\sigma/\sqrt{n}}$ $N(0,1)$
1標本平均（$\sigma^2$ 未知） $T = \dfrac{\bar{X}-\mu_0}{S/\sqrt{n}}$ $t(n-1)$
2標本平均（$\sigma^2$ 未知、等分散） $T = \dfrac{(\bar{X}_A - \bar{X}_B)-\delta_0}{S_p\sqrt{1/n_A+1/n_B}}$ $t(n_A+n_B-2)$
1標本分散 $V = \dfrac{(n-1)S^2}{\sigma_0^2}$ $\chi^2(n-1)$
2標本分散 $F = S_B^2 / S_A^2$ $F(n_B-1, n_A-1)$

例題・演習

問11.1：ポテトの重量検定（1標本 $t$ 検定）

大阪の店舗のポテト（$M$ サイズ120g）を10個測定。$\bar{x} = 125$g、$s = 10.0$g。

$H_0: \mu = 120$ vs $H_1: \mu > 120$（片側 $t$ 検定、有意水準2.5%）

(1) 検定統計量：

$$T = \frac{125 - 120}{10/\sqrt{10}} = \mathbf{1.581}$$

(2) 棄却限界値：

自由度9の $t$ 分布の上側2.5%点：$t_{0.025}(9) = \mathbf{2.262}$

(3) 判定：

$T = 1.581 < 2.262$ より、帰無仮説は棄却されない。

(4) 棄却される最小の標本サイズ：

$$\frac{125 - 120}{10/\sqrt{n}} \geq t_{0.025}(n-1)$$

反復的に解くと $n = \mathbf{18}$

問11.2：東京 vs 大阪の2標本 $t$ 検定

大阪：$n_A = 10$、$\bar{x}_A = 125$、$s_A = 10.0$。東京：$n_B = 10$、$\bar{x}_B = 115$、$s_B = 8.0$。

共通分散を仮定。$H_0: \mu_A = \mu_B$ vs $H_1: \mu_A \neq \mu_B$（両側、有意水準5%）

(1) プールした標本分散：

$$S_p^2 = \frac{9 \times 10^2 + 9 \times 8^2}{10 + 10 - 2} = \frac{900 + 576}{18} = \mathbf{82}$$

(2) 検定統計量：

$$T = \frac{125 - 115}{\sqrt{82}\sqrt{1/10 + 1/10}} = \frac{10}{\sqrt{82 \times 0.2}} = \mathbf{2.469}$$

(3) 棄却限界値：

自由度18の $t$ 分布の上側2.5%点：$t_{0.025}(18) = \mathbf{2.101}$

(4) 判定：

$|T| = 2.469 > 2.101$ より、帰無仮説は棄却される。

メモ

  • 分散が既知なら $Z$ 検定（標準正規分布）、未知なら $t$ 検定（$t$ 分布）を使う
  • 2標本で対応がある場合は差を取って1標本問題に帰着できる
  • 2標本 $t$ 検定では等分散の仮定が必要。等分散でない場合はベーレンス・フィッシャー問題（ウェルチの $t$ 検定などで近似）
  • $F$ 検定の統計量は分散の比 $S_B^2/S_A^2$ であり、$F$ 分布の自由度は $(n_B-1, n_A-1)$
  • 問11.1 では $n=10$ で棄却されなかったが、$n=18$ まで増やすと棄却される → 標本サイズの重要性
