概要

多変量解析では、すべての変数を用いるのではなく、何らかの規準に基づき適切な変数やモデルを選択したい場合がある。本章では、情報量規準（AIC、BIC）、クロスバリデーション、過学習の問題について学ぶ。

仮説検定（F検定）による変数選択とは異なり、尤度とパラメータ数に基づいた情報量規準によるモデル選択法が提案されている。データへの当てはまりのよさとモデルの簡潔さのバランスをとることが重要である。

重要概念

赤池情報量規準（AIC）

最大尤度 $L$、推定するパラメータ数 $k$ として：

$$\mathrm{AIC} = -2\log L + 2k$$

AICはカルバック・ライブラー情報量の期待値の漸近的な不偏推定量、あるいは期待平均対数尤度の漸近的な不偏推定量の観点で導かれたもの。AICが最小のモデルを選択する。導出には大標本の仮定を用いている。

BIC（ベイズ情報量規準）

$$\mathrm{BIC} = -2\log L + k\log n$$

AICと比較してペナルティが $2k$ から $k\log n$ に変化。$n \ge 8$ のとき BIC の方がより簡潔なモデルを選ぶ傾向がある。$n \to \infty$ のとき確率1で真のモデルを選択する（AICはそうとは限らない）。

その他の情報量規準

  • c-AIC：AICの有限修正。標本が小さいときも期待平均対数尤度の不偏推定量を与える
  • MDL（最小記述長）：minimum description length

クロスバリデーション（交差検証法）

データを $K$ 分割し、1セットをテスト用、$K-1$ セットで学習を行い、テスト用データで精度を評価。テスト用の選択を $K$ 通りすべて試して平均をとる。

  • $K$ 分割交差検証：データを $K$ グループに分けて繰り返す
  • Leave-one-out法：$K = n$ とした場合。1つだけ省いてテスト用とし、残り全てで学習

変数選択・モデル選択の規準としても用いられるが、計算量が大きくなることに注意。

過学習（Overfitting）

複雑なモデルほど学習データへの適合はよくなるが、将来のデータに対する予測精度はかえって落ちる現象。パラメータ数が大きすぎるモデルやデータ数に近いパラメータ数のモデルの使用を避け、当てはまりのよさと簡潔さのバランスをとることが重要。

関連：正則化によるモデル選択（16章参照）

公式・定理

重回帰分析におけるAIC

重回帰モデル $y = X\beta + \varepsilon$（$\varepsilon \sim N_n(0, \sigma^2 I_n)$）で、$\beta$ は $(p+1) \times 1$、$X$ は $n \times (p+1)$ とする。

最大尤度：

$$L = \left(\frac{2\pi S_e}{n}\right)^{-n/2} \exp\!\left(-\frac{n}{2}\right) \tag{30.2}$$

ただし $S_e$ は残差平方和 $S_e = y^T(I_n - X(X^TX)^{-1}X^T)y$。

推定パラメータ数は $\beta$ と $\sigma^2$ を合わせて $k + 2$ 個：

$$\mathrm{AIC}(k) = n\!\left(\log S_e^{(k)} + \log\frac{2\pi}{n} + 1\right) + 2(k+2)$$

選択する説明変数によって変化しない項を省くと $\mathrm{AIC}' = n\log S_e + 2p$ とも書ける。

AICとF検定統計量の関係

$\mathrm{AIC}(k) > \mathrm{AIC}(k+1)$ の条件は、$n$ が大きいとき、F検定統計量 $F > 2$ を基準とした変数選択とほぼ同じ結果を与える。すなわち：

$$F = \frac{S_e^{(k)} - S_e^{(k+1)}}{S_e^{(k+1)} / (n - k - 2)}$$

の棄却限界値が $(\exp(2/n) - 1)(n - k - 2) \to 2$（$n \to \infty$）となることから確かめられる。

重回帰分析におけるBIC

$$\mathrm{BIC}(k) = n\!\left(\log S_e^{(k)} + \log\frac{2\pi}{n} + 1\right) + (k+2)\log n$$

例題・演習

問30.1：最大尤度の導出

式(30.2)を示せ。

ヒント：$\hat{\beta} = (X^TX)^{-1}X^Ty$, $\hat{\sigma}^2 = S_e / n$

解答：$y \sim N_n(X\beta, \sigma^2 I_n)$ より最大尤度は

$$L = (2\pi)^{-n/2}(\hat{\sigma}^2)^{-n/2}\exp\!\left(-\frac{1}{2\hat{\sigma}^2}(y - X\hat{\beta})^T(y - X\hat{\beta})\right)$$

$\hat{\sigma}^2 = S_e/n$ および $(y - X\hat{\beta})^T(y - X\hat{\beta}) = S_e$ を代入すると

$$L = (2\pi)^{-n/2}\left(\frac{S_e}{n}\right)^{-n/2}\exp\!\left(-\frac{n}{2}\right) = \left(\frac{2\pi S_e}{n}\right)^{-n/2}\exp\!\left(-\frac{n}{2}\right)$$

問30.2：AICとBICによる変数選択

$n = 20$ のデータで、$y$ を目的変数、$x_1, x_2, x_3$ を説明変数の候補として回帰分析。

説明変数$k$$S_e$$\log S_e$
なし05318010.881
$x_1$14149910.633
$x_2$13033810.320
$x_3$14042010.607
$x_1, x_2$22972710.300
$x_1, x_3$22625210.176
$x_2, x_3$22692310.201
$x_1, x_2, x_3$32389310.081

$\log 2\pi = 1.838$, $\log 20 = 2.996$ として計算する。

解答：各組合せについて AIC$(k) = n(\log S_e^{(k)} + \log(2\pi/n) + 1) + 2(k+2)$ と BIC$(k) = n(\log S_e^{(k)} + \log(2\pi/n) + 1) + (k+2)\log n$ を計算：

  • AICによる変数選択：$\{x_1, x_3\}$ が選ばれる
  • BICによる変数選択：$\{x_2\}$ が選ばれる

BICの方がペナルティが大きいため、より簡潔なモデルが選ばれている。

メモ

  • AIC = $-2\log L + 2k$：当てはまりのよさ（尤度）とモデルの簡潔さ（パラメータ数の少なさ）のバランス
  • BIC = $-2\log L + k\log n$：$n \ge 8$ でAICより簡潔なモデルを選ぶ。$n \to \infty$ で確率1で真のモデルを選択
  • 重回帰のAIC：$n$ が大きいとき F > 2 の変数選択と近い結果
  • クロスバリデーション：学習データとテストデータの分離により過適合を検出。Leave-one-out（$K = n$）が特殊ケース
  • 過学習を避けるには、パラメータ数が大きすぎるモデルを避け、情報量規準などでバランスをとる
  • 問30.2の結果：AICは $\{x_1, x_3\}$、BICは $\{x_2\}$ → BICの方が簡潔なモデルを好む典型例
