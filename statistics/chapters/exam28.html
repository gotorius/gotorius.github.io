第28章「分割表」のキーポイント
  • 分割表（分割表）： 複数の質的変量の組合せごとの頻度を表にしたもの。$2\times 2$ 分割表が基本。
  • オッズ比 $\psi$： 2つの二項確率 $\theta_1, \theta_2$ のオッズの比。$\psi = \dfrac{\theta_1/(1-\theta_1)}{\theta_2/(1-\theta_2)}$。$H_0:\psi=1$ は独立性（一様性）と同値。
  • 前向き研究・後向き研究： 後向き（症例対照）研究では有病率の差や比は推定できないが、オッズ比は前向き研究のオッズ比と一致する（ベイズの定理より）。希少事象では $\psi \approx \theta_1/\theta_2$（相対リスクの近似）。
  • 適合度カイ二乗検定 $\chi^2$： 帰無仮説下の当てはめ値 $m_{ij} = x_{i\cdot}x_{\cdot j}/x_{\cdot\cdot}$ を用いる。$2\times 2$ では $\chi^2 = x_{\cdot\cdot}(x_{11}x_{22}-x_{12}x_{21})^2/(x_{1\cdot}x_{2\cdot}x_{\cdot 1}x_{\cdot 2})$。
  • 逸脱度 $G^2$： 尤度比検定統計量。$G^2 = 2\sum_{ij} x_{ij}\log(x_{ij}/m_{ij})$。自由度1のカイ二乗分布に漸近。
  • 対数オッズ比の信頼区間： 標本対数オッズ比 $\log(\widehat{OR})$ の標準誤差は $\sqrt{1/x_{11}+1/x_{12}+1/x_{21}+1/x_{22}}$。
  • グラフィカルモデル： 無向独立グラフの性質から条件付き独立性を読み取る。辺がない $\Leftrightarrow$ 他のすべての因子を条件付けて独立。分離する頂点集合を条件付ければ独立。

問28.1　症例対照研究と確率の解釈

問題

ある産婦人科では，出産における低出生体重児の原因として母親の喫煙習慣の影響を調べるために，以下のような調査を行った． まず，この産婦人科で過去に低出生体重児を出産した母親 12 名について，出産時の喫煙習慣の有無を調査した． 次に，過去にこの産婦人科で通常の出産を行った母親のなかから，低出生体重児を出産した母親 12 名のそれぞれと， 年齢・体重が比較的近い母親を 3 名ずつ選んだ．選ばれた 36 名をコントロール群とよび， コントロール群に対しても出産時の喫煙習慣の有無を調査した．調査結果をまとめたのが次の表である（数値は架空）．

低体重児コントロール計
喫煙習慣あり4610
喫煙習慣なし83038
計123648

この調査から読み取れる，出産における低体重児の発生の確率の解釈について， 次の①〜③のうちから最も適切なものを1つ選べ．

① 喫煙習慣がある母親に関する低出生体重児の割合は $4/10 = 0.40$ であり， 喫煙習慣のない母親に関する低出生体重児の割合は $8/38 = 0.21$ である． これらはそれぞれ，喫煙習慣がある母親とない母親のそれぞれに対する，低体重児の出生率の妥当な推定値である．
② 喫煙習慣がある母親に関する低出生体重児の割合は $4/10 = 0.40$ であり， 喫煙習慣のない母親に関する低出生体重児の割合は $8/38 = 0.21$ であり，その差はおよそ $0.19$ である． このことから，喫煙習慣があると，喫煙習慣がない場合に比べて，低体重児の出生率が 19 % 増えると推定できる．
③ 低体重児を出産した母親に関する喫煙歴ありのオッズ $(4/8 = 0.5)$ は， コントロール群に関する喫煙歴ありのオッズ $(6/30 = 0.2)$ の 2.5 倍である． 低体重児の出生率は小さい値であると知られているので， 喫煙習慣がある母親とない母親のそれぞれに対する低体重児の出生率の比（相対リスク）は， およそ 2.5 であると推定できる．
解答・解説
研究デザインの確認：後向き研究（症例対照研究）

この調査は，低体重児を出産した母親（症例群）と通常の出産をした母親（コントロール群）の 2 グループをあらかじめ用意し，そこから過去の喫煙歴を調べる研究です． このような研究デザインを後向き研究（retrospective study）， あるいは症例対照研究（case-control study）といいます．

後向き研究では，コントロール群の人数（36 名）は研究者が設定した固定値です． 実際の低体重児の発生率を反映したものではありません． したがって，行ごとの構成比（割合）には意味がなく，罹患率・発生率の差を推定することはできません．

① の評価

① 誤り

低体重児を出産した母親とコントロール群の標本サイズの比 $1:3$ は，研究デザインで定めたものです． したがって，「喫煙習慣あり 10 名のうち低体重児 4 名」という行ごとの比は， 喫煙習慣がある母親全体のなかでの低体重児の割合を反映していません． 後向き研究では，各行の合計（10, 38）はランダムに決まったものではなく，列の合計（12, 36）が固定されています．

② の評価

② 誤り

① と同じ理由で誤りです．後向き研究では，曝露要因（喫煙習慣）ごとの罹患率や， その差（19%増加）を推定することはできません． 割合の差 $0.40 - 0.21 = 0.19$ は研究デザインによって変化する値であり，意味をもちません．

③ の評価（正解）

③ 正しい

後向き研究において，オッズ比は前向き研究のオッズ比と一致するという重要な性質があります（ベイズの定理から導かれます）．

標本オッズ比の計算

低体重児群における「喫煙あり」のオッズ：

$$\text{オッズ}_{\text{低体重児}} = \frac{4}{8} = 0.5$$

コントロール群における「喫煙あり」のオッズ：

$$\text{オッズ}_{\text{コントロール}} = \frac{6}{30} = 0.2$$

したがって，標本オッズ比は

$$\widehat{OR} = \frac{4/8}{6/30} = \frac{0.5}{0.2} = 2.5$$

後向き研究のオッズ比 ＝ 前向き研究のオッズ比（ベイズの定理）

後向き研究での母数 $\pi_j = P(\text{喫煙あり} \mid B_j)$（$j=1$：低体重児群，$j=2$：コントロール群）について， ベイズの定理を用いると：

$$\frac{\pi_1/(1-\pi_1)}{\pi_2/(1-\pi_2)} = \frac{P(\text{低体重児}\mid\text{喫煙あり})/P(\text{低体重児}\mid\text{喫煙なし})}{P(\text{コントロール}\mid\text{喫煙あり})/P(\text{コントロール}\mid\text{喫煙なし})} = \frac{\theta_1/(1-\theta_1)}{\theta_2/(1-\theta_2)}$$

つまり，後向き研究で求めたオッズ比は，前向き研究のオッズ比 $\psi = \theta_1(1-\theta_2)/(\theta_2(1-\theta_1))$ に等しくなります．

希少事象における相対リスクの近似

低体重児の出生率 $\theta_1, \theta_2$ がいずれも非常に小さい（希少事象）とき， $1-\theta_1 \approx 1$，$1-\theta_2 \approx 1$ と近似できるので：

$$\psi = \frac{\theta_1/(1-\theta_1)}{\theta_2/(1-\theta_2)} \approx \frac{\theta_1}{\theta_2} = \text{相対リスク}$$

したがって，標本オッズ比 2.5 は，喫煙習慣がある母親の低体重児の出生率が， 喫煙習慣がない母親のそれの約 2.5 倍であるという相対リスクの推定値として使えます．

まとめ：後向き研究で推定できること・できないこと
指標後向き研究で推定できるか
各群の罹患率 $\theta_1, \theta_2$❌ できない（サンプルサイズが研究者の設定値）
罹患率の差 $\theta_1 - \theta_2$❌ できない
オッズ比 $\psi$✅ できる（前向き研究と同一値）
相対リスク $\theta_1/\theta_2$（希少事象）✅ オッズ比で近似できる
答え：③

③ のみが，この研究の設計（後向き研究）に即した正しい解釈です． 後向き研究ではオッズ比だけを推定でき，希少事象の仮定のもとで相対リスクの推定値として利用できます．

問28.2　適合度検定・逸脱度・信頼区間

問題

問28.1 の分割表データに対して，一様性の帰無仮説のもとでの当てはめ値， 適合度カイ二乗 $\chi^2$，逸脱度 $G^2$ を計算し，有意水準 5% で両側検定せよ． また，標本オッズ比に基づく母集団のオッズ比 $\psi$ の信頼係数 95% の近似的な信頼区間を構成せよ．

解答・解説

Step 1：一様性の帰無仮説のもとでの当てはめ値

当てはめ値の公式

$I \times J$ 分割表において，一様性の帰無仮説のもとでの当てはめ値（期待度数）$m_{ij}$ は次の式で与えられます：

$$m_{ij} = \frac{x_{i\cdot}\, x_{\cdot j}}{x_{\cdot\cdot}}$$

ここで $x_{i\cdot}$ は $i$ 行の合計，$x_{\cdot j}$ は $j$ 列の合計，$x_{\cdot\cdot}$ は総計です． この公式は「行周辺和 × 列周辺和 ÷ 総計」と覚えると便利です．

各セルの当てはめ値を計算する

問28.1 の分割表の周辺和：行の合計は 10, 38；列の合計は 12, 36；総計は 48．

$$m_{11} = \frac{10 \times 12}{48} = \frac{120}{48} = 2.5, \quad m_{12} = \frac{10 \times 36}{48} = \frac{360}{48} = 7.5$$ $$m_{21} = \frac{38 \times 12}{48} = \frac{456}{48} = 9.5, \quad m_{22} = \frac{38 \times 36}{48} = \frac{1368}{48} = 28.5$$

観測度数と当てはめ値を並べると次の通りです：

観測度数当てはめ値（期待度数）
低体重児コントロール低体重児コントロール
喫煙習慣あり462.57.5
喫煙習慣なし8309.528.5
計12361236

Step 2：適合度カイ二乗統計量 $\chi^2$

$2\times 2$ 分割表の $\chi^2$ の公式

$2\times 2$ 分割表では，一般式 $\chi^2 = \sum_{ij}(x_{ij}-m_{ij})^2/m_{ij}$ を変形すると，次の便利な表現が得られます：

$$\chi^2 = \frac{x_{\cdot\cdot}(x_{11}x_{22} - x_{12}x_{21})^2}{x_{1\cdot}\,x_{2\cdot}\,x_{\cdot 1}\,x_{\cdot 2}}$$

分子の $(x_{11}x_{22} - x_{12}x_{21})^2$ は対角積の差の2乗，分母は4つの周辺和の積です．

数値を代入して計算する

$x_{11}=4,\; x_{12}=6,\; x_{21}=8,\; x_{22}=30$ を代入：

$$x_{11}x_{22} - x_{12}x_{21} = 4\times 30 - 6\times 8 = 120 - 48 = 72$$
$$\chi^2 = \frac{48 \times 72^2}{10 \times 38 \times 12 \times 36} = \frac{48 \times 5184}{164160} = \frac{248832}{164160} \approx 1.516$$

Step 3：逸脱度 $G^2$

逸脱度（尤度比検定統計量）の公式

逸脱度 $G^2$ は，対数尤度比の2倍として定義されます：

$$G^2 = 2\sum_{ij} x_{ij}\log\!\left(\frac{x_{ij}}{m_{ij}}\right)$$

各セルで「観測値 × $\log$(観測値/期待値)」を計算し，その総和の2倍です． $x_{ij} = m_{ij}$ のセルでは $x_{ij}\log(x_{ij}/m_{ij}) = 0$ となります．

各セルの寄与を計算する

セル$x_{ij}$$m_{ij}$$x_{ij}/m_{ij}$$\log(x_{ij}/m_{ij})$$x_{ij}\log(\cdots)$
喫煙あり・低体重42.51.6000$\log 1.6 = 0.4700$$4 \times 0.4700 = 1.8800$
喫煙あり・コントロール67.50.8000$\log 0.8 = -0.2231$$6 \times (-0.2231) = -1.3389$
喫煙なし・低体重89.50.8421$\log 0.8421 = -0.1719$$8 \times (-0.1719) = -1.3748$
喫煙なし・コントロール3028.51.0526$\log 1.0526 = 0.0513$$30 \times 0.0513 = 1.5388$
$$G^2 = 2\,(1.8800 - 1.3389 - 1.3748 + 1.5388) = 2 \times 0.7051 \approx 1.4103$$

Step 4：仮説検定の判断

棄却域との比較

帰無仮説 $H_0:\theta_1=\theta_2$（一様性，すなわちオッズ比 $\psi=1$）のもとで， $\chi^2$ および $G^2$ はいずれも漸近的に自由度 1 のカイ二乗分布に従います．

有意水準 5% の両側棄却域は $\chi^2_{0.05}(1) = 3.84$ ですので：

$$\chi^2 = 1.516 < 3.84, \quad G^2 = 1.4103 < 3.84$$

いずれも棄却域に入らないため，有意水準 5% で帰無仮説は棄却されません（検定は有意ではない）．

Step 5：オッズ比 $\psi$ の 95% 信頼区間

標本オッズ比と対数オッズ比の標準誤差

標本オッズ比は $\widehat{OR} = x_{11}x_{22}/(x_{12}x_{21})$ です：

$$\widehat{OR} = \frac{4 \times 30}{6 \times 8} = \frac{120}{48} = 2.5$$

標本対数オッズ比の標準誤差の近似値は：

$$\widehat{SE}[\log\widehat{OR}] = \sqrt{\frac{1}{x_{11}} + \frac{1}{x_{12}} + \frac{1}{x_{21}} + \frac{1}{x_{22}}}$$

これは各セルの観測度数の逆数の平方根です（標本サイズが大きいほど小さくなります）．数値を代入：

$$\widehat{SE} = \sqrt{\frac{1}{4} + \frac{1}{6} + \frac{1}{8} + \frac{1}{30}} = \sqrt{\frac{30 + 20 + 15 + 4}{120}} = \sqrt{\frac{69}{120}} = \sqrt{0.575} \approx 0.758$$

対数オッズ比の 95% 信頼区間

標本対数オッズ比は $\log(\widehat{OR}) = \log(2.5) \approx 0.916$． 正規近似による 95% 信頼区間は：

$$\log\widehat{OR} \pm 1.96 \times \widehat{SE} = 0.916 \pm 1.96 \times 0.758 = 0.916 \pm 1.486 = (-0.570,\; 2.402)$$

オッズ比 $\psi$ の 95% 信頼区間（指数変換）

対数オッズ比の信頼区間を指数変換してオッズ比スケールに戻します：

$$\left(e^{-0.570},\; e^{2.402}\right) = (0.566,\; 11.04)$$
結果のまとめと解釈
検定統計量計算値棄却域（$\alpha=0.05$）判定
$\chi^2$1.516$> 3.84$有意でない
$G^2$1.4103$> 3.84$有意でない

オッズ比 $\psi$ の 95% 信頼区間 $(0.566,\; 11.04)$ に $\psi = 1$ が含まれています． これは「帰無仮説 $\psi = 1$ が有意に棄却されなかった」という検定結果と整合しています．

標本オッズ比 $\widehat{OR} = 2.5$ は 1 よりかなり大きいですが，標本サイズが小さい（合計 48 名）ため， 統計的に有意とはいえない結果となっています．

補足：対数オッズ比の標準誤差の導出

$\log\widehat{OR} = \log x_{11} - \log x_{12} - \log x_{21} + \log x_{22}$ と書けます． 各 $x_{ij}$ は（大標本では）近似的に正規分布に従い，その分散は二項分布の性質から $\text{Var}[\log x_{ij}] \approx 1/x_{ij}$ と近似できます． 独立性の仮定のもとで分散を加算すると：

$$\text{Var}[\log\widehat{OR}] \approx \frac{1}{x_{11}} + \frac{1}{x_{12}} + \frac{1}{x_{21}} + \frac{1}{x_{22}}$$

問28.3　グラフィカルモデルと条件付き独立性

問題

無向独立グラフが図 28.1 で与えられるグラフィカルモデルについて，以下の問いに答えよ．

図 28.1 ：4頂点の無向独立グラフ

枝（辺）：A–B，B–C，B–D，C–D　　枝なし：A–C，A–D

[1] 独立な因子は存在するか．

[2] 因子 A と因子 C にはどのような条件付き独立性が成り立つか．

[1] 解答・解説
グラフィカルモデルの性質の復習

無向独立グラフ $G=(V,E)$ を持つグラフィカルモデルについて， 因子の独立性は次の3つの性質で特徴づけられます：

性質グラフの条件独立性の結論
性質1頂点 $A, B$ が連結でない$A$ と $B$ は（周辺）独立：$A \perp\!\!\!\perp B$
性質2$(A,B) \notin E$（辺がない）$A \perp\!\!\!\perp B \mid V\setminus\{A,B\}$（他のすべての因子を条件付けて独立）
性質3集合 $S \subset V$ が $A$ と $B$ を分離する$A \perp\!\!\!\perp B \mid S$（$S$を条件付けて独立）

連結性の確認

グラフ中の各頂点の隣接関係を確認します：

  • A の隣接頂点：B（辺 A–B）
  • B の隣接頂点：A，C，D（辺 A–B，B–C，B–D）
  • C の隣接頂点：B，D（辺 B–C，C–D）
  • D の隣接頂点：B，C（辺 B–D，C–D）

任意の2頂点間に道（辺の列）が存在するかを確認します：

  • A–B：直接つながっている ✓
  • A–C：A–B–C の道 ✓
  • A–D：A–B–D の道 ✓
  • B–C, B–D, C–D：直接つながっている ✓

4つの頂点はすべて互いに連結しているため，性質1の条件（連結でない頂点対が存在する）は満たされません．

結論

4つの頂点はすべて互いに連結しているので，独立な因子は存在しない（性質1）．

[2] 解答・解説

性質2の適用：A と C の間に辺がない

図 28.1 では，頂点 A と頂点 C の間に辺がありません（$(A,C) \notin E$）． したがって性質2より：

$$A \perp\!\!\!\perp C \mid \{B, D\}$$

すなわち，因子 A と因子 C は，他のすべての因子（B と D）を条件付けたとき，条件付き独立となります．

性質3の適用：{B} が A と C を分離する

集合 $\{B\}$ が頂点 A と頂点 C を分離するかどうかを確認します．

「$S$ が $a$ と $b$ を分離する」とは，$a$ から $b$ へ向かう任意の道が必ず $S$ の元を通ることをいいます．

A から C への道を列挙します：

  • A–B–C：B を通る
  • A–B–D–C：B を通る

A と C を結ぶすべての道が B を通るため，$\{B\}$ は A と C を分離します． したがって性質3より：

$$A \perp\!\!\!\perp C \mid B$$

すなわち，因子 A と因子 C は，因子 B のみを条件付けるだけで条件付き独立となります（D を条件付ける必要はありません）．

結論

因子 A と因子 C には，以下の2種類の条件付き独立性が成り立ちます：

  1. 性質2より：$A \perp\!\!\!\perp C \mid \{B, D\}$
    （他のすべての因子 B, D を条件付けて独立）
  2. 性質3より：$A \perp\!\!\!\perp C \mid B$
    （因子 B のみを条件付けても独立；D を条件付ける必要はない）
補足：グラフィカルモデルの背景

図 28.1 のグラフのクリーク（極大完全部分グラフ）は $\{A,B\}$ と $\{B,C,D\}$ の2つです． 対応するグラフィカルモデルの生成集合は $AB/BCD$ と表され， 対数線形モデルとして次のように書かれます：

$$\log\theta_{ijkl} = \mu + \alpha_i + \beta_j + \gamma_k + \delta_l + (\alpha\beta)_{ij} + (\beta\gamma)_{jk} + (\beta\delta)_{jl} + (\gamma\delta)_{kl} + (\beta\gamma\delta)_{jkl}$$

このモデルでは，A と B の交互作用 $(\alpha\beta)_{ij}$ は存在しますが， A と C の交互作用 $(\alpha\gamma)_{ik}$ や A と D の交互作用 $(\alpha\delta)_{il}$ は存在しません（0 に固定）． これが条件付き独立性 $A\perp\!\!\!\perp C\mid\{B,D\}$，$A\perp\!\!\!\perp D\mid\{B,C\}$ に対応しています．

第28章全体のまとめ

第28章「分割表」の主要な流れ：

  1. 研究デザインの確認：前向き研究 vs. 後向き研究。後向き研究では罹患率の推定は不可，オッズ比のみ推定可。
  2. 当てはめ値の計算：帰無仮説（一様性・独立性）のもとで $m_{ij} = x_{i\cdot}x_{\cdot j}/x_{\cdot\cdot}$。
  3. 適合度検定：$\chi^2 = \sum(x_{ij}-m_{ij})^2/m_{ij}$，逸脱度 $G^2 = 2\sum x_{ij}\log(x_{ij}/m_{ij})$，いずれも自由度1（$2\times2$の場合）のカイ二乗分布に漸近。
  4. オッズ比の信頼区間：$\log\widehat{OR} \pm 1.96\sqrt{1/x_{11}+1/x_{12}+1/x_{21}+1/x_{22}}$ で構成し，指数変換。
  5. グラフィカルモデル：無向独立グラフの辺の有無・連結性・分離集合から条件付き独立性を読み取る。
