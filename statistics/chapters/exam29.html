第29章「不完全データの統計処理」のキーポイント
  • 欠測メカニズム（Missing Mechanism）：データがなぜ欠測するかの仕組み。統計量の偏りの有無を左右する最重要概念。
  • MCAR（Missing Completely At Random）：欠測が観測値・欠測値のどちらにも依存しない完全ランダム欠測。CC解析でも各統計量はほぼ不偏。
  • MAR（Missing At Random）：欠測が観測値には依存するが欠測値そのものには依存しない。回帰補完で平均の偏りは除去できるが、標準偏差・相関係数は過小・過大評価となる。
  • MNAR（Missing Not At Random）：欠測が欠測値そのものに依存する。単純な補完法では偏りを除去できない。
  • 打ち切り（Censoring）：値が閾値 $c$ を超えた個体数 $n-m$ が既知のケース。EMアルゴリズムで最尤推定が可能。
  • トランケーション（Truncation）：値が閾値 $c$ を超えた個体数が未知のケース。反復計算により最尤推定を行う。トランケーションでは推定の改善量が打ち切りより小さい。
  • 回帰補完：回帰直線を用いて欠測値を予測補完する方法。平均の偏りは除去できるが、予測値は回帰直線のまわりのばらつきを持たないため、標準偏差を過小評価・相関係数を過大評価する。

問29.1　2変量データの欠測とその影響

問題

ある大学の授業では，最初にスキルテスト（1回目）を行い，定められた基準 $c$ に満たない学生のみが授業を履修し， 基準を超えた学生は受講を免除される． 授業を履修した学生は最後の授業にあらためてスキルテスト（2回目）を受け，その成績で合否が定まる．

最初のスキルテストを受けた 65人中， 15人が基準 $c$ を超えて受講を免除された． 2回のテストを両方とも受験した学生のテストの点数を $(x_1, y_1), \ldots, (x_{50}, y_{50})$ とする． また，受講を免除された15人の学生の1回目のテストの点数を $x_{51}, \ldots, x_{65}$ とする．

ここで，全学生が2回テストを受けたとしたときの点数の分布は， 相関係数 $\rho$ が正の2変量正規分布 $N_2(\mu_X,\, \mu_Y,\, \sigma_X^2,\, \sigma_Y^2,\, \rho\sigma_X\sigma_Y)$ に従うとする．

状況の整理（図）
学生グループ人数観測データ
2回とも受験（授業を履修）50人$(x_i, y_i)$，$i=1,\ldots,50$
1回目のみ（受講免除）15人$x_i$，$i=51,\ldots,65$（$y_i$ は欠測）

ポイント：1回目の点数が基準 $c$ を超えた（つまり $x_i > c$ の）学生は2回目を受験しないため， $y_i$ が欠測。欠測するかどうかが $x_i$ の値（観測値）に依存している。

[1] TAのAさんの分析

授業のTA（Teaching Assistant）のAさんは， 2回とも受験した50人の点数 $(x_1, y_1), \ldots, (x_{50}, y_{50})$ のみを用いて， 2回目のテストの平均値 $\bar{y}_A$ と2回のテスト間の相関係数 $r_A$ を求めた．

$\bar{y}_A$ と $r_A$ の記述として，次の①〜⑤のうちから最も適切なものを1つ選べ．

① $\bar{y}_A$ は $\mu_Y$ を偏りなく推定し，$r_A$ は $\rho$ を概ね偏りなく推定する．
② $\bar{y}_A$ は $\mu_Y$ を偏りなく推定するが，$r_A$ は $\rho$ を過大評価する．
③ $\bar{y}_A$ は $\mu_Y$ を過小評価するが，$r_A$ は $\rho$ を概ね偏りなく推定する．
④ $\bar{y}_A$ は $\mu_Y$ を過小評価し，$r_A$ は $\rho$ を過大評価する．
⑤ $\bar{y}_A$ は $\mu_Y$ を過小評価し，$r_A$ は $\rho$ を過小評価する．

[2] TAのBさんの分析

別のTAのBさんは，2回とも受験した50人の点数 $(x_1, y_1), \ldots, (x_{50}, y_{50})$ を用いて 回帰直線 $\hat{y} = a + bx$ を求め， 2回目の点数の得られない15人の学生の点数の予測値 $\hat{y}_i = a + bx_i\;(i=51,\ldots,65)$ を計算した． そして，

$$(x_1,y_1),\ldots,(x_{50},y_{50}),\;(x_{51},\hat{y}_{51}),\ldots,(x_{65},\hat{y}_{65})$$

の65組のデータから，2回目のテストの平均値 $\bar{y}_B$ と 2回のテスト間の相関係数 $r_B$ を求めた．

$\bar{y}_B$ と $r_B$ の記述として，次の①〜⑤のうちから最も適切なものを1つ選べ．

① $\bar{y}_B$ は $\mu_Y$ を偏りなく推定し，$r_B$ は $\rho$ を概ね偏りなく推定する．
② $\bar{y}_B$ は $\mu_Y$ を偏りなく推定するが，$r_B$ は $\rho$ を過大評価する．
③ $\bar{y}_B$ は $\mu_Y$ を過小評価するが，$r_B$ は $\rho$ を概ね偏りなく推定する．
④ $\bar{y}_B$ は $\mu_Y$ を過小評価し，$r_B$ は $\rho$ を過大評価する．
⑤ $\bar{y}_B$ は $\mu_Y$ を過小評価し，$r_B$ は $\rho$ を過小評価する．
[1] 解答・解説
まず状況を整理する：これはどの欠測メカニズムか？

2回目の点数 $y_i$ が欠測するのは， 1回目の点数 $x_i$ が基準 $c$ を超えた（$x_i > c$）ためです． $x_i$ は観測されているので，欠測は観測値 $x_i$ に依存しています．

しかし，$y_i$ が欠測するかどうかは $y_i$ そのものの値には直接依存していません（$y_i$ は観測されない）． したがって，これは MAR（Missing At Random） に近い状況です．

ただし，実質的には「$x_i > c$ だった学生は $y_i$ も高かった可能性が高い（正の相関があるため）」という形で， 欠測している $y_i$ の値が系統的に大きいという選択バイアス（トランケーション）が生じています．

Aさんの分析の評価：$\bar{y}_A$ について

なぜ $\bar{y}_A$ は $\mu_Y$ を過小評価するのか？

Aさんは，2回とも受験した50人，すなわち「1回目の点数が基準 $c$ 以下だった学生」だけで 2回目の平均を計算しています．

2変量の相関係数 $\rho > 0$ なので， 「1回目が高い学生は2回目も高い傾向がある」という関係があります． 受講を免除された15人（$x_i > c$）は，もし2回目を受験したとすると， 平均より高い $y_i$ を得ていた可能性が高いです．

この「高い $y_i$ を持つはずだったデータ」が丸ごと除外されているため， 50人だけで平均 $\bar{y}_A$ を計算すると 真の平均 $\mu_Y$ より低い値になります（過小評価）．

イメージ（散布図で考える）

散布図上で考えると，1回目の点数 $x$ が高い領域（右側）のデータが すべて削除されているような状況です：

削除されたデータの特徴影響
$x_i > c$（1回目が高得点）$\rho > 0$ より $y_i$ も高い傾向 → 削除で $\bar{y}$ が下がる
分布の右端・上端付近のデータ散布図の「外側」の点が削除され，残ったデータは中心に偏る

Aさんの分析の評価：$r_A$ について

なぜ $r_A$ は $\rho$ を過小評価するのか？（トランケーションの効果）

相関係数は，散布図での「データの広がりの方向への密集度」を測る指標です． データが全範囲にわたって散らばっているほど，相関係数は大きく計算されます．

今回，$x_i > c$ の領域（散布図の右端・上端）のデータがすべて欠測により除去されています． これは分布の上端・右端を切り落とす（トランケーション）に相当します．

散布図の外側（高 $x$，高 $y$）の点を除くと，残ったデータは散布図の 左下部分に集中します．このように分布の広がりが失われると， 相関係数の値は真の値より小さく計算されます（過小評価）．

トランケーションと相関係数の過小評価（直感的説明）

例として，身長と体重の散布図を考えます．全身長の範囲でデータが取られていれば高い相関が観察されます． しかし，身長が高い人だけを除外すると（右端を切り落とす）， 残ったデータの身長の範囲が狭くなり，見かけの相関係数は小さくなります．

この問題でも同様のことが起きており，$r_A < \rho$（過小評価）となります．

答え：⑤

$\bar{y}_A$ は $\mu_Y$ を過小評価し，$r_A$ は $\rho$ を過小評価する．

  • $\bar{y}_A$ の過小評価：$\rho > 0$ のため，除外された15人（$x_i > c$）は $y_i$ も高かった可能性が高い．その分平均が下がる．
  • $r_A$ の過小評価：分布の右端・上端（高 $x$，高 $y$ の領域）が除去されることでトランケーションが生じ，散布図の広がりが失われる．

過大評価（②④）ではなく過小評価（③⑤）であることに注意．$\rho > 0$ のときのトランケーションは「下方向」ではなく，分布端の切り落としによる相関の縮小をもたらす．

[2] 解答・解説
Bさんの手順の確認
  1. Step 1：2回とも受験した50人 $(x_1,y_1),\ldots,(x_{50},y_{50})$ から回帰直線 $\hat{y}=a+bx$ を求める
  2. Step 2：受講免除の15人の2回目の点数を予測値 $\hat{y}_i = a + bx_i\;(i=51,\ldots,65)$ で補完
  3. Step 3：65組の「疑似完全データ」から $\bar{y}_B$ と $r_B$ を計算する

これは回帰代入（regression imputation）と呼ばれる補完法です．

$\bar{y}_B$ の評価：偏りなく推定できる理由

2変量正規分布の性質：回帰直線の不変性

全学生の点数が2変量正規分布 $N_2(\mu_X, \mu_Y, \sigma_X^2, \sigma_Y^2, \rho\sigma_X\sigma_Y)$ に従うとしたとき， $X$ を与えた条件のもとでの $Y$ の条件付き分布は：

$$Y \mid X=x \;\sim\; N\!\left(\mu_Y + \frac{\rho\sigma_Y}{\sigma_X}(x - \mu_X),\; \sigma_Y^2(1-\rho^2)\right)$$

すなわち，$Y \mid X=x$ の条件付き期待値は $x$ の線形関数です． この回帰直線は $x$ の観測・欠測に依存しません（$X=x$ が与えられたもとで $Y$ の欠測がランダム = MAR）．

重要な結論：2回とも受験した50人から求めた回帰直線 $\hat{y}=a+bx$ は， 全65人が受験したとしたら得られる回帰直線とほぼ一致します．

$\bar{y}_B$ が不偏である理由

65組の疑似データの $y$ 方向の平均を計算します：

$$\bar{y}_B = \frac{1}{65}\left(\sum_{i=1}^{50} y_i + \sum_{i=51}^{65} \hat{y}_i\right) = \frac{1}{65}\left(\sum_{i=1}^{50} y_i + \sum_{i=51}^{65} (a + bx_i)\right)$$

回帰直線が全体を代表する直線と一致するなら，予測値 $\hat{y}_i = a + bx_i$ は 「もし受験したとしたときの $y_i$ の期待値」を正しく推定しています． したがって，65組の $y$ の平均 $\bar{y}_B$ は母平均 $\mu_Y$ を偏りなく推定します．

$r_B$ の評価：過大評価となる理由

回帰予測値にはばらつきがない

真の関係では，$x$ が同じ値でも $y$ には確率的なばらつきがあります． 2変量正規分布では，$X=x$ のもとで $Y$ は残差分散 $\sigma_Y^2(1-\rho^2)$ を持ちます：

$$Y = \underbrace{a + bX}_{\text{回帰直線}} + \underbrace{\varepsilon}_{\sigma_Y^2(1-\rho^2)\text{のノイズ}}$$

ところが，Bさんが補完に使った予測値 $\hat{y}_i = a + bx_i$ は 回帰直線上の点だけであり，残差 $\varepsilon$ に相当するランダムな変動が含まれていません．

なぜ相関係数が過大評価されるのか？（直感的説明）

補完された15組の $(x_i, \hat{y}_i)$ は，すべてが回帰直線上にぴったり乗っています． 回帰直線上にある点は，$x$ と $y$ が完全に線形の関係（相関係数 $= 1$）を持ちます．

本来，本物の点は回帰直線からばらついているはずです． しかしBさんの疑似データでは，15点が「理想的に直線上に並んだ点」となっており， 65点全体で見たとき，点が実際より直線にきれいに揃っているように見えます． その結果，$r_B > \rho$（過大評価）となります．

数式で確認する（イメージ）

相関係数は「$x$ の変動と $y$ の変動の共変動 / （$x$ の標準偏差 $\times$ $y$ の標準偏差）」で計算されます． 補完された $\hat{y}_i = a + bx_i$ は $x_i$ と完全に連動しているため：

  • 共分散 $S_{xy}$ は適切な値（回帰直線の傾きに対応）に近い値
  • $y$ の標準偏差 $S_y$ は過小評価（回帰予測値のばらつきは $y$ の実際のばらつきより小さい）
  • 結果として $r_B = S_{xy}/(S_x S_y)$ の分母が小さくなり，$r_B$ は過大評価
答え：②

$\bar{y}_B$ は $\mu_Y$ を偏りなく推定するが，$r_B$ は $\rho$ を過大評価する．

  • $\bar{y}_B$ の不偏性：全学生が2変量正規分布に従うため，50人から求めた回帰直線が全体の回帰直線と一致し，補完された予測値も含めた平均は $\mu_Y$ を正しく推定する．
  • $r_B$ の過大評価：補完された15点は回帰直線上に乗っており，$y$ 方向のばらつきを持たない．そのため，$y$ の標準偏差が過小評価され，相関係数が過大評価される．

第29章全体のまとめ

第29章「不完全データの統計処理」の主要な流れ：

  1. 欠測メカニズムの分類：MCAR（完全ランダム欠測）・MAR（ランダム欠測）・MNAR（非ランダム欠測）。どのメカニズムかによって対処法と偏りの有無が変わる。
  2. CC解析・AC解析：MCARならCC解析で十分。MARでは偏りが生じるため補完が必要。MNARは問題ごとの個別対応が必要。
  3. 打ち切りとトランケーション：閾値 $c$ を超えた個体数が既知なら打ち切り（EMアルゴリズム），未知ならトランケーション（反復スキーム）。どちらも過小評価を改善できる。
  4. 回帰補完の特性：MAR下では $y$ の平均の偏りを除去できるが，$y$ の標準偏差を過小評価・相関係数を過大評価する。
  5. 多重代入法：補完値に確率的なばらつきを加えることで，標準誤差の過小評価も含めた偏りを除去する高度な手法。
