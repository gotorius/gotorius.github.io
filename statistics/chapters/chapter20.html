はじめに：「複数グループの平均を比べる」

「3種類の肥料のどれが最も収穫量を増やすか？」 「4つの工場の製品品質に差はあるか？」 こういった3つ以上のグループの平均値に統計的な差があるかどうかを調べる方法が 分散分析（ANOVA: Analysis of Variance）です。

名前に「分散」とありますが、検定の本質は「グループ間のばらつき」と「グループ内のばらつき」を比べることです。

なぜt検定を繰り返すのでだめなのか？

グループが3つある場合、ペアごとにt検定（A-B、A-C、B-C）すると合計3回行うことになります。 各検定の有意水準が5%でも、3回繰り返すと「少なくとも1回は偶然有意になる確率」は約14%に膨れ上がります （多重比較問題）。分散分析はこれを一度の検定でまとめて扱います。

また、効率的な実験の設計法（どの条件を組み合わせて試すか）が実験計画法です。

キーワード：フィッシャーの三原則、1元配置、2元配置、乱塊法、直交表

フィッシャーの三原則

良い実験を設計するための3つの原則

統計学者ロナルド・フィッシャーは、バイアスのない信頼できる実験を行うための 3つの基本原則を提唱しました。農業実験（どの肥料が最も収量を上げるか）を背景に生まれた考え方ですが、 医学・工学・社会科学すべての実験に当てはまります。

①反復（Replication）

同じ条件で複数の実験を行う。これにより：

  • 1回の偶然による外れ値の影響を小さくできる
  • 実験誤差（within-group variability）の推定ができる
  • サンプルサイズが増え、統計的検出力が上がる

反復なしの問題

もし肥料Aと肥料Bをそれぞれ1区画ずつしか試さなかったら、 A区画の土が偶然に良い土だった場合、それが「肥料の効果」か「土の差」かを区別できません。

②無作為化（Randomization）

実験単位（区画・患者・工場など）への処理（肥料の種類・薬の投与）の割り当てをランダムに行う。

  • 未知の交絡変数（confounding variable）の影響を無作為化によって均等に分散させる
  • 研究者の主観的な割り当てを排除して、統計的推論の前提（独立性）を満たす

③局所管理（Local Control）

実験誤差を小さくするために、実験単位をできるだけ均一なグループ（ブロック）にまとめる。

  • たとえば農場を土壌の肥よくさによってブロック分けし、各ブロック内で全処理を割り当てる
  • ブロック内の変動は「土の差」として取り除けるので、処理効果の検出力が上がる

1元配置分散分析

設定：1つの因子・a水準

因子（Factor）が1つの場合を1元配置分散分析（one-way ANOVA）といいます。 たとえば「肥料の種類（A, B, C）が収穫量に影響するか？」で、 因子は「肥料の種類」、水準（level）は「A, B, C」（3水準）です。

統計モデル

$$y_{ij} = \mu + \alpha_i + \varepsilon_{ij}, \quad \varepsilon_{ij} \overset{\text{i.i.d.}}{\sim} N(0, \sigma^2)$$
  • $y_{ij}$：$i$ 番目の水準、$j$ 番目の反復の観測値
  • $\mu$：全体平均
  • $\alpha_i$：水準 $i$ の効果（全体平均からのずれ。$\sum_i \alpha_i = 0$）
  • $\varepsilon_{ij}$：誤差（偶然変動）

変動の分解（平方和の分解）

全変動 = グループ間の変動 + グループ内の変動 という分解が鍵です：

$$SS_T = SS_A + SS_e$$
変動の名前計算式意味
全変動 $SS_T$$\sum_{i,j}(y_{ij} - \bar{y})^2$全データが全体平均からどれだけ散らばっているか
グループ間 $SS_A$$n \sum_i (\bar{y}_{i.} - \bar{y})^2$グループ平均同士の差（処理効果）
グループ内（誤差） $SS_e$$\sum_{i,j}(y_{ij} - \bar{y}_{i.})^2$グループ内での個体差（偶然誤差）

F検定

「グループ間のばらつき」が「グループ内のばらつき」より十分大きければ、 グループ間に平均の差があると判断できます：

$$F = \frac{MS_A}{MS_e} = \frac{SS_A / (a-1)}{SS_e / (n-a)} \sim F(a-1,\; n-a)$$
  • $a$：水準の数、$n$：全データ数
  • $MS_A = SS_A/(a-1)$：グループ間分散（平均平方）
  • $MS_e = SS_e/(n-a)$：誤差分散（平均平方）
  • $F$ が大きいほど「グループ間に差がある」証拠が強い

2元配置分散分析

2因子の効果と交互作用

因子が2つある場合が2元配置分散分析（two-way ANOVA）です。 たとえば「薬の種類（A, B）と投与量（低、中、高）が血圧低下量に影響するか？」では 因子が「薬の種類」と「投与量」の2つです。

2元配置で重要なのが交互作用（interaction）の概念です。

交互作用とは？

「薬Aは低投与量でも高投与量でも同じ効果だが、薬Bは投与量が増えると効果が急増する」という場合、 薬の種類と投与量の掛け合わせ（交互作用）が血圧に影響しています。
→ どちらか一方の因子だけでは説明できない「2つの因子が一緒に働くことで生じる追加効果」。

均衡2元配置モデル

$$y_{ijk} = \mu + \alpha_i + \beta_j + (\alpha\beta)_{ij} + \varepsilon_{ijk}$$
  • $\alpha_i$：因子Aの主効果（水準 $i$ の効果）
  • $\beta_j$：因子Bの主効果（水準 $j$ の効果）
  • $(\alpha\beta)_{ij}$：AとBの交互作用効果

変動の分解

$$SS_T = SS_A + SS_B + SS_{AB} + SS_e$$

各ソースについてF比を計算し、それぞれ独立にF検定します。

繰り返しのない2元配置（交互作用の検定不可）

各セルに1観測しかない場合（繰り返しなし）、誤差分散を推定できないため交互作用の検定ができません。 モデルは $y_{ij} = \mu + \alpha_i + \beta_j + \varepsilon_{ij}$ となり、主効果のみを推定します。

乱塊法

「ノイズ源」をブロックとして取り除く

実験の誤差の一部が「既知のノイズ源」（例：農場の土質、患者の年齢層、機械の個体差）に起因する場合、 そのノイズ源をブロック（block）として明示的にモデルに組み込むことで、 残差（誤差）が小さくなり、処理効果の検定精度が向上します。

タイヤの例

タイヤの摩耗比較

4種類のタイヤブランド（A, B, C, D）の摩耗を比較したい。 しかし「どの車に装着するか」が結果に影響する（車によって走行パターンが異なる）。
解決策：各車に4種類すべてのタイヤを1本ずつ装着し、車（ブロック）の効果を取り除く。

→ 乱塊法（randomized block design）：各ブロック内でランダムに処理を割り当て。

統計モデル

$$y_{ij} = \mu + \alpha_i + \beta_j + \varepsilon_{ij}$$
  • $\alpha_i$：処理効果（タイヤブランド $i$ の効果）
  • $\beta_j$：ブロック効果（車 $j$ の効果）。これを取り除くことで $SS_e$ が縮小し、検出力が上がる。

変動の分解：$SS_T = SS_A + SS_B + SS_e$（ブロック変動を分離）

直交表・一部実施要因計画

「全組み合わせ」は無理でも「賢いサンプリング」で解決

7つの因子それぞれが2水準なら、全組み合わせは $2^7 = 128$ 通り。 すべて試す「完全実施要因計画（full factorial design）」は多くの場合コストが高すぎます。 そこで、「特定の組み合わせだけ試しても、各因子の主効果が推定できる」という 巧妙な実験計画が一部実施要因計画（fractional factorial design）です。

L8直交表

$L_8(2^7)$ 直交表は「8回の実験で7つの2水準因子の主効果を推定できる」設計です。 8 $= 2^3$ 回だけで、$2^7 = 128$ 回必要な情報の主要部分を得られます。

直交表の「直交」とは？

直交表では、どの2列をとっても「$(1,1), (1,2), (2,1), (2,2)$ の組み合わせが同数現れる」 という性質（直交性）があります。これにより、ある因子の効果を推定する際に 他の因子の影響がキャンセルされ、各因子の主効果を独立に推定できます。

交絡（confounding）

一部実施では「$A$ の主効果」と「交互作用 $BC$」が同じ列に割り当てられることがあります（交絡）。 「交互作用が無視できる（主効果が支配的）」という仮定のもとで主効果のみを推定します。

例題・演習

問20.1：1元配置 ANOVA

問題

3種類の肥料（A, B, C）について、それぞれ4区画で米の収穫量（kg）を測定した。 次の分散分析表を完成させ、肥料の効果があるか5%水準で検定せよ。

変動源SSdfMSF値
肥料間120???
誤差135??—
合計255?——

（$F_{0.05}(2, 9) = 4.26$）

解答

水準数 $a = 3$、各4反復なので $n = 12$ の1元配置です。

変動源SSdfMSF値
肥料間120$3-1=2$$120/2=60$$60/15=4.0$
誤差135$12-3=9$$135/9=15$—
合計255$12-1=11$——

$F_0 = 4.0 < F_{0.05}(2,9) = 4.26$ なので、有意水準5%では肥料の効果は有意でない（帰無仮説を棄却できない）。

問20.2：乱塊法

問題

タイヤの摩耗量を比較する実験で、4種類のタイヤ（処理1〜4）を4台の車（ブロック1〜4）に1本ずつ装着して走行し、摩耗量（mm）を測定した。

車1車2車3車4
タイヤA13101211
タイヤB17141615
タイヤC15121413
タイヤD16131514

乱塊法の分散分析を行い、タイヤ間に差があるか検定せよ（$F_{0.05}(3,9) = 3.86$）。

解答

全体平均：$\bar{y} = (13+10+\cdots+14)/16 = 210/16 = 13.125$

各タイヤの平均（行平均）：

  • タイヤA：$(13+10+12+11)/4 = 46/4 = 11.5$
  • タイヤB：$(17+14+16+15)/4 = 62/4 = 15.5$
  • タイヤC：$(15+12+14+13)/4 = 54/4 = 13.5$
  • タイヤD：$(16+13+15+14)/4 = 58/4 = 14.5$

タイヤ間SS（$SS_A$）：

$$SS_A = 4\left[(11.5-13.125)^2 + (15.5-13.125)^2 + (13.5-13.125)^2 + (14.5-13.125)^2\right]$$ $$= 4[2.641 + 5.641 + 0.141 + 1.891] = 4 \times 10.314 = 41.25$$

各車の平均（列平均）：車1=15.25, 車2=12.25, 車3=14.25, 車4=13.25

ブロック（車）SS（$SS_B$）：

$$SS_B = 4[(15.25-13.125)^2 + (12.25-13.125)^2 + (14.25-13.125)^2 + (13.25-13.125)^2] = 4 \times 5.1875 = 20.75$$

全体SS： $SS_T = \sum_{i,j}(y_{ij}-13.125)^2 = 62.75$

誤差SS： $SS_e = SS_T - SS_A - SS_B = 62.75 - 41.25 - 20.75 = 0.75$

変動源SSdfMSF値
タイヤ（処理）41.25313.75$13.75/0.0833 = 165$
車（ブロック）20.7536.91783.1
誤差0.7590.0833—
合計62.7515——

$F_0 = 165 \gg F_{0.05}(3,9) = 3.86$ なので、タイヤ間の摩耗量の差は有意（5%水準）。

問20.4：2元配置（交互作用の解釈）

問題

薬の種類（A, B）と投与量（低, 高）の2因子について、血圧低下量（mmHg）を測定した。 各セルに2反復として次の結果を得た：

投与量：低投与量：高
薬A10, 1214, 16
薬B8, 1020, 22

（1）各セルの平均を求め、交互作用があるかをグラフ（交互作用プロット）的に判断せよ。

（2）交互作用がある場合、主効果をどのように解釈すべきか述べよ。

解答

(1) セル平均：

低投与高投与差（高−低）
薬A1115+4
薬B921+12

薬Aでは投与量が増えると効果が+4増えるのに対し、薬Bでは+12増える。 「投与量を上げたときの追加効果」が薬の種類によって大きく違う → 交互作用が存在すると判断できます。

(2) 交互作用がある場合の解釈：

交互作用が有意なとき、主効果（「薬全体の効果」「投与量全体の効果」）を 単独で解釈することはミスリーディングになります。 なぜなら、ある薬の効果は投与量によって大きく異なるからです。 正しい解釈は「低投与量ではA・B両方に効果があるが差は小さい。 高投与量では薬Bの方が顕著に効く」という条件付きの記述です。

問20.5：L8直交表の割り付け

問題

製品の中に含まれる不良品の割合を下げるため、製造工程の7因子（A〜G、各2水準）を $L_8(2^7)$ 直交表を使って評価した。なぜ128回の実験をせず8回で主効果を推定できるのか説明せよ。

解答

$L_8(2^7)$ 直交表の各列はどの2列をとっても直交（どの組み合わせも同数現れる）しています。 この直交性により、あるの因子の主効果を計算する際に他の因子の影響が等しく分散され、 互いにキャンセルされます。

ただし交互作用は主効果と交絡しているため、 「交互作用が無視できる（主効果が支配的）」という仮定のもとで初めて主効果の独立推定が成り立ちます。 この仮定が成り立つ場合は8回の実験だけで7因子の主効果が推定可能です。

問20.6・20.7：多重比較

問題

問20.1（肥料A・B・C）でF検定が有意だった場合、どのペアの平均が異なるかを調べるには何を行えばよいか。ボンフェローニ補正の考え方も含めて説明せよ。

解答

F検定が有意 → 「3グループのどこかに差がある」ことはわかるが、具体的にどのペアが違うかはわからない。 そこで多重比較（multiple comparison）を行います。

代表的な方法：

  • ボンフェローニ法：比較するペアの数 $m = 3$ として、各検定の有意水準を $\alpha/m = 0.05/3 \approx 0.0167$ に厳しくしてt検定を行う。シンプルだが保守的（真の差を見落としやすい）。
  • Tukey-HSD法：全ペアを同時に検定する方法。第一種過誤の全体的確率を $\alpha$ に保てる。実践でよく使われる。

ボンフェローニ補正の理論：各検定の過誤を $\alpha/m$ にすると、 $m$ 回行っても全体の第一種過誤が $m \cdot (\alpha/m) = \alpha$ 以下になる（ボンフェローニの不等式）。

まとめ・ポイント

分散分析・実験計画法のまとめ

手法設定特徴
1元配置ANOVA1因子・複数水準グループ間のF検定
2元配置ANOVA2因子主効果＋交互作用を分離して検定
乱塊法処理＋ブロック既知のノイズ源を取り除いて検出力UP
一部実施要因計画多因子・実験数削減直交表で主効果のみ効率よく推定
  • 分散分析のF比 = 「グループ間MS / 誤差MS」。これが大きいほどグループ間に差がある証拠
  • 2元配置では交互作用が有意なら主効果単独の解釈に注意
  • 乱塊法はブロック変動を除くことで誤差MSが小さくなり、検定精度が改善
  • 直交表は「交互作用が無視できる」という前提が必要。大きな交互作用があると結果が誤る
  • F検定後に有意差があればTukey-HSD法などで多重比較を行う
