概要

第20章では分散分析と実験計画法を扱う。フィッシャーの三原則に基づく 実験計画、1元配置・2元配置分散分析、乱塊法、直交表を用いた一部実施要因計画を学ぶ。

キーワード：フィッシャーの三原則、1元配置、2元配置、乱塊法、直交表、交互作用、交絡

重要概念

フィッシャーの三原則

  1. 反復（繰り返し）の原則：同じ条件で複数回実験し、偶然誤差を推定
  2. ランダム化の原則：実験順序・割り当てを乱数で決定し、系統的偏りを排除
  3. 局所管理の原則：均一とみなせるブロックに分け、各ブロック内で一揃いの水準を実験

1元配置分散分析

モデル：$y_{ij} = \mu + \alpha_i + \varepsilon_{ij}$, $\varepsilon_{ij} \sim N(0, \sigma^2)$

  • 帰無仮説 $H_0: \alpha_1 = \cdots = \alpha_a = 0$
  • 平方和の分解：$S_T = S_A + S_E$
  • F検定：$F = V_A / V_E \sim F(\phi_A, \phi_E)$

2元配置分散分析

モデル：$y_{ijk} = \mu + \alpha_i + \beta_j + (\alpha\beta)_{ij} + \varepsilon_{ijk}$

  • 主効果 $\alpha_i, \beta_j$ と交互作用 $(\alpha\beta)_{ij}$
  • 平方和の分解：$S_T = S_A + S_B + S_{A \times B} + S_E$
  • 交互作用が有意なら、主効果だけでなく水準の組み合わせで評価

乱塊法

ブロック因子を導入し、ブロック間変動を誤差から分離：

$y_{ijk} = \mu + \alpha_i + \beta_j + (\alpha\beta)_{ij} + \gamma_k + \varepsilon_{ijk}$

  • ブロック因子による変動が大きい場合、誤差分散が小さくなり検出力が向上
  • ブロック因子の変動がほとんどない場合は、誤差の自由度が減り検出しにくくなる

直交表と一部実施要因計画

  • $L_8(2^7)$ 直交表：2水準7因子を8回の実験で評価
  • 因子を列に割り付け、主効果と交互作用の推定を行う
  • 交絡（別名関係）：主効果と交互作用、または交互作用同士が区別できない関係
  • 成分記号：列の積で交互作用が現れる列を特定

一部実施要因計画 $2^{k-1}$

  • $D = A \times B \times C$：主効果と2因子交互作用の交絡なし、2因子交互作用同士が交絡
  • $D = A \times B$：一部の主効果と2因子交互作用が交絡
  • 目的に応じた使い分けが重要

公式・定理

1元配置の平方和

$$S_T = \sum_{i=1}^{a}\sum_{j=1}^{n_i}(y_{ij}-\bar{y})^2 = S_A + S_E$$
$$F = \frac{S_A / (a-1)}{S_E / (n-a)} = \frac{V_A}{V_E} \sim F(a-1, n-a)$$

2元配置の平方和分解

$$S_T = S_A + S_B + S_{A \times B} + S_E$$

ネイマン配分法（層化抽出法）

$$n_h = \frac{N_h \sigma_h}{\sum_{k=1}^{L} N_k \sigma_k} \times n$$

例題・演習

問20.1：フィッシャーの三原則

(1) ランダム化の原則 → 苗の大小の影響を除くためランダムに割当て

(2) 局所管理の原則 → 均一な区画内で $A_1, A_2$ の一揃いを実験

(3) 反復の原則＋局所管理 → 複数区画で $A_1, A_2$ の一揃いをランダムに実施

問20.2：研磨機械の1元配置分散分析

(1) モデル：$y_{ij} = \mu + \alpha_i + \varepsilon_{ij}$, $H_0: \alpha_1 = \cdots = \alpha_4 = 0$

(2) 分散分析表：$S_A = 21.47$, $\phi_A = 3$, $V_A = 7.157$。$S_E = 19.00$, $\phi_E = 13$, $V_E = 1.462$。$F = 4.897$。$F_{0.05}(3,13)=3.41$ より $H_0$ 棄却。機械により生産個数の母平均は異なる。

(3) $\hat{\mu} = 17.5$, 95%信頼区間：$17.5 \pm 2.16\sqrt{1.462/4} = (16.19,\; 18.81)$

問20.5：タイヤ耐摩耗（乱塊法）

3元配置（ブロック因子なし）：$A, B$ の主効果、$A \times B$ は5%で有意とならない。

乱塊法（ブロック因子あり）：$A, B, V$ の主効果、$A \times B$ が5%で有意。ブロック因子による変動を分離したことで検出力が向上。

問20.6：$2^{4-1}$ 一部実施要因計画

$D=A \times B \times C$：主効果と2因子交互作用の交絡なし。2因子交互作用同士が交絡（$A \times B$ と $C \times D$ など）。

$D=A \times B$：$A$ と $B \times D$, $B$ と $A \times D$, $D$ と $A \times B$ が交絡。

特定の因子に興味があり交互作用を調べたい場合は $D=A \times B$、まんべんなく交絡を避けたい場合は $D=A \times B \times C$ を使用。

メモ

  • 分散分析はF検定に基づく → 平方和の分解が基本
  • 交互作用が有意なら主効果だけでなく、水準の組み合わせで最適条件を判断
  • 乱塊法：ブロック変動が大きいほど有効 → 誤差分散が減り検出力が上がる
  • 直交表：少ない実験回数で多くの因子の効果を推定。ただし交絡に注意
  • $2^{k-1}$ 計画で $D=A \times B \times C$ は解像度IV → 主効果と2因子交互作用は交絡しない
  • 誤差の自由度が小さい場合は $F=2$ 程度でも考慮（AICとの対応）
