問12.1 — 曜日別問い合わせ数の一様性検定

問題

次の表は，小売店Sへの「問い合わせ」の回数について，ある1週間の調査結果を示したものである．曜日によって「問い合わせ」の回数に差があるかどうかを考える．「曜日によって問い合わせの回数が異ならない」という帰無仮説を立て，一様性の検定を有意水準5%で行う．

曜日 月火 水木 金土 日合計
回数 7 3 5 2 7 12 13 49
  1. この検定を行うためのカイ二乗統計量の値を求めよ．
  2. カイ二乗分布表より棄却限界値を求め，この検定の結論を述べよ．
解答・解説
一様性検定（適合度検定）とは

「全カテゴリで確率が等しい（一様分布に従う）」という帰無仮説を $\chi^2$ 検定で調べます。

カテゴリ数を $I$，全観測度数の合計を $n$，$i$ 番目カテゴリの観測度数を $O_i$，期待度数を $E_i$ とすると，検定統計量は

$$T = \sum_{i=1}^{I} \frac{(O_i - E_i)^2}{E_i}$$

帰無仮説が正しいとき，$T$ は近似的に自由度 $I - 1$ の $\chi^2$ 分布に従います。

[1] カイ二乗統計量の計算

期待度数の設定：7曜日すべて等しいという帰無仮説なので，各曜日の期待度数は

$$E_i = \frac{49}{7} = 7 \quad (\text{回/曜日})$$

各曜日の寄与（乖離度）を計算：

曜日 観測 $O_i$ 期待 $E_i$ $O_i - E_i$ $(O_i-E_i)^2$ $\dfrac{(O_i-E_i)^2}{E_i}$
月77000/7 = 0
火37−41616/7
水57−244/7
木27−52525/7
金77000/7 = 0
土127+52525/7
日137+63636/7
合計 49 49 — 106 106/7
$$T = \frac{0 + 16 + 4 + 25 + 0 + 25 + 36}{7} = \frac{106}{7} \approx 15.14$$
$$\boxed{T \approx 15.14}$$

[2] 棄却限界値と検定の結論

自由度の求め方：カテゴリ数 $I = 7$（月〜日）なので，自由度は $I - 1 = 6$ です。
「なぜ $I-1$?」：7つの期待度数の合計は観測総数 49 に固定されているため，7つ目は残り6つが決まれば自動的に決まります。独立に動けるカテゴリは6つ，つまり自由度6です。

自由度6の $\chi^2$ 分布の上側5%点：

$$\chi^2_{0.05}(6) = 12.59$$

棄却域：$T > 12.59$

$$T = 15.14 > 12.59 = \chi^2_{0.05}(6)$$
結論

検定統計量 $T \approx 15.14$ が棄却限界値 $12.59$ を上回るため，帰無仮説を棄却します。

すなわち，有意水準5%で「曜日によって問い合わせ回数が異なる」といえます。

特に土曜（12回）・日曜（13回）が期待値7回を大きく上回っており，週末に問い合わせが集中している傾向が示されています。

問12.2 — 台風上陸数のポアソン分布適合度検定

問題

AさんとBさんは，台風が来たときの備えのため，1年間にどの程度の数の台風が上陸するかを考えることにした。次の表は，1951年から2019年までの69年間の年ごとの台風の日本本土への上陸数（回）をまとめたものである。この表から求めた69年間の上陸数の平均は2.99（回），標準偏差は1.70（回）である。

上陸数（回） 012 345 678 910以上計
観測度数（年） 47 1718 127 30 00 169

（資料：気象庁「台風の上陸数」）

Bさんは，この表をもとに，1年間の台風上陸数がポアソン分布に従うかどうかを吟味したいうえで，1年間に7回以上上陸する確率を求めるほうがよいのではないかと提案した。また，平均と標準偏差の値からもポアソン分布に従うのではないかという意見を述べた。Bさんの考えをもとに，1年間の台風の上陸数がポアソン分布に従うかどうかの適合度検定によって考察する。なお，パラメータ $\lambda$ のポアソン分布の確率関数は次のとおりである。

$$p(x) = \frac{\lambda^x}{x!} e^{-\lambda} \quad (x = 0, 1, 2, \ldots)$$
  1. Bさんが述べた「平均が2.99（回），標準偏差が1.70（回）であることからもポアソン分布に従うのではないか」という意見が意味するところを示せ。
  2. 次の表は，1年間の上陸数がパラメータ $\lambda = 2.99$ のポアソン分布に従うと仮定したとき，69年間の上陸数の期待度数（年）を求めたものである。この表の求め方を説明せよ。
    上陸数 012 345
    期待度数 3.4710.37 15.5115.46 11.566.91
    上陸数 678 910以上計
    期待度数 3.441.47 0.550.18 0.0769.00
  3. カイ二乗統計量を求めたところ16.37であった。適合度検定を有意水準5%で行ったときの棄却域の設定に必要となる数値を示し述べよ。また，統計量の値を大きくしている原因について考察せよ。
  4. Bさんは，1年間に上陸数6回以上の期待度数が小さいので，これらを「上陸数6回以上」とまとめたほうがよいのではないかと提案した。上陸数6回以上をまとめたときの検定の結果と上陸数6回以上をまとめないときの結果を比較し，あてはまりのよさについて述べよ。
解答・解説

[1] ポアソン分布の平均＝分散という性質

ポアソン分布の特徴

パラメータ $\lambda$ のポアソン分布 $\mathrm{Po}(\lambda)$ では，

$$\text{平均} = \lambda, \quad \text{分散} = \lambda$$

つまり平均と分散が等しいという特徴があります。

与えられたデータでは：

  • 平均 $= 2.99$（回）
  • 標準偏差 $= 1.70$（回） → 分散 $= 1.70^2 = 2.89$（回²）

$2.99 \approx 2.89$ と平均と分散がほぼ等しいことから，ポアソン分布に近い可能性が示唆されます。これがBさんの意見の根拠です。

[2] 期待度数の求め方

上陸数 $x$ 回の期待度数は次の手順で求めます：

$$E_x = n \cdot p(x) = 69 \times \frac{2.99^x}{x!} e^{-2.99}$$

ここで $n = 69$（観測年数），$p(x)$ は $\lambda = 2.99$ のポアソン分布の確率関数です。

「10以上」の期待度数の扱い：上陸数9回以下の期待度数を全て合計し，69から引くことで求めます（確率の合計が1になるため）。具体的には： $$E_{10+} = 69 - (3.47 + 10.37 + \cdots + 0.18) = 0.07$$ この「差し引きで求める」際に丸め誤差に注意が必要です。

例として $x = 0$ の場合を計算で確認します：

$$E_0 = 69 \times \frac{2.99^0}{0!} e^{-2.99} = 69 \times e^{-2.99} = 69 \times 0.0502 \approx 3.47 \checkmark$$

[3] 棄却域の設定と統計量が大きくなる原因

自由度の求め方（ポアソン適合度検定）：カテゴリ数 $I$ に対して，パラメータ $\lambda$ を標本平均で推定しているため自由度が1つ減ります。 $$d = (I - 1) - 1 = I - 2$$ カテゴリ数は 0, 1, 2, …, 9, 10以上 の $I = 11$ なので，自由度 $d = 11 - 2 = 9$ です。

自由度9の $\chi^2$ 分布の上側5%点：

$$\chi^2_{0.05}(9) = 16.92$$

棄却域：$T > 16.92$

$$T = 16.37 < 16.92 = \chi^2_{0.05}(9)$$

検定統計量 $T = 16.37$ が棄却限界値 $16.92$ より小さいため，帰無仮説は棄却されません（ただし棄却限界値にほぼ近い値です）。

統計量が大きくなる原因：

「10以上」のカテゴリの期待度数が $E_{10+} = 0.07$ と極めて小さいのに，観測度数が $O_{10+} = 1$ です。このカテゴリの寄与は：

$$\frac{(1 - 0.07)^2}{0.07} = \frac{0.93^2}{0.07} = \frac{0.8649}{0.07} \approx 12.36$$
期待度数が小さいカテゴリの問題：$\chi^2$ 検定は「期待度数が5以上」という目安があります。期待度数が極めて小さい（例：0.07）場合，観測度数との比 $(O-E)^2/E$ が爆発的に大きくなり，統計量全体を歪める原因となります。

この「10以上」カテゴリ一つで $T = 16.37$ のうち約 $12.36$（75%以上）を占めており，統計量を大きく押し上げています。これが，統計量が大きくなる主因です。

[4] 6回以上をまとめた場合との比較

6回以上をまとめた場合の期待度数：

$$E_{6+} = 3.44 + 1.47 + 0.55 + 0.18 + 0.07 = 5.71$$

6回以上をまとめた場合の観測度数：

$$O_{6+} = 3 + 0 + 0 + 0 + 1 = 4$$

まとめた場合のカイ二乗統計量：

カテゴリ 観測 $O_i$ 期待 $E_i$ $(O_i-E_i)^2/E_i$
0回43.47$(4-3.47)^2/3.47 \approx 0.081$
1回710.37$(7-10.37)^2/10.37 \approx 1.095$
2回1715.51$(17-15.51)^2/15.51 \approx 0.143$
3回1815.46$(18-15.46)^2/15.46 \approx 0.418$

まとめた場合の自由度：カテゴリ数は 0, 1, 2, 3, 4, 5, 6以上 の $I = 7$，パラメータ $\lambda$ を推定しているので自由度 $d = 7 - 2 = 5$。自由度5の $\chi^2$ の上側10%点は $9.24$，同5%点は $11.07$。

結論あてはまりが悪いあてはまりが良い

P値で比較すると，6回以上をまとめた場合（$T = 2.27$，自由度5）のほうがはるかに大きなP値（≫10%）となり，ポアソン分布へのあてはまりがよいといえます。期待度数が極めて小さいカテゴリは統計量を歪めるため，まとめることで適切な評価ができます。

問12.3 — Instagram 利用率の男女差の検定

問題

次の表は，ソーシャルネットワークサービス「Instagram」の20代男女の利用者数を整理したクロス集計表である。

利用している 利用していない 計
20代男 40 74 114
20代女 62 45 107
計 102 119 221

Instagramの利用率に男女差があるかどうかを調べるために，検定統計量 $Z$ を用いて，利用率に男女差がないという帰無仮説に対する有意水準 $\alpha$ の両側検定を行うことにした。

$$|Z| \geq z_{\alpha/2} \Rightarrow \text{利用率に男女差がある}$$

ただし $z_{\alpha/2}$ は標準正規分布の上側 $100\alpha/2$% 点である。検定統計量 $Z$ として適切なものを求めよ。

解答・解説
2標本の母比率の差の検定

2つの母集団の比率 $\theta_1, \theta_2$ を比較する検定です。

帰無仮説 $H_0: \theta_1 = \theta_2$ のもとで，次の検定統計量を使います：

$$Z = \frac{\hat{\theta}_1 - \hat{\theta}_2}{\sqrt{\hat{\theta}_* (1 - \hat{\theta}_*)\left(\dfrac{1}{n_1} + \dfrac{1}{n_2}\right)}}$$

ここで

  • $\hat{\theta}_1 = $ 男性の標本比率，$\hat{\theta}_2 = $ 女性の標本比率
  • $n_1 = $ 男性サンプルサイズ，$n_2 = $ 女性サンプルサイズ
  • $\hat{\theta}_* = \dfrac{n_1 \hat{\theta}_1 + n_2 \hat{\theta}_2}{n_1 + n_2}$：2群を合わせたプール推定量（$H_0$ のもとで共通の比率を推定）

各値を代入します：

  • $\hat{\theta}_1 = \dfrac{40}{114}$（男性の利用率）
  • $\hat{\theta}_2 = \dfrac{62}{107}$（女性の利用率）
  • $n_1 = 114$，$n_2 = 107$
  • $\hat{\theta}_* = \dfrac{40 + 62}{114 + 107} = \dfrac{102}{221}$（全体の利用率）
  • $1 - \hat{\theta}_* = \dfrac{119}{221}$
$$Z = \frac{\dfrac{40}{114} - \dfrac{62}{107}}{\sqrt{\left(\dfrac{1}{114} + \dfrac{1}{107}\right) \times \dfrac{102}{221} \times \dfrac{119}{221}}}$$
なぜ「プール推定量」$\hat{\theta}_*$ を使うのか？
帰無仮説 $H_0: \theta_1 = \theta_2 = \theta$（共通）のもとで標準誤差を計算するとき，$\theta$ の推定には2群を合わせたプール推定量 $\hat{\theta}_* = \frac{n_1\hat{\theta}_1 + n_2\hat{\theta}_2}{n_1+n_2}$ を使うのが最も効率的です。全体のデータを使って $\theta$ を推定することで，標準誤差の精度が上がります。

数値を確認：

  • $\hat{\theta}_1 = 40/114 \approx 0.351$（男性利用率 35.1%）
  • $\hat{\theta}_2 = 62/107 \approx 0.579$（女性利用率 57.9%）
  • 差 $= 40/114 - 62/107 \approx 0.351 - 0.579 = -0.228$（女性のほうが高い）
  • $\hat{\theta}_* = 102/221 \approx 0.462$

$|Z|$ は大きな値になることが予想され，有意差があると判定されます（20代女性のほうがInstagram利用率が有意に高い）。

答：検定統計量 $Z$
$$Z = \frac{\dfrac{40}{114} - \dfrac{62}{107}}{\sqrt{\left(\dfrac{1}{114} + \dfrac{1}{107}\right) \times \dfrac{102}{221} \times \dfrac{119}{221}}}$$

これは母比率の差の検定（式(12.4)）に $\hat{\theta}_1 = 40/114$，$\hat{\theta}_2 = 62/107$，$n_1 = 114$，$n_2 = 107$，$\hat{\theta}_* = 102/221$ を代入したものです。式(12.2)（等分散を仮定しない式）を用いても誤りではありません。
