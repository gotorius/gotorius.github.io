概要

第19章では回帰分析その他のトピックとして、打ち切りデータを扱う トービットモデル、生存時間解析（ハザード関数、比例ハザードモデル、カプラン・マイヤー推定量）、 およびニューラルネットワークモデルを学ぶ。

キーワード：打ち切り、トービットモデル、比例ハザードモデル、ニューラルネットワークモデル

重要概念

トービットモデル（打ち切り回帰モデル）

被説明変数 $y_i$ に打ち切りが発生する場合の線形モデル。潜在変数 $y_i^*$ との関係：

  • 左打ち切り：$y_i = y_i^*$ ($y_i^* > L$), $y_i = L$ ($y_i^* \leq L$)
  • 右打ち切り：$y_i = y_i^*$ ($y_i^* < U$), $y_i = U$ ($y_i^* \geq U$)
  • 両側打ち切り：上記の組み合わせ

タイプIトービットモデル：$y_i^* = x_i^T\beta + \varepsilon_i$, $\varepsilon_i \sim N(0, \sigma^2)$

生存時間解析

ある時点からイベント（死亡・完治など）までの時間 $T \geq 0$ を分析。

  • 生存関数：$S(t) = P(T > t) = \int_t^\infty f(u|\theta) du$
  • ハザード関数：$h(t) = -S'(t)/S(t)$（瞬間的なイベント発生率）

ハザード関数の例

分布$h(t)$特徴
指数分布$\lambda$（一定）健康状態が安定な人のモデル
ワイブル分布$\lambda p(\lambda t)^{p-1}$$p>1$: 単調増加, $p<1$: 単調減少
対数正規分布増加後減少慢性疾患のモデル

Cox比例ハザードモデル

$$h(t; z) = h_0(t)\exp(z^T\beta) \quad (h_0(t) > 0)$$
  • $h_0(t)$：基準ハザード（baseline hazard） — 形を指定しない → セミパラメトリック
  • 比例ハザード性：$\frac{h(t;z)}{h(t;z^*)} = \exp\{(z-z^*)^T\beta\}$（時間に依存しない）
  • 比例ハザード性の検証：$\log(-\log S(t;z)) = z^T\beta + \log H_0(t)$

カプラン・マイヤー推定量（積極限推定量）

右打ち切りがある場合の生存関数のノンパラメトリック推定量：

$$\hat{S}(t) = \prod_{i: t_i \leq t}\frac{n_i - d_i}{n_i}$$

$n_i$：時刻 $t_i$ の直前まで生存していた人数、$d_i$：時刻 $t_i$ での死亡数

ニューラルネットワークモデル

  • 入力 $x$、出力 $y$、活性化関数 $f$ による非線形回帰
  • 活性化関数：シグモイド $f(z) = \frac{1}{1+e^{-z}}$、ReLU $f(z) = \max\{0, z\}$
  • 中間層（隠れ層）を持つ合成関数：$y = h_\theta(g_\theta(f_\theta(x)))$
  • 深層ニューラルネットワーク（DNN）：中間層の数が多いモデル

公式・定理

トービットモデルの尤度関数（左打ち切り, $L=0$）

$$L(\beta, \sigma) = \prod_{y_i > 0}\frac{1}{\sigma}\phi\!\left(\frac{y_i - x_i^T\beta}{\sigma}\right) \prod_{y_i \leq 0}\Phi\!\left(\frac{-x_i^T\beta}{\sigma}\right)$$

比例ハザードモデルの生存関数

$$S(t; z) = \exp\left\{-H_0(t)e^{z^T\beta}\right\}$$

ここで $H_0(t) = \int_0^t h_0(u)du$ は累積基準ハザード。

カプラン・マイヤー推定量の条件付き表現

$$\hat{S}(t_j) = \hat{S}(t_{j-1}) \times \hat{P}(T > t_j \mid T \geq t_j)$$

ワイブル分布の生存関数

$$S(t) = e^{-(\lambda t)^p}$$

例題・演習

問19.1：越後湯沢の積雪量（トービットモデル）

2018年12月〜2019年1月の62日間の積雪量を平均気温・日照時間で回帰（左打ち切り $L=0$）。

(1) 尤度関数：

$L(\beta,\sigma) = \prod_{y_i>0}\frac{1}{\sigma}\phi\!\left(\frac{y_i - \beta_0 - \beta_1 x_{i1} - \beta_2 x_{i2}}{\sigma}\right)\prod_{y_i=0}\Phi\!\left(\frac{-\beta_0 - \beta_1 x_{i1} - \beta_2 x_{i2}}{\sigma}\right)$

(2) AICによるモデル選択：

説明変数パラメータ数AIC
日照時間＋平均気温4532.69
日照時間＋平均気温＋最高気温5533.81
日照時間＋平均気温＋最低気温5533.95
日照時間＋平均気温＋最低気温＋最高気温6535.72

AICが最小の「日照時間＋平均気温」が最適モデル。

問19.2：指数分布のハザード関数

$f(t) = \lambda e^{-\lambda t}$ → $S(t) = e^{-\lambda t}$ → $h(t) = -\frac{S'(t)}{S(t)} = \frac{\lambda e^{-\lambda t}}{e^{-\lambda t}} = \lambda$

問19.3：比例ハザードモデルの対数変換

$S(t;z) = \exp\{-H_0(t)e^{z^T\beta}\}$ より：

$-\log S = H_0(t)e^{z^T\beta}$ → $\log(-\log S(t;z)) = z^T\beta + \log H_0(t)$

問19.4：白血病データの比例ハザード性

(1) 比例ハザード性の妥当性：

$\log(-\log \hat{S})$ の2重対数プロットで治療群と対照群がほぼ平行 → 比例ハザード性の仮定は妥当。

(2) 2曲線の距離の意味：

$\beta = \log(-\log S(t;1)) - \log(-\log S(t;0))$

2つのKM推定量の2重対数間の距離は、治療効果 $\beta$ の大きさを表す。

メモ

  • トービットモデル：打ち切りのある連続値応答に対する回帰 → 打ち切られた部分は $\Phi$ で処理
  • Cox比例ハザードモデルが広く使われる理由：基準ハザードの形を指定しないセミパラメトリック性 → 頑健
  • 比例ハザード性のチェック：$\log(-\log S)$ プロットで2群が平行かどうか
  • カプラン・マイヤー推定量は右打ち切りデータを扱える万能な生存関数推定法
  • NNモデルは活性化関数の合成 → 非線形回帰の柔軟なフレームワーク
  • AICの計算：$\text{AIC} = -2 \times \text{最大対数尤度} + 2k$（小さいほど良い）
