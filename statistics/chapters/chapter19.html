はじめに：特殊な構造を持つデータ

通常の回帰分析では「応答変数はどんな実数値もとれる」「データは打ち切られていない（fully observed）」「変数間の関係が線形」などを前提としています。 しかし現実のデータには、こうした前提が成り立たない場合が多くあります。

  • トービットモデル：応答変数に「床（または天井）」があり、一定値以下（以上）が観測されない（打ち切りデータ）
  • 生存時間解析：「いつ事象が起きるか」という時間を分析。まだ事象が起きていない観測（打ち切り）を正しく扱う
  • Cox比例ハザードモデル：生存時間の分布を仮定せずに、グループ間のハザード比を推定
  • カプラン＝マイヤー推定量：ノンパラメトリックな生存関数の推定
  • ニューラルネットワーク：非線形な関係を多層の計算で近似

キーワード：打ち切り、生存関数、ハザード関数、比例ハザード仮定、ニューラルネット

トービットモデル

「床効果」のある応答変数

雪の量を考えてみましょう。「今日の積雪量は-5cm」とはなりません。0cmより少なくなれないという制約（床）があります。 消費額も同様で、「ある商品を購入しない人」の消費額はぴったり0であり、 「マイナスの消費」は観測されません。

このような場合、潜在的な意向（$Y^*$）は正にも負にもなれるが、 実際の観測値 $Y$ は $Y^* > 0$ のときだけ観測できる（それ以外は0になる）と考えます。 これがトービットモデル（censored regression model）です。

3種類の打ち切り

打ち切りの種類観測される条件具体例
下方打ち切り（Left censoring）$Y^* > L$ のとき $Y = Y^*$、それ以外 $Y = L$消費額（最低0円）
上方打ち切り（Right censoring）$Y^* < U$ のとき $Y = Y^*$、それ以外 $Y = U$試験の満点上限
区間打ち切り（Interval censoring）$L < Y^* < U$ のとき $Y = Y^*$、それ以外は境界年収（上下限あり）

標準的なトービットモデル

潜在変数 $Y_i^*$ が正規分布に従うと仮定：

$$Y_i^* = \beta_0 + \beta_1 x_{1i} + \cdots + \beta_p x_{pi} + \varepsilon_i, \quad \varepsilon_i \sim N(0, \sigma^2)$$
$$Y_i = \begin{cases} Y_i^* & \text{if } Y_i^* > 0 \\ 0 & \text{if } Y_i^* \leq 0 \end{cases}$$

普通の最小二乗法は「$Y = 0$」の観測を無視するバイアスが生じます。 正しくは最尤法で推定します。0の観測には「$Y^* \leq 0$ である確率」を、正の観測には正規分布の密度を使います。

OLSはなぜダメか？

$Y = 0$ の観測があるとき、OLSで全データを使うと係数が過小推定されます。 例えば「年収が0の人（無職）」を「収入が本当に0の人」として扱うと、 年収に影響する変数の係数が実態より小さく推定されてしまいます。

生存時間解析

「いつ起きるか」を分析する

医学では「がん治療後の再発までの時間」、工学では「機械が壊れるまでの時間」、 経済学では「失業から再就職までの時間」など、 ある事象が「いつ」起きるかを分析したい場面があります。 これを生存時間解析（survival analysis）またはイベント時間分析といいます。

特徴的なのが打ち切り（censoring）の問題です。

打ち切りとは？

研究終了時点でまだ事象（死亡・再発・故障）が起きていない人・機械は「打ち切り」と呼ばれます。 たとえば、5年間の調査で終了時点まで生存していた患者は「5年以上生きた」ことはわかりますが、 正確な生存時間は不明です。このデータを「捨てる」のは情報の無駄→打ち切りを正しく扱う必要があります。

生存関数とハザード関数

時刻 $t$ を超えて生存（事象が起きない）する確率を生存関数といいます：

$$S(t) = P(T > t)$$

特性：$S(0) = 1$（最初は全員生存）、$t$ が増えるにつれ単調に減少し $S(\infty) = 0$。

ハザード関数（hazard function） $h(t)$ は「時刻 $t$ まで生存した条件のもとで、次の瞬間に事象が起きる瞬間的な確率率」です：

$$h(t) = \lim_{\Delta t \to 0} \frac{P(t \leq T < t + \Delta t \mid T \geq t)}{\Delta t}$$

直感的には「今この瞬間の死亡（故障）リスクの強度」です。

代表的な分布

分布$S(t)$ の形ハザード $h(t)$ の特性用途例
指数分布$e^{-\lambda t}$定数（$h(t) = \lambda$）：時間によらず一定電子部品の故障
ワイブル分布$e^{-(t/\lambda)^k}$単調増加（$k>1$）/定数（$k=1$）/単調減少（$k<1$）機械の摩耗・老化
ログ正規分布$\Phi$で表現先に増加後に減少（山型）がんの再発時間

Cox比例ハザードモデル

分布を仮定しない「半パラメトリック」な方法

指数分布やワイブル分布を使うと「基準となるハザード関数（ベースラインハザード）の形を仮定する」必要があります。 しかし、実際に分布が正しいかどうか検証は難しい。 Coxモデルでは、ベースラインハザードの形を一切仮定せず、 グループ間のハザード比（どちらが何倍危険か）だけを推定します。

比例ハザードモデル

$$h(t \mid x) = h_0(t) \cdot \exp(\beta_1 x_1 + \beta_2 x_2 + \cdots + \beta_p x_p)$$

$h_0(t)$：ベースラインハザード（$x_1=\cdots=x_p=0$ の基準グループのハザード）。
$\exp(\beta_1 x_1 + \cdots)$：共変量による「倍率」。

重要な仮定が比例ハザード仮定（proportional hazards assumption）： 2つのグループのハザード比 $h(t|x_A)/h(t|x_B)$ が時間 $t$ によらず一定である、ということです。

比例ハザード仮定のイメージ

「男性の死亡リスクは女性の2倍」というとき、 比例ハザード仮定は「30歳でも50歳でも70歳でも常に2倍」を意味します。 年齢によってこの比率が変わる場合は仮定が成り立ちません。

偏尤度による推定

ベースラインハザード $h_0(t)$ を推定せずに係数 $\beta$ だけを推定するため、偏尤度（partial likelihood）を使います。 各「死亡イベント」が起きた時刻 $t_j$ において、「リスクセットにいる人の中でこの人が死亡した条件付き確率」の積をとります：

$$L(\beta) = \prod_{j=1}^{D} \frac{\exp(x_j^T \beta)}{\sum_{i \in R(t_j)} \exp(x_i^T \beta)}$$

$R(t_j)$：時刻 $t_j$ にまだ「生存かつ未打ち切り」の人の集合（リスクセット）。

カプラン＝マイヤー推定量

生存関数をノンパラメトリックに推定する

Coxモデルは回帰（共変量の影響を見る）のためでしたが、 まず「集団全体の生存率がどう変化するか」を視覚化したい場合は カプラン＝マイヤー（KM）推定量を使います。 分布の仮定は一切不要です。

計算の仕組み：積限推定量

各「死亡（事象発生）時刻 $t_j$」において、生存確率が少しずつ下がります。 死亡が起きた時刻だけで更新され、打ち切りがあっても正しく扱えます：

$$\hat{S}(t) = \prod_{t_j \leq t} \left(1 - \frac{d_j}{n_j}\right)$$
  • $t_j$：$j$ 番目の死亡時刻
  • $d_j$：時刻 $t_j$ に死亡した人数
  • $n_j$：時刻 $t_j$ 直前のリスクセット人数（まだ生存かつ未打ち切り）

ミニ例：5人のデータ（◆=死亡、○=打ち切り）

観察時間（月）結果リスクセット $n_j$死亡 $d_j$$(1-d_j/n_j)$$\hat{S}(t)$
2◆死亡51$4/5=0.800$0.800
4○打切——生存続く0.800
7◆死亡31$2/3=0.667$$0.800 \times 0.667 = 0.533$
10◆死亡21$1/2=0.500$$0.533 \times 0.500 = 0.267$
12○打切——生存続く0.267

月4と月12の打ち切り観察は、その時点まで「生きていた」としてリスクセットに含めるが、 その後は分母から除外。これで打ち切りを正しく処理できます。

グループ間の生存曲線を比較するにはログランク検定（log-rank test）を使います。 「2つのグループの生存曲線に差がない」という帰無仮説を検定します。

ニューラルネットワーク

脳のニューロンを数式でまねる

ニューラルネットワークは、人間の脳の神経細胞（ニューロン）のつながりをコンピュータ上でまねた数理モデルです。 複雑な非線形関係も学習できる汎用的な予測モデルです。

基本構造：入力層・隠れ層・出力層

3層ニューラルネット

  • 入力層（Input Layer）：説明変数 $x_1, x_2, \ldots, x_p$ を受け取るノード。計算はしない。
  • 隠れ層（Hidden Layer）：重み（weight） $w_{jk}$ でつながれたノード。各ノードで「重み付き和 $z = \sum w_k x_k + b$ → 活性化関数 $f(z)$」を計算する。
  • 出力層（Output Layer）：最後の予測値 $\hat{y}$ を出力。回帰なら恒等関数、分類ならソフトマックスなど。

活性化関数

各ノードで線形変換の後に適用する（非線形を導入する）関数：

活性化関数式特徴
シグモイド$1/(1+e^{-z})$出力が (0,1)。勾配消失問題あり
ReLU$\max(0, z)$現在最もよく使われる。計算効率◎
tanh$(e^z - e^{-z})/(e^z + e^{-z})$出力が (−1,1)

学習：誤差逆伝播法

ニューラルネットのパラメータ（重み $w$ とバイアス $b$）は、誤差逆伝播法（backpropagation）で学習します：

  1. 入力 $x$ を順番に流し予測 $\hat{y}$ を計算（順伝播）
  2. 損失関数 $L = (y - \hat{y})^2$ を計算
  3. 連鎖律（chain rule）で各層の重みに対する勾配 $\partial L / \partial w$ を逆算（逆伝播）
  4. 勾配降下法で $w \leftarrow w - \eta \cdot \partial L/\partial w$ と更新（$\eta$: 学習率）

公式まとめ

トービットモデルの対数尤度

$$\ell(\beta,\sigma) = \sum_{y_i>0}\log\phi\!\left(\frac{y_i - x_i^T\beta}{\sigma}\right) - \sum_{y_i=0} \log\Phi\!\left(\frac{x_i^T\beta}{\sigma}\right)$$

$\phi$: 標準正規密度、$\Phi$: 標準正規CDF（累積分布関数）。

カプラン＝マイヤー推定量

$$\hat{S}(t) = \prod_{t_j \leq t} \frac{n_j - d_j}{n_j}$$

Cox比例ハザードモデルのハザード

$$h(t \mid x) = h_0(t) \exp(x^T \beta)$$

ハザード比（2グループ比較）

$$\text{HR} = \frac{h(t \mid x_A)}{h(t \mid x_B)} = \exp\left(\hat{\beta}(x_A - x_B)\right)$$

例題・演習

問19.1：女性の稼ぎとトービットモデル

問題

夫の収入（$x_1$）、子供の数（$x_2$）、妻の教育年数（$x_3$）が 妻の年間労働所得（万円）に与える影響を調べる。 妻が働いていない場合（所得=0）が全体の40%ある。 なぜトービットモデルが適切か。また、OLSが問題をとなる理由を述べよ。

解答

トービットが適切な理由： 妻の所得は「働く意欲・能力」という潜在変数 $Y^*$ によって決まるが、 $Y^* \leq 0$（就労しない状態）の場合は実際の所得は0で観測される。 つまり応答変数に下方打ち切りがある。

OLSの問題： OLSで「所得 > 0」の人だけを使うと、「働く意欲が高い人」だけのサンプルになり、 係数が選択バイアスを持つ（Heckman選択バイアス）。 全データを使ったOLSでも、0のパイルアップがあると残差が正規分布に従わず、 推定量も一致性を失う。

問19.2：2群の生存時間比較

問題

2つの治療法（A群・B群）の5年生存率を比較する研究で、 次のカプラン＝マイヤー推定量の結果を得た：

月A群 $\hat{S}(t)$B群 $\hat{S}(t)$
01.0001.000
120.8200.760
240.6800.580
360.5400.420
480.4100.290
600.3100.190

5年（60か月）時点での2群の生存率の差を述べよ。また、どのような統計検定でこの差を評価するか。

解答

5年生存率の差： A群の5年生存率は 31.0%、B群は 19.0%。A群の方が 12.0ポイント高い。

統計検定： ログランク検定（log-rank test）を用いる。 観測された死亡数 $O_k$ と、帰無仮説（2群で生存曲線が同じ）のもとで期待される死亡数 $E_k$ を比較し、 カイ二乗統計量 $\chi^2 = \sum_k (O_k - E_k)^2 / E_k$ で判定する。

問19.3：Cox比例ハザードモデルの解釈

問題

心筋梗塞患者のデータにCoxモデルを適用した結果：

変数$\hat{\beta}$SEp値HR（$e^{\hat{\beta}}$）
年齢（歳）0.04230.0110.0011.043
喫煙（1=あり）0.48120.1980.0151.618
治療（1=新薬）$-0.3219$0.1520.0340.724
  1. 喫煙のハザード比を解釈せよ。
  2. 新薬投与のハザード比を解釈せよ。
解答

(1) 喫煙： HR $= e^{0.4812} = 1.618$。 他の条件（年齢・治療法）が同じ場合、喫煙者は非喫煙者に比べて 死亡リスクが約1.62倍（62%高い）。$p = 0.015 < 0.05$ なので有意。

(2) 新薬： HR $= e^{-0.3219} = 0.724$。 新薬を投与された患者は従来治療の患者に比べて 死亡リスクが約0.72倍（28%低い）→ 新薬は生存を延ばす効果がある。$p = 0.034 < 0.05$ なので有意。

問19.4：ニューラルネットワークの構造

問題

入力層3ノード・隠れ層1層4ノード（シグモイド活性化）・出力層1ノード（恒等）のニューラルネットワークを考える。 (1) パラメータ数（重みとバイアスの総数）はいくつか。 (2) 「過学習」とは何か、そのモデルではなぜ起きやすいか。

解答

(1) パラメータ数：

  • 入力層→隠れ層：$3 \times 4 = 12$（重み）$+ 4$（バイアス）$= 16$
  • 隠れ層→出力層：$4 \times 1 = 4$（重み）$+ 1$（バイアス）$= 5$
  • 合計：$16 + 5 = \mathbf{21}$ パラメータ

(2) 過学習：

過学習（overfitting）とは、モデルが訓練データのノイズまで学習してしまい、 新しいデータへの予測精度（汎化能力）が落ちること。 ニューラルネットはパラメータ数が多く複雑な形を表現できるため、 データ数が少ない場合に過学習しやすい。 対策：ドロップアウト（Dropout）、L1/L2正則化、データ拡張など。

まとめ・ポイント

4つのモデルの使い分け

データの特性使うモデル
応答変数に下方（上方）打ち切りがあるトービットモデル
事象が起きるまでの時間、ノンパラメトリック推定カプラン＝マイヤー
生存時間に対する共変量の影響（分布仮定なし）Cox比例ハザード
複雑な非線形関係ニューラルネットワーク
  • トービットでOLSを使うとバイアスが生じる→ 最尤法を使う
  • 生存時間分析では打ち切り（censoring）を正しく扱うことが最重要
  • CoxモデルはベースラインHの形を指定しない半パラメトリック→ 柔軟で広く使われる
  • KMはステップ関数で生存曲線を描く。死亡が起きた時刻だけ下がる
  • ニューラルネットはどんな関数でも近似できる（普遍近似定理）が、解釈しにくく過学習しやすい
