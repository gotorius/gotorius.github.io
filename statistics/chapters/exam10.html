問10.1 — 支持率検定の検出力とサンプルサイズ

問題

ある政党に対する支持率調査を行ったところ、先月の支持率は0.45であった。 この政党の党首は「今月は先月よりも支持率5ポイントが上がり0.50である」という主張をしている。 なお、支持率調査において、全国の有権者より無作為に抽出された $n$ 人から有効な回答を得ることができるとする。

  1. 帰無仮説 $H_0: p_0 = 0.45$ に対する対立仮説 $H_1: p_1 = 0.50$ の一標本の両側検定を考える。 有意水準5%として、$n = 600$ のとき、この検定の検出力はいくらか。
  2. 検出力を80%以上とするために必要なサンプルサイズを求めよ。
解答・解説

問題の設定の整理

検出力（検定力）とは、「対立仮説が真であるときに帰無仮説を正しく棄却できる確率（$1 - \beta$）」のことです。 ここでは $p_1 = 0.50$ が真の値のとき、$H_0: p_0 = 0.45$ を棄却できる確率を求めます。

検出力の考え方

有意水準 $\alpha$ の両側検定では、帰無仮説のもとで $|Z_0| > z_{\alpha/2} = 1.96$ のとき棄却します。 対立仮説が真（$p = p_1$）のとき、検定統計量は $H_0$ のもとでの正規分布からズレます。 このズレの大きさ（非心度）が検出力を決めます。

(1) $n = 600$ のときの検出力

答：$0.6915$

帰無仮説 $H_0: p_0 = 0.45$ のもとで、検定統計量は：

$$Z_0 = \frac{\hat{p} - 0.45}{\sqrt{0.45 \times 0.55 / n}}$$

有意水準5%の両側検定なので、棄却域は $|Z_0| > z_{0.025} = 1.96$ です。

検出力の計算：対立仮説 $p_1 = 0.50$ が真であるとき、 $\hat{p}$ は近似的に $N(0.50,\ 0.50 \times 0.50 / n)$ に従います。 棄却される確率（検出力）は：

$$1 - \beta = P\!\left(|Z_0| > 1.96 \mid p = 0.50\right)$$

検出力の計算に用いる公式：両側検定の検出力の近似式として、 非心パラメータを用いた次の式が使えます。 $z_{\alpha/2}$ と $z_\beta$ の間には次の関係があります：

$$z_{\alpha/2} + z_\beta = \frac{|p_1 - p_0|}{\sqrt{p_0(1-p_0)/n}}$$

$n = 600$、$p_0 = 0.45$、$p_1 = 0.50$、$z_{\alpha/2} = 1.96$ を代入します：

$$z_{\alpha/2} + z_\beta = \frac{0.50 - 0.45}{\sqrt{0.45 \times 0.55 / 600}} = \frac{0.05}{\sqrt{0.2475/600}} = \frac{0.05}{\sqrt{0.0004125}} = \frac{0.05}{0.02031} = 2.4618$$
$$z_\beta = 2.4618 - 1.96 = 0.5018$$

$z_\beta = 0.5018$ は「$\beta$（第2種の過誤の確率）に対応する標準正規分布の上側確率の点」です。 $\Phi(0.50) = 0.6915$（標準正規分布の累積分布関数）より：

$$\beta = P(Z > 0.5018) \approx 1 - \Phi(0.50) = 1 - 0.6915 = 0.3085$$
$$\text{検出力} = 1 - \beta = 1 - 0.3085 = \mathbf{0.6915}$$
標準正規分布表の使い方

$z_\beta = 0.50$ に対し、$P(Z \leq 0.50) = \Phi(0.50) = 0.6915$ を標準正規分布表から読み取ります。 $\beta = P(Z > z_\beta) = 1 - \Phi(z_\beta)$ なので、 検出力 $1 - \beta = \Phi(z_\beta) = \Phi(0.50) = 0.6915$ となります。

(2) 検出力80%以上に必要なサンプルサイズ

答：$n = 777$

検出力 $1 - \beta \geq 0.80$ とするには $z_\beta \geq z_{0.20}$ が必要です。 $z_{0.20} = 0.84$（標準正規分布で上側20%点）。

先ほどの関係式 $z_{\alpha/2} + z_\beta = |p_1 - p_0| / \sqrt{p_0(1-p_0)/n}$ より：

$$z_{\alpha/2} + z_\beta = \frac{0.50 - 0.45}{\sqrt{0.45 \times 0.55/n}} = 1.96 + 0.84 = 2.80$$

$n$ について解きます：

$$\sqrt{\frac{0.45 \times 0.55}{n}} = \frac{0.05}{2.80}$$ $$\frac{0.45 \times 0.55}{n} = \left(\frac{0.05}{2.80}\right)^2$$ $$n = \frac{(z_{\alpha/2} + z_\beta)^2 \times p_0(1-p_0)}{(p_1 - p_0)^2} = \frac{(1.96 + 0.84)^2 \times 0.45 \times 0.55}{(0.50-0.45)^2}$$ $$= \frac{2.80^2 \times 0.2475}{0.0025} = \frac{7.84 \times 0.2475}{0.0025} = \frac{1.9404}{0.0025} = 776.16$$

小数点以下を切り上げて $n = \mathbf{777}$ となります。

サンプルサイズ設計の重要性

検出力が低い（例：69%）ということは、真の効果（支持率の変化）が存在していても見逃す確率が31%もあることを意味します。 研究や調査を設計する段階で必要サンプルサイズを求めておくことが重要です。 医学研究などでは検出力80〜90%が標準とされています。

問10.2 — 分散の信頼区間とF検定（等分散検定）

問題

あるメーカーは既製品Aに対して、新製品Bを開発した。開発の目的は、既製品Aより重さの分散を小さくすることである。 ここで、既製品A、新製品Bの重さはそれぞれ独立に正規分布 $N(\mu_A,\, \sigma_A^2)$、$N(\mu_B,\, \sigma_B^2)$ に従うとする。 このメーカーが、既製品A、新製品Bを16個ずつ無作為に選んで重さを測定したところ、 標本平均からの偏差平方和はそれぞれ、$T_A^2 = 300$、$T_B^2 = 100$ であった。

  1. 新製品Bの母分散 $\sigma_B^2$ の95%信頼区間を求めよ。
  2. このメーカーは、既製品Aの重さの分散 $\sigma_A^2$ より新製品Bの重さの分散 $\sigma_B^2$ のほうが小さいと主張している。 この主張を有意水準5%で片側検定することにした。次の文章の ア・イ について選択肢から適切な項目を選択せよ。
    「この片側検定の帰無仮説は $H_0: \sigma_A^2 = \sigma_B^2$、対立仮説は $H_1: \sigma_A^2 > \sigma_B^2$ である。 検定統計量 $F = T_A^2/T_B^2$ とおくと、帰無仮説が正しい場合、$F$ は自由度 （ア、ア）のF分布に従う。測定の結果より統計量の値は $F = 3.0$ となる。自由度 （ア、ア）のF分布の上側5%点と比較すると、帰無仮説は イ。」

    選択肢 — ア：15、16、30　　イ：棄却できる、棄却できない、どちらともいえない

解答・解説

(1) 新製品Bの母分散 $\sigma_B^2$ の95%信頼区間

答：$(3.64,\ 15.97)$（正確には $(100/27.49,\ 100/6.26) \approx (3.64,\ 15.97)$）

正規分布 $N(\mu_B, \sigma_B^2)$ からの標本サイズ $n = 16$ の標本で、標本平均からの偏差平方和が $T_B^2 = 100$ のとき、次の統計量がカイ二乗分布に従うことを使います：

$$\frac{T_B^2}{\sigma_B^2} = \frac{\sum_{i=1}^{n}(X_i - \bar{X})^2}{\sigma_B^2} \;\sim\; \chi^2(n-1) = \chi^2(15)$$

自由度15のカイ二乗分布の上側2.5%点と下側2.5%点を使います。 問題文より $\chi^2_{0.025}(15) = 27.49$、$\chi^2_{0.975}(15) = 6.26$ です。

信頼区間の導出：

$$P\!\left(\chi^2_{0.975}(15) \leq \frac{T_B^2}{\sigma_B^2} \leq \chi^2_{0.025}(15)\right) = 0.95$$ $$P\!\left(6.26 \leq \frac{100}{\sigma_B^2} \leq 27.49\right) = 0.95$$ $$P\!\left(\frac{100}{27.49} \leq \sigma_B^2 \leq \frac{100}{6.26}\right) = 0.95$$
$$\frac{100}{27.49} \approx 3.638, \quad \frac{100}{6.26} \approx 15.974$$

したがって $\sigma_B^2$ の95%信頼区間は $(3.64,\ 15.97)$ です。

なぜカイ二乗分布を使うのか

正規分布から得た $n$ 個の標本について、偏差平方和 $\sum(X_i - \bar{X})^2 = (n-1)S^2$ を $\sigma^2$ で割ると自由度 $n-1$ のカイ二乗分布に従います。 これは不偏分散 $S^2 = T^2/(n-1)$ を使うと $T^2/\sigma^2 \sim \chi^2(n-1)$ と書けます（$n-1 = 15$）。

(2) ア・イの選択

答：ア = 15、イ = 棄却できる

ア の解説（自由度の決定）

等分散検定のF統計量 $F = T_A^2/T_B^2$ の分布を求めます。 帰無仮説 $H_0: \sigma_A^2 = \sigma_B^2 = \sigma^2$ のもとで：

$$\frac{T_A^2}{\sigma^2} \sim \chi^2(n_A - 1) = \chi^2(15), \quad \frac{T_B^2}{\sigma^2} \sim \chi^2(n_B - 1) = \chi^2(15)$$

F分布の定義は「独立な2つのカイ二乗変数の比をそれぞれの自由度で割ったもの」：

$$F = \frac{T_A^2/T_B^2 \cdot \sigma^2/\sigma^2}{1} = \frac{T_A^2/(n_A-1)}{T_B^2/(n_B-1)} \cdot \frac{n_B-1}{n_A-1}$$

より正確には、$F = (T_A^2/15) / (T_B^2/15)$ はF分布 $F(n_A-1,\ n_B-1) = F(15,\ 15)$ に従います。 よって ア = 15 です。

F統計量の値の確認

$$F = \frac{T_A^2/15}{T_B^2/15} = \frac{300/15}{100/15} = \frac{20}{100/15} = \frac{300}{100} = 3.0$$

イ の解説（棄却判定）

有意水準5%の片側検定（$H_1: \sigma_A^2 > \sigma_B^2$、すなわち $F$ が大きいほうを棄却）なので、 棄却域は $F > F_{0.05}(15, 15)$ です。

$F(15, 15)$ の上側5%点（問題より）は $F_{0.05}(15, 15) = 2.40$ 程度です （教科書付表では約2.40）。

$$F = 3.0 > F_{0.05}(15, 15) \approx 2.40$$

検定統計量 $F = 3.0$ は上側5%点を超えているため、帰無仮説は 棄却できる。 すなわち「新製品Bの分散は既製品Aの分散より有意に小さい」という主張が有意水準5%で支持されます。

項目内容
帰無仮説 $H_0$$\sigma_A^2 = \sigma_B^2$
対立仮説 $H_1$$\sigma_A^2 > \sigma_B^2$（片側）
検定統計量$F = T_A^2/T_B^2 = 3.0$（自由度 $(15, 15)$）
棄却域$F > F_{0.05}(15, 15) \approx 2.40$
判定$3.0 > 2.40$ ∴ 棄却できる

問10.3 — 二項分布と生産者危険・消費者危険

問題

箱のなかから5個の商品を抜き取り、そのなかの2個以上が不良品ならその箱の商品をすべて不合格とする抜取調査について考える。 次の表は、不良品率が $p$ のとき、商品を5個抜き取り、不良品が $r$ 個発見される確率である。 たとえば、$r = 1$、$p = 0.2$ に対応するセルの値 $0.41$ は、不良品率が0.2のとき、5個抜き取った商品のなかで不良品が1個発見される確率である。 ただしここでは、小数点以下第3位を四捨五入している。

$p = 0.1$ $p = 0.2$ $p = 0.3$ $p = 0.4$ $p = 0.5$
$r = 0$0.590.33ア0.080.03
$r = 1$0.330.41イ0.260.16
$r = 2$0.070.20ウ0.350.31
$r = 3$0.010.050.130.230.31
$r = 4$0.000.010.030.080.16
$r = 5$0.000.000.000.010.03
  1. 空欄 ア〜ウ を埋めよ。
  2. ある箱の不良品率が0.1以下であればその箱を合格とし、そうでない場合は不合格とする消費者がいる。 不良品率が0.1以下であるにもかかわらず、抜き取ったなかの不良品が2個以上であったために不合格となる確率を「生産者危険」といい、 不良品率が0.1を超えるにもかかわらず、抜き取ったなかの不良品が1個以下であったため合格となる確率を「消費者危険」という。 不良品率が0.1のときの「生産者危険」と、不良品率が0.2のときの「消費者危険」を求めよ。
解答・解説

(1) 空欄ア〜ウの計算

答：ア = $0.17$、イ = $0.36$、ウ = $0.31$

不良品率 $p$ のとき、5個の商品を抜き取って不良品が $r$ 個見つかる確率は、 二項分布 $\mathrm{Bin}(5, p)$ に従います：

$$P(X = r \mid p) = {}_{5}C_{r} p^r (1-p)^{5-r}, \quad r = 0, 1, \ldots, 5$$

$p = 0.3$ のとき、各 $r$ の確率を計算します（小数点以下第3位を四捨五入）：

ア：$P(X = 0 \mid p=0.3)$

$$P(X=0) = {}_{5}C_{0}(0.3)^0(0.7)^5 = 1 \times 1 \times 0.7^5 = 0.16807 \approx \mathbf{0.17}$$

イ：$P(X = 1 \mid p=0.3)$

$$P(X=1) = {}_{5}C_{1}(0.3)^1(0.7)^4 = 5 \times 0.3 \times 0.2401 = 0.36015 \approx \mathbf{0.36}$$

ウ：$P(X = 2 \mid p=0.3)$

$$P(X=2) = {}_{5}C_{2}(0.3)^2(0.7)^3 = 10 \times 0.09 \times 0.343 = 0.3087 \approx \mathbf{0.31}$$

確認として確率の合計が1に近いことを確かめます：

$$0.17 + 0.36 + 0.31 + 0.13 + 0.03 + 0.00 = 1.00 \checkmark$$

(2) 生産者危険と消費者危険

答：生産者危険（$p = 0.1$）= $0.08$、消費者危険（$p = 0.2$）= $0.74$

生産者危険・消費者危険とは

抜取検査における2種類の誤りです。検定の文脈では、第1種・第2種の過誤に対応します。

名称真の状態判定対応する検定の誤り
生産者危険良品（合格品）不合格と判定第1種の過誤（$\alpha$）
消費者危険不良品（不合格品）合格と判定第2種の過誤（$\beta$）

生産者危険（$p = 0.1$ のとき）

不良品率 $p = 0.1$ の箱は本来「合格」ですが、5個中2個以上が不良品だった場合に誤って「不合格」とされます。

$$\text{生産者危険} = P(X \geq 2 \mid p = 0.1) = P(X=2) + P(X=3) + P(X=4) + P(X=5)$$ $$= 0.07 + 0.01 + 0.00 + 0.00 = \mathbf{0.08}$$

消費者危険（$p = 0.2$ のとき）

不良品率 $p = 0.2$ の箱は本来「不合格」ですが、5個中1個以下しか不良品が見つからなかった場合に誤って「合格」とされます。

$$\text{消費者危険} = P(X \leq 1 \mid p = 0.2) = P(X=0) + P(X=1)$$ $$= 0.33 + 0.41 = \mathbf{0.74}$$
生産者・消費者危険のトレードオフ

判定基準を厳しくする（例：「1個以上不良品で不合格」）と生産者危険は増えますが消費者危険は減ります。 基準を緩める（例：「3個以上不良品で不合格」）と生産者危険は減りますが消費者危険が増えます。 この関係は検定の第1種・第2種の過誤のトレードオフと全く同じ構造です。 サンプルサイズ（今回は5個）を増やすことが、両方の危険を同時に小さくする唯一の方法です。
