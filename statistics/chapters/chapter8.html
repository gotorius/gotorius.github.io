概要

第8章では統計的推定の基礎を扱う。 統計量・順序統計量の概念から始まり、最尤法・モーメント法・最小二乗法などの点推定法、 不偏性・十分性・有効性などの推定量の評価基準、クラーメル・ラオの不等式、 フィッシャー情報量、最尤推定量の漸近正規性、ジャックナイフ法を学ぶ。

キーワード：統計量、順序統計量、点推定、最尤法、モーメント法、線形モデル、最小二乗法、不偏性、十分性、有効性、クラーメル・ラオの不等式、フィッシャー情報量、最尤推定量の漸近正規性、ジャックナイフ法

重要概念

統計量とパラメータ

  • 統計量（statistic）：未知の $\theta$ に依存しない、標本 $X_1, \ldots, X_n$ のみの関数
  • 順序統計量（order statistic）：標本を昇順に並べた $X_{(1)} \leq X_{(2)} \leq \cdots \leq X_{(n)}$
  • 推定量（estimator）：$\hat\theta = h(X_1, \ldots, X_n)$。観測値を代入した $h(x_1, \ldots, x_n)$ を推定値（estimate）という

各種推定法

  • 最尤法（maximum likelihood）：尤度関数 $L(\theta) = \prod_{i=1}^n f(x_i; \theta)$ を最大化する $\hat\theta$。対数尤度 $\ell(\theta) = \log L(\theta)$ で計算すると楽
  • モーメント法：モーメントの理論値と標本モーメントを等しいとおき $\theta$ を解く
  • 最小二乗法：残差二乗和 $\sum(y_i - \beta x_i)^2$ を最小化する回帰パラメータを求める

推定量の評価基準

  • 不偏性：$E_\theta[\hat\theta] = \theta$（バイアス $b_\theta(\hat\theta) = E_\theta[\hat\theta] - \theta = 0$）
  • 平均二乗誤差（MSE）：$\mathrm{MSE}_\theta(\hat\theta) = E_\theta[(\hat\theta - \theta)^2]$
  • バイアス・バリアンス分解：$$\mathrm{MSE}_\theta(\hat\theta) = (b_\theta(\hat\theta))^2 + V_\theta[\hat\theta]$$
  • 一様最小分散不偏推定量（UMVUE）：不偏推定量のなかで分散を最小化
  • 相対効率：$e(\hat\theta_1, \hat\theta_2) = \mathrm{MSE}(\hat\theta_2) / \mathrm{MSE}(\hat\theta_1)$

十分統計量

  • $T(X)$ で条件付けた $X$ の分布がパラメータに依存しないとき、$T(X)$ を十分統計量という
  • フィッシャー・ネイマンの分解定理：$f(x; \theta) = h(x)\, g(T(x), \theta)$ と分解できることが十分統計量の必要十分条件
  • 最尤推定量は十分統計量のみの関数となる

公式・定理

クラーメル・ラオの不等式

フィッシャー情報量 $J_n(\theta) > 0$ のとき、任意の不偏推定量 $\hat\theta$ に対して

$$V[\hat\theta] \geq J_n(\theta)^{-1}$$

フィッシャー情報量の定義：

$$J_n(\theta) = E_\theta\!\left[\left(\frac{\partial}{\partial\theta}\log f(X_1, \ldots, X_n; \theta)\right)^2\right] = -E_\theta\!\left[\frac{\partial^2}{\partial\theta^2}\log f(X_1, \ldots, X_n; \theta)\right]$$

標本が独立同一のとき $J_n(\theta) = n\, J_1(\theta)$（標本サイズに比例）

等号を満たす不偏推定量を有効推定量という。有効推定量は UMVUE である。

ガウス・マルコフの定理

線形単回帰モデル $y_i = \beta x_i + \varepsilon_i$ で、$E[\varepsilon_i] = 0$、$V[\varepsilon_i] < \infty$、$E[\varepsilon_i \varepsilon_j] = 0\ (i \neq j)$ のとき、最小二乗推定量 $\hat\beta$ は不偏な線形推定量のなかで分散最小（BLUE：Best Linear Unbiased Estimator）

最尤推定量の漸近正規性

$$\sqrt{n}(\hat\theta - \theta) \xrightarrow{d} N(0, J_1(\theta)^{-1})$$

適当な正則条件のもとで、最尤推定量は一致性と漸近有効性を持つ。

ジャックナイフ法

$i$ 番目の標本を除いた推定量を $\hat\theta_{(i)}$、その平均を $\hat\theta_{(\cdot)}$ とすると

$$\hat\theta_{\mathrm{jack}} = n\hat\theta - (n-1)\hat\theta_{(\cdot)}$$

バイアスが $O(1/n)$ の推定量に対して有効なバイアス補正法。

例題・演習

問8.1：統計量・十分統計量の判定

$N(\mu, \sigma^2)$ からの標本 $X_1, \ldots, X_n$ に対し、正しい記述を選ぶ。

正しいもの：①②④

  • ① $\sum(X_i - \mu)^2$ は未知パラメータ $\mu$ を含むので統計量ではない
  • ② 順序統計量 $(X_{(1)}, \ldots, X_{(n)})$ は $(\mu, \sigma^2)$ の十分統計量
  • ③ 標本分散 $\frac{1}{n}\sum(X_i - \bar{X})^2$ は $\sigma^2$ の不偏推定量ではない（不偏分散は $n-1$ で割る）
  • ④ 標本分散の平方根は標準偏差の最尤推定量
  • ⑤ 標本分散は $\sigma^2$ の一致推定量であり、漸近有効推定量でもある

問8.2：ポアソン分布の推定

平均 $\lambda$ のポアソン分布からの標本 $X_1, \ldots, X_n$。

(1) 最尤推定量：$\hat\lambda = \bar{X}$（標本平均）

(2) フィッシャー情報量：$J_n(\lambda) = n/\lambda$

(3) 有効推定量であることの証明：$V[\bar{X}] = \lambda/n = J_n(\lambda)^{-1}$ よりクラーメル・ラオの下限を達成

(4) 漸近正規性：中心極限定理より $\sqrt{n}(\bar{X} - \lambda) \xrightarrow{d} N(0, \lambda)$ であり、漸近分散 $\lambda$ は $J_1(\lambda)^{-1} = \lambda$ と一致するので漸近有効性も成立

問8.3：コインの面積のバイアス補正

5つのコインの半径データ：$8.4, 9.0, 8.2, 10.4, 9.2$ (mm)。面積 $\pi\mu^2$ の推定。

(1) バイアス：$T_1 = \pi\bar{X}^2$ のバイアスは $\pi\sigma^2/n$、$T_2 = \pi\overline{X^2}$ のバイアスは $\pi\sigma^2$

(3) バイアス補正推定値：$\bar{X} = 9.04$、$U = 0.748$ より

$$\pi(\bar{X}^2 - U/n) = 3.14 \times (9.04^2 - 0.748/5) \approx \mathbf{256}$$

(4) ジャックナイフ推定値：同じく 256

(5) デルタ法：$g(x) = x^2$ に対し漸近分散は $(2\mu)^2\sigma^2 = 4\mu^2\sigma^2$

メモ

  • 最尤推定量は大標本で最良（一致性＋漸近有効性）だが、有限標本では不偏とは限らない
  • クラーメル・ラオの下限を使えば、ある推定量が UMVUE かどうか判定できる
  • フィッシャー・ネイマンの分解定理は十分統計量を見つける実用的なツール
  • ジャックナイフ法は推定量が $\bar{X}^2$ のような場合でもバイアス補正可能で汎用性が高い
  • ガウス・マルコフの定理は「不偏な線形推定量」の範囲内での最良性を保証する
