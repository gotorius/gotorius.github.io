統計的推定の基礎――「データから未知のパラメータを推測する」

7章まででは「確率的な仕組み」を学びました。 しかし実際の問題では「仕組みのパラメータ（例：コインの表が出る確率 $p$）は未知」であり、 観測データから推測する作業が必要です。これが統計的推定（statistical estimation）です。

「どの推定量を使うべきか？」「この推定量はどれくらい信頼できるか？」を評価するためには、 推定量の性質を理論的に評価する枠組みが必要です。本章ではその枠組みを学びます。

1. 統計量と推定量――データの要約と推測

統計量（statistic）とは？

データ $X_1, \ldots, X_n$ の関数のうち、未知パラメータ $\theta$ を含まない関数を統計量といいます。 「計算に $\theta$ が必要ない」つまりデータだけから計算できるのが統計量の条件です。

統計量の例
  • 標本平均 $\bar{X} = \frac{1}{n}\sum_{i=1}^n X_i$ — 統計量 ⭕
  • 不偏分散 $S^2 = \frac{1}{n-1}\sum_{i=1}^n (X_i - \bar{X})^2$ — 統計量 ⭕
  • $\sum_{i=1}^n (X_i - \mu)^2$（$\mu$ が未知のとき）— 統計量 ❌（$\mu$ を含む）

順序統計量（order statistics）

データを昇順に並べた $X_{(1)} \leq X_{(2)} \leq \cdots \leq X_{(n)}$ を順序統計量といいます。 最小値 $X_{(1)}$、最大値 $X_{(n)}$、中央値などはすべて順序統計量から定義されます。

推定量（estimator）と推定値（estimate）

推定量と推定値の区別

$\hat\theta = h(X_1, \ldots, X_n)$ という確率変数を推定量といいます（大文字 $X$ のまま）。 実際にデータ $x_1, \ldots, x_n$ を観測した後の具体的な値 $h(x_1, \ldots, x_n)$ を推定値といいます。

区別のポイント：推定量は「次の標本でまた変わりうる確率変数」であり、推定値は「今回の標本から得た1つの数値」です。 推定量の性質（不偏性・分散など）を評価するには、確率変数として扱う必要があります。

2. 推定の方法――最尤法・モーメント法・最小二乗法

最尤法（Maximum Likelihood Method）

「今観測されたデータが出やすいようなパラメータを選ぶ」という考え方が最尤法です。 尤度関数 $L(\theta) = \prod_{i=1}^n f(x_i; \theta)$（全データが同時に観測される確率（密度）の積）を最大にする $\hat\theta$ を最尤推定量（MLE）といいます。

最尤法の手順
  1. 尤度関数を書く：$L(\theta) = \prod_{i=1}^n f(x_i;\theta)$
  2. 対数尤度関数を書く：$\ell(\theta) = \log L(\theta) = \sum_{i=1}^n \log f(x_i;\theta)$（積が和になって扱いやすい）
  3. $\dfrac{\partial \ell(\theta)}{\partial \theta} = 0$ を解く（尤度方程式）
  4. 解が最大点かどうか確認（2次微分が負など）

なぜ対数をとるのか：積の微分より和の微分の方が簡単。また $\log$ は単調増加なので最大化点は変わりません。

例：正規分布 $N(\mu, \sigma^2)$ のMLE（両パラメータ未知）

$\ell(\mu, \sigma^2) = -\frac{n}{2}\log(2\pi\sigma^2) - \frac{1}{2\sigma^2}\sum(x_i - \mu)^2$ を偏微分してゼロとおくと：

$$\hat\mu = \bar{x} = \frac{1}{n}\sum x_i, \quad \hat\sigma^2 = \frac{1}{n}\sum(x_i - \bar{x})^2$$

モーメント法（Method of Moments）

「理論的なモーメント（$E[X^k]$）と標本モーメント（$\frac{1}{n}\sum x_i^k$）を等しいとおいて $\theta$ を解く」方法です。 最尤法より簡単な計算で済むことが多いですが、最尤推定量より統計的効率が低いことがあります。

例：ガンマ分布 $\mathrm{Ga}(a, b)$ のモーメント法

$E[X] = ab$、$E[X^2] = (a+a^2)b^2 = ab^2(1+a)$。標本モーメントと等置して $\hat{a}, \hat{b}$ を解く。

最小二乗法（Least Squares Method）

線形モデル $y_i = \beta x_i + \varepsilon_i$ において、残差二乗和 $\sum_{i=1}^n (y_i - \beta x_i)^2$ を最小化する $\hat\beta$ を求める方法です。

$$\hat\beta = \frac{\sum x_i y_i}{\sum x_i^2} \quad \text{（通常のOLS推定量）}$$
ガウス・マルコフの定理（最小二乗法の根拠）

線形モデルで $E[\varepsilon_i] = 0$、$V[\varepsilon_i] = \sigma^2 < \infty$、$\mathrm{Cov}(\varepsilon_i, \varepsilon_j) = 0$（$i \neq j$）のとき、

最小二乗推定量 $\hat\beta$ は不偏な線形推定量のなかで最小分散（BLUE: Best Linear Unbiased Estimator）です。

「最小二乗法は単なるカーブフィットではなく、統計的に根拠のある最良推定法」ということを保証します。

3. 推定量の評価基準――バイアス・MSE・効率性

不偏性（unbiasedness）

不偏性の定義

どんな $\theta$ でも $E_\theta[\hat\theta] = \theta$ が成り立つとき、$\hat\theta$ を不偏推定量といいます。

バイアス（偏り）：$b_\theta(\hat\theta) = E_\theta[\hat\theta] - \theta$

不偏推定量では $b_\theta(\hat\theta) = 0$（バイアスなし）。

例：$X_1, \ldots, X_n \overset{\text{i.i.d.}}{\sim} N(\mu, \sigma^2)$ のとき

$$E[\bar{X}] = \mu \quad \text{（不偏）}, \quad E\!\left[\frac{1}{n-1}\sum(X_i-\bar{X})^2\right] = \sigma^2 \quad \text{（不偏）}$$ $$E\!\left[\frac{1}{n}\sum(X_i-\bar{X})^2\right] = \frac{n-1}{n}\sigma^2 \neq \sigma^2 \quad \text{（バイアスあり）}$$

「なぜ不偏分散は $n-1$ で割るのか」：$n$ 個の偏差 $X_i - \bar{X}$ は $\sum(X_i - \bar{X}) = 0$ という制約で1自由度失われているため。

平均二乗誤差（MSE: Mean Squared Error）

バイアス・バリアンス分解
$$\mathrm{MSE}_\theta(\hat\theta) = E_\theta[(\hat\theta - \theta)^2] = (b_\theta(\hat\theta))^2 + V_\theta[\hat\theta]$$

MSE は「真の値からの2乗誤差の平均」で、推定の精度を総合評価します。

バイアス・バリアンスのトレードオフ：バイアスを減らすと分散が増えることがあります。 不偏推定量（バイアス0）が常に最良とは限りません。 少しバイアスを許してでも分散を大幅に減らせるなら、MSE は小さくなります。

一様最小分散不偏推定量（UMVUE）

不偏推定量のなかで「すべての $\theta$ で分散が最小になる」推定量をUMVUE（Uniformly Minimum Variance Unbiased Estimator）といいます。 UMVUE が「不偏推定量の中での最良」です。 クラーメル・ラオの下限を達成する不偏推定量があれば、それが UMVUE です。

相対効率

$$e(\hat\theta_1, \hat\theta_2) = \frac{\mathrm{MSE}(\hat\theta_2)}{\mathrm{MSE}(\hat\theta_1)}$$

$e > 1$ なら $\hat\theta_1$ が $\hat\theta_2$ より効率的。不偏推定量同士なら分散の比として計算できます。

4. 十分統計量――「情報を失わないデータ圧縮」

直感：何が「十分」なのか？

$n$ 個のデータすべてを保存せずとも、1つの統計量 $T(X)$ に「パラメータについての情報が全部詰まっている」なら、 $T$ だけ見ればよいはずです。 これが十分統計量（sufficient statistic）の概念です。

十分統計量の定義

$T(X)$ で条件付けた $X$ の条件付き分布がパラメータ $\theta$ に依存しないとき、$T(X)$ を $\theta$ の十分統計量といいます。

$$P(X = x \mid T(X) = t;\; \theta) \text{ が $\theta$ に依存しない}$$

直感：「$T$ を知ったとき、$\theta$ についての追加情報は $X$ のどこにもない」。 言い換えると「$T$ がすべての情報を持っている」ということです。

フィッシャー・ネイマンの分解定理（実用的な判定法）

$T(X)$ が $\theta$ の十分統計量 $\iff$ 尤度関数が次の形に分解できる

$$L(\theta; x) = \prod_{i=1}^n f(x_i;\theta) = h(x) \cdot g(T(x),\; \theta)$$

つまり「$\theta$ が関係する部分はすべて $T(x)$ を通してのみ現れる」という条件です。

応用例：正規分布 $N(\mu, \sigma^2)$（$\sigma^2$ 既知）では $\bar{X}$ が $\mu$ の十分統計量です。

$$L(\mu; x) \propto \exp\!\left(-\frac{n(\bar{x}-\mu)^2}{2\sigma^2}\right) \cdot \exp\!\left(-\frac{\sum(x_i-\bar{x})^2}{2\sigma^2}\right)$$

ここで最初の因子が $T(x) = \bar{x}$ と $\mu$ に依存し、2番目は $\mu$ に依存しない（$h(x)$ の役割）。

💡 十分統計量の重要性

最尤推定量は常に十分統計量の関数になります（フィッシャー・ネイマンの定理の系）。 また、Rao-Blackwell の定理により「不偏推定量を十分統計量で条件付き期待値をとるとMSEが改善する」という結果があります。 十分統計量は UMVUE を構成するための基礎でもあります。

5. クラーメル・ラオの不等式――「推定量の分散には下限がある」

動機：どこまで精度を上げられるか？

推定量の分散は小さければ小さいほどよいですが、情報量に限界がある以上「どれだけ頑張っても超えられない下限」があります。 それを与えるのがクラーメル・ラオの不等式です。

フィッシャー情報量（Fisher information）

「データが $\theta$ についてどれだけ情報を持っているか」を定量化したものがフィッシャー情報量です。

フィッシャー情報量の定義（同値な2つの表現）
$$J_1(\theta) = E_\theta\!\left[\left(\frac{\partial}{\partial\theta}\log f(X;\theta)\right)^2\right] = -E_\theta\!\left[\frac{\partial^2}{\partial\theta^2}\log f(X;\theta)\right]$$

第1式：スコア関数の2乗の期待値（スコア = 対数尤度の微分）。
第2式：対数尤度の2階微分の負の期待値（曲率）。
等号は正則条件のもとで成り立ちます。

直感：$J_1(\theta)$ が大きいほど、尤度曲線の「山」が急峻で $\theta$ を精度よく推定できます。 $n$ 個の独立同一標本では $J_n(\theta) = n J_1(\theta)$（情報量はサンプルサイズに比例）。

クラーメル・ラオの不等式

正則条件のもとで、任意の不偏推定量 $\hat\theta$ に対して

$$V_\theta[\hat\theta] \geq \frac{1}{J_n(\theta)} = \frac{1}{n J_1(\theta)}$$

右辺をクラーメル・ラオの下限（CRB）といいます。

等号が成り立つ不偏推定量を有効推定量（efficient estimator）といい、これが UMVUE です。

応用：CRB を達成できない推定量は「まだ改善の余地がある」ので、CRB はベンチマークとして機能します。

具体例：ポアソン分布

$X \sim \mathrm{Po}(\lambda)$ のとき $f(x;\lambda) = e^{-\lambda}\lambda^x/x!$ なので

$$\log f = -\lambda + x\log\lambda - \log x!, \quad \frac{\partial}{\partial\lambda}\log f = -1 + \frac{x}{\lambda}$$ $$\frac{\partial^2}{\partial\lambda^2}\log f = -\frac{x}{\lambda^2}, \quad J_1(\lambda) = E\!\left[\frac{X}{\lambda^2}\right] = \frac{1}{\lambda}$$
$$\text{CRB} = \frac{1}{nJ_1(\lambda)} = \frac{\lambda}{n}$$

$V[\bar{X}] = \lambda/n$ でこの下限を達成 $\Rightarrow$ $\bar{X}$ は有効推定量（かつ UMVUE）。

6. 最尤推定量の漸近理論――大標本での最良性

有限標本での MLE の問題点

MLE は有限標本では必ずしも不偏とは限りません（例：正規分布の分散の MLE $\hat\sigma^2 = S_n^2 = \frac{1}{n}\sum(X_i-\bar{X})^2$ はバイアスあり）。 しかし「標本サイズが大きくなるにつれて」MLE は非常に良い性質を持ちます。

MLE の漸近正規性

適当な正則条件のもとで、最尤推定量 $\hat\theta_n$ は

$$\sqrt{n}\,(\hat\theta_n - \theta) \xrightarrow{d} N\!\left(0,\; J_1(\theta)^{-1}\right)$$

この結果から：

  • 一致性：$\hat\theta_n \xrightarrow{p} \theta$（$n \to \infty$ で真の値に収束）
  • 漸近不偏性：大標本では偏りが消える
  • 漸近有効性：漸近分散 $J_1(\theta)^{-1}/n$ がクラーメル・ラオの下限（CRB）と一致 ← 大標本での意味での UMVUE!

「MLE は大標本で最良」というのはこの漸近有効性を指しています。

漸近正規性を使った信頼区間の構成

漸近正規性から、近似的な $100(1-\alpha)\%$ 信頼区間が構成できます。

$$\hat\theta_n \pm z_{\alpha/2} \cdot \frac{1}{\sqrt{n\, J_1(\hat\theta_n)}}$$

ここで $z_{\alpha/2}$ は $N(0,1)$ の上側 $\alpha/2$ 点です。$J_1(\theta)$ の代わりに $J_1(\hat\theta_n)$（推定値）を使う点がポイントです。

7. ジャックナイフ法――「1つずつ抜いて」バイアスを補正する

動機：非線形推定量のバイアス補正

$\bar{X}^2$ や $\log\bar{X}$ のような非線形推定量には一般にバイアスがあります。 ジャックナイフ法は、「$i$ 番目のデータを除いた推定量を利用してバイアスを補正する」汎用的な手法です。

ジャックナイフ推定量の構成

$i$ 番目のデータを除いた推定量 $\hat\theta_{(i)}$（$i = 1, \ldots, n$）を求め、 その平均 $\hat\theta_{(\cdot)} = \frac{1}{n}\sum_{i=1}^n \hat\theta_{(i)}$ とする。

$$\hat\theta_{\mathrm{jack}} = n\hat\theta - (n-1)\hat\theta_{(\cdot)}$$

なぜこれでバイアスが消えるか？
元の推定量が $E[\hat\theta] = \theta + b/n + O(n^{-2})$（bias $= O(1/n)$）のとき、
$E[\hat\theta_{(i)}] = \theta + b/(n-1) + O(n^{-2})$ より
$E[\hat\theta_{\mathrm{jack}}] = n(\theta + b/n) - (n-1)(\theta + b/(n-1)) + O(n^{-2}) = \theta + O(n^{-2})$
となり、バイアスが $O(1/n)$ から $O(1/n^2)$ に改善されます。

💡 ジャックナイフ法の特徴と限界

利点：分布の仮定なし、解析的な計算なしにバイアス補正できる汎用性の高い手法です。
限界：分位点や中央値のような「非滑らか」な推定量には有効でない場合があります（その場合はブートストラップ法などを使う）。

8. 例題・演習

問8.1：統計量・十分統計量・不偏推定量に関する正誤判定

問題

未知の平均 $\mu$、分散 $\sigma^2$ をもつ正規分布 $N(\mu, \sigma^2)$ から独立・同一に得られた標本を $X_1, \ldots, X_n$ とし、その標本平均を $\bar{X}$ と書く。 以下の①〜⑤の文章のうち、正しいものをすべて選べ（複数回答あり）。

  1. $\displaystyle\sum_{i=1}^n (X_i - \mu)^2$ は統計量である。
  2. 順序統計量を並べたベクトル $(X_{(1)}, \ldots, X_{(n)})$ はパラメータ $(\mu, \sigma^2)$ の十分統計量である。
  3. $\displaystyle\frac{1}{n}\sum_{i=1}^n(X_i - \bar{X})^2$ は $\sigma^2$ の不偏推定量である。
  4. $\displaystyle\sqrt{\frac{1}{n}\sum_{i=1}^n(X_i - \bar{X})^2}$ は標準偏差 $\sigma$ の最尤推定量である。
  5. 標本分散 $S^2 = \frac{1}{n-1}\sum(X_i-\bar{X})^2$ は $\sigma^2$ の漸近有効推定量ではない。
解答

✅ 正しいのは ②・④

  • ① 誤り：$\sum(X_i - \mu)^2$ は未知パラメータ $\mu$ を含むため、データだけの関数ではない → 統計量ではない。
  • ② 正しい：順序統計量はデータ全体の情報を失わずにソートしただけなので、常に十分統計量になる。
  • ③ 誤り：$\frac{1}{n}\sum(X_i-\bar{X})^2$ の期待値は $\frac{n-1}{n}\sigma^2 \neq \sigma^2$ なので不偏ではない。$\frac{1}{n-1}$ で割ると不偏になる。
  • ④ 正しい：正規分布の MLE に不変性（$g(\hat\theta)$ は $g(\theta)$ の MLE）が成立。$\hat{\sigma}^2 = \frac{1}{n}\sum(X_i-\bar{X})^2$ が $\sigma^2$ の MLE なので、その平方根は $\sigma$ の MLE。
  • ⑤ 誤り：$S^2$ も大標本では漸近有効推定量である（フィッシャー情報量下限（CRB）を漸近的に達成する）。

問8.2：ポアソン分布のMLE・フィッシャー情報量・有効推定量

問題

ある交差点で1時間あたりの事故件数は平均 $\lambda > 0$ のポアソン分布に従うとし、 独立に $n$ 時間分の観測データ $X_1, \ldots, X_n$ が得られているとする。 ポアソン分布の確率関数：$f(x; \lambda) = \dfrac{e^{-\lambda}\lambda^x}{x!}$（$x = 0, 1, 2, \ldots$）

  1. パラメータ $\lambda$ の最尤推定量 $\hat{\lambda}$ を求めよ。
  2. 1標本のフィッシャー情報量 $J_1(\lambda)$ を求めよ。
  3. $\hat{\lambda} = \bar{X}$ が有効推定量（クラーメル・ラオ下限を達成する推定量）であることを示せ。
  4. 中心極限定理を用いて $\hat{\lambda}$ の漸近正規性と漸近有効性を示せ。
解答

(1) 最尤推定量の導出

対数尤度関数：

$$\ell(\lambda) = \log L(\lambda) = \sum_{i=1}^n \log f(X_i;\lambda) = -n\lambda + \left(\sum_{i=1}^n X_i\right)\log\lambda - \sum_{i=1}^n \log X_i!$$

$\partial\ell/\partial\lambda = -n + (\sum X_i)/\lambda = 0$ を解くと：

$$\hat{\lambda} = \frac{\sum_{i=1}^n X_i}{n} = \bar{X}$$

(2) フィッシャー情報量

$\log f(x;\lambda) = -\lambda + x\log\lambda - \log x!$ より

$$\frac{\partial^2}{\partial\lambda^2}\log f(x;\lambda) = -\frac{x}{\lambda^2}$$ $$J_1(\lambda) = -E\!\left[\frac{\partial^2}{\partial\lambda^2}\log f(X;\lambda)\right] = \frac{E[X]}{\lambda^2} = \frac{\lambda}{\lambda^2} = \frac{1}{\lambda}$$

(3) 有効性の確認

$n$ 標本のクラーメル・ラオ下限：$\mathrm{CRB} = 1/(nJ_1(\lambda)) = \lambda/n$。

$$V[\bar{X}] = \frac{V[X_1]}{n} = \frac{\lambda}{n} = \mathrm{CRB} \quad \Rightarrow \text{ 有効推定量}$$

(4) 漸近正規性と漸近有効性

$X_1, \ldots, X_n$ は独立で $E[X_i] = V[X_i] = \lambda$ なので、中心極限定理より：

$$\sqrt{n}(\bar{X} - \lambda) \xrightarrow{d} N(0, \lambda) = N\!\left(0, J_1(\lambda)^{-1}\right)$$

漸近分散 $J_1(\lambda)^{-1} = \lambda$ がフィッシャー情報量の逆数と一致するため、漸近有効性も成立しています。

問8.3：コインの面積推定――バイアス・バイアス補正・デルタ法

問題

コインを製造する工場で5つの円形のコインを試作し、その半径（mm）を測定したところ、 次のデータが得られた： $$8.4, \quad 9.0, \quad 8.2, \quad 10.4, \quad 9.2 \text{（単位: mm）}$$ コインは真円とし、半径の真の平均を $\mu$（未知）、分散 $\sigma^2$（未知）とする。 コインの面積の真の値 $\pi\mu^2$ を推定したい。ただし $\pi = 3.14$ を使え。

  1. $T_1 = \pi\bar{X}^2$ のバイアス（$E[T_1] - \pi\mu^2$）を求めよ。
  2. $(1)$ の結果を説明として使い、「各コインの面積の平均値」を推定値として使うことの問題点を述べよ。
  3. $(1)$ で求めたバイアスに標本平均 $\bar{X}$ と不偏分散 $U = S^2$ を代入してバイアス補正した推定値を計算せよ。
  4. デルタ法を使い、$T_1 = \pi\bar{X}^2$ の漸近分散を求めよ（$\sigma^2$・$\mu$ で表せ）。
解答

基本量の計算

$$\bar{X} = \frac{8.4 + 9.0 + 8.2 + 10.4 + 9.2}{5} = \frac{45.2}{5} = 9.04\text{ mm}$$
$$U = S^2 = \frac{1}{4}\sum_{i=1}^5(X_i - \bar{X})^2$$

各偏差の2乗：$(8.4-9.04)^2 = 0.4096$、$(9.0-9.04)^2 = 0.0016$、$(8.2-9.04)^2 = 0.7056$、$(10.4-9.04)^2 = 1.8496$、$(9.2-9.04)^2 = 0.0256$

$$U = \frac{0.4096 + 0.0016 + 0.7056 + 1.8496 + 0.0256}{4} = \frac{2.992}{4} = 0.748\text{ mm}^2$$

(1) $T_1 = \pi\bar{X}^2$ のバイアス

$$E[T_1] = \pi E[\bar{X}^2] = \pi\!\left(V[\bar{X}] + (E[\bar{X}])^2\right) = \pi\!\left(\frac{\sigma^2}{n} + \mu^2\right) = \pi\mu^2 + \frac{\pi\sigma^2}{n}$$
$$\mathrm{Bias}(T_1) = E[T_1] - \pi\mu^2 = \frac{\pi\sigma^2}{n} > 0$$

$T_1 = \pi\bar{X}^2$ は面積 $\pi\mu^2$ を過大評価します（正のバイアス）。

(2) 問題点の説明

「各コインの面積を計算して平均する」推定量は $\frac{1}{n}\sum \pi X_i^2 = \pi\overline{X^2}$ です。 $E\!\left[\pi\overline{X^2}\right] = \pi E[X_i^2] = \pi\!\left(\sigma^2 + \mu^2\right) = \pi\mu^2 + \pi\sigma^2$ となり、バイアス $\pi\sigma^2$ があります（$T_1$ のバイアスより大きい）。 コインの半径にばらつきがある限り、面積の直接平均は真の面積 $\pi\mu^2$ を過大評価します。

(3) バイアス補正推定値

$\sigma^2/n$ を $U/n = S^2/n$ で推定してバイアスを引きます：

$$\tilde{T}_1 = \pi\!\left(\bar{X}^2 - \frac{U}{n}\right) = 3.14 \times \!\left(9.04^2 - \frac{0.748}{5}\right)$$ $$= 3.14 \times (81.7216 - 0.1496) = 3.14 \times 81.572 \approx \mathbf{256.1\text{ mm}^2}$$

(4) 漸近分散（デルタ法）

$g(x) = \pi x^2$ とおくと $g'(\mu) = 2\pi\mu$。デルタ法より $T_1 = g(\bar{X})$ の漸近分散：

$$[g'(\mu)]^2 \cdot \frac{\sigma^2}{n} = (2\pi\mu)^2 \cdot \frac{\sigma^2}{n} = \frac{4\pi^2\mu^2\sigma^2}{n}$$

つまり $\sqrt{n}(T_1 - \pi\mu^2) \xrightarrow{d} N(0, 4\pi^2\mu^2\sigma^2)$ となります。

9. まとめ・ポイント

  • 統計量と推定量の区別：$\theta$ を含まない関数が統計量。実際に観測値を代入した値が推定値。「推定量は確率変数」という視点が重要。
  • 最尤法は推定の基本：対数尤度の最大化。MLE は有限標本で不偏とは限らないが、大標本では漸近有効性を持つ最良推定量になる。
  • クラーメル・ラオの下限は「達成可能な最小分散」の基準：不偏推定量の分散はこの下限以上。下限を達成する推定量が UMVUE。
  • フィッシャー情報量はサンプルサイズに比例：$J_n(\theta) = nJ_1(\theta)$。データを増やせばより精度よく推定できるという定量的な表現。
  • フィッシャー・ネイマンの分解定理は実用的ツール：尤度が $h(x) \cdot g(T(x), \theta)$ に分解できれば $T$ が十分統計量。MLE は十分統計量の関数。
  • ガウス・マルコフの定理は「線形推定量の範囲での最良性」：不偏な線形推定量のなかでは最小二乗推定量（OLS）が BLUE。しかし非線形推定量や不偏性を外した推定量が MLE に勝る場合もある。
  • ジャックナイフ法は汎用バイアス補正法：$O(1/n)$ のバイアスを $O(1/n^2)$ に改善。分布の仮定不要で使いやすい。
