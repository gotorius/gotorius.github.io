問16.1 [1] — 自由度調整済み決定係数によるモデル選択

解答・解説
自由度調整済み決定係数とは

通常の決定係数（Multiple R-squared）$R^2$ は，説明変数を増やすだけで必ず上がります．これはモデルが「過学習」している可能性を示せません．

そこで，説明変数の数（モデルの複雑さ）によるペナルティを加えた自由度調整済み決定係数（Adjusted R-squared）$\bar{R}^2$ が使われます：

$$\bar{R}^2 = 1 - \frac{n-1}{n-p-1}(1-R^2)$$

ここで $n$ はサンプルサイズ，$p$ は説明変数の数です．$\bar{R}^2$ が大きいほどモデルが優れています．

各モデルの自由度調整済み決定係数を比較する

モデル 説明変数 変数の数 $p$ Multiple R² Adjusted R²
モデル 0 （なし） 0 — —
モデル 1 PR, DIAP, QRS 3 0.4448 0.3167
モデル 2 GEN, AMT 2 0.787 0.7566
モデル 3 GEN, AMT, PR, DIAP, QRS 5 0.8764 0.8202

自由度調整済み決定係数はモデル 3 が $\bar{R}^2 = 0.8202$ で最大です．

自由度調整済み決定係数が最大のモデルはモデル 3 である．

なぜモデル 3 ＞ モデル 2 なのか？
モデル 2 には 2 変数，モデル 3 には 5 変数があります．モデル 3 は変数が増えたぶんペナルティもありますが，それでも当てはまりの改善（$R^2$：0.787 → 0.8764）がペナルティを上回るため，調整済み $R^2$ もモデル 3 のほうが高くなります．

問16.1 [2] — モデル 3 で有意な変数の選択（$\alpha = 0.05$）

解答・解説
重回帰における各係数の有意性検定

各説明変数 $x_k$ に対して，帰無仮説 $H_0 : \beta_k = 0$（その変数は $y$ の予測に寄与しない）を検定します．

$t$ 統計量 $= \hat{\beta}_k / \text{Std. Error}$ を計算し，$t$ 分布の両側確率 $\mathrm{Pr}(>|t|)$（P 値）を求めます．

P 値 $< \alpha = 0.05$ ならば，その変数は有意と判断されます．

問16.2 [1] — Day の係数推定量と $t$ 統計量

解答・解説
最小二乗推定量と $t$ 統計量の読み方

Rの lm() 出力において：

  • Estimate：各係数の最小二乗推定値 $\hat{\beta}_k$
  • Std. Error：推定値の標準誤差（$\hat{\beta}_k$ の推定標準偏差）
  • $t$ 統計量 $= \dfrac{\hat{\beta}_k}{\text{Std. Error}}$（帰無仮説 $H_0:\beta_k=0$ のもとで $t$ 分布に従う）

Day の推定値と $t$ 統計量を計算する

モデル 2 の出力より：

  • 推定値：$\hat{\beta}_5 = 0.27388$
  • 標準誤差：$\mathrm{Std. Error} = 0.22967$
$$t = \frac{\hat{\beta}_5}{\mathrm{Std.\ Error}} = \frac{0.27388}{0.22967} \approx 1.1925$$
$$\boxed{\hat{\beta}_5 = 0.27388, \qquad t \approx 1.1925}$$

有意性の判断

自由度 $111 - 5 - 1 = 105$ の $t$ 分布における両側確率（P 値）は，$|t| \approx 1.19$ に対して P 値 $\approx 0.236 > 0.1$ となります．

有意水準 $\alpha = 0.1$（$|t| \geq$ 約 1.66 が棄却域）としても $t$ 統計量 1.19 は棄却域に入りません．

Day の係数は有意ではない（有意水準 0.1 でも帰無仮説 $\beta_5 = 0$ を棄却できない）．

別の見方：モデル 2 には Day の係数 $0.27388$ という正の推定値がありますが，その不確実性（Std. Error = 0.22967）が大きく，「Day が Ozone の予測に本当に寄与するか」という統計的根拠が弱いことを意味します。

問16.2 [2] — 予測誤差の観点からのモデル選択

解答・解説
AIC（赤池情報量基準）とは

AIC（Akaike Information Criterion）は，モデルの当てはまりと複雑さのトレードオフを表す指標です：

$$\mathrm{AIC} = -2\ell(\hat{\theta}) + 2p$$

ここで $\ell(\hat{\theta})$ は最大対数尤度，$p$ は推定したパラメータの数です．AIC は小さいほどモデルが優れています．

予測誤差の観点では，決定係数（訓練データへの当てはまり）よりも AIC（汎化性能＝新しいデータへの予測精度）を使うことが適切です．

各モデルの指標を比較する

モデル 変数の数 $R^2$ Adjusted $R^2$ AIC F 統計量（p 値）
モデル 0 0 — — 1096.073 —
モデル 1 4 0.6199 0.6055 996.71 43.21（$<2.2\times10^{-16}$）
モデル 2 5 0.6249 0.6071 997.22 34.99（$<2.2\times10^{-16}$）

各指標から何がわかるか

決定係数 $R^2$：モデル 2 のほうが大きい（0.6249 > 0.6199）．しかしこれは Day を追加した分だけ機械的に上がっており，当てはまりの「見かけ上の改善」にすぎない可能性があります．

AIC：モデル 1 のほうが小さい（996.71 < 997.22）．AIC はパラメータ数のペナルティを考慮するため，Day の追加によって AIC が悪化していることは，Day が予測に寄与しないことを示唆します．

F 統計量：モデル 1 の F 値（43.21）はモデル 2（34.99）より大きく，モデル 1 のほうが全体的な有意性の観点からも優れています．

予測誤差の観点からは，AIC 最小のモデル 1 を用いることが望ましい．

まとめ：決定係数はモデル 2 が最も大きいが，予測誤差という観点からは AIC 最小のモデル 1 のほうが望ましい．観測データへの当てはまりが最良であることと，予測誤差最小化は必ずしも一致しないことを示している．

問題

Aquatic Toxicity データは，322 個の化合物の毒性を調べたデータセットである．ここでは，"原子対記述子" とよばれる 468 次元の説明変数を用いて毒性を予測する．$i$ 番目の化合物の毒性を $y_i$ として，

$$y_i = \beta_0 + \sum_{k=1}^{468}\beta_k x_{i,k} + \varepsilon_i$$

なるモデルを考える．ここで，$x_{i,k}$ は $i$ 番目の化合物の各原子対記述子を表し，$\varepsilon_i$ は独立同一に正規分布 $N(0, \sigma^2)$ に従う誤差項である．このモデルの推定に，Elastic-Net 正則化を用いる．Elastic-Net 正則化による推定は $\lambda \geq 0$ と $0 \leq \alpha \leq 1$ を用いて

$$\hat{\boldsymbol{\beta}} = \arg\min_{\boldsymbol{\beta} \in \mathbb{R}^{469}} \left(\frac{1}{322}\sum_{i=1}^{322}\left(y_i - \beta_0 - \sum_{k=1}^{468}\beta_k x_{i,k}\right)^2 + \lambda\sum_{k=1}^{468}\left(\alpha|\beta_k| + \frac{1}{2}(1-\alpha)\beta_k^2\right)\right)$$

で与えられる．

  1. $\alpha = 0$ として，正則化パラメータ $\lambda$ を動かして平均二乗誤差 $\left(\frac{1}{322}\sum_{i=1}^{322}\left(y_i - \beta_0 - \sum_{k=1}^{468}\hat{\beta}_k x_{i,k}\right)^2\right)$ および交差検証法によるスコアを計算したところ，下図のようになった．この図は横軸に正則化パラメータ $\lambda$ の自然対数，縦軸に平均二乗誤差および交差検証スコアをプロットしている．このグラフから予測誤差の観点から望ましい正則化パラメータ $\lambda$ は $e^{-6},\, e^{-4},\, 1,\, e^6$ のうちどれか．
  2. $\lambda = 0$ もしくは，$\lambda = e^{-2}$ かつ $\alpha = 0,\, 0.5,\, 1$ なる設定で推定を行った．推定の結果得られた回帰係数を下図にプロットした．なお，$\lambda = 0$ のときは最小二乗解のなかでも最小ノルム解を用いた．図の (a), (b), (c), (d) はそれぞれ，$\lambda = 0$ および $\lambda = e^{-2}$ かつ $\alpha = 0,\, \alpha = 0.5,\, \alpha = 1$ のどれに対応するか答えよ．
  3. $\alpha = 0,\, 0.5,\, 1$ に対して，$\lambda$ を動かして推定された回帰係数の解パスと非ゼロ要素の数を下図にプロットした．このとき，図 (a), (b), (c) はそれぞれどの $\alpha$ に対応するか答えよ．

問16.3 [1] — 予測誤差最小の正則化パラメータ $\lambda$

解答・解説
正則化パラメータと過学習・過正則化のトレードオフ

$\alpha = 0$（Ridge 回帰）において，$\lambda$ を変えると：

  • $\lambda \to 0$（正則化なし）：推定値が最小二乗解に近くなり，訓練データへの過学習が起きる → 残差（Residual）は小さいが，交差検証スコアは悪化
  • $\lambda \to \infty$（強い正則化）：全ての係数が 0 に近づく → 予測が切片のみになり過正則化 → 残差も交差検証スコアも悪化
  • 最適な $\lambda$：交差検証スコアが最小になる点

グラフの形状（問題の図 Residual および cv）から読み取る

図：残差（Residual）と交差検証スコア（cv）の概念図（横軸：$\log\lambda$）

グラフから読み取れること

  • Residual（残差）グラフ：$\lambda$ が大きいほど（$\log\lambda$ が大きいほど）残差が単調に増加します．これは強い正則化が当てはまりを悪化させるためです．
  • cv（交差検証スコア）グラフ：$\log\lambda = 0$（すなわち $\lambda = 1$）付近で最小値を取る U 字型になっています．左側（$\lambda$ が小さすぎ）は過学習，右側（$\lambda$ が大きすぎ）は過正則化でそれぞれ予測精度が悪化します．

予測誤差（交差検証スコア）最小の $\lambda$ を選びます：

候補 $e^{-6},\, e^{-4},\, 1,\, e^6$ のなかで，交差検証スコアが最小となるのは $\log\lambda = 0$ すなわち $\boxed{\lambda = 1}$ である．

直感的な説明：$\lambda = e^{-6}$ は正則化が弱く訓練データへの過学習が起きており，$\lambda = e^{-4}$ もまだ過学習気味です．$\lambda = 1$（$\log\lambda = 0$）が cv スコアの最小点であり，最も汎化性能が高い．$\lambda = e^6$ は強すぎる正則化で係数がほぼ 0 になり予測精度が大きく落ちます．

係数のばらつき具合（スパース性）で図を識別します：

  • $\lambda = e^{-2},\, \alpha = 0$（Ridge）：$L_2$ 正則化のみ．係数は 0 に向けて縮小されますが，完全に 0 にはなりません（スパース性なし）．全体的にばらつきは小さく，0 に近いが非ゼロの点が均一に分布 → (a)
  • $\lambda = e^{-2},\, \alpha = 0.5$（Elastic-Net）：$L_1$ と $L_2$ の混合．中程度のスパース性（Ridge より多くの係数が 0 に，Lasso より少ない非ゼロ係数が残る）→ (d)
  • $\lambda = e^{-2},\, \alpha = 1$（Lasso）：$L_1$ 正則化のみ．強いスパース性（多くの係数が正確に 0，少数の係数だけが非ゼロ）→ (c)
$$\boxed{(a) = \lambda=e^{-2},\,\alpha=0 \quad (b) = \lambda=0 \quad (c) = \lambda=e^{-2},\,\alpha=1 \quad (d) = \lambda=e^{-2},\,\alpha=0.5}$$
判断のポイント：
「(b) が最もノルムが大きくばらついている → $\lambda=0$ が対応」：$\lambda=0$ は最小二乗解でノルムが最大になります．
「スパース性の弱い順：$\lambda=0$（すべて非ゼロ）> $\alpha=0$ > $\alpha=0.5$ > $\alpha=1$（最もスパース）」の順に並べて，図の点の散らばり具合と対応させます．

問16.3 [3] — 解パス図(a)(b)(c) と $\alpha$ の対応

解答・解説
解パス（coefficient path）とは

$\lambda$ を変えたときに，各回帰係数 $\hat{\beta}_k$ がどのように変化するかを示したグラフです．横軸は $\log\lambda$，縦軸は各係数の値，468 本の折れ線が描かれます．

また，非ゼロ要素数（# of nonzeros）は，$\lambda$ の関数として推定された非ゼロ係数の個数です．

  • $\alpha = 0$（Ridge）：係数は 0 に向けて縮小されるが，完全に 0 にはならない → 非ゼロ要素数は常に 468（変化しない）
  • $\alpha = 1$（Lasso）：$L_1$ 正則化の効果で係数が順次 0 になる → 非ゼロ要素数が $\lambda$ とともに急速に減少
  • $\alpha = 0.5$（Elastic-Net）：中間的な挙動 → 非ゼロ要素数は Lasso と Ridge の中間的に減少
図：解パスと非ゼロ要素数（概念図）
(a)
$\alpha = 0$（Ridge）
(b)
$\alpha = 1$（Lasso）
(c)
$\alpha = 0.5$（Elastic-Net）

スパース性から判断する

問題の図（a）はまったくスパースになっていない（非ゼロ要素数が $\lambda$ によらず一定）．図（b）と（c）はともに $\lambda$ が大きくなるにつれて非ゼロ要素数が減っているが，（b）のほうがより小さな $\lambda$ で非ゼロ要素数が小さくなっている（より強い $L_1$ 正則化効果）．

  • 図 (a)：まったくスパースにならない → $\alpha = 0$（Ridge）
  • 図 (b)：より小さい $\lambda$ で非ゼロ数が減る → $L_1$ 正則化の効果が強い → $\alpha = 1$（Lasso）
  • 図 (c)：(b) よりも大きい $\lambda$ まで非ゼロ数が多く残る → $L_1$ 効果が弱い → $\alpha = 0.5$（Elastic-Net）
$$\boxed{\text{図}(a) \leftrightarrow \alpha = 0, \quad \text{図}(b) \leftrightarrow \alpha = 1, \quad \text{図}(c) \leftrightarrow \alpha = 0.5}$$
