はじめに

この章で学ぶこと

これまでの章では、量的データ（連続値）に対する統計手法を学んできた。しかし現実には「病気あり／なし」「商品の好み（A, B, C）」「薬の効果（あり、なし）」のような質的データ（カテゴリカルデータ）も非常に多い。

この章では、複数の質的変量のデータを分割表（contingency table）に集約し、変量間の関連を分析する手法を学ぶ。特に2×2分割表から出発し、一般のI×J分割表、3元分割表（3因子）へと拡張する。

なぜ分割表が重要なのか

分割表の分析は、医学・疫学・社会科学など広い分野で不可欠なツールである。具体的な応用例を見てみよう：

分野 因子A（行） 因子B（列） 興味のある問い
医学 新薬 vs 従来薬 効果あり vs なし 薬の種類と治療効果は関連するか？
疫学 喫煙習慣 肺がん罹患 喫煙は肺がんリスクを高めるか？
マーケティング 年齢層 商品の選好 年齢層によって好む商品は異なるか？
社会学 学歴 政党支持 学歴と政治的選好は独立か？

分割表の基本的な構造

2つの質的変量（因子）A（$I$ 水準）と B（$J$ 水準）に対して、データを集計した表が分割表（contingency table）である。因子Aの第 $i$ 水準、因子Bの第 $j$ 水準に入る観測個数を $x_{ij}$ と書く。

I×J分割表の記法

$x_{ij}$：第$(i,j)$セルの頻度、$x_{i.} = \sum_j x_{ij}$：行和、$x_{.j} = \sum_i x_{ij}$：列和、$x_{..} = \sum_{i,j} x_{ij}$：総度数

2群比較と2×2分割表

2群比較の統計モデル

最も基本的な分割表の状況として、2つの群（A₁ vs A₂）での応答（B₁：有/ B₂：無）の比較を考える。

たとえば新薬（A₁）と従来薬（A₂）の各群から、$n_1$人、$n_2$人の患者を観察し、改善（B₁）の人数 $X_1, X_2$ をカウントする。自然な統計モデルは：

独立二項分布モデル（2群比較）

$$X_i \sim \text{Bin}(n_i, \theta_i), \quad i=1,2, \quad X_1 \perp X_2$$

$\theta_i = P(B_1 | A_i)$：群 $i$ での $B_1$ の確率（生起確率）

パラメータ $\theta_i$ の最尤推定量（同時に不偏推定量）：

$$\hat{\theta}_i = \frac{X_i}{n_i} = \frac{x_i}{n_i}, \quad i=1,2$$

2群比較の帰無仮説は $H_0: \theta_1 = \theta_2$（一様性の仮説）であり、対立仮説は $H_1: \theta_1 \neq \theta_2$（両側）または片側検定となる。

当てはめ値の計算

検定のためには帰無仮説のもとでの「当てはめ値」（fitted value）が必要である。$H_0: \theta_1 = \theta_2 = \theta$ のもとで最尤推定量は：

$$\hat{\theta} = \frac{x_1 + x_2}{n_1 + n_2} = \frac{x_{.1}}{x_{..}}$$

このとき当てはめ値は：

独立性・一様性の帰無仮説のもとでの当てはめ値

$$m_{ij} = \frac{x_{i.} \cdot x_{.j}}{x_{..}}$$

直感的な解釈：「行和 × 列和 ÷ 総度数」。この形は、多項分布モデルや独立ポアソン分布モデルなど、いずれの設定でも同じ形となる（重要）。

オッズ比と研究デザイン

オッズとオッズ比

2群の差を測る指標として、$\theta_1 - \theta_2$（差）や $\theta_1/\theta_2$（相対リスク）もあるが、オッズ比が特に重要な理由がある。

オッズとオッズ比

オッズ（odds）：$\frac{\theta_i}{1-\theta_i}$（「起こる確率」対「起こらない確率」の比）

オッズ比（odds ratio, OR）：

$$\psi = \frac{\theta_1/(1-\theta_1)}{\theta_2/(1-\theta_2)}$$

$\psi = 1$：2群で差なし、$\psi > 1$：群1の方がリスク大、$\psi < 1$：群1の方がリスク小

標本オッズ比（最尤推定量）：

$$\widehat{OR} = \frac{x_{11}x_{22}}{x_{12}x_{21}}$$

対数オッズ比の近似標準誤差：

$$\widehat{\text{se}}(\log\widehat{OR}) \approx \sqrt{\frac{1}{x_{11}}+\frac{1}{x_{12}}+\frac{1}{x_{21}}+\frac{1}{x_{22}}}$$

前向き研究と後向き研究

医学・疫学では、オッズ比を特に重要視する理由がある。それは研究デザインに依存しないという性質である。

2つの研究デザインの違い

研究タイプ 方法 固定されるもの 推定できるもの
前向き研究
（コホート研究）		 喫煙者 $n_1$ 人、非喫煙者 $n_2$ 人を用意し、将来の罹患を追跡 行和（暴露群のサイズ） $\theta_i = P(B_1|A_i)$（罹患確率）
後向き研究
（患者対照研究）		 がん患者 $m_1$ 人、健常者 $m_2$ 人を集め、過去の喫煙歴を調査 列和（結果群のサイズ） $\pi_i = P(A_1|B_i)$（喫煙率）

後向き研究では $\theta_i$ を直接推定できない。しかし、ベイズの定理を用いると、後向き研究のオッズ比は前向き研究のオッズ比と一致する：

$$\frac{\pi_1/(1-\pi_1)}{\pi_2/(1-\pi_2)} = \frac{\theta_1/(1-\theta_1)}{\theta_2/(1-\theta_2)} = \psi$$

つまり、後向き研究のデータからも、真に知りたいオッズ比 $\psi$ を推定できる。

希少事象と相対リスク

がんのような希少な疾患（$\theta_1, \theta_2 \ll 1$）では、オッズ $\approx$ 確率 となるため：

$$\psi = \frac{\theta_1/(1-\theta_1)}{\theta_2/(1-\theta_2)} \approx \frac{\theta_1}{\theta_2} = \text{相対リスク（RR）}$$

したがって希少事象では、オッズ比は相対リスクの良い近似値として使える。

検定手法

適合度カイ二乗検定（ピアソンカイ二乗検定）

帰無仮説 $H_0: \theta_1 = \theta_2$（一様性または独立性）のもとで、観測値 $x_{ij}$ と当てはめ値 $m_{ij}$ のずれを測る検定統計量：

適合度カイ二乗統計量

$$\chi^2 = \sum_{i}\sum_{j} \frac{(x_{ij} - m_{ij})^2}{m_{ij}}$$

帰無仮説のもとで漸近的に $\chi^2(1)$ 分布（2×2分割表の場合）に従う。

2×2分割表については次の式変形も よく使われる：

$$\chi^2 = \frac{x_{..}(x_{11}x_{22} - x_{12}x_{21})^2}{x_{1.}x_{2.}x_{.1}x_{.2}}$$

尤度比検定（逸脱度）

第12章で学んだ対数尤度比検定をここでも適用できる。帰無仮説のもとと対立仮説のもとでの最尤推定量を代入した対数尤度の比の2倍が逸脱度（deviance）：

逸脱度（尤度比検定統計量）

$$G^2 = 2\log\Lambda = 2\sum_{i}\sum_{j} x_{ij}\log\frac{x_{ij}}{m_{ij}}$$

帰無仮説のもとで漸近的に $\chi^2(1)$ 分布に従う（2×2の場合）。$G^2$ は加法性を持つため、モデルの比較・分解が容易である。

フィッシャーの正確検定

カイ二乗検定や尤度比検定は漸近分布を用いるため、サンプルサイズが小さいと正確でない。フィッシャーの正確検定は漸近論を使わず、超幾何分布に基づく。

フィッシャーの正確検定

行和 $x_{1.}, x_{2.}$ と列和 $x_{.1}, x_{.2}$（すべての周辺和）を固定したとき、$X_{11}$ の条件付き分布は超幾何分布：

$$P(X_{11} = k) = \frac{\binom{x_{.1}}{k}\binom{x_{.2}}{x_{1.}-k}}{\binom{x_{..}}{x_{1.}}}$$

片側仮説 $H_1: \psi > 1$ に対するP値：$P(X_{11} \geq x_{11})$。この検定は、サンプルサイズが小さいときや、当てはめ値が5未満のセルがある場合に特に有用。

フィッシャー正確検定の重要な特徴は、2群比較・多項分布モデル・独立ポアソン分布モデルのいずれの設定でも、同じ検定式が得られることである（条件付き分布が一致する）。

I×J分割表と独立性

一般のI×J分割表

2×2分割表を一般化すると、因子Aが $I$ 水準、因子Bが $J$ 水準のI×J分割表となる。

自然な統計モデルは多項分布（multinomial distribution）：観測値 $(x_{11}, x_{12}, \ldots, x_{IJ})$ の同時分布が多項分布に従う。総度数 $x_{..} = n$ を固定したとき：

独立性の仮説（I×J分割表）

$$H_0: \theta_{ij} = \theta_{i.} \cdot \theta_{.j} \quad \text{（全ての } i,j \text{ に対して）}$$

当てはめ値：$m_{ij} = x_{i.}x_{.j}/x_{..}$（2×2と同じ形）

カイ二乗統計量の自由度：$(I-1)(J-1)$

逸脱度 $G^2$ も同じ自由度 $(I-1)(J-1)$ に従う。

対数線形モデルと3元分割表

対数線形モデルとは

ポアソン分布モデルのもとで、2×2分割表の期待値を対数スケールでモデル化したものを対数線形モデル（log-linear model）という。

2×2分割表の飽和対数線形モデル

$$\log\theta_{ij} = \mu + \alpha_i + \beta_j + \gamma_{ij}, \quad \sum_i\alpha_i = \sum_j\beta_j = \sum_i\gamma_{ij} = \sum_j\gamma_{ij} = 0$$

$\mu$：全体平均効果、$\alpha_i$：因子Aの主効果、$\beta_j$：因子Bの主効果、$\gamma_{ij}$：A×B交互作用

独立性仮説（交互作用なし）は $H_0: \gamma_{ij} = 0$（主効果モデル）と等価。

3元分割表と独立性の種類

3つの因子A, B, Cがある場合、独立性の概念が複数ある。

3元分割表における独立性の種類

名称 意味 成立条件
完全独立
$A \perp B \perp C$		 3因子すべて独立 $\theta_{ijk} = \theta_{i..}\theta_{.j.}\theta_{..k}$
周辺独立
$A \perp (B,C)$		 AとBC全体が独立 $\theta_{ijk} = \theta_{i..}\theta_{.jk}$
条件付き独立
$A \perp B | C$		 Cを与えたとき、AとBが独立 $\theta_{ijk} = \theta_{i.k}\theta_{.jk}/\theta_{..k}$

条件付き独立は「見せかけの相関」を検出するうえで重要である。たとえば、A（喫煙）とB（がん）が強く相関していても、C（年齢）で条件付けると独立になるかもしれない（年齢が交絡因子の場合）。

階層モデルと生成集合

対数線形モデルで交互作用を含む場合、その低次の項もすべて含めるという制約を階層性という。階層モデルは生成集合（generators）で表す：

  • AB/BC：因子A×B交互作用とB×C交互作用を含む（自動的に主効果A, B, Cも含む）
  • ABC：3因子交互作用まで含む（飽和モデル）
  • A/B/C：交互作用なし（完全独立モデル）

グラフィカルモデル

グラフィカルモデルとは

変数（因子）を頂点、関係（交互作用）を辺で表す無向グラフを用いて統計モデルを視覚化したものがグラフィカルモデル（graphical model）である。25章で多変量正規分布のグラフィカルモデルを学んだが、ここでは分割表への応用を扱う。

グラフィカルモデルの基本性質

グラフ $G = (V, E)$（$V$：頂点集合＝因子、$E$：辺集合）について：

  1. 性質1 （非連結）：頂点 $A$ と $B$ の間にパスがない
    → $A \perp B$（A と B は独立）
  2. 性質2 （辺なし）：$A$-$B$ 間に辺がない
    → $A \perp B \mid V \setminus \{A,B\}$（他のすべての変数で条件付けると独立）
  3. 性質3 （分離）：頂点集合 $S$ を取り除くと $A$ と $B$ が非連結
    → $A \perp B \mid S$（$S$ で条件付けると独立）

クリークと生成集合の対応

クリーク（clique）とは、グラフにおける極大な完全部分グラフ（全頂点間に辺がある部分グラフで、それ以上拡張できないもの）である。

グラフィカルモデルに対応する階層モデルは、クリークを生成集合とするモデルである。

例：4因子 {A, B, C, D} で辺が A-B, B-C, B-D, C-D の場合：

  • クリーク：{B,C,D}（三角形），{A,B}
  • 生成集合表記：BCD/AB
  • A-C 間に辺がない → $A \perp C \mid \{B, D\}$（性質2）
  • B は A と C の間を分離する → $A \perp C \mid B$（性質3）

グラフィカルモデルを使う利点は、変数の数が多い場合でもグラフの形（クリーク構造）から条件付き独立関係を一目で読み取れることである。これにより、実際のデータから変数間の依存関係の構造を探索するモデル選択が効率的に行える。

公式まとめ

当てはめ値（独立性・一様性の帰無仮説）

$$m_{ij} = \frac{x_{i.} \cdot x_{.j}}{x_{..}}$$

適合度カイ二乗統計量（一般）

$$\chi^2 = \sum_{i}\sum_{j} \frac{(x_{ij} - m_{ij})^2}{m_{ij}} \sim \chi^2((I-1)(J-1)) \quad \text{（漸近的）}$$

2×2分割表の場合：

$$\chi^2 = \frac{x_{..}(x_{11}x_{22} - x_{12}x_{21})^2}{x_{1.}x_{2.}x_{.1}x_{.2}}$$

逸脱度（尤度比検定統計量）

$$G^2 = 2\sum_{i}\sum_{j} x_{ij}\log\frac{x_{ij}}{m_{ij}} \sim \chi^2((I-1)(J-1)) \quad \text{（漸近的）}$$

標本オッズ比と対数オッズ比の標準誤差

$$\widehat{OR} = \frac{x_{11}x_{22}}{x_{12}x_{21}}, \quad \widehat{\text{se}}(\log\widehat{OR}) \approx \sqrt{\frac{1}{x_{11}}+\frac{1}{x_{12}}+\frac{1}{x_{21}}+\frac{1}{x_{22}}}$$

OR の95%信頼区間：$\exp\!\left(\log\widehat{OR} \pm 1.96 \cdot \widehat{\text{se}}(\log\widehat{OR})\right)$

フィッシャー正確検定（超幾何分布）

$$P(X_{11} = k) = \frac{\binom{x_{.1}}{k}\binom{x_{.2}}{x_{1.}-k}}{\binom{x_{..}}{x_{1.}}}, \quad \max(0, x_{1.}+x_{.1}-x_{..}) \leq k \leq \min(x_{1.}, x_{.1})$$

飽和対数線形モデル（2×2分割表）

$$\log\theta_{ij} = \mu + \alpha_i + \beta_j + \gamma_{ij}, \quad \sum_i\alpha_i = \sum_j\beta_j = \sum_i\gamma_{ij} = \sum_j\gamma_{ij} = 0$$

3元分割表の条件付き独立

$$A \perp B \mid C \iff \theta_{ijk} = \frac{\theta_{i.k}\cdot\theta_{.jk}}{\theta_{..k}} \quad \text{（すべての } i,j,k \text{ に対して）}$$

例題・演習

問28.1 後向き研究（低出生体重児12名 vs コントロール36名）のオッズ比解釈

③が正解。後向き研究では各行の比（罹患率）を直接推定できないが、オッズ比は前向き研究のものと一致する。希少事象では相対リスクの推定値としても使える。

①②は誤り：標本サイズ比1:3は研究デザインで定めたもので、罹患率の推定には使えない。

問28.2 同データに対する検定とオッズ比の信頼区間

当てはめ値：$m_{11}=2.5$, $m_{12}=7.5$, $m_{21}=9.5$, $m_{22}=28.5$

  • $\chi^2 = 1.516$、$G^2 = 1.403$（いずれも $\chi^2_{0.05}(1)=3.84$ より小 → 有意でない）
  • $\widehat{OR} = (4 \times 30)/(6 \times 8) = 2.5$
  • $\widehat{\text{se}}(\log\widehat{OR}) = \sqrt{1/4+1/6+1/8+1/30} = 0.758$
  • $\log(2.5) \pm 1.96 \times 0.758 = (-0.570, 2.402)$
  • オッズ比の95%信頼区間：$(e^{-0.570}, e^{2.402}) = (0.566, 11.04)$（1を含む → 有意でない）

問28.3 4頂点 {A,B,C,D} の無向独立グラフ（辺：A-B, B-C, B-D, C-D）

[1] 全頂点が連結 → 独立な因子は存在しない（性質1）

[2] A-C間に辺なし → $A \perp C | \{B,D\}$（性質2）。さらに $\{B\}$ で分離 → $A \perp C | B$（性質3）

まとめ・ポイント

この章の重要ポイント

  • 当てはめ値の普遍性：2群比較（独立二項）・多項分布・独立ポアソンなど、どのモデル設定でも帰無仮説のもとで $m_{ij} = x_{i.}x_{.j}/x_{..}$ となる（重要!）
  • 自由度：I×J分割表の独立性検定は $(I-1)(J-1)$
  • オッズ比の優越性：前向き・後向き研究の両方で一致する推定量。希少事象では相対リスクの近似としても使える
  • $\chi^2$ vs $G^2$：$\chi^2$（ピアソン）は計算が分かりやすく $G^2$（逸脱度）は加法性を持ちモデル比較に有利。大標本では両者は漸近的に等しくなる
  • フィッシャー検定の使いどころ：サンプルが少ない（セルに5未満の当てはめ値がある）場合に漸近論に頼らない正確検定として使う
  • 条件付き独立の見方：交絡因子の影響を取り除いた関係を見るために必要。3元分割表で完全独立・周辺独立・条件付き独立を区別する
  • グラフィカルモデルの強み：変数間の依存構造を無向グラフで視覚化し、クリーク構造から条件付き独立を読み取れる
  • 階層原則：対数線形モデルで交互作用を含む場合、それより低次の項も必ず含める（階層性）
