概要

分割表（contingency table）は、複数の質的変量に関するデータの頻度を集計した表である。本章では、2×2分割表から一般の I×J 分割表、さらに3元分割表へと拡張しながら、オッズ比、適合度カイ二乗検定、尤度比検定、フィッシャー正確検定を学ぶ。

さらに、対数線形モデル・階層モデル・グラフィカルモデルを用いた分割表の体系的なモデリング手法を扱う。グラフィカルモデルでは、無向グラフのクリーク構造から変数間の条件付き独立性を視覚的に読み取ることができる。

重要概念

2群比較と2×2分割表

  • 独立二項分布モデル：$X_i \sim \text{Bin}(n_i, \theta_i)$, $i=1,2$, $X_1$ と $X_2$ は独立
  • オッズ比（odds ratio）：$\psi = \frac{\theta_1/(1-\theta_1)}{\theta_2/(1-\theta_2)}$
  • 相対リスク（relative risk）：$\theta_1/\theta_2$。希少事象では $\psi \approx \theta_1/\theta_2$
  • 前向き研究（prospective study）と後向き研究（retrospective study）：オッズ比は両方で一致

検定手法

  • 適合度カイ二乗検定：$\chi^2 = \sum \frac{(x_{ij} - m_{ij})^2}{m_{ij}}$（漸近的に $\chi^2$ 分布）
  • 尤度比検定（逸脱度）：$G^2 = 2\sum x_{ij}\log\frac{x_{ij}}{m_{ij}}$
  • フィッシャー正確検定：超幾何分布に基づく条件付き検定（漸近論を使わない）

対数線形モデルと3元分割表

  • 対数線形モデル：$\log\theta_{ij} = \mu + \alpha_i + \beta_j + \gamma_{ij}$
  • 完全独立 $A \perp B \perp C$：$\theta_{ijk} = \theta_{i..}\theta_{.j.}\theta_{..k}$
  • 周辺独立 $A \perp (B,C)$：$\theta_{ijk} = \theta_{i..}\theta_{.jk}$
  • 条件付き独立 $A \perp B | C$：$\theta_{ijk} = \theta_{i.k}\theta_{.jk}/\theta_{..k}$
  • 階層モデル：交互作用を含むなら低次の項もすべて含む
  • 生成集合：極大な交互作用の項で階層モデルを表記（例：AC/BC）

グラフィカルモデル

  • クリーク（clique）：極大な完全部分グラフ
  • クリークの集合が生成集合となる階層モデル
  • 性質1：非連結 → 独立
  • 性質2：辺なし → 他の全因子で条件付けて条件付き独立
  • 性質3：$S$ で分離 → $S$ で条件付けて条件付き独立

公式・定理

当てはめ値（一様性・独立性の帰無仮説）

$$m_{ij} = \frac{x_{i.} \cdot x_{.j}}{x_{..}}$$

適合度カイ二乗統計量（2×2分割表）

$$\chi^2 = \frac{x_{..}(x_{11}x_{22} - x_{12}x_{21})^2}{x_{1.}x_{2.}x_{.1}x_{.2}}$$

逸脱度（尤度比検定統計量）

$$G^2 = 2\sum_{i}\sum_{j} x_{ij}\log\frac{x_{ij}}{m_{ij}}$$

標本オッズ比と対数オッズ比の標準誤差

$$\widehat{OR} = \frac{x_{11}x_{22}}{x_{12}x_{21}}, \quad \widehat{\text{se}}(\log\widehat{OR}) \approx \sqrt{\frac{1}{x_{11}}+\frac{1}{x_{12}}+\frac{1}{x_{21}}+\frac{1}{x_{22}}}$$

フィッシャー正確検定（超幾何分布）

$$P(X_{11} = x_{11}) = \frac{\binom{x_{.1}}{x_{11}}\binom{x_{.2}}{x_{1.}-x_{11}}}{\binom{x_{..}}{x_{1.}}}$$

例題・演習

問28.1 後向き研究（低出生体重児12名 vs コントロール36名）のオッズ比解釈

③が正解。後向き研究では各行の比（罹患率）を直接推定できないが、オッズ比は前向き研究のものと一致する。希少事象では相対リスクの推定値としても使える。

①②は誤り：標本サイズ比1:3は研究デザインで定めたもので、罹患率の推定には使えない。

問28.2 同データに対する検定とオッズ比の信頼区間

当てはめ値：$m_{11}=2.5$, $m_{12}=7.5$, $m_{21}=9.5$, $m_{22}=28.5$

  • $\chi^2 = 1.516$、$G^2 = 1.403$（いずれも $\chi^2_{0.05}(1)=3.84$ より小 → 有意でない）
  • $\widehat{OR} = (4 \times 30)/(6 \times 8) = 2.5$
  • $\widehat{\text{se}}(\log\widehat{OR}) = \sqrt{1/4+1/6+1/8+1/30} = 0.758$
  • $\log(2.5) \pm 1.96 \times 0.758 = (-0.570, 2.402)$
  • オッズ比の95%信頼区間：$(e^{-0.570}, e^{2.402}) = (0.566, 11.04)$（1を含む → 有意でない）

問28.3 4頂点 {A,B,C,D} の無向独立グラフ（辺：A-B, B-C, B-D, C-D）

[1] 全頂点が連結 → 独立な因子は存在しない（性質1）

[2] A-C間に辺なし → $A \perp C | \{B,D\}$（性質2）。さらに $\{B\}$ で分離 → $A \perp C | B$（性質3）

メモ

  • オッズ比の重要な性質：前向き研究と後向き研究で一致。希少事象では相対リスクの近似値
  • 多項分布・独立ポアソン分布・行ごとの多項分布のいずれでも、当てはめ値 $m_{ij} = x_{i.}x_{.j}/x_{..}$ は同じ
  • 自由度は $(I-1)(J-1)$ でモデルの設定によらず一致
  • 階層モデルの中でグラフィカルモデルに絞ることで、因子数が多い場合のモデル選択が容易になる
  • 多変量正規分布のグラフィカルモデルは25章参照
