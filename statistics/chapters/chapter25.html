はじめに：「見えない背景にある力」を統計で探す

動機：なぜ変数どうしが「なぜか似た動きをする」のか？

多変量データを分析していると、「なぜかたくさんの変数が一緒に動く」現象に出くわします。 たとえば、あるクラスで数学の試験（「方程式」「微積分」「複素数」「三角関数」「図形」の5問）を行ったとしましょう。 採点結果を眺めると、「方程式が得意な生徒は微積分や複素数も得意」「三角関数が得意な生徒は図形も得意」という 一定のパターンが浮かび上がってきます。

これは単なる偶然でしょうか？いいえ――背景には明快な理由があります。 「解析的な数学能力（計算・論理）」と「幾何的な直感（図形・空間認識）」という、 テストでは直接測れない2つの潜在的な能力が、各問題の点数を決めているからです。

つまり、5つの変数（各問題の得点）の背後に、2つの「見えない変数」（共通因子）が潜んでいると考えることができます。 この構造を明らかにするのが 因子分析（factor analysis） という手法です。

因子分析とは何か（一言定義）

因子分析とは、$p$ 個の観測変数の変動を、 少数の「共通因子（common factor）」と各変数に固有の「独自性因子（unique factor）」 に分解・説明するための多変量解析手法です。 「なぜ変数間に相関があるのか」という生成メカニズムをモデル化します。

主成分分析（23章）との根本的な違い

23章で学んだ主成分分析（PCA）と因子分析はよく混同されますが、目的と考え方が根本的に異なります。

観点 主成分分析（PCA） 因子分析（FA）
目的分散を最大化する軸を求める（次元圧縮）観測変数の背後の共通因子を推定（構造発見）
モデルの仮定不要（記述的・機械的な変換）「因子が変数を生成する」という確率モデルを仮定
独自性（誤差）主成分で説明できない分も全部取り込む各変数固有の「独自性因子」を明示的にモデル化
解釈第1主成分・第2主成分は「最大分散の方向」因子は「解釈可能な共通の潜在概念」
回転通常行わない（固有ベクトルが一意）解釈を容易にするため「回転」を行う（必須）

PCAは「変数の総変動を最もよく表す方向」を探す手法であり、因子の意味は定義しません。 一方、因子分析では「なぜ変数間に相関があるのか」を潜在変数による生成メカニズムで説明しようとします。

因子分析が必要になる典型的な場面

因子分析は以下のような場面で特に力を発揮します：

  • 心理・教育測定：IQテストの各問題が「言語能力」「空間認識」「計算能力」という潜在的な知性を反映しているとモデル化する
  • マーケティング調査：顧客満足度調査の多数の質問項目を「サービス品質」「価格満足」「ブランドイメージ」などの少数の因子にまとめる
  • 経済・金融：多くの経済指標の共変動を「景気循環」「インフレ」などの共通マクロ因子で説明する
  • 神経科学・医学：fMRIデータの多地点活動を少数の「機能的ネットワーク」として捉える

本章で学ぶ内容のロードマップ

本章はさらに範囲を広げて、因子分析に関連する2つの重要な発展的トピックも扱います。

本章の構成

  1. 1因子モデル：最も基本的な因子分析モデル。共通性・独自性・因子スコアを理解する。
  2. K因子モデルと回転：複数の因子を使うとき、解釈しやすい「回転解」を求める手順。
  3. 構造方程式モデル（SEM）：変数間の因果関係をパス図で表現し、操作変数法でパス係数を推定する。
  4. グラフィカルモデル：「条件付き独立」を無向グラフで視覚化し、精度行列との関係を理解する。

因子分析は SEM の特殊ケース（潜在変数 → 観測変数の有向グラフ）であり、SEM はさらにグラフィカルモデルの考え方へと繋がります。本章を通じて「潜在変数と条件付き独立」という統計学の深い概念を理解してください。

① 1因子モデル――基本の仕組みを理解する

モデルの式

$n$ 人の被験者 ($i = 1, \ldots, n$) が $p$ 個の変数 ($j = 1, \ldots, p$) で測られているとします。 各変数はあらかじめ 平均 0・分散 1 に標準化 されているものとします。

1因子モデルは次の式で定義されます。

$$y_{ij} = a_j f_i + d_j u_{ij}$$

各記号の意味：

  • $y_{ij}$：第 $i$ 番目の人の第 $j$ 変数の（標準化後の）値
  • $f_i$：第 $i$ 番目の人の 因子スコア（factor score）。その人が共通因子をどれだけ持っているかを表す潜在変数。$f_i \sim N(0, 1)$ i.i.d. と仮定
  • $a_j$：因子負荷量（factor loading）。第 $j$ 変数が共通因子からどれだけ影響を受けるかを表す係数
  • $u_{ij}$：独自性因子。その変数だけに固有の誤差成分。$u_{ij} \sim N(0, 1)$ i.i.d. で $f_i$ とも他の $u_{i,j'}$ とも独立
  • $d_j$：独自性の大きさを調整するスケール係数

直感的なイメージ

式を日本語で読むと：

「変数 $j$ の点数 $y_{ij}$ は、共通因子 $f_i$（例：数学力）からの影響 $a_j f_i$と、 その変数特有のノイズ $d_j u_{ij}$ の 足し算 である」

因子負荷量 $a_j$ が大きいほど、変数 $j$ は共通因子の影響を強く受けています。 たとえば「方程式」の因子負荷量が 0.92 なら、 ほぼ「数学力」で点数が決まると言えます。 一方「平面図形」の因子負荷量が 0.37 なら、 「数学力」以外の要素（センスや暗記など）が大きく影響しています。

分散の分解と共通性・独自性

$y_{ij}$ の分散はどうなるでしょうか。$f_i$ と $u_{ij}$ は独立で、両者の分散は 1 です。

$$V[y_{ij}] = V[a_j f_i + d_j u_{ij}] = a_j^2 V[f_i] + d_j^2 V[u_{ij}]$$ $$= a_j^2 \cdot 1 + d_j^2 \cdot 1 = a_j^2 + d_j^2$$

さらに $y_{ij}$ は標準化されているので $V[y_{ij}] = 1$ でした。よって：

$$a_j^2 + d_j^2 = 1$$

この式は、変数 $j$ の分散（＝1）が
「共通因子で説明できる部分 $a_j^2$（＝共通性 communality）」と
「共通因子では説明できない固有の部分 $d_j^2 = 1 - a_j^2$（＝独自性 uniqueness）」
に分解されることを意味します。

共通性 $a_j^2$ は「変数 $j$ の変動のうち、共通因子が説明できる割合」です。 $a_j^2$ が 1 に近いほど、その変数は共通因子によくフィットしていることを意味します。

🔁 符号の不定性

因子分析には一つ注意点があります。$(a_j, f_i)$ を解としたとき、$(-a_j, -f_i)$ もまったく同じ $y_{ij} = a_j f_i = (-a_j)(-f_i)$ を与えます。 つまり「因子スコアの正負」と「因子負荷量の正負」は同時に反転させても区別がつきません。 これを 符号の不定性 といいます。解釈のときは文脈から自然な向きを選びます。

因子スコアの推定

因子スコア $f_i$ は直接観測できない潜在変数ですが、データ $y_{ij}$ から推定できます。 簡単な推定式として、加重平均を使う方法があります：

$$\hat{f}_i = \frac{\sum_{j=1}^p a_j y_{ij}}{\sum_{j=1}^p a_j^2}$$

解釈：因子負荷量 $a_j$ を重みとして各変数の値 $y_{ij}$ を加重平均する。 因子負荷量が大きい変数ほど、その人の因子スコア推定に大きく貢献する。

具体例で確認

数学テスト（5問）で1因子モデルを推定したとします。因子負荷量が次のように推定されたとしましょう。

問題 因子負荷量 $a_j$ 共通性 $a_j^2$ 独自性 $d_j^2$
方程式0.92$0.92^2 = 0.846$$1 - 0.846 = 0.154$
微積分0.84$0.84^2 = 0.706$$1 - 0.706 = 0.294$
複素数0.89$0.89^2 = 0.792$$1 - 0.792 = 0.208$
三角関数0.48$0.48^2 = 0.230$$1 - 0.230 = 0.770$
平面図形0.37$0.37^2 = 0.137$$1 - 0.137 = 0.863$

「方程式」「微積分」「複素数」はすべての因子負荷量が 0.8 台と高いです。 これらは「解析的な数学能力」を反映していると考えられます。 一方「三角関数」「平面図形」は因子負荷量が低く（0.5 未満）、 共通因子（数学力）とは別の要因（幾何的センスなど）も大きく影響しています。 → 共通因子は 「解析的な数学力」 と解釈できます。

② K因子モデルと回転――複数の因子を解釈する

K因子モデルへの拡張

実際のデータでは、1つの共通因子だけでは変数間の相関をうまく説明できないことが多くあります。 先ほどの数学テストでも、「解析の能力」と「幾何の能力」という 2つの 潜在的な能力があると考えるほうが自然かもしれません。 これを K因子モデル に一般化します。

$$y_{ij} = a_{j1} f_{i1} + a_{j2} f_{i2} + \cdots + a_{jK} f_{iK} + d_j u_{ij} = \sum_{k=1}^K a_{jk} f_{ik} + d_j u_{ij}$$

$k = 1, \ldots, K$：第 $k$ 因子
$f_{ik} \sim N(0, 1)$ i.i.d.（各因子スコアは独立・標準正規）
$a_{jk}$：変数 $j$ に対する第 $k$ 因子の負荷量
因子が互いに独立のとき（直交回転の場合）、共通性は $$a_{j1}^2 + a_{j2}^2 + \cdots + a_{jK}^2$$

行列による簡潔な表現

$n$ 人 × $p$ 変数のデータ全体を行列でまとめると：

$$X = F A^T + U D$$
$X$$n \times p$ のデータ行列（各行が1人分のデータ）
$F$$n \times K$ の因子スコア行列（各行が1人の因子スコアのベクトル）
$A$$p \times K$ の因子負荷行列（各行が1変数の因子負荷量のベクトル）
$U$$n \times p$ の独自性因子行列
$D$$\text{diag}(d_1, \ldots, d_p)$：$p \times p$ 対角行列

回転の不定性――初期解はなぜ解釈しにくいのか？

K因子モデルには、1因子モデルの「符号の不定性」をさらに広げた問題があります。 $T$ を「$T^T T$ の対角成分がすべて 1」という条件を満たす任意の $K \times K$ 正則行列とすると、

$$X = FA^T + UD = (FT)\bigl(T^{-1}A^T\bigr) + UD$$

この式を見ると、$\tilde{F} = FT$（新しい因子スコア行列）、 $\tilde{A} = A(T^{-1})^T$（新しい因子負荷行列）としても、 まったく同じ $X$ を生成できることがわかります。 つまり $K \geq 2$ のとき、因子スコアと因子負荷量の取り方は無数に存在 します。

💡 幾何的なイメージ

因子負荷量 $(a_{j1}, a_{j2})$ を2次元平面の点でプロットすることを想像してください。 「$T$ を変える」ということは、そのプロットはそのままに、 座標軸（因子の向き）だけを回転させる ことに対応します。 プロットの形（各変数間の相対的位置関係）は変わりませんが、 座標軸の向きによって「各軸が何を意味するか」の解釈が変わります。

最尤推定で得られた最初の解を 初期解 といいますが、 初期解は必ずしも解釈しやすい向きになっているとは限りません。 そこで「解釈しやすい」軸の向きを探す操作が 回転（rotation） です。

直交回転とバリマックス回転

回転の方法には大きく2種類あります。

  • 直交回転（orthogonal rotation）： $T$ を直交行列（$T^T T = I$）に限る回転。 回転後も各因子は互いに直交（独立）のままです。 因子間の相関がないと仮定できる場合に使います。
  • 斜交回転（oblique rotation）： 直交行列以外の $T$ を使う回転。 因子間の相関を許す場合に使います（解釈は複雑になります）。

直交回転の代表例が バリマックス回転（varimax rotation） です。 その定義は：

バリマックス回転：因子負荷行列 $A$ の各要素を2乗してできた $p \times K$ 行列の、各列の分散を最大にする直交回転

👉 直感的には「各因子について、因子負荷量が少数の変数に集中（大きい or 小さい）するような軸の向きを選ぶ」ということです。 こうすると「因子1は変数A・B・Cに効く」「因子2は変数D・Eに効く」という明快な解釈が得られやすくなります。

回転解の例（数学テスト・2因子モデル）

先ほどの数学テストで2因子モデルを推定し、バリマックス回転をかけた結果が以下だとします。

問題 初期解 $\hat{a}_{j1}$ 初期解 $\hat{a}_{j2}$ 回転解 $\tilde{a}_{j1}$ 回転解 $\tilde{a}_{j2}$
方程式0.190.110.230.02
微積分0.310.180.380.03
複素数0.420.190.45−0.02
三角関数−0.290.11−0.030.29
平面図形−0.380.17−0.010.31

初期解では全変数が両方の因子に中途半端にかかっていて解釈が難しいです。 回転解を見ると：

  • 第1因子：「方程式（0.23）」「微積分（0.38）」「複素数（0.45）」に正の影響 → 「解析の能力」
  • 第2因子：「三角関数（0.29）」「平面図形（0.31）」に正の影響 → 「幾何の能力」

このように、回転によって各因子の意味が明確になります。

③ 構造方程式モデル――変数間の因果をパス図で表す

パス図とは何か？

因子分析のパス図のように、頂点（変数）と有向辺（矢線）からなる図を 有向グラフ（directed graph） といいます。 矢線は「Aが Bに影響を与える」という方向を表します。

有向グラフが定義するモデル全体を 構造方程式モデル（structural equation model） といいます。 イメージとしては、矢線の向きに目的変数・説明変数を設定した 複数の線形回帰モデルの組み合わせ です。

具体例：3変数のパス図

標準化された3変数 $X_1, X_2, X_3$（平均0・分散1）が $X_1 \to X_2 \to X_3$ という有向グラフで関係しているとします。 対応する構造方程式は：

$$X_2 = a X_1 + v, \qquad X_3 = b X_2 + u \tag{1}$$
  • $a$：$X_1$ が $X_2$ に与える影響の強さ（パス係数 path coefficient）
  • $b$：$X_2$ が $X_3$ に与える影響の強さ（パス係数）
  • $v$：$X_2$ に固有の誤差項。$X_1$ と無相関
  • $u$：$X_3$ に固有の誤差項。$X_1, X_2$ と無相関

操作変数法――パス係数をどうやって推定するか？

$a$ や $b$ は未知なので、データから推定する必要があります。 使う技法が 操作変数法（instrumental variable method） です。 アイデアは単純です：

「方程式の両辺に適切な変数を掛けて、期待値（平均）をとる」

掛ける変数を 操作変数 といいます。目標はゼロにしたい項（誤差項との積の期待値）を消すことです。

パス係数 $a$ の推定

式 (1) の第1式の両辺に $X_1$ を掛けて期待値をとります。

$$E[X_1 \cdot X_2] = E[X_1 \cdot (aX_1 + v)]$$ $$= a \cdot E[X_1^2] + E[X_1 v]$$

ここで： $E[X_1^2] = V[X_1] = 1$（標準化されているため）、 $E[X_1 v] = 0$（$v$ は $X_1$ と無相関のため）

したがって：

$$\rho_{12} = a \cdot 1 + 0 = a \qquad \therefore\ a = \rho_{12}$$

$\rho_{12}$ は $X_1$ と $X_2$ の（標準化後の）共分散 ＝ 相関係数。標本相関係数で推定できます。

パス係数 $b$ の推定

式 (1) の第2式の両辺に $X_2$ を掛けて期待値をとります。

$$E[X_2 \cdot X_3] = E[X_2 \cdot (bX_2 + u)]$$ $$= b \cdot E[X_2^2] + E[X_2 u]$$

$E[X_2^2] = 1$（標準化）、$E[X_2 u] = 0$（$u$ は $X_2$ と無相関）なので：

$$\rho_{23} = b \cdot 1 + 0 = b \qquad \therefore\ b = \rho_{23}$$

別の操作変数を使う場合――過剰識別

パス係数 $b$ は別の方法でも推定できます。 式 (1) の第2式の両辺に、今度は $X_1$ を操作変数として掛けます：

$$E[X_1 \cdot X_3] = E[X_1 \cdot (bX_2 + u)] = b \cdot E[X_1 X_2] + E[X_1 u]$$ $$\rho_{13} = b \cdot \rho_{12} + 0 \qquad \therefore\ b = \frac{\rho_{13}}{\rho_{12}}$$

これで $b$ について 2通りの推定式 $b = \rho_{23}$ と $b = \rho_{13}/\rho_{12}$ が出てきました。 このように1つのパラメータに複数の推定式が存在する状況を 過剰識別（over-identification） といいます。

💡 過剰識別の重要な意味

このモデルが正しければ、2つの推定式は同じ値を与えるはずです。 すなわち $\rho_{23} = \rho_{13}/\rho_{12}$、つまり $\rho_{13} = \rho_{12}\rho_{23}$ が成立します。 これはパス図 $X_1 \to X_2 \to X_3$ における「間接効果の公式」です：

$$\rho_{13} = a \cdot b$$

$X_1$ と $X_3$ は直接の矢線でつながっておらず、$X_2$ を経由してのみ相関します。 その相関の強さはパス係数の積 $a \times b$ になります（間接効果 indirect effect）。

④ グラフィカルモデル――「条件付き独立」を図で表す

「条件付き独立」とは何か？

前節では有向グラフ（矢線）で変数間の因果を表しました。 今度は視点を変えて、「ある変数を知っているとき（条件付けたとき）に、他の2変数が独立になるか」という関係を考えます。 これを 条件付き独立（conditional independence） といいます。

パス図 $X_1 \to X_2 \to X_3$ の例で考えます。

  • $X_2$ を知らない場合：$X_1$ が高い（勉強時間が長い）と $X_3$ も高い（試験が良い）傾向があります。つまり $X_1$ と $X_3$ は相関する。
  • $X_2$ の値を固定（条件付け）した場合：「理解度が同じ人たちの中」で比べると、勉強時間が長くても試験点数に差はなくなります。→ $X_1$ と $X_3$ は 条件付き独立。

無向グラフとグラフィカルモデルの定義

すべての辺に向きがないグラフを 無向グラフ（undirected graph） といいます。 多変量正規分布 $N(\mathbf{0}, \Sigma)$ に従う変数ベクトル $(X_1, \ldots, X_d)$ を考えます。 無向グラフ $G = (V, E)$（$V$ は頂点集合、$E$ は辺の集合）上で、

グラフィカルモデル（graphical model）の定義：

「$X_i$ と $X_j$ の間に辺がない」 $\Leftrightarrow$ 「$X_i$ と $X_j$ は、それ以外の全変数で条件付けたときに条件付き独立」

グラフィカルモデルは、変数間の条件付き独立構造を無向グラフで視覚化 したものです。 辺なし ＝ 条件付き独立、辺あり ＝ 条件付き依存と読めます。

精度行列との関係

多変量正規分布の場合、グラフィカルモデルを精度行列（共分散行列の逆行列）で特徴付けられます。

$$\Sigma^{-1} = (\sigma^{ij})_{1 \leq i,j \leq d}$$ $$X_i \perp X_j \mid \text{その他すべて} \quad \Longleftrightarrow \quad \sigma^{ij} = 0$$

つまり、精度行列のゼロ成分と辺の欠落が1対1で対応します。 精度行列の $(i,j)$ 成分が 0 ならば $X_i$ と $X_j$ の間に辺がなく、 0 でなければ辺があります。

例えば $X_1 - X_2 - X_3$ という鎖型の無向グラフ（$X_1$ と $X_3$ の間には辺なし）では、 精度行列の $(1,3)$ 成分が 0 になります。

モラルグラフ――有向グラフから無向グラフへの変換

有向グラフ（構造方程式モデル）も、条件付き独立の関係を持ちます。 その関係を無向グラフで表現するための変換が モラルグラフ（moral graph） です。

有向グラフ $G = (V, E)$ のモラルグラフ $\bar{G}$ は次の手順で構成します：

モラルグラフの作り方（3ステップ）

  1. 頂点（変数）はそのままコピーする。
  2. 有向グラフで 矢線が引かれている頂点ペアに、無向辺を引く（矢線を「辺」に変える）。
  3. ある頂点 $v$ に向かって 2本以上の矢線が集まっている（共通の子がいる） 場合、 その「親どうし」の間にも無向辺を引く（「結婚させる」＝ moral の語源）。

例1：鎖型 $X_1 \to X_2 \to X_3$

  • 矢線：$X_1 \to X_2$、$X_2 \to X_3$ → 無向辺：$X_1 - X_2$、$X_2 - X_3$
  • $X_3$ の親は $X_2$ のみ（1本）→ 親どうしの辺は不要
  • モラルグラフ：$X_1 - X_2 - X_3$（$X_1$ と $X_3$ の間には辺がない）
  • 読み取れる条件付き独立：$X_1 \perp X_3 \mid X_2$

例2：合流型（共通の子）$X_1 \to X_3 \leftarrow X_2$（$X_1$ と $X_2$ が独立で $X_3$ を共同で生む）

  • 矢線：$X_1 \to X_3$、$X_2 \to X_3$ → 無向辺：$X_1 - X_3$、$X_2 - X_3$
  • $X_3$ の親は $X_1$ と $X_2$（2本）→ 親どうし $X_1 - X_2$ にも辺を追加
  • モラルグラフ：$X_1 - X_2$、$X_1 - X_3$、$X_2 - X_3$（完全グラフ）
  • 辺なしのペアがないので、条件付き独立なペアはない

✅ 重要な定理：有向グラフ $G$ が定義する構造方程式では、 モラルグラフ $\bar{G}$ が表す条件付き独立関係が成立する。

まとめ――本章の3つのモデルの位置づけ

モデル グラフの種類 表現するもの
因子分析モデル 有向グラフ（特殊） 潜在変数（因子）→ 観測変数の影響
構造方程式モデル 有向グラフ 変数間の因果関係（パス係数）
グラフィカルモデル 無向グラフ 変数間の条件付き独立構造

つまり：因子分析 ⊂ 構造方程式モデル（有向）⟶ モラルグラフ変換 ⟶ グラフィカルモデル（無向） という関係にあります。

⑤ 公式まとめ

1因子モデル

$$y_{ij} = a_j f_i + d_j u_{ij}, \quad V[y_{ij}] = a_j^2 + d_j^2 = 1$$

共通性 $= a_j^2$、独自性 $= d_j^2 = 1 - a_j^2$

$$\hat{f}_i = \frac{\displaystyle\sum_j a_j y_{ij}}{\displaystyle\sum_j a_j^2} \quad \text{（因子スコアの簡易推定）}$$

K因子モデルの行列表現

$$X = FA^T + UD, \quad X = (FT)(T^{-1}A^T) + UD$$

共通性（直交回転時）$= a_{j1}^2 + a_{j2}^2 + \cdots + a_{jK}^2$

構造方程式と操作変数法

$$X_2 = aX_1 + v,\ X_3 = bX_2 + u$$ $$a = \rho_{12},\quad b = \rho_{23} = \frac{\rho_{13}}{\rho_{12}}$$ $$\rho_{13} = a \cdot b \quad \text{（間接効果）}$$

グラフィカルモデル（多変量正規分布）

$$X_i \perp X_j \mid \text{その他} \quad \Longleftrightarrow \quad (\Sigma^{-1})_{ij} = 0$$

例題・演習

以下の問題はすべて自作問題です。教科書の内容を参考にしながら、オリジナルの数値で構成しています。問題文をよく読んでから解答を確認してください。

問1（1因子モデル）

ある高校の理科テストでは、「力学」「電磁気」「熱力学」「光学」の4問題からなる試験を10名の生徒が受験した。 各変数はすでに平均0・分散1に標準化されているものとする。 1因子モデル $y_{ij} = a_j f_i + d_j u_{ij}$ を当てはめたところ、 推定された因子負荷量は以下の表のようになった。

問題 因子負荷量 $a_j$
力学0.8
電磁気0.8
熱力学0.6
光学0.6

次の問いに答えよ。

  1. 各問題の共通性 $a_j^2$ と独自性 $d_j^2 = 1 - a_j^2$ を求めよ。
  2. 共通因子はどのような能力を表していると解釈できるか、理由とともに述べよ。
  3. 3名の生徒 A, B, C の各問題の標準化スコアが以下の表の通りであった。 因子スコアの推定値 $\hat{f}_i = \dfrac{\sum_j a_j y_{ij}}{\sum_j a_j^2}$ を用いて、各生徒の因子スコアを求めよ。
    生徒 力学 $y_1$ 電磁気 $y_2$ 熱力学 $y_3$ 光学 $y_4$
    A1.51.01.51.0
    B−1.0−1.5−1.0−0.5
    C0.50.0−0.51.0

【解答】問1

(1) 共通性と独自性

各問題について $a_j^2$ と $d_j^2 = 1 - a_j^2$ を計算する。

問題 $a_j$ 共通性 $a_j^2$ 独自性 $d_j^2$
力学0.8$0.8^2 = 0.64$$1 - 0.64 = 0.36$
電磁気0.8$0.8^2 = 0.64$$1 - 0.64 = 0.36$
熱力学0.6$0.6^2 = 0.36$$1 - 0.36 = 0.64$
光学0.6$0.6^2 = 0.36$$1 - 0.36 = 0.64$

解釈：「力学」「電磁気」の共通性は 0.64 と高く、「熱力学」「光学」は 0.36 とやや低い。 すなわち、力学と電磁気の変動の 64% が共通因子で説明されるのに対し、熱力学と光学は 36% にとどまる。 独自性が高い問題は、その問題に特有の要因（例：光学は色覚など）が影響していることを示す。

(2) 共通因子の解釈

4問すべての因子負荷量が正（力学・電磁気は 0.8、熱力学・光学は 0.6）であることから、 共通因子はすべての科目に同じ方向の影響を与えている。 特に力学と電磁気への影響が強く、これらは物理の「計算・論理的思考」に関わる問題である。 したがって、共通因子は 「物理全般の基礎的な理解力（理系思考力）」 と解釈できる。

(3) 因子スコアの推定

まず分母の $\sum_j a_j^2$ を計算する：

$$\sum_j a_j^2 = 0.64 + 0.64 + 0.36 + 0.36 = 2.00$$

生徒A（$y_1=1.5,\ y_2=1.0,\ y_3=1.5,\ y_4=1.0$）：

$$\sum_j a_j y_{Aj} = 0.8 \times 1.5 + 0.8 \times 1.0 + 0.6 \times 1.5 + 0.6 \times 1.0$$ $$= 1.20 + 0.80 + 0.90 + 0.60 = 3.50$$ $$\hat{f}_A = \frac{3.50}{2.00} = 1.75$$

生徒B（$y_1=-1.0,\ y_2=-1.5,\ y_3=-1.0,\ y_4=-0.5$）：

$$\sum_j a_j y_{Bj} = 0.8 \times (-1.0) + 0.8 \times (-1.5) + 0.6 \times (-1.0) + 0.6 \times (-0.5)$$ $$= -0.80 - 1.20 - 0.60 - 0.30 = -2.90$$ $$\hat{f}_B = \frac{-2.90}{2.00} = -1.45$$

生徒C（$y_1=0.5,\ y_2=0.0,\ y_3=-0.5,\ y_4=1.0$）：

$$\sum_j a_j y_{Cj} = 0.8 \times 0.5 + 0.8 \times 0.0 + 0.6 \times (-0.5) + 0.6 \times 1.0$$ $$= 0.40 + 0.00 - 0.30 + 0.60 = 0.70$$ $$\hat{f}_C = \frac{0.70}{2.00} = 0.35$$
✅ まとめ：$\hat{f}_A = 1.75 > \hat{f}_C = 0.35 > \hat{f}_B = -1.45$
生徒Aが最も「物理全般の理解力」が高く、生徒Bが最も低い。 因子スコアが正の生徒は平均より理解力が高く、負の生徒は低いことを意味する。

問2（2因子モデルと回転解の解釈）

あるファミリーレストランチェーンの顧客満足度調査で、 5項目（接客態度・対応速度・メニューの豊富さ・価格の手ごろさ・清潔感）について アンケートを実施した。2因子モデルをバリマックス回転で推定したところ、 因子負荷行列の回転解は次の通りであった（値はすべて小数第2位まで）。

項目 第1因子 $\hat{a}_{j1}$ 第2因子 $\hat{a}_{j2}$
接客態度0.850.10
対応速度0.780.15
メニューの豊富さ0.120.82
価格の手ごろさ−0.080.76
清潔感0.450.48

次の問いに答えよ。ただし因子は互いに独立と仮定する（直交回転）。

  1. 各項目の共通性 $a_{j1}^2 + a_{j2}^2$ を求めよ。
  2. 第1因子・第2因子それぞれがどのような概念を表すか、回転解の数値をもとに解釈せよ。
  3. 「清潔感」の共通性が他の項目より低いことの意味を述べよ。

【解答】問2

(1) 各項目の共通性

直交回転を用いているので、共通性 $= a_{j1}^2 + a_{j2}^2$ となる。

項目 $a_{j1}^2$ $a_{j2}^2$ 共通性
接客態度 $0.85^2 = 0.7225$ $0.10^2 = 0.0100$ 0.7325
対応速度 $0.78^2 = 0.6084$ $0.15^2 = 0.0225$ 0.6309
メニューの豊富さ $0.12^2 = 0.0144$ $0.82^2 = 0.6724$ 0.6868
価格の手ごろさ $(-0.08)^2 = 0.0064$ $0.76^2 = 0.5776$ 0.5840
清潔感 $0.45^2 = 0.2025$ $0.48^2 = 0.2304$ 0.4329

(2) 各因子の解釈

回転解を見ると：

  • 第1因子：「接客態度（0.85）」「対応速度（0.78）」が大きな正の値を持ち、「清潔感（0.45）」も中程度の値。 「メニューの豊富さ」「価格の手ごろさ」はほぼ0。 → 第1因子は「接客・サービスの質」を表していると解釈できる。
  • 第2因子：「メニューの豊富さ（0.82）」「価格の手ごろさ（0.76）」が大きな正の値を持ち、「清潔感（0.48）」も中程度。 「接客態度」「対応速度」はほぼ0。 → 第2因子は「コストパフォーマンス（費用対効果）」を表していると解釈できる。

(3) 「清潔感」の共通性が低いことの意味

「清潔感」の共通性は 0.4329 であり、他の項目（0.58〜0.73）と比べて低い。 これは「清潔感」の変動の約 43% しか2因子（接客の質・コスパ）で説明できていないことを意味する。 残りの約 57%（独自性 $d_j^2 = 1 - 0.4329 = 0.5671$）は、 これら2因子に含まれない「清潔感」固有の要因（例：厨房・トイレの衛生管理など）によるものである。 2因子では「清潔感」を十分に説明できていないことを示唆しており、 もう1つ追加の因子（例：「店舗環境の質」）が存在するかもしれない。

✅ ポイント：共通性が低い変数は、抽出した因子では説明できない固有の情報を多く含んでいる。 因子数を増やすことで共通性が上がる可能性がある。

問3（構造方程式モデルと操作変数法）

3変数 $X_1$（1日の勉強時間）、$X_2$（内容の理解度）、$X_3$（定期試験の得点）の関係を 次の有向グラフが定義する構造方程式を考える。

$X_1 \;\longrightarrow\; X_2 \;\longrightarrow\; X_3$

構造方程式は次の通りとする。

$$X_2 = a X_1 + v, \qquad X_3 = b X_2 + u \tag{*}$$

ここで $X_1, X_2, X_3$ は平均0・分散1に標準化されており、 $v$ は $X_1$ と無相関、$u$ は $X_1, X_2$ と無相関と仮定する。 また、3変数の相関係数（＝共分散）が以下の通りであった。

$$\rho_{12} = 0.60, \qquad \rho_{23} = 0.70, \qquad \rho_{13} = 0.42$$

次の問いに答えよ。

  1. 操作変数法を用いて、パス係数 $a$ および $b$ を推定せよ。
    ヒント：$(*)$ の第1式の両辺に $X_1$ を掛けて期待値をとると $a$ が求まる。同様に第2式で $b$ が求まる。
  2. $\rho_{13} = a \cdot b$ が成立することを確かめよ。
  3. 上記の有向グラフのモラルグラフ（無向グラフ）を描き、 そこから読み取れる条件付き独立関係を述べよ。

【解答】問3

(1) パス係数の推定（操作変数法）

パス係数 $a$ の推定（第1式：$X_2 = aX_1 + v$）

両辺に $X_1$ を掛けて期待値をとる：

$$E[X_1 X_2] = a \cdot E[X_1^2] + E[X_1 v]$$

ここで $X_1$ は標準化されているので $E[X_1^2] = V[X_1] = 1$、 また $v$ は $X_1$ と無相関なので $E[X_1 v] = 0$。よって：

$$\rho_{12} = a \cdot 1 \quad \Longrightarrow \quad \boxed{a = \rho_{12} = 0.60}$$

パス係数 $b$ の推定（第2式：$X_3 = bX_2 + u$）

両辺に $X_2$ を掛けて期待値をとる：

$$E[X_2 X_3] = b \cdot E[X_2^2] + E[X_2 u]$$

$E[X_2^2] = 1$（標準化）、$E[X_2 u] = 0$（$u$ と $X_2$ が無相関）より：

$$\rho_{23} = b \cdot 1 \quad \Longrightarrow \quad \boxed{b = \rho_{23} = 0.70}$$

(2) $\rho_{13} = a \cdot b$ の確認

$a \cdot b$ を計算し、観測された $\rho_{13}$ と比較する：

$$a \cdot b = 0.60 \times 0.70 = 0.42 = \rho_{13} \quad \checkmark$$

直感的な説明：$X_1$ と $X_3$ の間には直接の矢線がなく、$X_2$ を経由してのみつながっている。 したがって $X_1$ と $X_3$ の相関は、パス係数の積 $a \cdot b$ に等しくなる。 これは 間接効果（indirect effect） の考え方であり、 「勉強時間が試験点数に与える影響は、理解度を通じた間接的な影響のみ」を数値で表している。

なお、$X_1$ を操作変数として第2式の両辺に掛けると：

$$E[X_1 X_3] = b \cdot E[X_1 X_2] + E[X_1 u]$$ $$\rho_{13} = b \cdot \rho_{12} + 0 \quad \Longrightarrow \quad b = \frac{\rho_{13}}{\rho_{12}} = \frac{0.42}{0.60} = 0.70$$

これは $(1)$ で求めた $b = \rho_{23} = 0.70$ と一致する（過剰識別）。

(3) モラルグラフの構成と条件付き独立

有向グラフ $X_1 \to X_2 \to X_3$ からモラルグラフを構成する手順：

  • 同じ頂点集合 $\{X_1, X_2, X_3\}$ を持つ無向グラフを作る。
  • 有向グラフで矢線が引かれている頂点間に無向辺を引く：$X_1$–$X_2$、$X_2$–$X_3$。
  • 各頂点の「親」が複数いる場合は親どうしも結ぶ（今回は親が1つなので省略）。

モラルグラフ：

$X_1 \;{-}\; X_2 \;{-}\; X_3$　（無向辺）

このグラフでは $X_1$ と $X_3$ の間に直接の辺がない。 グラフィカルモデルの定義より、 $X_1$ と $X_3$ は $X_2$ を与えたとき（$X_2$ で条件付けたとき）条件付き独立 である。

数式で確認：精度行列 $\Sigma^{-1}$ の $(1,3)$ 成分が 0 であることが条件付き独立に対応する。

✅ まとめ
$a = 0.60$（勉強時間 → 理解度）、$b = 0.70$（理解度 → 試験点数）
$\rho_{13} = 0.60 \times 0.70 = 0.42$ ✓（間接効果のみ）
条件付き独立：$X_1 \perp X_3 \mid X_2$（理解度を知っていれば、勉強時間は試験点数に追加情報を与えない）

まとめ・ポイント

  • 因子分析は主成分分析（23章）と異なり、背景にモデルを仮定する。共通因子による説明と独自性に分離するのが特徴
  • $K \geq 2$ の場合、初期解は解釈が困難なので回転が必要。バリマックス回転が代表的
  • パス図では矢線の向き（因果の方向）が重要。構造方程式は回帰モデルの集まりと考えればよい
  • グラフィカルモデル：$\Sigma^{-1}$ のゼロ要素 $\leftrightarrow$ 条件付き独立 $\leftrightarrow$ 辺なし
  • 分割表のグラフィカルモデルは28章参照
