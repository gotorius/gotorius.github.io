はじめに：なぜ「全数調査」ではなく「標本調査」なのか

統計学を学んでいると「母集団のパラメータ（平均・分散など）を知りたい」という目標が繰り返し登場します。 では、なぜ全国民・全製品・全患者をすべて調べないのでしょうか？ この問いへの答えが標本調査法の出発点です。

全数調査（悉皆調査）の限界

  • コストと時間：日本全国の有権者（約1億人）に調査票を送り、回収・集計するには莫大な費用と時間がかかります
  • 破壊検査の問題：電球の寿命を調べるには点灯させて壊すしかなく、全製品を検査したら売る物がなくなります
  • 動的母集団：調査中に新たに生まれる・死亡するなど、母集団が動き続ける場合は全数把握が不可能です

したがって「母集団の一部（標本）を上手く取り出し、そこから母集団全体を推測する」 標本調査（sample survey）が科学・行政・医療・工業のあらゆる場面で使われています。

標本の「代表性」と無作為性

どんな方法で個体を選んでも良いわけではありません。 「道を歩いていた人に声をかけた」「雑誌の読者アンケートの返信のみを集計した」という方法は 母集団全体を代表していない可能性があります（標本の偏り＝バイアス）。

有名な歴史的失敗例：1936年のアメリカ大統領選挙で、大手雑誌が240万通のアンケートを元に「ランドン勝利」と予測しましたが、 実際にはルーズヴェルトが圧勝しました。雑誌購読者は富裕層に偏っており、母集団（全有権者）を代表していなかったのです。

無作為（ランダム）抽出の意義

バイアスをなくすには、母集団のすべての個体に均等な「選ばれる確率」を与える必要があります。 これが無作為抽出（random sampling）の基本的な考え方です。 抽出に確率的なメカニズムを導入することで、区間推定・仮説検定の理論的根拠が得られます。

一方、調査者の判断で「代表的」と思われる個体を選ぶ 有意抽出法（purposive sampling）はバイアスが入りやすく、 統計的推論の保証がありません。

この章で学ぶこと

  1. 単純無作為抽出法：最も基本的な抽出法。非復元の場合の標本平均の分散（有限修正）と必要サンプルサイズ
  2. さまざまな抽出法：集落・二段・層化・系統的抽出法の特徴と違い
  3. 層化抽出法の詳細：各層から独立抽出する際の推定量と分散公式
  4. 標本配分法：各層に何人割り当てるか（比例・等・ネイマン配分）

キーワード：単純無作為抽出法、非復元抽出、復元抽出、有限修正、層化抽出法、二段抽出法、集落抽出法、系統的抽出法、比例配分法、ネイマン配分法

単純無作為抽出法

定義

大きさ $N$ の母集団から大きさ $n$ の標本を抽出するとき、 すべての $\binom{N}{n}$ 通りの標本が同じ確率で選ばれる抽出方法を 単純無作為抽出法（simple random sampling）という。

  • 非復元抽出（without replacement）：一度選んだ個体を戻さず抽出。同一個体は1回しか選ばれない
  • 復元抽出（with replacement）：選んだ個体を戻してから次を抽出。同一個体が複数回選ばれる可能性あり

標本平均の分散（非復元抽出）

母平均 $\mu$、母分散 $\sigma^2$ の母集団から非復元抽出した $x_1, \ldots, x_n$ の標本平均 $\bar{x}$ について：

$$E[\bar{x}] = \mu, \quad V[\bar{x}] = \frac{N-n}{N-1}\cdot\frac{\sigma^2}{n}$$
  • $\frac{N-n}{N-1}$：有限修正（finite correction）項
  • $n=N$（全数調査）のとき有限修正項＝0 → 分散もゼロ（確実に母平均と一致）
  • 復元抽出・無限母集団の場合：$V[\bar{x}] = \sigma^2/n$（有限修正なし）

有限修正の直感

母集団10人から10人全員調べれば、標本平均は必ず母平均と一致し分散は0。 母集団100万人から100人取っても0.01%しかカバーできず有限修正項は約1（ほぼ無視）。 有限修正は $n/N > 0.1$ くらいから実質的な効果が出てくる。

必要サンプルサイズ

$V[\bar{x}] \leq C$（許容誤差）を満たすには：

$$n \geq \frac{N\sigma^2}{\sigma^2 + (N-1)C}$$

$N \to \infty$ の極限では $n \geq \sigma^2/C$ となり、母集団の大きさによらない （超大規模調査でも必要標本サイズは変わらない）。

さまざまな標本抽出法

単純無作為抽出は理論的に明快ですが、母集団が大規模だと実施コストが高くなります。 また推定精度をさらに上げたい・調査コストを下げたい場合には以下の方法が使われます。

①集落抽出法（cluster sampling）

母集団をあらかじめ集落（クラスター）というグループに分け、 いくつかの集落を抽出して選ばれた集落のすべての調査単位を調査する。

例：学校の全生徒調査

ある県の全高校生から1000人を調査したいとき、各クラスを集落とし、無作為に10クラスを選んでクラス全員を調査する（1クラス100人と仮定）。 → 調査する生徒がまとまっているためコスト削減。ただしどのクラスが選ばれるかで結果が大きく変わる可能性がある。

②二段抽出法（two-stage sampling）

第1次抽出単位（グループ）を選び、その中からさらに一部の調査単位（第2次抽出単位）を選ぶ。

  • 集落との違い：集落は「全員」調査するが、二段抽出はさらに「一部のみ」抽出
  • 例：都市の世帯調査で、まず町丁目を抽出し（第1段）、各町丁目から一部の世帯を抽出（第2段）
  • さらに第3・第4段まで繰り返す「多段抽出法（multistage sampling）」も大規模調査で使われる

③層化抽出法（stratified sampling）

母集団をあらかじめ層（stratum）というグループに分け、 すべての層から決められた大きさの標本を抽出する。

  • 集落・二段抽出と異なり母集団全体から満遍なく抽出できる
  • 層の作り方の原則：層内はできるだけ均質に、異なる層はできるだけ異質に → 分散が小さくなり精度向上
  • 例：有権者調査で小選挙区を層とし、各選挙区から一定数を抽出

④系統的抽出法（systematic sampling）

調査単位がリスト化されている場合、リスト上で等間隔に選ぶ。

  • 例：名簿1万人から100人 → 乱数で1〜100の中から開始番号を決め（例：51番）、以後100番おきに151, 251, …と選ぶ
  • 乱数を何度も使わず実施が簡単。母集団全体からある程度満遍なく取れる
  • 注意：リストが周期的に並んでいると偏りが生じる（例：曜日ごとの売上リストで7の倍数おきに抽出）

抽出法まとめ

抽出法特徴メリット注意点
単純無作為全員に等確率理論が明快、バイアスなし大規模では費用大
集落抽出クラスター丸ごと調査コスト低クラスター間差が大きいと精度低
二段抽出2段階に分けて抽出柔軟、コスト適度第1・第2段の設計が複雑
層化抽出全層から抽出精度向上、代表性高層の設定が重要
系統的抽出等間隔実施が簡単リストの周期性に注意

層化抽出法の詳細

設定

大きさ $N$ の母集団が $L$ 個の層に分けられており、層 $h$ の大きさは $N_h$（$\sum_{h=1}^{L}N_h = N$）。 層 $h$ から独立に大きさ $n_h$ の標本を非復元無作為抽出し、得られた値を $y_{hi}$（$i=1,\ldots,n_h$）とする。

推定量と分散

母平均の推定量として層化推定量を用いる：

$$\bar{x}_{\text{st}} = \sum_{h=1}^{L}\frac{N_h}{N}\bar{y}_h, \quad \bar{y}_h = \frac{1}{n_h}\sum_{i=1}^{n_h}y_{hi}$$

この推定量は不偏（$E[\bar{x}_{\text{st}}] = \mu$）であり、その分散は：

$$V[\bar{x}_{\text{st}}] = \sum_{h=1}^{L}\left(\frac{N_h}{N}\right)^2\cdot\frac{N_h - n_h}{N_h - 1}\cdot\frac{\sigma_h^2}{n_h}$$

ここで $\sigma_h^2$ は層 $h$ の母分散、$\frac{N_h-n_h}{N_h-1}$ は各層の有限修正項。

標本配分法

各層から何人抽出するか（$n_h$ の決め方）を標本配分法（sample allocation）という。合計 $n = \sum_{h=1}^{L}n_h$。

①比例配分法（proportional allocation）

各層の標本サイズを母集団での層の大きさに比例させる：

$$n_h = \frac{N_h}{N} \times n$$

自然で直感的な方法。有限修正を無視すると $V[\bar{x}_{\text{st}}] \leq V[\bar{x}]$ が保証される（単純無作為以上の精度）。

②等配分法（equal allocation）

$$n_h = \frac{n}{L}$$

各層の推定精度を均等に保ちたい場合（例：各都道府県の推定値の精度を揃えたい）に有効。ただし層サイズが大きく異なると全体精度が低下することもある。

③ネイマン配分法（Neyman allocation）＝最適配分法

「大きい層からも、ばらつきの大きい層からも多く取る」→ $V[\bar{x}_{\text{st}}]$ を最小化：

$$n_h = \frac{N_h\sigma_h}{\sum_{k=1}^{L}N_k\sigma_k} \times n$$

3つの配分法の比較

配分法$n_h$ の決め方使いどころ
比例配分$\propto N_h$層のばらつきが均等な場合など一般的
等配分各層 $n/L$ 人各層の推定精度を揃えたい場合
ネイマン配分$\propto N_h\sigma_h$全体の推定精度最大（$\sigma_h$ が既知か推定できる場合）

ネイマン配分は分散最小だが、$\sigma_h$ が事前にわからないと使えない → 過去の調査値で代用することが多い。

公式まとめ

非復元単純無作為抽出の標本平均

$$E[\bar{x}] = \mu, \quad V[\bar{x}] = \frac{N-n}{N-1}\cdot\frac{\sigma^2}{n}$$

必要サンプルサイズ（$V[\bar{x}] \leq C$）

$$n \geq \frac{N\sigma^2}{\sigma^2 + (N-1)C}$$

層化推定量と分散

$$\bar{x}_{\text{st}} = \sum_{h=1}^{L}\frac{N_h}{N}\bar{y}_h, \quad V[\bar{x}_{\text{st}}] = \sum_{h=1}^{L}\left(\frac{N_h}{N}\right)^2\frac{N_h-n_h}{N_h-1}\cdot\frac{\sigma_h^2}{n_h}$$

配分法まとめ

配分法$n_h$
比例配分$(N_h/N)\times n$
等配分$n/L$
ネイマン配分$\dfrac{N_h\sigma_h}{\sum_k N_k\sigma_k}\times n$

例題・演習

問21.1：集落抽出法の識別

問題

ある都市でいくつかの世帯を抽出して世帯主を調査したい。「集落抽出法」を説明している記述を①〜⑤から選べ。

  1. 市内の世帯に連番を付け、コンピュータでランダムな番号を発生させ、その番号の世帯を抽出する。
  2. 市内をいくつかのグループにまとめ、第一段階としてグループを抽出し、選ばれたグループに含まれる世帯をすべて調査する。
  3. 属性（地域・家族構成・職業など）によりグループに分け、それぞれのグループから世帯を無作為に抽出する。
  4. 市内の世帯の中で、役員経験のある世帯を抽出する。
  5. 市内をいくつかのグループにまとめ、第一段階としてグループを抽出し、選ばれたグループの中からさらに世帯を抽出する。
解答

答：②

  • ①→ 単純無作為抽出法
  • ②→ 集落抽出法（グループ選択→全員調査）
  • ③→ 層化抽出法（属性で層化し各層から一部抽出）
  • ④→ 有意抽出法（主観的選択）
  • ⑤→ 二段抽出法（グループ選択→さらに一部抽出）

②と⑤の違い：②は選んだグループの全員を調査（集落）、⑤はさらに一部のみ抽出（二段）。

例1：英語テストの標本設計

問題

$N=1600$ 人の母集団から非復元単純無作為抽出で英語テストを受験させる。母分散 $\sigma^2=120$ と仮定する。

  1. $n=100$ 人を抽出するときの標本平均の分散を求めよ（小数第4位まで）。
  2. 標本平均の分散を0.5以下にするには何人以上の抽出が必要か。
解答

(1)

$$V[\bar{x}] = \frac{1600-100}{1600-1}\cdot\frac{120}{100} = \frac{1500}{1599}\times 1.2 \approx \mathbf{1.1257}$$

(2) $V[\bar{x}] \leq 0.5$ となるには：

$$n \geq \frac{1600\times 120}{120 + (1600-1)\times 0.5} = \frac{192000}{919.5} \approx 208.8$$

よって 209人以上を抽出する必要がある。

例2：層化抽出の比例・ネイマン配分

問題

2つの層があり $N_1=500, N_2=500, \sigma_1=6, \sigma_2=18$。合計 $n=20$ を抽出したい。 比例配分法とネイマン配分法でそれぞれの層から何人抽出するか求めよ。

解答

比例配分：$n_1 = \dfrac{500}{1000}\times 20 = 10, \quad n_2 = 10$

ネイマン配分（有限修正無視）：

$$n_1 = \frac{500\times 6}{500\times 6 + 500\times 18}\times 20 = \frac{3000}{12000}\times 20 = 5$$

$n_2 = 20 - 5 = 15$

層2のばらつきが層1の3倍（$\sigma_2/\sigma_1=3$）なので、ネイマン配分では層2から3倍多く取る。これにより $V[\bar{x}_{\text{st}}]$ が最小化される。

問21.2：有限修正と分散の比較

問題

$N=9585, n=600, \sigma^2=420$ のとき、非復元抽出の場合の標本平均の分散 $V_1$ と復元抽出（無限母集団）の場合の分散 $V_2$ を求めよ。

解答

$V_1 = \dfrac{9585-600}{9585-1}\cdot\dfrac{420}{600} = \dfrac{8985}{9584}\times 0.7 \approx \mathbf{0.6563}$

$V_2 = \dfrac{420}{600} = \mathbf{0.7}$

有限修正項 $\approx 0.9375$ → 非復元抽出の分散が復元抽出より約6.25%小さい。

まとめ・ポイント

  • 有限修正項：$n/N$ が小さいときはほぼ1（影響ほぼなし）。$n/N > 0.1$ くらいから効いてくる
  • 比例配分法は単純無作為抽出と比べて推定精度が悪化しない（$V_{\text{st}} \leq V_{\text{srs}}$）
  • ネイマン配分法は分散最小だが $\sigma_h$ が既知でないと使えない → 過去の調査値で代用
  • 層化抽出の原則「層内均質・層間異質」を守るほど精度が上がる
  • 集落抽出法と二段抽出法の違い：集落は全員調査、二段はさらに一部抽出
  • 公的統計（国勢調査・家計調査等）では層化二段抽出法が広く使われる
  • 系統的抽出法：実施は簡単だがリストの周期性に注意
