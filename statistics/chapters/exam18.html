第18章「質的回帰」のキーポイント

第18章では，応答変数が2値（0/1）やカウント（非負整数）などの「質的」な場合に使う回帰モデルを学びます．

モデル 応答変数の型 リンク関数 確率のモデル式
ロジスティック回帰 2値（0/1） ロジット $\log\dfrac{\pi}{1-\pi}$ $\pi = \dfrac{e^{\eta}}{1+e^{\eta}}$
プロビットモデル 2値（0/1） プロビット $\Phi^{-1}(\pi)$ $\pi = \Phi(\eta)$
ポアソン回帰 カウント（$\geq 0$） 対数 $\log\mu$ $\mu = e^{\eta}$

ここで $\eta = \beta_0 + \beta_1 x_1 + \cdots + \beta_d x_d$ は線形予測子，$\Phi$ は標準正規分布の累積分布関数です．

問18.1 [1] — 寛解確率が 0.5 となる LI 値・LI = 30 の寛解確率

解答・解説
ロジスティック回帰モデルの基礎

2値の応答変数 $y \in \{0, 1\}$ に対し，$y=1$ となる確率 $\pi$ を説明変数 $x$ でモデル化するのがロジスティック回帰です．

$$\log\frac{\pi}{1-\pi} = \beta_0 + \beta_1 x$$

左辺 $\log\dfrac{\pi}{1-\pi}$ をロジット（logit）といいます．推定値 $\hat\beta_0,\,\hat\beta_1$ が得られたとき，確率の推定値は：

$$\hat\pi = \frac{e^{\hat\beta_0 + \hat\beta_1 x}}{1 + e^{\hat\beta_0 + \hat\beta_1 x}} = \frac{1}{1 + e^{-(\hat\beta_0 + \hat\beta_1 x)}}$$

推定された回帰式（推定量を代入）：

$$\log\frac{\hat\pi}{1-\hat\pi} = -3.77714 + 0.14486 \times \text{LI}$$

寛解確率 $\hat\pi = 0.5$ となる LI 値を求める

$\hat\pi = 0.5$ のとき，オッズは $\dfrac{0.5}{1-0.5} = 1$ となり，$\log(\text{オッズ}) = 0$ です．よって右辺を 0 とおいて：

$$-3.77714 + 0.14486 \times \text{LI} = 0$$ $$\text{LI} = \frac{3.77714}{0.14486} \approx 26.07$$

LI = 30 のときの寛解確率を推定する

まずロジット値を計算する：

$$\log\frac{\hat\pi}{1-\hat\pi} = -3.77714 + 0.14486 \times 30 = -3.77714 + 4.3458 = 0.56866$$

次に確率に変換する：

$$\hat\pi = \frac{e^{0.56866}}{1 + e^{0.56866}} = \frac{1.766}{2.766} \approx 0.638$$

寛解確率が 0.5 となるのは LI 値がおよそ 26 のときである．また，LI 値が 30 の患者の寛解確率はおよそ 0.64 と推定される．

問18.1 [2] — LI 値が 2 増えるときのオッズ倍率

解答・解説
オッズ比（Odds Ratio）の考え方

ロジスティック回帰では，説明変数が 1 増えたときのオッズの変化は $e^{\hat\beta_1}$ 倍です．これをオッズ比といいます．一般に，LI が $k$ 増えたときのオッズ倍率は $e^{\hat\beta_1 \times k}$ です．

LI が 2 増えたときのオッズ比を計算する

オッズの関係式を書くと：

$$\frac{\hat\pi}{1-\hat\pi} = e^{-3.77714 + 0.14486 \times \text{LI}} = e^{-3.77714} \times \bigl(e^{0.14486}\bigr)^{\text{LI}} = e^{-3.77714} \times (1.15588)^{\text{LI}}$$

LI が $k$ から $k+2$ に変化したときの比は：

$$\frac{e^{-3.77714} \times (1.15588)^{k+2}}{e^{-3.77714} \times (1.15588)^{k}} = (1.15588)^2 \approx 1.336$$

LI 値が 2 増えるとき，寛解のオッズはおよそ 1.34 倍になると考えられる．

オッズ比の直感的な解釈
オッズ比が 1.34 倍ということは，「LI 値が 2 高いと，寛解する確率が寛解しない確率に対して1.34倍大きい」という意味です．確率そのものは「LI に関係なく一定に増える」わけではなく，オッズが一定の倍率で増えます（この点がロジスティック回帰の特徴）．

問18.2 [1] — 喫煙・肥満・いびきがすべて「あり」の男性が高血圧である確率

解答・解説
多変数ロジスティック回帰の基礎

説明変数が複数の場合も，ロジット変換した確率が線形モデルになるという構造は同じです：

$$\log\frac{\pi}{1-\pi} = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + \cdots + \beta_d x_d$$

喫煙（$x_1$），肥満（$x_2$），いびき（$x_3$）とすると，推定された回帰式：

$$\log\frac{\hat\pi}{1-\hat\pi} = -2.37766 - 0.06777\,x_1 + 0.69531\,x_2 + 0.87194\,x_3$$

$x_1 = x_2 = x_3 = 1$ を代入する

$$\log\frac{\hat\pi}{1-\hat\pi} = -2.37766 - 0.06777 + 0.69531 + 0.87194 = -0.87818$$ $$\hat\pi = \frac{e^{-0.87818}}{1 + e^{-0.87818}} = \frac{0.41556}{1.41556} \approx 0.2936$$

喫煙，肥満，いびきがすべて「あり」の男性が高血圧である確率の推定値はおよそ 0.29（約29%）である．

問18.2 [2] — 肥満でない男性に対する肥満の男性の高血圧リスク（オッズ比）

解答・解説

$x_1, x_3$ を固定して $x_2$ の効果を取り出す

喫煙（$x_1$）といびき（$x_3$）の値を任意の定数 $a, c$ に固定すると：

肥満あり（$x_2 = 1$）のオッズ： $\displaystyle\frac{\hat\pi}{1-\hat\pi}\Big|_{x_2=1} = e^{-2.37766 - 0.06777a + 0.69531 + 0.87194c}$

肥満なし（$x_2 = 0$）のオッズ： $\displaystyle\frac{\hat\pi}{1-\hat\pi}\Big|_{x_2=0} = e^{-2.37766 - 0.06777a + 0.87194c}$

比をとると：

$$\text{オッズ比} = \frac{\text{肥満ありのオッズ}}{\text{肥満なしのオッズ}} = e^{0.69531} \approx 2.00$$

肥満でない男性に対する肥満の男性の高血圧リスク（オッズ比）の推定値は $e^{0.69531} \approx \mathbf{2.00}$ 倍である．

他の変数を固定したオッズ比
ロジスティック回帰では，他の説明変数の値によらず，$x_2$ が 0 から 1 に変わったときのオッズ比は常に $e^{\hat\beta_2}$ になります（この値が $x_1, x_3$ の値に依存しない点がロジスティック回帰の特徴です）．

問18.2 [3] — 肥満に対する高血圧のオッズ比の近似的な 95% 信頼区間

解答・解説
オッズ比の信頼区間の求め方

$\beta_k$ の 95% 信頼区間 $[\hat\beta_k - 1.96 \cdot \text{SE}_k,\; \hat\beta_k + 1.96 \cdot \text{SE}_k]$ を指数変換するとオッズ比の 95% 信頼区間が得られます：

$$\text{OR の 95\% CI} = \left[e^{\hat\beta_k - 1.96\, \text{SE}_k},\; e^{\hat\beta_k + 1.96\, \text{SE}_k}\right]$$

係数 $\hat\beta_2$ の 95% 信頼区間を計算し，指数変換する

出力結果より，$\hat\beta_2 = 0.69531$，$\text{SE}(\hat\beta_2) = 0.26509$．

$\hat\beta_2$ の近似的な 95% 信頼区間（正規近似）：

$$\hat\beta_2 \pm 1.96 \times \text{SE}(\hat\beta_2) = 0.695 \pm 1.96 \times 0.265 = 0.695 \pm 0.519$$ $$\Rightarrow [0.695 - 0.519,\; 0.695 + 0.519] = [0.176,\; 1.214]$$

オッズ比の 95% 信頼区間：

$$\left[e^{0.176},\; e^{1.214}\right] \approx [1.19,\; 3.37]$$

肥満に対する高血圧のオッズ比の近似的な 95% 信頼区間はおおむね $[\mathbf{1.2},\; \mathbf{3.4}]$ 程度となる．

信頼区間が 1 を含まない ⟺ 有意水準 5% で有意
オッズ比の 95% 信頼区間が 1 を含まないということは，$\beta_2$ の 95% 信頼区間が 0 を含まないことと同値です．これは $z$ 値を用いた近似的な検定の $p$ 値が 0.05 未満であることに対応し，「肥満は高血圧に有意に関連する」と判断できます（出力では $p = 0.0147 < 0.05$）．

問18.3 [1] — 喫煙・肥満・いびきがすべて「あり」の男性の高血圧確率

解答・解説
プロビットモデルの基礎

プロビットモデルは，確率 $\pi$ を標準正規分布の累積分布関数 $\Phi$ でモデル化します：

$$\pi = \Phi(\beta_0 + \beta_1 x_1 + \cdots + \beta_d x_d) = \Phi(\eta)$$

ロジスティック回帰とプロビットモデルの違いはリンク関数だけで，分析結果（有意な変数，符号の方向）は通常ほぼ一致します．

推定された回帰式（プロビット）：

$$\hat\pi = \Phi(-1.37312 - 0.03865\,x_1 + 0.39996\,x_2 + 0.46508\,x_3)$$

$x_1 = x_2 = x_3 = 1$ を代入して線形予測子を計算する

$$\eta = -1.37312 - 0.03865 + 0.39996 + 0.46508 = -0.54673$$

標準正規分布表より：

$$\hat\pi = \Phi(-0.5467) \approx \Phi(-0.55) = 0.291$$

喫煙，肥満，いびきがすべて「あり」の男性が高血圧である確率の推定値はおよそ 0.29 である．

これはロジスティック回帰の結果（$\approx 0.29$）と一致する．プロビットモデルとロジスティック回帰は異なるリンク関数を使うが，得られる確率の推定値は通常ほぼ同じになる．

問18.3 [2] — 肥満でない男性の高血圧確率と限界効果

解答・解説
限界効果（marginal effect）とは

限界効果：$x_k$ が 1 単位増えたときの確率の変化量の推定値です．

$$\frac{\partial \pi}{\partial x_k} = \varphi(\beta_0 + \beta_1 x_1 + \cdots + \beta_d x_d) \times \beta_k$$

ここで $\varphi = \Phi'$ は標準正規分布の確率密度関数（PDF）です．

$x_1 = x_2 = x_3 = 0$（喫煙なし，肥満なし，いびきなし）のときの確率

$$\eta = -1.37312 + 0 + 0 + 0 = -1.37312$$ $$\hat\pi\big|_{x_2=0} = \Phi(-1.37312) \approx \Phi(-1.37) = 0.0853$$

限界効果（$x_1 = x_3 = 0$ のもとで $x_2 = 0$ の点で評価）

肥満（$x_2$）の限界効果は：

$$\frac{\partial \hat\pi}{\partial x_2}\bigg|_{x_1=x_3=0,\, x_2=0} = \varphi(\hat\beta_0)\times\hat\beta_2 = \varphi(-1.37312) \times 0.39996$$

$\varphi(-1.373) = 0.155$（問題文より）を使うと：

$$= 0.155 \times 0.39996 \approx 0.062$$

喫煙・いびきが「なし」，肥満も「なし」のときの高血圧確率の推定値はおよそ 0.085（約 8.5%）である．

限界効果はおよそ 0.062，すなわち「肥満でない → 肥満あり」に変化するとき，高血圧の確率が約 6.2 ポイント増加すると推定される．

ダミー変数の場合は「確率の差」にも注目する
肥満のような 2 値のダミー変数では，$x_2 = 0$ から $x_2 = 1$ に変わったときの確率の直接の差を見ることが多いです： $$\Phi(-1.37312 + 0.39996) - \Phi(-1.37312) = \Phi(-0.97) - \Phi(-1.37) \approx 0.166 - 0.085 = 0.081$$ この約 0.08（8 ポイント増加）が「確率の差」です．これは限界効果（0.062）とは異なりますが，どちらも肥満が高血圧リスクを高めることを示しています．

ポアソン回帰モデルの基礎

$$\log \mu_i = \beta_0 + \beta_1 x_{i1} + \beta_2 x_{i2} + \cdots$$

係数の解釈：$x_k$ が 1 増えると，平均 $\mu$ は $e^{\beta_k}$ 倍になります．

推定された対数線形モデルを確認する

$$\log \hat\mu = 6.340 - 0.08126 \times \text{Dist} + 0.01959 \times \text{Sprint}$$

各係数の意味を解釈する

Dist の係数 $\hat\beta_{\text{Dist}} = -0.08126$：

他の変数（Sprint）を固定したとき，チーム走行距離合計が 1 km 増えると，平均得点数は $e^{-0.08126} \approx 0.922$ 倍，すなわち約 8% 減少になると推定される．

Sprint の係数 $\hat\beta_{\text{Sprint}} = +0.01959$：

他の変数（Dist）を固定したとき，スプリント回数が 1 回増えると，平均得点数は $e^{0.01959} \approx 1.020$ 倍，すなわち約 2% 増加になると推定される．

推定された対数線形モデルは $\log\hat\mu = 6.34 - 0.0813 \times \text{Dist} + 0.0196 \times \text{Sprint}$ である．

  • チーム走行距離合計が 1 km 増えると，得点数の期待値は約 0.92 倍に減少すると推定される
  • スプリント回数が 1 回増えると，得点数の期待値は約 1.02 倍に増加すると推定される

問18.4 [2] — チーム走行距離合計とスプリント回数の説明力の比較

解答・解説

各変数の有意性検定の結果を確認する
