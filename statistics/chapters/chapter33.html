はじめに――32章の旅路を振り返る

この学習ノートでは、全32章にわたって統計学の世界を歩いてきました。 出発点は「コインを投げたとき表が出る確率は？」という素朴な問い（第1章）であり、 到着点は「コンピュータで乱数を生成し、推定量の分布をシミュレーションで近似する」（第32章）という高度な応用でした。

振り返ると、32章の内容は大きく 7つの部 に分けて捉えることができます。

  1. 第Ⅰ部：確率論の基盤（第1〜7章）――確率・分布・極限定理
  2. 第Ⅱ部：統計的推測（第8〜13章）――推定・検定・ノンパラメトリック法
  3. 第Ⅲ部：確率過程（第14〜15章）――マルコフ連鎖・ブラウン運動
  4. 第Ⅳ部：回帰分析（第16〜19章）――線形モデルから質的回帰・生存分析まで
  5. 第Ⅴ部：実験・調査の設計（第20〜21章）――分散分析・標本調査法
  6. 第Ⅵ部：多変量解析（第22〜28章）――次元削減・分類・時系列・分割表
  7. 第Ⅶ部：発展的トピック（第29〜32章）――不完全データ・モデル選択・ベイズ法・シミュレーション

この章では、各部・各章を詳しく振り返りつつ、全体としてのストーリーを描きます。 単なる要約の羅列ではなく、「なぜこの順番で学ぶのか」「この章の道具はどこで再登場するのか」という 縦の糸（学びの流れ）と横の糸（章と章のつながり）を意識して読んでください。

第Ⅰ部：確率論の基盤（第1〜7章）

統計学のすべての理論は確率論を基礎としています。第Ⅰ部では、 「不確かなできごとを数学的に記述する言語」としての確率の体系を最初から丁寧に立ち上げていきます。

第1章：事象と確率

すべての出発点です。 事象・確率・余事象・積事象・和事象という集合論の言葉を導入し、 「不確かさ」を数学的に扱う基本文法を身につけます。

最初に学ぶ重要なツールは 包除原理 $P(A \cup B) = P(A) + P(B) - P(A \cap B)$。 「どちらか一方が起きる確率」を重複なく計算する方法です。 次に、「ある情報が得られたとき確率はどう更新されるか」を表す 条件付き確率 $P(B|A)$ と、 2つの事象が互いに影響を及ぼさないことを表す 独立性 を定義します。

そしてこの章の最大のハイライトは ベイズの定理 です。 「原因→結果」の確率がわかっているとき、データ（結果）から原因の確率を逆算できる。 この「逆向きの推論」は第31章のベイズ統計学の核心へと直結します。

後半では 確率変数・確率関数（離散型）・確率密度関数（連続型）を導入し、 期待値 $E[X]$ と 分散 $V[X]$ を定義します。 $V[X] = E[X^2] - (E[X])^2$ という分散の計算公式はこの先のあらゆる章で繰り返し使われます。

🔗 つながり：条件付き確率・ベイズの定理 → 第3章（条件付き期待値）、第31章（ベイズ統計学）。期待値・分散 → 第2章以降すべての章で使用。

第2章：確率分布と母関数

第1章で「確率変数が何か」を定義したら、次は「確率変数の分布全体をどう見渡すか」です。

累積分布関数（CDF） $F(x) = P(X \leq x)$ は、離散でも連続でも統一的に定義でき、 分布の全体像をひとつの曲線で表します。 寿命データの文脈では、CDF の裏返しの 生存関数 $S(x) = 1 - F(x)$ と、 「今まで生き延びた人のうち、直後に死亡する割合」を表す ハザード関数 が重要です。

2つ以上の確率変数を同時に扱うために、同時分布・周辺分布・条件付き分布を導入します。 「同時分布から片方だけ取り出す＝周辺分布」、「一方を固定して見た分布＝条件付き分布」です。

この章の核心は 母関数 です。 整数値確率変数には 確率母関数（PGF） $G(s) = E[s^X]$、 一般の確率変数には モーメント母関数（MGF） $m(\theta) = E[e^{\theta X}]$ を使います。 「微分して特定の値を代入するだけ」で期待値・分散が得られ、 「独立な変数の和の MGF ＝ 各変数の MGF の積」という性質のおかげで、 和の分布を積分計算なしに特定できます。

🔗 つながり：母関数 → 第5章・第6章（具体的な分布の性質の導出）、第7章（中心極限定理の証明）。生存関数・ハザード関数 → 第19章（比例ハザードモデル）。

第3章：分布の特性値

分布の「個性」を数値ひとつで言い表す方法を学ぶ章です。

位置の指標として、期待値・中央値・最頻値があります。 左右対称な分布ではこの3つは一致しますが、右裾長の分布（所得データなど）では期待値が引っ張られるため、中央値のほうが「典型値」を表します。

散らばりの指標としては 標準偏差・四分位範囲・変動係数。 変動係数 $CV = \sigma / \mu$ は単位に依存しないため、「身長のばらつき」と「体重のばらつき」を同じ尺度で比較できます。

形状の指標として 歪度（分布の左右非対称性）と 尖度（裾の重さ）があります。 正規分布の尖度は3であり、3より大きいと「裾が重い（外れ値が出やすい）」分布です。

2変数の関係は 共分散・相関係数 で測ります。 ただし、第3の変数 $Z$ が $X$ と $Y$ の両方に影響しているだけの 擬似相関 には注意が必要です。 $Z$ の影響を取り除いた 偏相関係数 を使えば「本当の関係」を見抜けます。

さらに、条件付き期待値 $E[Y|X=x]$ と 条件付き分散 $V[Y|X=x]$ も重要な概念です。 条件付き期待値は「$X$ の値がわかったとき $Y$ の予測をどう更新するか」を表し、 回帰分析（第16章〜）の根幹をなす概念です。

🔗 つながり：相関係数・偏相関係数 → 第16章（回帰分析）、第25章（因子分析・グラフィカルモデル）。条件付き期待値 → 回帰分析の理論的基礎。変動係数 → 第21章（標本調査の設計）で効率的配分に使用。

第4章：変数変換

確率変数 $X$ の分布がわかっているとき、$Y = g(X)$ の分布はどうなるか？ 核心のアイデアは 「確率は変換しても保存される」 です。

1変数の場合は $f_Y(y) = f_X(g^{-1}(y)) \cdot |dg^{-1}(y)/dy|$。 2変数では「面積のスケール変化率」を表す ヤコビアン（行列式）で割ります。 たとえば標準正規分布に従う $X$ に対して $Y = X^2$ の分布がカイ二乗分布（自由度1）になることは、この変数変換の公式で導出できます。

独立な確率変数の和 $Z = aX + bY$ の分布は、母関数のかけ算（第2章）で求める方法と、 ヤコビアンを使って直接積分する方法の2通りがあります。 正規分布の和が正規分布に、ポアソン分布の和がポアソン分布になるのは、母関数法で簡潔に示せます。

実データの分析においては 対数変換・Box-Cox 変換・ロジット変換・プロビット変換 といった「データを変換してから分析する」アプローチが頻出します。 右裾の長い所得データには対数変換、0〜1の確率値にはロジット変換が有効で、 これらは第18章のロジスティック回帰・プロビットモデルに直結します。

🔗 つながり：ヤコビアン → 第6章（標本分布の導出）。対数変換・ロジット変換 → 第18章（質的回帰）。Box-Cox 変換 → 第17章（回帰診断）。

第5章：離散型確率分布

ここまでの道具立て（確率関数、母関数、変数変換）を使い、 代表的な離散型分布の「カタログ」を整備する章です。

離散一様分布（サイコロの目など）、 ベルヌーイ分布（成功/失敗の1回の試行）、 二項分布 $\text{Bin}(n,p)$（成功回数の分布）、 超幾何分布（非復元抽出）、 ポアソン分布（稀な事象の回数）、 幾何分布（初めて成功するまでの試行回数）、 負の二項分布（$r$回成功するまでの試行回数）、 多項分布（3カテゴリ以上への分類）。

これらの分布は互いに関連しています。 たとえば、二項分布で $n \to \infty, p \to 0, np \to \lambda$ とするとポアソン分布に収束します（第7章の少数法則）。 また超幾何分布で母集団サイズ $N \to \infty$ とすれば二項分布になります。

🔗 つながり：二項分布 → 第9章（母比率の区間推定）、第12章（二項分布に関する検定）。ポアソン分布 → 第12章（ポアソン分布に関する検定）、第15章（ポアソン過程）、第18章（ポアソン回帰）。多項分布 → 第28章（分割表）。

第6章：連続型確率分布

連続版の「分布カタログ」です。ここが第Ⅰ部のクライマックスの一つです。

正規分布 $N(\mu, \sigma^2)$ は統計学の最重要分布。中心極限定理（第7章）により、 どんな分布の平均でもサンプルサイズが大きければ近似的に正規分布に従います。 2変量正規分布・多変量正規分布は第16章以降の回帰分析・多変量解析の前提です。

正規分布から派生する 標本分布 が極めて重要です：

  • $Z^2$（標準正規の2乗の和）→ カイ二乗分布 $\chi^2(n)$
  • $Z / \sqrt{\chi^2/n}$ → $t$ 分布
  • $(\chi^2_1/n_1) / (\chi^2_2/n_2)$ → $F$ 分布

これらは第9章（区間推定）・第11章（正規分布の検定）・第16章（回帰の $F$ 検定）・第20章（分散分析）で繰り返し登場します。

他にも 指数分布（ポアソン過程の待ち時間）、ガンマ分布（指数分布の和）、 ベータ分布（0〜1に制限された変量）、コーシー分布（期待値が存在しない重い裾の分布）、 対数正規分布（対数をとると正規分布にしたがう）など、応用で頻出する分布を扱います。

🔗 つながり：$\chi^2$・$t$・$F$ 分布 → 第9〜11章（推定・検定）、第16章・第20章。指数分布 → 第15章（ポアソン過程の待ち時間）。ベータ分布 → 第31章（ベイズの事前分布として）。

第7章：極限定理，漸近理論

「サンプルサイズが大きくなるとき、推定量はどう振る舞うのか」を理論的に保証する章です。

まず確率変数列の 収束 の種類を整理します： 概収束 → 確率収束 → 分布収束の順に弱くなります。 大数の（弱）法則 は「標本平均はサンプルサイズ $n \to \infty$ で母平均に確率収束する」ことを保証します。

そして統計学で最も重要な定理のひとつ、中心極限定理（CLT）：

$$\frac{\bar{X}_n - \mu}{\sigma / \sqrt{n}} \xrightarrow{d} N(0,1)$$

元の分布が何であれ、標本平均の標準化は正規分布に収束する。 これが第9章以降の区間推定・検定で「正規近似」が使える理論的根拠です。

さらに デルタ法（$g(\bar{X})$ の漸近分布を求める手法）や 少数法則（二項分布のポアソン近似）、 極値分布（最大値・最小値の分布）などを扱います。

🔗 つながり：中心極限定理 → 第8章以降のすべての漸近的推測。大数の法則 → 第32章（モンテカルロ法の理論的根拠）。デルタ法 → 第8章（最尤推定量の漸近分散）。

第Ⅱ部：統計的推測（第8〜13章）

第Ⅰ部で確率論の「言語」を習得したら、次は「データから未知のパラメータについて何が言えるか」 という 統計的推論 に踏み出します。推定→検定→ノンパラメトリック法という流れで進みます。

第8章：統計的推定の基礎

データ $X_1, \ldots, X_n$ が得られたとき、母集団のパラメータ $\theta$ をどう推定するか？ これが 点推定 の問題です。

まず 統計量（データの関数）と 推定量（パラメータの推定に使う統計量）の定義を確認します。 代表的な推定法として 最尤法（尤度を最大化）と モーメント法（母モーメント＝標本モーメント）を学びます。

推定量の「よさ」を測る基準として：

  • 不偏性：$E[\hat{\theta}] = \theta$（平均的に正しい）
  • 有効性：分散が最小（精度が最高）
  • 十分性：データの情報を余すところなく使っている
  • 一致性：$n \to \infty$ で真の値に収束

クラメール・ラオの不等式は「不偏推定量の分散は $1/nI(\theta)$ 以下にはならない」ことを示し、 フィッシャー情報量 $I(\theta)$ がデータに含まれる情報の量を測る尺度です。 最尤推定量は漸近的にこの下限を達成する（漸近有効）ことが知られています。

🔗 つながり：最尤法 → 第12章（尤度比検定）、第18章（ロジスティック回帰の推定）、第30章（AICの導出）。フィッシャー情報量 → 第31章（ベイズの事後分布の正規近似）。十分統計量 → 第10章（検定法の導出）。

第9章：区間推定

点推定は「最良の1点」を与えますが、それだけでは「どのくらい信頼できるか」がわかりません。 信頼区間 は「この範囲に真のパラメータが含まれる確率が95%（または99%）」という区間です。

正規母集団で分散既知なら正規分布を、分散未知なら $t$ 分布 を使って母平均の信頼区間を作ります。 母分散の信頼区間は カイ二乗分布 を使います。 標本サイズが大きければ、中心極限定理（第7章）による正規近似で一般の母集団にも適用でき、 二項分布の母比率の信頼区間もここで扱います。

🔗 つながり：第6章の $t$ 分布・$\chi^2$ 分布を実際に使う場面。第7章の中心極限定理による正規近似。第10章の検定との「表裏の関係」（信頼区間と検定は対応する）。

第10章：検定の基礎と検定法の導出

区間推定が「パラメータの範囲を推定する」のに対し、 仮説検定 は「パラメータについての仮説を、データに基づいて採択するか棄却するか決める」手続きです。

帰無仮説 $H_0$（「差はない」など）と 対立仮説 $H_1$ を設定し、 検定統計量 の値が 棄却域 に入れば $H_0$ を棄却します。 2種類の間違い――帰無仮説を誤って棄却する 第一種の過誤（偽陽性）と、 対立仮説が正しいのに棄却しない 第二種の過誤（偽陰性）のバランスが重要です。

P値は「帰無仮説のもとで、観測データ以上に極端な値が出る確率」であり、 有意水準 $\alpha$ と比較して棄却を判断します。 検定法の導出には ネイマン・ピアソンの補題（最強力検定）や 尤度比検定 の考え方が使われます。

🔗 つながり：尤度比検定 → 第12章で一般化。P値・検出力 → 第11章以降のすべての検定で使用。「信頼区間に含まれない ⇔ 検定で棄却される」という対応は第9章と表裏一体。

第11章：正規分布に関する検定

第10章の理論を 正規母集団 という最も基本的な設定に適用する実践の章です。

1標本の母平均の検定（分散既知→ $Z$ 検定、分散未知→ $t$ 検定）、 2標本の母平均の差の検定（等分散→ プールされた $t$ 検定、等分散でない→ ウェルチの $t$ 検定）、 母分散の検定（$\chi^2$ 検定）、 2群の分散の等しさの検定（$F$ 検定） を扱います。片側検定と両側検定の使い分けも重要です。

🔗 つながり：$t$ 検定 → 第16章（回帰係数の検定）、第20章（分散分析の対比較）。$F$ 検定 → 第16章（回帰の有意性検定）、第20章（分散分析の F 検定）。

第12章：一般の分布に関する検定法

正規分布を離れ、より一般的な母集団に対する検定を扱います。

母比率の検定（二項分布に基づく）、ポアソン母数の検定、 データが特定の分布に従うかを調べる 適合度検定（$\chi^2$ 適合度検定）、 そして汎用的な 尤度比検定 を学びます。 尤度比検定統計量 $-2\log\Lambda$ は、サンプルサイズが大きいとき漸近的にカイ二乗分布に従います。

🔗 つながり：適合度検定 → 第28章（分割表の独立性検定）。尤度比検定 → 第30章（モデル選択）。母比率の検定 → 第21章（標本調査法）の設計に関連。

第13章：ノンパラメトリック法

「母集団の分布を特定しない」検定法です。 $t$ 検定は正規分布を仮定しますが、データが正規からかけ離れている場合には信頼性が下がります。 ノンパラメトリック法は データの順位 に基づいて検定統計量を作るため、分布の仮定が不要です。

ウィルコクソンの順位和検定（2群の位置の差）、 符号付き順位検定（対応ありデータの位置の差）、 符号検定、 クラスカル・ウォリス検定（3群以上への拡張）、 順位相関係数（スピアマンの $\rho$、ケンドールの $\tau$） を扱います。

🔗 つながり：並べ替え検定の考え方 → 第32章（シミュレーションによる帰無分布の構成）。順位相関 → 第3章の相関係数の「ロバスト版」。

第Ⅲ部：確率過程（第14〜15章）

ここまでの章では「独立同一分布（i.i.d.）に従うデータ」を前提としていました。 しかし現実のデータ――株価、気温、来客数など――は 時間とともに変動し、前後の値に依存関係があります。 第Ⅲ部では、こうした時間的な依存構造をもつ確率変数の列、すなわち 確率過程 の基礎を学びます。

第14章：マルコフ連鎖

マルコフ連鎖 とは「次の状態が現在の状態のみに依存して決まる」確率過程です。 過去の履歴に依存しないという性質を マルコフ性（無記憶性）と呼びます。

有限の状態空間と 推移確率行列 $P$ で完全に定義され、 $n$ ステップ後の状態分布は $P^n$ の行列の掛け算で求まります。 十分長い時間が経つと、初期状態によらず特定の分布に収束する（定常分布）ことがあり、 これが エルゴード性 です。

🔗 つながり：マルコフ連鎖 → 第31章（MCMC法でベイズ推定に利用）。推移確率行列 → 第27章（時系列の状態空間モデル）。

第15章：確率過程の基礎

マルコフ連鎖は離散時間・離散状態でしたが、この章では 連続時間 の確率過程を扱います。

ブラウン運動（ウィーナー過程）は連続時間の確率過程の代表格で、 独立定常増分をもち、増分が正規分布に従います。 金融工学のブラック・ショールズモデルの基礎です。

ポアソン過程 は「ランダムなタイミングでイベントが発生する」モデルです。 一定期間のイベント数がポアソン分布に従い、イベント間の待ち時間が指数分布に従います（第5章・第6章との架け橋）。

🔗 つながり：ブラウン運動 → 金融時系列（第27章の応用）。ポアソン過程 → 第5章（ポアソン分布）・第6章（指数分布）。

第Ⅳ部：回帰分析（第16〜19章）

回帰分析は「説明変数 $x$ から応答変数 $y$ を予測する」ための枠組みであり、 統計学の応用の中で最も広く使われる手法群です。 第3章の条件付き期待値 $E[Y|X]$ を具体的なモデルとして定式化したのが回帰分析です。

第16章：重回帰分析

ガウス・マルコフモデル $\boldsymbol{y} = X\boldsymbol{\beta} + \boldsymbol{\varepsilon}$ のもとで、 回帰係数 $\boldsymbol{\beta}$ を 最小二乗法（OLS）で推定します。 $\hat{\boldsymbol{\beta}} = (X^\top X)^{-1} X^\top \boldsymbol{y}$ はガウス・マルコフの定理により「不偏線形推定量の中で分散が最小」です。

誤差が正規分布に従うと仮定すれば、個々の回帰係数に対する $t$ 検定（第11章）と、 回帰式全体の有意性に対する $F$ 検定（第6章の $F$ 分布を利用）が構成できます。

説明変数間に強い相関がある 多重共線性 の問題に対しては、 リッジ回帰（$L_2$ 正則化）や Lasso（$L_1$ 正則化）が有効です。 Lasso は不要な説明変数の係数をちょうど0にする 変数選択 の効果を持ちます。

🔗 つながり：最小二乗法 → 第8章（点推定の特殊ケース）。$t$・$F$ 検定 → 第11章。正則化 → 第30章（モデル選択の一手法）。

第17章：回帰診断法

回帰分析は万能ではありません。モデルの仮定（独立性・等分散性・正規性）が満たされていなければ、 推定結果は信頼できなくなります。この章では、仮定が成り立っているかを チェックする手法 を学びます。

残差プロット（等分散性・非線形性の発見）、 正規 Q-Q プロット（誤差の正規性の確認）、 てこ比（leverage）（影響の大きい説明変数の値の検出）、 Cook の距離（1つのデータ点を除いたときの推定値の変化）、 ダービン・ワトソン比（誤差の自己相関のチェック） などの診断ツールを扱います。

🔗 つながり：正規性の確認 → 第4章（Box-Cox 変換で改善）。自己相関 → 第27章（時系列分析）。外れ値 → 第13章（ノンパラメトリック法で対処）。

第18章：質的回帰

応答変数が「成功/失敗」「カテゴリ」のように質的（離散的）な場合、 通常の線形回帰は適用できません。この章では応答変数が非連続な状況をモデル化します。

ロジスティック回帰は「成功確率」をロジット変換（第4章）で実数全体に写し、 線形モデルに載せます。プロビットモデルは標準正規分布の累積分布関数を使う別のアプローチです。 ポアソン回帰は応答が「回数」のとき使います。

これらは 一般化線形モデル（GLM） という枠組みで統一的に扱えます。 「応答の分布（正規・二項・ポアソンなど）」と「リンク関数（恒等・ロジット・対数など）」の2つの選択で多様なモデルが作れます。

🔗 つながり：ロジット変換 → 第4章。ポアソン分布 → 第5章。最尤推定 → 第8章。GLM のモデル選択 → 第30章（AIC による比較）。

第19章：回帰分析その他

応答変数のさらに特殊な構造を持つ回帰モデルを扱います。

トービットモデルは応答変数に「打ち切り」がある場合です。 たとえば家計の耐久消費財支出は0以下にはならない（左打ち切り）。 比例ハザードモデル（Cox モデル）は生存時間データの回帰で、 ハザード関数（第2章）を説明変数で表現します。 ニューラルネットワークは非線形回帰の一般化として、複雑な関数関係をデータから直接学習します。

🔗 つながり：打ち切り → 第29章（不完全データ）。ハザード関数 → 第2章。ニューラルネット → 第22章の自己符号化器（次元削減への応用）。

第Ⅴ部：実験・調査の設計（第20〜21章）

データは分析する前に「正しく集める」ことが重要です。 この部では、データ収集の方法論として実験計画法と標本調査法を学びます。

第20章：分散分析と実験計画法

分散分析（ANOVA）は $t$ 検定の拡張で、「3群以上の平均の間に差があるか」を検定します。 全体の変動を「群間変動」と「群内変動」に分解し、その比が $F$ 分布 に従うことを利用します。

1元配置（因子が1つ）と 2元配置（因子が2つ、交互作用あり/なし）を扱います。

実験計画法 では、フィッシャーの三原則（反復・ランダム化・局所管理）を学び、 ブロック因子 による誤差の削減、直交表 を用いた一部実施要因計画など、 効率的な実験の組み方を扱います。

🔗 つながり：分散分析 → 第16章（回帰の ANOVA 表）。$F$ 検定 → 第6章・第11章。ランダム化 → 第21章（標本の無作為抽出）。

第21章：標本調査法

実験が不可能な場合（国民全体の健康状態など）に、 母集団から 標本を抽出して母集団の特性を推定 する方法論です。

単純無作為抽出法（復元/非復元）の推定量の期待値・分散を導出し、 有限母集団ならではの 有限修正 を扱います。 層化抽出法 は母集団を同質な層に分けてから抽出する方法で、 層内の変動が小さいほど精度が上がります。 ネイマン配分は各層の標準偏差に比例して標本を割り当てる最適な方法です。 二段抽出法・集落抽出法も実務で重要です。

🔗 つながり：有限修正 → 第5章（超幾何分布との関連）。推定量の分散 → 第8章（推定量の評価）。ランダム化 → 第20章（実験計画法）。

第Ⅵ部：多変量解析（第22〜28章）

現実のデータは多数の変数を同時にもっています。 第Ⅵ部では、多変量データから 構造を読み解く ための解析手法を学びます。 「次元削減」「分類」「構造モデリング」「時間依存」「カテゴリデータ」と多岐にわたるテーマをカバーします。

第22章：主成分分析

多くの変数を 少数の「主成分」に要約 する手法です。 分散共分散行列（または相関行列）の 固有値・固有ベクトル を計算し、 分散が最大になる方向を第1主成分、次に大きい方向を第2主成分…とします。

寄与率（各主成分が全体の分散の何%を説明するか）と 累積寄与率 で次元削減の妥当性を判断します。 特異値分解（SVD） による計算法や、ニューラルネットの 自己符号化器（主成分分析の非線形拡張）にも触れます。

🔗 つながり：分散共分散行列 → 第3章、第6章（多変量正規分布）。固有値 → 第25章（因子分析でも使用）。次元削減のアイデア → 第26章（多次元尺度法・対応分析）。

第23章：判別分析

「新しいデータがどのグループに属するかを予測する」分類（classification）の手法です。

フィッシャーの線形判別関数 は、群間分散と群内分散の比を最大にする線形結合を求めます。 多変量正規分布を仮定すれば 2次判別（共分散行列が異なる場合）に拡張できます。

分類器の精度は 混同行列（正しく分類/誤分類の表）、 ROC 曲線と AUC（曲線下面積）で評価します。 サポートベクターマシン（SVM） は「マージン最大化」の考え方に基づく非線形分類手法です。

🔗 つながり：線形判別 → 第16章（線形結合という同じアイデア）、第22章（主成分と似た固有値問題）。ロジスティック回帰（第18章）による分類との比較。

第24章：クラスター分析

判別分析はグループのラベルが既知でしたが、 クラスター分析 はラベルが未知のまま、データの類似度だけでグループを発見する 教師なし学習 です。

階層的手法 はデータを順番にまとめあげ、デンドログラム（樹形図）で可視化します。 結合方法（最短距離法・最長距離法・ウォード法など）と距離尺度（ユークリッド距離・マハラノビス距離など）の選択が結果に大きく影響します。

非階層的手法の代表は K-means で、$K$ 個のクラスタ重心を反復的に更新します。 混合分布モデル は「各クラスタが確率分布（正規分布など）を表す」と仮定し、EM アルゴリズムで推定します。

🔗 つながり：マハラノビス距離 → 第23章（判別分析でも使用）。混合分布 → 第6章（混合正規分布）、第29章（EM アルゴリズム）。

第25章：因子分析・グラフィカルモデル

主成分分析が「変数を合成する」のに対し、 因子分析 は「観測変数の背後に隠れた 共通因子 がある」と仮定し、その構造を推定します。

$X_i = \ell_{i1} F_1 + \cdots + \ell_{im} F_m + \varepsilon_i$ で、$\ell_{ij}$ が 因子負荷量、$F_j$ が共通因子、$\varepsilon_i$ が独自因子です。 因子負荷行列から「どの変数がどの因子に強く関わるか」を読み取ります。

グラフィカルモデル は変数間の条件付き独立関係をグラフ（ネットワーク）で表現します。 辺がない2変数は、他の変数を所与とすると独立です。 偏相関係数（第3章）がゼロであることと、グラフで辺がないことが対応します。

🔗 つながり：因子負荷量 → 第22章（主成分負荷量との比較）。条件付き独立 → 第1章・第2章。グラフィカルモデル → 第28章（対数線形モデルの階層構造）。

第26章：その他の多変量解析手法

標準的な手法にはまらないがよく使われる多変量手法を集めた章です。

多次元尺度法（MDS）は「個体間の距離・非類似度」から低次元空間に配置します。 主成分分析が変数×変数の共分散行列を使うのに対し、MDS は個体×個体の距離行列を使います。

正準相関分析 は2群の変数セットの間の相関を最大にする線形結合を見つけます。 数量化法（I, II, III, IV 類）は質的変数を数値化して回帰・判別・主成分分析に持ち込む手法です。 対応分析 はクロス集計表を低次元に可視化する方法で、第28章（分割表）と深く関連します。

🔗 つながり：MDS → 第22章（主成分分析のデュアル）。対応分析 → 第28章（分割表）。数量化法 → 第16章（回帰）・第23章（判別）の質的変数版。

第27章：時系列解析

時間の経過とともに観測されるデータの分析手法です。 第14章・第15章で確率過程の基礎を学びましたが、ここではデータ分析としての時系列モデルを扱います。

自己共分散・自己相関係数（ACF）・偏自己相関係数（PACF）で時間的依存構造を把握します。 定常性（統計的性質が時間に依存しない）の概念は時系列モデルの前提です。

AR（自己回帰）モデル：$X_t = \phi_1 X_{t-1} + \cdots + \phi_p X_{t-p} + \varepsilon_t$、 MA（移動平均）モデル：$X_t = \varepsilon_t + \theta_1 \varepsilon_{t-1} + \cdots + \theta_q \varepsilon_{t-q}$、 そしてこれらを組み合わせた ARMA モデル が基本です。 非定常データには差分をとって定常化する ARIMA モデル を使います。

周波数領域の分析として スペクトラム（パワースペクトル密度）と ペリオドグラム、 そして 状態空間モデル（カルマンフィルタ）も重要です。

🔗 つながり：AR モデル → 第14章（マルコフ連鎖の連続版）。ダービン・ワトソン検定 → 第17章（回帰の自己相関診断）。モデル選択 → 第30章（AIC による次数の選択）。

第28章：分割表

質的変量（カテゴリ変数）のクロス集計表 = 分割表 の分析です。

2元分割表の 独立性の検定 には $\chi^2$ 検定・尤度比検定・フィッシャーの正確検定 が使えます。 オッズ比 は2つのカテゴリ比較で「効果の大きさ」を測る指標です。

多元分割表では 対数線形モデル で因子間の関係を階層的に表現し、 条件付き独立 の検証を行います。 これはグラフィカルモデル（第25章）の離散版ともいえる考え方です。

🔗 つながり：$\chi^2$ 検定 → 第12章（適合度検定）。条件付き独立 → 第25章（グラフィカルモデル）。対応分析 → 第26章。多項分布 → 第5章。

第Ⅶ部：発展的トピック（第29〜32章）

最終部では、現実データの「理想的でない」側面に対処し、 分析の精度を高めるための高度な手法を学びます。

第29章：不完全データの統計処理

現実のデータには 欠測値（回答拒否、脱落、測定不能など）がつきものです。 また第19章で触れた打ち切り（寿命データで追跡終了時点で生存中）も不完全データの一種です。

欠測のメカニズムを3段階に分類します：

  • MCAR（完全にランダム）：欠測はデータと無関係
  • MAR（観測データに条件付きでランダム）：欠測の有無は観測済み変数で説明可能
  • MNAR（非ランダム）：欠測そのものが欠測値に依存

対処法として 完全ケース分析（リストワイズ削除）、 単一代入法・多重代入法（MI）、 そして EM アルゴリズム（E ステップで欠測値の期待値を計算、M ステップで最尤推定）を学びます。

🔗 つながり：打ち切り → 第19章（トービットモデル、Cox モデル）。EM アルゴリズム → 第24章（混合分布モデル）、第8章（最尤法の反復版）。

第30章：モデル選択

統計分析では「どのモデルを使うか」の選択が結果を大きく左右します。 複雑なモデルほどデータへの当てはまりは良くなりますが、 過学習（未知データに対する予測力が低下すること）のリスクが高まります。

赤池情報量規準（AIC）は「モデルの予測力」を推定する情報量規準です。 $\text{AIC} = -2\log L(\hat{\theta}) + 2k$（$k$ はパラメータ数）。 これは カルバック・ライブラー情報量（2つの分布の「距離」）の不偏推定量として導出されます。

BIC（ベイズ情報量規準）は $-2\log L + k \log n$ で、AIC よりパラメータ数にペナルティを強く課します。 クロスバリデーションは実際にデータを分割して予測力を検証する方法です。

🔗 つながり：最尤推定 → 第8章。尤度比検定 → 第12章（入れ子モデルの比較には検定、非入れ子には AIC）。正則化 → 第16章（Lasso も一種のモデル選択）。ベイズ → 第31章（BIC はベイズ的解釈をもつ）。

第31章：ベイズ法

第1章で学んだベイズの定理が、ここで本格的に活躍します。 頻度論的統計学ではパラメータは「未知の固定値」でしたが、 ベイズ統計学 ではパラメータ自体を確率変数とみなし、事前分布 を与えます。

データ（尤度）が得られると、ベイズの定理により事前分布が更新され 事後分布 になります：

$$p(\theta | \boldsymbol{x}) \propto L(\boldsymbol{x} | \theta) \cdot \pi(\theta)$$

事後分布が解析的に求まらない場合は、MCMC（マルコフ連鎖モンテカルロ法）でサンプリングします。 メトロポリス・ヘイスティングス（MH）アルゴリズムと ギブスサンプリング が代表的です。 ここで第14章のマルコフ連鎖の理論が土台になります。

🔗 つながり：ベイズの定理 → 第1章。事前分布（ベータ分布・ガンマ分布など）→ 第6章。マルコフ連鎖 → 第14章。モンテカルロ法 → 第32章。BIC → 第30章（ベイズ的モデル選択）。

第32章：シミュレーション

統計学の最後のピースは コンピュータの力 を使うことです。 解析的に解けない問題も、乱数を大量に生成してシミュレーションすれば数値的に解けます。

擬似乱数（メルセンヌツイスタなど）と一様乱数からの変換（逆関数法、棄却法）に始まり、 モンテカルロ法（乱数を使った数値積分）で期待値計算や複雑な確率の近似を行います。 大数の法則（第7章）がモンテカルロ法の精度を保証する理論的根拠です。

ジャックナイフ法（1個ずつデータを除いて推定量のバイアス・分散を推定）、 ブートストラップ法（データから復元抽出で再標本を作り、推定量の分布を近似）は、 分布の仮定なしに信頼区間や標準誤差を求められる強力な手法です。

🔗 つながり：大数の法則 → 第7章。MCMC → 第31章（ベイズ推定）、第14章（マルコフ連鎖）。ブートストラップ → 第13章（ノンパラメトリックな精度評価）。逆関数法 → 第2章（累積分布関数の逆関数）・第4章（変数変換）。

全体のつながり――統計学はひとつの物語

縦の流れ：基礎から応用へ

32章の旅を一本のストーリーとして振り返ると、大きく3つのフェーズに分かれます。

フェーズ1：確率論の言語を学ぶ（第1〜7章）

確率・確率変数・分布・母関数・特性値・変数変換・極限定理。 これは「道具箱」を作るフェーズです。 ここで作った道具がなければ、第8章以降の推定・検定・モデリングは語れません。

フェーズ2：データから知識を引き出す（第8〜28章）

推定・検定（第8〜13章）でパラメータについて語り、回帰分析（第16〜19章）で変数間の関係を定量化し、 実験・調査（第20〜21章）でデータの質を自ら設計し、多変量解析（第22〜28章）で複雑なデータの構造に切り込む。 このフェーズでは「具体的な問題にどの手法を使うか」を判断する力が求められます。

フェーズ3：限界を超える（第29〜32章）

不完全データ・モデル選択・ベイズ法・シミュレーション。 理想的でないデータ、解析的に解けないモデル、パラメータの不確実性そのものを扱う。 これらは「フェーズ2の手法がうまくいかないとき」の切り札です。

横の糸：繰り返し登場する共通テーマ

32章を横断する共通テーマがいくつかあります。

  • 正規分布の普遍性： 第6章で定義され、第7章の中心極限定理で「なぜどこにでも出てくるのか」が説明され、 第9〜11章の推定・検定、第16章の回帰分析、第20章の分散分析、第22章の主成分分析、 第23章の判別分析…とほぼすべての応用で前提や近似として使われます。
  • 最尤法の一貫した力： 第8章で紹介された最尤推定は、第12章（尤度比検定）、第16章（回帰の推定）、 第18章（GLM）、第24章（混合分布の EM）、第27章（時系列の推定）、 第30章（AIC の構成要素）を通じて繰り返し姿を現します。
  • 条件付き確率の深い含意： 第1章のベイズの定理→第2章（条件付き分布）→第3章（条件付き期待値）→第16〜18章（回帰＝条件付き期待値のモデル化） →第25章（条件付き独立＝グラフィカルモデル）→第28章（分割表の条件付き独立）→第31章（ベイズ統計学）。
  • 「分布を仮定するか、しないか」という対立軸： パラメトリック法（第8〜12章、第16〜18章）vs. ノンパラメトリック法（第13章）、 モデルの選択（第30章）、ブートストラップによる分布フリーの推定（第32章）。
  • シミュレーション＝最後の武器： 解析的に解けない問題はモンテカルロで解け（第32章）、 事後分布のサンプリングは MCMC（第14章＋第31章）で解け、 ブートストラップ（第32章）は分布の仮定なしに信頼区間を作れる。

最後に

統計学は「不確かな世界でデータに基づいて判断する技術」です。 確率という言語で不確かさを記述し（第Ⅰ部）、推定・検定で仮説を吟味し（第Ⅱ部）、 回帰と多変量解析で複雑な関係を紐解き（第Ⅳ〜Ⅵ部）、ベイズ法やシミュレーションで限界を超える（第Ⅶ部）。

全32章は独立した「部品」ではなく、互いにつながりあったひとつの体系です。 この章が、いつでも全体像を見渡せる「地図」として役に立つことを願います。
