概要

時系列解析（time-series analysis）は、時間の経過とともに観測されたデータ（時系列データ）を分析する手法である。時系列データは自己共分散・自己相関係数で特徴づけられ、定常性の有無で適用するモデルが異なる。

代表的なモデルとして、AR（自己回帰）過程、MA（移動平均）過程、ARMA過程、ARIMA過程がある。時間領域での分析に加え、スペクトラム・ペリオドグラムによる周波数領域での分析もある。また、状態空間モデルやダービン・ワトソン検定など、実用的なツールも多い。

重要概念

基本概念

  • 自己共分散（autocovariance）：$\gamma_h = \text{Cov}[Y_t, Y_{t-h}]$
  • 自己相関係数（autocorrelation coefficient）：$\rho_h = \gamma_h / \gamma_0$
  • 共分散定常過程（弱定常過程）：$E[Y_t] = \mu$（一定）、$\text{Cov}[Y_t, Y_{t-h}] = \gamma_{|h|}$（時点に依存しない）
  • 強定常過程：任意の時間差に対して同時分布が時点に依存しない
  • ホワイトノイズ：平均0、$h \neq 0$ のすべての自己共分散が0

AR過程

  • AR(1)：$Y_t = c + \phi_1 Y_{t-1} + U_t$。$|\phi_1| < 1$ で定常
  • 平均 $\mu = c/(1-\phi_1)$、分散 $\gamma_0 = \sigma^2/(1-\phi_1^2)$、自己相関 $\rho_h = \phi_1^h$
  • AR(p)：定常性条件は特性多項式 $1 - \phi_1 z - \cdots - \phi_p z^p = 0$ の解がすべて単位円外

MA過程

  • MA(1)：$Y_t = \mu + U_t + \theta_1 U_{t-1}$。常に定常
  • MA(q)：$q$ 次以降の自己共分散が0（重要な特徴）

ARMA・ARIMA

  • ARMA(p,q)：AR過程とMA過程の組合せ
  • ARIMA(p,1,q)：階差 $\Delta Y_t = Y_t - Y_{t-1}$ がARMA(p,q)に従う
  • 単位根検定（DF検定・拡張DF検定）：$\phi_1 = 1$ を検定（特別な分布表が必要）

その他の手法

  • ラグ多項式：$\phi(L)Y_t = c + \theta(L)U_t$（$L$：ラグオペレーター、$LY_t = Y_{t-1}$）
  • スペクトラム密度：$f(\lambda) = \frac{1}{2\pi}\sum_{h=-\infty}^{\infty}\gamma_h e^{-i\lambda h}$
  • ペリオドグラム：標本自己共分散を用いたスペクトラムの推定量（一致推定量ではない）
  • 状態空間モデル：観測方程式＋状態方程式、カルマンフィルターで推定
  • ダービン・ワトソン検定：回帰モデルの誤差項の系列相関を検定

公式・定理

AR(1)過程の性質（$|\phi_1| < 1$）

$$\mu = \frac{c}{1-\phi_1}, \quad \gamma_0 = \frac{\sigma^2}{1-\phi_1^2}, \quad \gamma_h = \phi_1^h \gamma_0, \quad \rho_h = \phi_1^h$$

MA(q)過程の自己共分散

$$\gamma_h = \begin{cases} (1+\theta_1^2+\cdots+\theta_q^2)\sigma^2 & (h=0) \\ (\theta_h + \theta_1\theta_{h+1} + \cdots + \theta_{q-h}\theta_q)\sigma^2 & (1 \leq h \leq q) \\ 0 & (h > q) \end{cases}$$

AR(1)のスペクトラム

$$f(\lambda) = \frac{\sigma^2}{2\pi(1+\phi_1^2-2\phi_1\cos\lambda)}$$

ダービン・ワトソン比

$$DW = \frac{\sum_{t=2}^{T}(\hat{U}_t - \hat{U}_{t-1})^2}{\sum_{t=1}^{T}\hat{U}_t^2} \approx 2(1-\hat{\rho})$$

$\hat{\rho}$：1次の自己相関係数の推定量。$DW \approx 2$：相関なし、$DW < 2$：正の相関、$DW > 2$：負の相関

次数選択の目安

自己相関係数偏自己相関係数選択モデル
2次以降0ゆっくり減衰MA(1)
ゆっくり減衰2次以降0AR(1)
ゆっくり減衰ゆっくり減衰ARMA(1,1)

例題・演習

問27.1 AR(1)過程 $Y_t = 2(1-\phi_1) + \phi_1 Y_{t-1} + U_t$ のグラフ対応

[1] $\phi_1 = -0.8, 0, 0.7, 1$ のグラフ対応

  • (a) $\phi_1 = 0.7$：隣接データが似た値、変動が穏やか
  • (b) $\phi_1 = 0$：ホワイトノイズ、不規則変動
  • (c) $\phi_1 = 1$：非定常、平均が一定でない
  • (d) $\phi_1 = -0.8$：上下に激しく変動（符号が交互）

[2] DW比

正の自己相関がある場合、最小二乗推定量は不偏だが最小分散ではない。DW比は0に近い値をとる。

問27.2 MA(2)過程 $Y_t = U_t + \theta_1 U_{t-1} + \theta_2 U_{t-2}$ の共分散

$$\gamma_0 = (1+\theta_1^2+\theta_2^2)\sigma^2, \quad \gamma_1 = (\theta_1+\theta_1\theta_2)\sigma^2, \quad \gamma_2 = \theta_2\sigma^2, \quad \gamma_h = 0 \ (h>2)$$

メモ

  • AR過程の自己相関は幾何級数的に減衰、MA(q)の自己共分散は $q$ 次で打ち切り → 標本からの次数選択に利用
  • 偏自己相関係数はAR次数選択に有用：AR(p)は $p+1$ 次以降が0
  • DW検定の注意：検定不能領域あり、被説明変数の過去値を含むモデルには適用不可
  • ペリオドグラムは漸近不偏だが一致推定量ではない → 平滑化が必要
  • 状態空間モデルは可変パラメータモデルや潜在変数モデルなど柔軟な表現が可能
