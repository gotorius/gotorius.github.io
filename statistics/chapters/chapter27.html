はじめに

この章で学ぶこと

これまでの章では、複数の変量を同時に観測した横断面データ（cross-sectional data）を扱ってきた。しかし現実世界のデータには、時間の経過とともに繰り返し観測されるデータが多く存在する。株価、GDP、気温、売上高など、時間的順序を持つデータを「時系列データ」と呼び、その分析手法が時系列解析（time-series analysis）である。

この章では、時系列解析の理論的基礎を学ぶ。具体的には、定常性・自己相関という基本概念から出発し、AR・MA・ARMAなどの代表的なパラメトリックモデルを理解し、それらをデータから識別・推定するための手法を習得する。

普通の回帰分析と時系列分析の違い

章の内容に入る前に、なぜ通常の統計手法では時系列データをそのまま扱えないのかを理解しておこう。

通常の回帰分析では、観測値 $Y_1, Y_2, \ldots, Y_n$ は互いに独立と仮定する（独立同一分布 i.i.d.）。しかし時系列データでは、連続する観測値には系列相関（serial correlation）が存在することが多い。たとえば「今月の売上が高ければ来月の売上も高い傾向がある」というように、隣り合う時点の値が似た値をとりやすい。

このような時系列の特徴は、以下の問題を引き起こす：

  • OLS推定量の効率性が失われる：誤差項に系列相関があると、OLSは不偏だが最小分散ではなくなる
  • 標準誤差の推定が偏る：通常のt検定・F検定の結果が信頼できなくなる
  • 見せかけの回帰（spurious regression）：非定常な時系列同士を回帰すると、無関係なのに有意な関係があるように見えてしまう

そのため、時系列データの分析には専用の理論とモデルが必要となる。

なぜ定常性が重要なのか

時系列解析で最も基本的かつ重要な概念が定常性（stationarity）である。直感的には、「時間が経過しても統計的な性質が変わらない」という状態を指す。

たとえば、サイコロを振り続けた結果の系列は定常である（平均・分散・自己相関は時点によらず一定）。一方、毎年インフレで価格が上昇し続ける物価データは非定常である（平均が時点とともに変化する）。

定常性が重要な理由は、定常過程であれば以下が成立するからである：

  • 1つの時系列から時点をずらしながら標本を集めることで、母平均・母分散を推定できる（エルゴード性）
  • ARMAなどのモデルが数学的に well-defined になる
  • 予測や推論の理論が整備されている

定常性の基礎

自己共分散と自己相関係数

時系列データ $Y_1, Y_2, \ldots, Y_T$ では、各時点 $t$ での $Y_t$ は確率変数とみなされる。通常の統計では異なる変数間の共分散を考えるが、時系列では同じ変数の異なる時点間の共分散を考える。これを自己共分散（autocovariance）という。

自己共分散の定義

$$\gamma_{t,h} = \text{Cov}[Y_t, Y_{t-h}] = E[(Y_t - \mu_t)(Y_{t-h} - \mu_{t-h})]$$

$h$ はラグ（lag）と呼ばれ、2時点間の時間差を表す。自己共分散を $\gamma_0 = \text{Var}[Y_t]$ で割ったものが自己相関係数 $\rho_{t,h} = \gamma_{t,h}/\sqrt{\gamma_{t,0}\cdot\gamma_{t-h,0}}$ である。

一般に、平均 $\mu_t$ や分散 $\sigma_t^2$、自己共分散 $\gamma_{t,h}$ は観測時点 $t$ に依存するため、計量的な分析が困難になる。そこで「定常性」という仮定を置く。

共分散定常過程（弱定常過程）

共分散定常過程の定義

$\{Y_t\}$ が共分散定常過程（covariance stationary process）または弱定常過程（weakly stationary process）であるとは、以下の3条件を満たすことをいう：

  1. $E[Y_t] = \mu$（平均が時点によらず一定）
  2. $\text{Var}[Y_t] = \gamma_0 < \infty$（有限の分散）
  3. $\text{Cov}[Y_t, Y_{t-h}] = \gamma_{|h|}$（自己共分散が時点 $t$ によらずラグ $h$ のみに依存）

条件3が成立するとき、自己相関係数も $\rho_h = \gamma_h / \gamma_0$ と表せ、時点によらず一定となる。

強定常過程との違い

より強い仮定として、強定常過程（strongly stationary process）がある。これは、任意の整数 $h_1, h_2, \ldots, h_m$ に対して、$(Y_t, Y_{t-h_1}, \ldots, Y_{t-h_m})$ の同時分布が時点 $t$ によらない場合を指す。

種別 条件 備考
強定常過程 同時分布が時点に依存しない 分布形状を含む全情報が不変
弱定常過程（共分散定常） 平均・分散・自己共分散が時点に依存しない 1・2次のモーメントのみ不変

平均と分散が有限な強定常過程は弱定常過程でもあるが、逆は必ずしも成り立たない。工学や経済学では通常「定常」と言えば弱定常を指すことが多い。

ホワイトノイズ

共分散定常過程の中で最も単純なものがホワイトノイズ（white noise）である。

ホワイトノイズの定義

$\{U_t\}$ がホワイトノイズであるとは：

  • $E[U_t] = 0$（平均0）
  • $\text{Var}[U_t] = \sigma^2$（一定の有限分散）
  • $\text{Cov}[U_t, U_{t-h}] = 0$ （$h \neq 0$ のとき、系列無相関）

記号では $U_t \sim \text{WN}(0, \sigma^2)$ と書く。正規分布の場合は $U_t \sim \text{NID}(0, \sigma^2)$（正規独立同一分布）とも書く。

ホワイトノイズは定常であり、過去の値から将来を予測できない（各時点が独立）。時系列モデルでは、予測できない「ショック」（外生的なランダム変動）の表現として使われる。

AR過程（自己回帰過程）

AR(1)過程とは

最もシンプルな時系列モデルの1つが1次の自己回帰過程（AR(1)過程）である。直感的には「現在の値が1期前の値に依存する」という構造を持つ。

AR(1)過程（定数項あり）

$$Y_t = c + \phi_1 Y_{t-1} + U_t \quad (U_t \sim \text{WN}(0, \sigma^2))$$

ここで $c$ は定数項（切片）、$\phi_1$ は自己回帰係数（AR係数）である。$|\phi_1| < 1$ のとき、$\{Y_t\}$ は共分散定常過程となる。

AR(1)過程の平均・分散・自己相関

$|\phi_1| < 1$ の定常条件を満たす場合、両辺の期待値をとると $\mu = c + \phi_1 \mu$、すなわち $\mu = c/(1 - \phi_1)$ が得られる。

分散については、$(Y_t - \mu) = \phi_1(Y_{t-1} - \mu) + U_t$ の両辺を2乗して期待値をとると：

AR(1)過程の平均・分散・自己相関（$|\phi_1| < 1$ のとき）

$$\mu = \frac{c}{1-\phi_1}, \quad \gamma_0 = \frac{\sigma^2}{1-\phi_1^2}, \quad \gamma_h = \phi_1^h \gamma_0, \quad \rho_h = \phi_1^h \quad (h \geq 0)$$

自己相関係数 $\rho_h = \phi_1^h$ は、$|\phi_1| < 1$ のとき $h$ の増加とともに指数的に減衰する。

$\phi_1$ の値による系列の性質の違いをまとめる：

$\phi_1$ の値 定常性 系列の見た目
$0 < \phi_1 < 1$ 定常 なだらかに変動（正の自己相関）
$\phi_1 = 0$ 定常（ホワイトノイズ） 不規則に変動
$-1 < \phi_1 < 0$ 定常 上下に激しく変動（符号交互）
$\phi_1 = 1$ 非定常（単位根） ランダムウォーク（平均が変化）

AR(p)過程への一般化

$p$期前までの値に依存する場合、$p$次の自己回帰過程（AR(p)過程）に拡張できる：

AR(p)過程

$$Y_t = c + \phi_1 Y_{t-1} + \phi_2 Y_{t-2} + \cdots + \phi_p Y_{t-p} + U_t$$

定常性条件：$p$ 次多項式（特性多項式）

$$1 - \phi_1 z - \phi_2 z^2 - \cdots - \phi_p z^p = 0$$

のすべての解（複素解を含む）が複平面上で単位円の外側にある（$|z| > 1$）ことが必要十分条件。

このとき平均は $\mu = c/(1 - \phi_1 - \cdots - \phi_p)$。

ラグオペレーターによる表現

ラグオペレーター $L$（$LY_t = Y_{t-1}$, $L^h Y_t = Y_{t-h}$）を用いると、AR(p)過程を簡潔に表現できる：

$$\phi(L)Y_t = c + U_t, \quad \phi(L) = 1 - \phi_1 L - \phi_2 L^2 - \cdots - \phi_p L^p$$

MA過程（移動平均過程）

MA(1)過程とは

AR過程では「現在の値が過去の観測値に依存する」モデルだった。これに対して移動平均過程（MA過程）は、「現在の値が現在と過去のショックの加重和で生成される」モデルである。

MA(1)過程

$$Y_t = \mu + U_t + \theta_1 U_{t-1} \quad (U_t \sim \text{WN}(0, \sigma^2))$$

$E[Y_t] = \mu$ であり、分散と自己共分散は次のようになる：

$$\gamma_0 = (1 + \theta_1^2)\sigma^2, \quad \gamma_1 = \theta_1 \sigma^2, \quad \gamma_h = 0 \; (h \geq 2)$$

重要な性質：MA(1)過程は常に（$\theta_1$ の値によらず）定常である。

MA(q)過程への一般化

MA(q)過程

$$Y_t = \mu + U_t + \theta_1 U_{t-1} + \theta_2 U_{t-2} + \cdots + \theta_q U_{t-q}$$

自己共分散：

$$\gamma_h = \begin{cases} (1+\theta_1^2+\cdots+\theta_q^2)\sigma^2 & (h=0) \\ (\theta_h + \theta_1\theta_{h+1} + \cdots + \theta_{q-h}\theta_q)\sigma^2 & (1 \leq h \leq q) \\ 0 & (h > q) \end{cases}$$

最重要な特徴：$h > q$ のとき $\gamma_h = 0$。すなわち、MA(q)過程の自己共分散は $q$ 次で打ち切られる（cut-off）。これはAR過程の自己相関が指数的にゆっくり減衰するのと対照的である。

AR過程とMA過程の対比

性質 AR(p)過程 MA(q)過程
定常性 特性多項式の根が単位円外の場合 常に定常
自己相関の挙動 指数的に減衰（ゆっくり） $q$ 次以降で完全に0（打ち切り）
偏自己相関の挙動 $p$ 次以降で0 指数的に減衰
次数の識別 偏自己相関の打ち切り次数を見る 自己相関の打ち切り次数を見る

ARMA・ARIMAと単位根

ARMA(p,q)過程

AR過程とMA過程を組み合わせると、より柔軟なARMA（自己回帰移動平均）過程が得られる。

ARMA(p,q)過程

$$Y_t = c + \phi_1 Y_{t-1} + \cdots + \phi_p Y_{t-p} + U_t + \theta_1 U_{t-1} + \cdots + \theta_q U_{t-q}$$

ラグ多項式を用いると：

$$\phi(L)Y_t = c + \theta(L)U_t, \quad \theta(L) = 1 + \theta_1 L + \cdots + \theta_q L^q$$

定常性条件：AR部分の特性多項式 $\phi(z) = 0$ の解がすべて単位円外にあること。比較的小さい $(p,q)$ でも複雑な自己共分散構造を表現できるため、実証研究でよく使われる。

非定常系列と階差

実際のデータ（GDP、株価等）はしばしば非定常である。典型的な非定常過程として、$\phi_1 = 1$ の場合のAR(1)過程、すなわちランダムウォークがある：

$$Y_t = Y_{t-1} + U_t \quad \Rightarrow \quad Y_t = Y_0 + \sum_{s=1}^t U_s$$

この場合 $\text{Var}[Y_t] = t\sigma^2$ と時点とともに増加するため、定常でない。

このような場合、階差（difference）$\Delta Y_t = Y_t - Y_{t-1}$ をとることで定常化できることがある。

ARIMA(p,1,q)過程

$\{Y_t\}$ 自体は非定常だが、1階差分 $\Delta Y_t = Y_t - Y_{t-1}$ がARMA(p,q)過程に従う場合：

$$\Delta Y_t = \phi_1 \Delta Y_{t-1} + \cdots + \phi_p \Delta Y_{t-p} + U_t + \theta_1 U_{t-1} + \cdots + \theta_q U_{t-q}$$

これを ARIMA(p,1,q)（$d=1$ の自己回帰和分移動平均）過程と呼ぶ。

たとえば株式の対数価格 $Y_t = \log P_t$ は通常 ARIMA(p,1,0) に近く、対数収益率 $\Delta Y_t = \log P_t - \log P_{t-1}$ は定常とみなされることが多い。

単位根検定

系列が非定常（単位根あり）かどうかを統計的に検定する手法が単位根検定である。

ディッキー・フラー検定（DF検定）と拡張DF検定（ADF検定）

AR(1)過程 $Y_t = c + \phi_1 Y_{t-1} + U_t$ に対して：

  • 帰無仮説 $H_0: \phi_1 = 1$（単位根あり＝非定常）
  • 対立仮説 $H_1: |\phi_1| < 1$（定常）

式を変形すると $\Delta Y_t = c + (\phi_1 - 1)Y_{t-1} + U_t = c + \delta Y_{t-1} + U_t$ となり、$H_0: \delta = 0$ を検定する。

ただし $H_0$ のもとでの検定統計量は通常の $t$ 分布に従わず、専用のディッキー・フラー臨界値表を使用する必要がある（臨界値は通常の $t$ 分布より小さい）。

高次のAR過程に対応した拡張DF検定（ADF検定）では、$\Delta Y_{t-1}, \ldots, \Delta Y_{t-p}$ を追加した回帰を行う。

スペクトラム解析と次数決定

スペクトラム密度

これまでは時間領域（自己相関など）で時系列の性質を記述してきたが、周波数領域でも分析できる。スペクトラム密度（spectral density）は、自己共分散のフーリエ変換として定義される。

スペクトラム密度

$$f(\lambda) = \frac{1}{2\pi}\sum_{h=-\infty}^{\infty} \gamma_h e^{-i\lambda h}, \quad \lambda \in [-\pi, \pi]$$

逆に、自己共分散はスペクトラム密度から：

$$\gamma_h = \int_{-\pi}^{\pi} e^{i\lambda h} f(\lambda)\, d\lambda$$

特に AR(1) のスペクトラム密度は：

$$f(\lambda) = \frac{\sigma^2}{2\pi(1 + \phi_1^2 - 2\phi_1\cos\lambda)}$$

ペリオドグラム

スペクトラム密度を標本から推定したものがペリオドグラム（periodogram）である。標本自己共分散 $\hat{\gamma}_h$ を用いて計算するが、ペリオドグラムは漸近不偏推定量であるものの一致推定量ではない（標本サイズが増えても推定量の分散が収束しない）。そのため実用上は平滑化が必要である。

コレログラムと次数決定

標本自己相関係数 $\hat{\rho}_h$ を $h$ に対してプロットしたグラフをコレログラム（correlogram）、または自己相関関数（ACF）プロットという。同様に、偏自己相関係数（PACF）プロットも次数決定に用いる。

コレログラムによる次数決定の目安

自己相関係数（ACF） 偏自己相関係数（PACF） 有望なモデル
$q$ 次以降ほぼ0（打ち切り） ゆっくり減衰 MA(q)
ゆっくり減衰 $p$ 次以降ほぼ0（打ち切り） AR(p)
ゆっくり減衰 ゆっくり減衰 ARMA(p,q)

次数を複数候補から選ぶ際には、AIC（赤池情報量規準）や BIC（ベイズ情報量規準）が用いられる。ARMAモデルの場合：

$$\text{AIC} = -2\log\hat{L} + 2k, \quad \text{BIC} = -2\log\hat{L} + k\log T$$

$\hat{L}$：最尤推定量における尤度、$k$：パラメータ数、$T$：標本サイズ。AIC・BICが最小のモデルを選択する。

ダービン・ワトソン検定

回帰モデルの誤差項に1次の系列相関があるかどうかを検定する手法がダービン・ワトソン検定（Durbin-Watson test）である。

ダービン・ワトソン比

$$DW = \frac{\sum_{t=2}^{T}(\hat{U}_t - \hat{U}_{t-1})^2}{\sum_{t=1}^{T}\hat{U}_t^2} \approx 2(1 - \hat{\rho}_1)$$

$\hat{\rho}_1$：残差の1次自己相関係数の推定量。

  • $DW \approx 2$：系列相関なし（$\hat{\rho}_1 \approx 0$）
  • $DW < 2$（$DW \to 0$）：正の系列相関（$\hat{\rho}_1 > 0$）
  • $DW > 2$（$DW \to 4$）：負の系列相関（$\hat{\rho}_1 < 0$）

注意：DW検定は被説明変数の過去値を含む動的回帰モデルには適用不可（推定量が偏る）。また、検定不能領域（inconclusive region）も存在する。

公式まとめ

AR(1)過程（$|\phi_1| < 1$）の諸量

$$\mu = \frac{c}{1-\phi_1}, \quad \gamma_0 = \frac{\sigma^2}{1-\phi_1^2}, \quad \gamma_h = \phi_1^h \gamma_0 \; (h \geq 0), \quad \rho_h = \phi_1^h$$

AR(p)過程の定常性条件

特性多項式 $1 - \phi_1 z - \phi_2 z^2 - \cdots - \phi_p z^p = 0$ のすべての解が $|z| > 1$（単位円の外側）。

平均：$\mu = c/(1 - \phi_1 - \cdots - \phi_p)$

MA(q)過程の自己共分散

$$\gamma_h = \begin{cases} (1+\theta_1^2+\cdots+\theta_q^2)\sigma^2 & (h=0) \\ (\theta_h + \theta_1\theta_{h+1} + \cdots + \theta_{q-h}\theta_q)\sigma^2 & (1 \leq h \leq q) \\ 0 & (h > q) \end{cases}$$

ARMA(p,q)過程とラグ多項式

$$\phi(L)Y_t = c + \theta(L)U_t, \quad \phi(L) = 1 - \sum_{j=1}^p \phi_j L^j, \quad \theta(L) = 1 + \sum_{j=1}^q \theta_j L^j$$

AR(1)のスペクトラム密度

$$f(\lambda) = \frac{\sigma^2}{2\pi(1+\phi_1^2-2\phi_1\cos\lambda)}, \quad \lambda \in [-\pi, \pi]$$

ダービン・ワトソン比

$$DW = \frac{\sum_{t=2}^{T}(\hat{U}_t - \hat{U}_{t-1})^2}{\sum_{t=1}^{T}\hat{U}_t^2} \approx 2(1-\hat{\rho}_1)$$

$DW \approx 2$：系列相関なし、$DW \to 0$：正の相関、$DW \to 4$：負の相関

AIC・BICによる次数選択

$$\text{AIC} = -2\log\hat{L} + 2k, \quad \text{BIC} = -2\log\hat{L} + k\log T$$

値が最小となるモデルを選択する。

例題・演習

問27.1 AR(1)過程 $Y_t = 2(1-\phi_1) + \phi_1 Y_{t-1} + U_t$ のグラフ対応

[1] $\phi_1 = -0.8, 0, 0.7, 1$ のグラフ対応

  • (a) $\phi_1 = 0.7$：隣接データが似た値、変動が穏やか
  • (b) $\phi_1 = 0$：ホワイトノイズ、不規則変動
  • (c) $\phi_1 = 1$：非定常、平均が一定でない
  • (d) $\phi_1 = -0.8$：上下に激しく変動（符号が交互）

[2] DW比

まとめ・ポイント

この章の重要ポイント

  • 定常性が基盤：時系列解析の理論はすべて「定常性」を前提とする。非定常データには階差などの変換が必要
  • AR vs MA の識別：AR過程は自己相関がゆっくり減衰・偏自己相関が打ち切り。MA過程は自己相関が打ち切り・偏自己相関がゆっくり減衰。コレログラムで識別する
  • 単位根の危険性：非定常系列を定常と思って分析すると「見せかけの回帰」が起きる。単位根検定（ADF検定等）で確認が重要
  • ARMA(p,q)の柔軟性：低次のARMAモデルで複雑な自己相関構造を表現できる。パラメータ数節約のために有用
  • 次数選択はAIC/BICで：コレログラムで候補を絞り、AIC・BICで最終選択する
  • DW検定の限界：有効な検定条件は回帰モデルに被説明変数の過去値が含まれない場合。含む場合はBreusch-Godfrey検定が適切
  • ペリオドグラムの注意：漸近不偏だが一致推定量でない。実用にはスペクトラムの平滑化推定量を使う
  • 状態空間モデルの拡張性：観測方程式＋状態方程式の枠組みは、時変パラメータモデルや潜在変数モデルなど多様な時系列モデルを統一的に表現できる
