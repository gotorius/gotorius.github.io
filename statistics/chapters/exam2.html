第2章「確率分布と母関数」のキーポイント
  • 同時確率密度関数 $f(x,y)$：2つの連続確率変数 $X, Y$ の分布を表す関数．全体を積分すると 1 になる（規格化条件）．
  • 周辺確率密度関数：同時密度関数をもう一方の変数について積分して得る．$f_X(x) = \displaystyle\int_{-\infty}^\infty f(x,y)\,dy$
  • 条件付き確率密度関数：$f_{Y|X}(y|x) = \dfrac{f(x,y)}{f_X(x)}$（$X=x$ を与えたときの $Y$ の分布）．
  • 確率母関数 $G(s)$：非負整数値をとる離散確率変数に対して $G(s) = E[s^X] = \displaystyle\sum_x s^x p(x)$．$G'(1) = E[X]$，$V[X] = G''(1) + G'(1) - (G'(1))^2$
  • モーメント母関数 $m(\theta)$：$m(\theta) = E[e^{\theta X}]$．$m'(0) = E[X]$，$m''(0) = E[X^2]$，$V[X] = m''(0) - (m'(0))^2$
  • 幾何分布 $Ge(p)$：$p(x) = p(1-p)^x$（$x=0,1,2,\ldots$）．期待値 $(1-p)/p$，分散 $(1-p)/p^2$
  • 指数分布 $Ex(\lambda)$：$f(x) = \lambda e^{-\lambda x}$（$x \geq 0$）．期待値 $1/\lambda$，分散 $1/\lambda^2$

問2.1　単位正方形上の同時確率密度関数

問題

$xy$-平面の単位正方形 $\{(x,y) \mid 0 \leq x \leq 1,\; 0 \leq y \leq 1\}$ 上の確率密度関数 $f(x,y)$ を

$$f(x,y) = c(x+y)$$

とおく．

[1]　基準化定数 $c$ を求めよ．

[2]　$X$ の周辺確率密度関数を求めよ．

[3]　$X$ を与えたときの $Y$ の条件付き確率密度関数を求めよ．

[1] 解答・解説
基準化定数とは

確率密度関数は「全体を積分すると 1 になる」という条件（規格化条件）を満たす必要があります． $c$ はこの条件を満たすように決める定数です：

$$\int_0^1\!\int_0^1 f(x,y)\,dx\,dy = 1$$

Step 1：二重積分を計算する

$f(x,y) = c(x+y)$ を代入します：

$$\int_0^1\!\int_0^1 c(x+y)\,dx\,dy = c\int_0^1\!\int_0^1 (x+y)\,dx\,dy$$

$x$ と $y$ の対称性（$x$ と $y$ を入れ替えても積分領域・被積分関数とも変わらない）を利用します：

$$\int_0^1\!\int_0^1 (x+y)\,dx\,dy = 2\int_0^1\!\int_0^1 x\,dx\,dy$$

内側の $x$ についての積分：$\displaystyle\int_0^1 x\,dx = \left[\frac{x^2}{2}\right]_0^1 = \frac{1}{2}$ 外側の $y$ についての積分：$\displaystyle\int_0^1 \frac{1}{2}\,dy = \frac{1}{2}$ したがって：$2 \times \dfrac{1}{2} = 1$

Step 2：$c$ を求める

$$c \times 1 = 1 \implies c = 1$$
答え

$c = 1$，すなわち $f(x,y) = x + y \quad (0 \leq x \leq 1,\; 0 \leq y \leq 1)$

[2] 解答・解説
周辺確率密度関数の求め方

同時確率密度関数 $f(x,y)$ から $X$ だけの分布を取り出すには， $y$ についてすべての範囲で積分して「$y$ を消す」操作をします：

$$f_X(x) = \int_{-\infty}^{\infty} f(x,y)\,dy$$

この $f_X(x)$ を $X$ の周辺確率密度関数といいます．

$y$ について積分する

$f(x,y) = x+y$ を $0 \leq y \leq 1$ で積分します：

$$f_X(x) = \int_0^1 (x+y)\,dy = \left[xy + \frac{y^2}{2}\right]_0^1 = x \cdot 1 + \frac{1}{2} - 0 = x + \frac{1}{2}$$
答え
$$f_X(x) = x + \frac{1}{2} \quad (0 \leq x \leq 1)$$

確認：$\displaystyle\int_0^1\!\left(x+\frac{1}{2}\right)dx = \left[\frac{x^2}{2}+\frac{x}{2}\right]_0^1 = \frac{1}{2}+\frac{1}{2} = 1$ ✓

[3] 解答・解説
条件付き確率密度関数の求め方

$X = x$ という値が与えられたときの $Y$ の分布を表す条件付き確率密度関数は：

$$f_{Y|X}(y|x) = \frac{f(x,y)}{f_X(x)}$$

「全体の密度を $x$ が固定されたときの $X$ の密度で割る」ことで，$X=x$ を条件としたときの $Y$ の分布が得られます．

分子・分母を代入する

$f(x,y) = x+y$，$f_X(x) = x+\dfrac{1}{2}$ を代入します：

$$f_{Y|X}(y|x) = \frac{x+y}{x+\dfrac{1}{2}}$$
答え
$$f_{Y|X}(y|x) = \frac{x+y}{x+\dfrac{1}{2}} \quad (0 \leq y \leq 1)$$

確認：$\displaystyle\int_0^1 \frac{x+y}{x+\frac{1}{2}}\,dy = \frac{1}{x+\frac{1}{2}}\left[xy+\frac{y^2}{2}\right]_0^1 = \frac{x+\frac{1}{2}}{x+\frac{1}{2}} = 1$ ✓

$x$ が大きいほど $X$ と $Y$ が独立（$f_{Y|X}(y|x) \to 1$）に近づく一方， $x$ が 0 に近づくほど $f_{Y|X}(y|x)$ は $y$ の影響を強く受けます．

問2.2　幾何分布の確率母関数・期待値・分散

問題

$X$ を幾何分布に従う離散確率変数とする．すなわち $X$ は非負整数値をとりその確率関数を

$$p(x) = p(1-p)^x, \quad x = 0, 1, 2, \ldots$$

とする．ただし $0 < p < 1$ は幾何分布のパラメータである． $X$ の確率母関数 $G(s)$ を求めよ．また確率母関数を微分することにより $X$ の期待値と分散を求めよ．

解答・解説
幾何分布のイメージ

確率 $p$ で成功するベルヌーイ試行を繰り返すとき，最初に成功するまでの失敗の回数が幾何分布 $Ge(p)$ に従います． $p(x) = p(1-p)^x$ は「$x$ 回失敗して，次に成功する」確率です．

例：$p=0.5$ の場合，$p(0) = 0.5$（0回失敗で成功），$p(1) = 0.25$，$p(2) = 0.125$，…

Step 1：確率母関数 $G(s)$ の計算

確率母関数の定義を適用する

確率母関数は $G(s) = E[s^X] = \displaystyle\sum_{x=0}^\infty s^x p(x)$ です．$p(x) = p(1-p)^x$ を代入します：

$$G(s) = \sum_{x=0}^\infty s^x \cdot p(1-p)^x = p\sum_{x=0}^\infty \bigl((1-p)s\bigr)^x$$

これは初項 1，公比 $(1-p)s$ の等比級数です． $|(1-p)s| < 1$，すなわち $|s| < \dfrac{1}{1-p}$ のとき収束し：

$$G(s) = \frac{p}{1-(1-p)s}, \quad |s| < \frac{1}{1-p}$$

Step 2：$G'(s)$ と $G''(s)$ を計算して $s=1$ を代入

1階微分 $G'(s)$ を求め，$G'(1)$ を計算する

商の微分則（または合成関数の微分）を用います：

$$G'(s) = \frac{p(1-p)}{\bigl(1-(1-p)s\bigr)^2}$$

$s=1$ を代入すると，$1-(1-p)\cdot 1 = p$ なので：

$$G'(1) = \frac{p(1-p)}{p^2} = \frac{1-p}{p}$$

確率母関数の性質から $G'(1) = E[X]$ なので，期待値 $E[X] = \dfrac{1-p}{p}$

2階微分 $G''(s)$ を求め，$G''(1)$ を計算する

$$G''(s) = \frac{2p(1-p)^2}{\bigl(1-(1-p)s\bigr)^3}$$

$s=1$ を代入すると：

$$G''(1) = \frac{2p(1-p)^2}{p^3} = \frac{2(1-p)^2}{p^2}$$

Step 3：分散の公式で $V[X]$ を求める

確率母関数による分散の公式を適用する

確率母関数による分散の公式は：

$$V[X] = G''(1) + G'(1) - \bigl(G'(1)\bigr)^2$$

各値を代入します：

$$V[X] = \frac{2(1-p)^2}{p^2} + \frac{1-p}{p} - \left(\frac{1-p}{p}\right)^2$$

通分して計算します（分母を $p^2$ に統一）：

$$= \frac{2(1-p)^2}{p^2} + \frac{p(1-p)}{p^2} - \frac{(1-p)^2}{p^2} = \frac{(1-p)\bigl[2(1-p) + p - (1-p)\bigr]}{p^2}$$
$$= \frac{(1-p)(2-2p+p-1+p)}{p^2} = \frac{(1-p)(2p)}{p^2}$$

括弧内を展開すると：$2(1-p) + p - (1-p) = (1-p) + p = 1$

$$V[X] = \frac{1-p}{p^2}$$
答え
$$G(s) = \frac{p}{1-(1-p)s} \quad \left(|s| < \frac{1}{1-p}\right)$$
$$E[X] = \frac{1-p}{p}, \qquad V[X] = \frac{1-p}{p^2}$$
数値例で確認

$p = 0.5$ の場合：$E[X] = (1-0.5)/0.5 = 1$，$V[X] = (1-0.5)/0.25 = 2$

直感的には「成功確率が 50% なら，平均 1 回の失敗で成功する」ことに対応しています． また $p$ が小さいほど（成功しにくいほど），期待値・分散ともに大きくなります．

問2.3　指数分布のモーメント母関数・期待値・分散

問題

$X$ を指数分布に従う連続確率変数とする．すなわち $X \geq 0$ の確率密度関数を

$$f(x) = \lambda e^{-\lambda x}$$

とする．ただし $\lambda > 0$ はパラメータである． $X$ のモーメント母関数を求めよ．またモーメント母関数を微分することにより $X$ の期待値と分散を求めよ．

解答・解説
指数分布のイメージ

指数分布は「ある事象が次に起きるまでの待ち時間」をモデル化するのによく使われます． たとえば，コールセンターへの次の電話が来るまでの時間，放射性物質が次に崩壊するまでの時間などです． パラメータ $\lambda$ は単位時間あたりの「発生率」を表します．

Step 1：モーメント母関数 $m(\theta)$ の計算

定義に従って積分する

モーメント母関数は $m(\theta) = E[e^{\theta X}]$ です．$f(x) = \lambda e^{-\lambda x}$ を代入します：

$$m(\theta) = \int_0^\infty e^{\theta x}\cdot \lambda e^{-\lambda x}\,dx = \lambda\int_0^\infty e^{-(\lambda-\theta)x}\,dx$$

この積分は $\lambda - \theta > 0$，すなわち $\theta < \lambda$ のときに収束します． 指数関数の積分公式 $\displaystyle\int_0^\infty e^{-ax}\,dx = \dfrac{1}{a}$（$a > 0$）を用いると：

$$m(\theta) = \lambda \cdot \frac{1}{\lambda - \theta} = \frac{\lambda}{\lambda - \theta}, \quad \theta < \lambda$$

Step 2：$m'(\theta)$ を求めて期待値を得る

1階微分 $m'(\theta)$ を計算し，$\theta=0$ を代入

$m(\theta) = \lambda(\lambda-\theta)^{-1}$ を $\theta$ で微分します：

$$m'(\theta) = \lambda \cdot (-1) \cdot (\lambda-\theta)^{-2} \cdot (-1) = \frac{\lambda}{(\lambda-\theta)^2}$$

$\theta = 0$ を代入：

$$m'(0) = \frac{\lambda}{\lambda^2} = \frac{1}{\lambda}$$

モーメント母関数の性質から $m'(0) = E[X]$ なので，期待値 $E[X] = \dfrac{1}{\lambda}$

Step 3：$m''(\theta)$ を求めて $E[X^2]$ を得る

2階微分 $m''(\theta)$ を計算し，$\theta=0$ を代入

$$m''(\theta) = \frac{2\lambda}{(\lambda-\theta)^3}$$

$\theta = 0$ を代入：

$$m''(0) = \frac{2\lambda}{\lambda^3} = \frac{2}{\lambda^2}$$

モーメント母関数の性質から $m''(0) = E[X^2]$ なので，$E[X^2] = \dfrac{2}{\lambda^2}$

Step 4：分散を計算する

分散の公式 $V[X] = E[X^2] - (E[X])^2$ を用います：

$$V[X] = \frac{2}{\lambda^2} - \left(\frac{1}{\lambda}\right)^2 = \frac{2}{\lambda^2} - \frac{1}{\lambda^2} = \frac{1}{\lambda^2}$$
答え
$$m(\theta) = \frac{\lambda}{\lambda - \theta} \quad (\theta < \lambda)$$
$$E[X] = \frac{1}{\lambda}, \qquad V[X] = \frac{1}{\lambda^2}$$
指数分布の性質まとめ

指数分布では $E[X] = 1/\lambda$，$SD[X] = \sqrt{V[X]} = 1/\lambda$ となり， 期待値と標準偏差が等しいという特徴があります．

$\lambda$ が大きいほど（事象の発生率が高いほど）待ち時間の平均・ばらつきともに小さくなります．

$\lambda$$E[X] = 1/\lambda$$SD[X] = 1/\lambda$
$\lambda=1$$1$$1$
$\lambda=2$$0.5$$0.5$
$\lambda=0.5$$2$$2$
母関数の微分による期待値・分散の求め方（まとめ）
母関数の種類期待値分散
確率母関数 $G(s)$
（離散型に利用）		 $E[X] = G'(1)$ $V[X] = G''(1) + G'(1) - (G'(1))^2$
モーメント母関数 $m(\theta)$
（連続型に多く利用）		 $E[X] = m'(0)$ $V[X] = m''(0) - (m'(0))^2$
