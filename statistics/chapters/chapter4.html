概要

第4章では、確率変数の変数変換による確率密度関数の変化と、確率変数の線形結合の分布を求める方法、さらにデータの変換（対数変換・ベキ乗変換・Box-Cox変換・ロジット変換・プロビット変換）について扱う。 1変数・2変数の変数変換の手法はヤコビアンを用いた一般的な方法として統計学全体で頻出する。

キーワード：変数変換、確率変数の線形結合の分布、ヤコビアン、対数変換、ベキ乗変換、Box-Cox変換、ロジット変換、プロビット変換

重要概念

1変数の変数変換

  • 連続型確率変数 $X$ の確率密度関数を $f(x)$ とする
  • 新たな確率変数 $Y = g(X)$（$g$ は1対1の関数）の確率密度関数は： $$f_Y(y) = f\!\left(g^{-1}(y)\right) \left|\frac{d}{dy}g^{-1}(y)\right|$$
  • すなわち $f(x) / |g'(x)|$ の $x$ を $y$ に変換したもの

2変数の変数変換（ヤコビアン）

  • $(X, Y)$ の同時確率密度関数を $f(x, y)$ とし、1対1の変換 $(Z, W) = (u(X,Y),\, v(X,Y))$ を考える
  • 逆変換を $(X, Y) = (s(z, w),\, t(z, w))$ とする
  • ヤコビアン： $$J(X, Y) = \frac{\partial(u, v)}{\partial(x, y)} = \begin{vmatrix} \dfrac{\partial u}{\partial x} & \dfrac{\partial u}{\partial y} \\[6pt] \dfrac{\partial v}{\partial x} & \dfrac{\partial v}{\partial y} \end{vmatrix}$$
  • $(Z, W)$ の確率密度関数は $f(s(z, w),\, t(z, w)) / |J(s(z,w),\, t(z,w))|$

確率変数の線形結合の分布

  • 独立な $X, Y$ の線形結合 $aX + bY$ の分布を求める方法：
    1. 母関数を使う方法：$E[e^{\theta(aX+bY)}] = E[e^{\theta aX}] \cdot E[e^{\theta bY}]$ を計算し、既知の分布と比較
    2. 変数変換を使う方法：$Z = aX + bY,\; W = Y$ と置き、$W$ について積分して $Z$ の周辺分布を得る
  • 変数変換の方法では、逆変換 $(X, Y) = ((z - bw)/a,\; w)$ のヤコビアンは $|a|$
  • $Z$ の確率密度関数： $$f_Z(z) = \frac{1}{|a|} \int_{-\infty}^{\infty} f_X\!\left(\frac{z - bw}{a}\right) f_Y(w)\,dw$$

公式・定理

線形結合の重要な結果

  • $X \sim N(\mu_1, \sigma_1^2),\; Y \sim N(\mu_2, \sigma_2^2)$ が独立 $\Rightarrow$ $aX + bY + c \sim N(a\mu_1 + b\mu_2 + c,\; a^2\sigma_1^2 + b^2\sigma_2^2)$
  • $X \sim \mathrm{Bin}(n_1, p),\; Y \sim \mathrm{Bin}(n_2, p)$ が独立 $\Rightarrow$ $X + Y \sim \mathrm{Bin}(n_1 + n_2, p)$
  • $X \sim \mathrm{Po}(\lambda_1),\; Y \sim \mathrm{Po}(\lambda_2)$ が独立 $\Rightarrow$ $X + Y \sim \mathrm{Po}(\lambda_1 + \lambda_2)$

データの変換法

  • 対数変換：積が積み重なるデータ（人口・株価・所得）に使用。非負値を $(-\infty, \infty)$ に変換可能
  • ベキ乗変換：$x \mapsto x^a$。正規化のための変換
  • Box-Cox変換：パラメータ $\lambda$ に対して $$y = \begin{cases} \dfrac{x^\lambda - 1}{\lambda} & (\lambda \neq 0) \\[6pt] \log x & (\lambda = 0) \end{cases}$$ 注意：非負のデータしか変換できない
  • ロジット変換：$\log \dfrac{p}{1-p}$。確率 $p \in (0,1)$ を $(-\infty, \infty)$ に変換。ロジスティック回帰
  • ロジスティック変換：$\dfrac{1}{1 + e^{-(a + bx)}}$（ロジット変換の逆）
  • プロビット変換：標準正規分布の累積分布関数の逆関数 $\Phi^{-1}(p)$。詳細は第18章

例題・演習

例1：自由度1のカイ二乗分布

標準正規分布に従う $X$ に対する $Y = X^2$ の確率密度関数を求めよ。

解：標準正規分布に従う $X$ の確率密度関数は $\dfrac{1}{\sqrt{2\pi}} \exp\!\left(-\dfrac{x^2}{2}\right)$ である。$Y = X^2$ は1対1ではない。$Y = y$ となるケースは $X = \sqrt{y}$ と $X = -\sqrt{y}$ の2通りあるので、$X > 0$ の場合のみ考え、確率密度関数を2倍とすればよい。

$Y = X^2$ を $X$ で微分すると $2X$ となるので、

$$f_Y(y) = 2 \times \frac{1}{\sqrt{2\pi}} \exp\!\left(-\frac{x^2}{2}\right) \times \frac{1}{2x} = \frac{1}{\sqrt{2\pi y}} \exp\!\left(-\frac{y}{2}\right) \quad (y > 0)$$

問4.1：対数正規分布

確率変数 $X$ は正規分布 $N(\mu, \sigma^2)$ に従い、確率密度関数は $f(x) = \dfrac{1}{\sqrt{2\pi}\,\sigma} \exp\!\left[-\dfrac{(x - \mu)^2}{2\sigma^2}\right]$ である。$Y = \exp(X)$ とする（$Y$ の対数が正規分布に従うので、$Y$ の分布は対数正規分布と呼ばれる）。

(1) $Y$ の期待値を求めよ。

$$E[Y] = E[\exp(X)] = \exp\!\left(\mu + \frac{\sigma^2}{2}\right)$$

正規分布 $N(\mu, \sigma^2)$ のモーメント母関数は $E[\exp(\theta X)] = \exp(\mu\theta + \sigma^2\theta^2/2)$ であるので、$\theta = 1$ とすればよい。

(2) $Y$ の分散を求めよ。

$\theta = 2$ として $E[\exp(2X)] = E[Y^2] = \exp(2\mu + 2\sigma^2)$ となるので、

$$V[Y] = E[Y^2] - (E[Y])^2 = \exp(2\mu + 2\sigma^2) - \exp(2\mu + \sigma^2) = \exp(2\mu + \sigma^2)\big(\exp(\sigma^2) - 1\big)$$

(3) $Y$ の確率密度関数を求めよ。

$Y = \exp(X)$ を $X$ で微分すると $\exp(X)$ である。よって

$$f_Y(y) = \frac{1}{\sqrt{2\pi}\,\sigma\,y} \exp\!\left[-\frac{(\log y - \mu)^2}{2\sigma^2}\right] \quad (y > 0)$$

問4.2：独立な指数分布の和

確率変数 $X, Y$ が独立でそれぞれパラメータ $\lambda$ の指数分布に従う、つまり確率密度関数が $f(x) = \lambda e^{-\lambda x}$ で与えられるとする。$X + Y$ の確率密度関数を求めよ。

解：$Z = X + Y,\; W = Y$ とすることで、$(Z, W)$ の同時確率密度関数は

$$f_X(z - w) \cdot f_Y(w) = \lambda e^{-\lambda(z-w)} \cdot \lambda e^{-\lambda w} = \lambda^2 e^{-\lambda z}$$

ここで $X, Y$ ともに非負であるので、$z$ が与えられたもとで $w$ は $0$ から $z$ の値をとりうる。よって $Z$ の確率密度関数は

$$f_Z(z) = \int_0^z \lambda^2 e^{-\lambda z}\,dw = \lambda^2 z\, e^{-\lambda z} \quad (z \geq 0)$$

メモ

  • 1変数の変数変換：$f(x)/|g'(x)|$ の $x$ を $y$ に置換するだけ。$g$ が1対1でない場合は場合分けが必要（例1のカイ二乗分布）
  • 2変数の変数変換ではヤコビアンの絶対値で割る。行列式の計算を正確に
  • 線形結合の分布：母関数が既知なら母関数法が楽。未知なら変数変換法を用いる
  • 対数変換は「積の蓄積」による偏りを正規化する基本的な手法
  • Box-Cox変換は $\lambda = 0$ で対数変換、$\lambda = 1$ で線形変換に対応するファミリー
  • ロジット変換はロジスティック回帰（第18章）、プロビット変換はプロビット回帰で再登場
  • 問4.2の結果 $\lambda^2 z e^{-\lambda z}$ はガンマ分布 $\mathrm{Ga}(2, \lambda)$ の確率密度関数（第6章で詳しく扱う）
