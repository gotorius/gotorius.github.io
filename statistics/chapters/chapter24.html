クラスター分析とは？――直感的なイメージから

まずクラスター分析（cluster analysis）が何をするものか、日常的な例で考えてみましょう。

たとえばあなたが図書館で本を整理するとします。最初は本にジャンルのラベルが貼られていません。 それでも「似たような内容の本を近くに並べよう」と考えながら棚に分類するはずです。 これがクラスター分析の本質です。
ラベル（答え）なしで、似ているもの同士をグループ（クラスター）にまとめる方法です。

🔑 判別分析との違い

  • 判別分析：「この本は理工書」「この本は小説」などラベルが最初から付いている。新しい本のラベルを予測するルールを学ぶ。
    → 教師あり学習
  • クラスター分析：ラベルがない。データの似ている・似ていないだけを手がかりにグループを作る。
    → 教師なし学習

クラスタリング手法は大きく 2 種類に分かれます。

  • 階層的手法：まず全員をバラバラ（個別）として、近い者から順番にくっつけていく。デンドログラム（樹状図）で可視化できる。
  • 非階層的手法：最初に「グループ数K」を決めてから、各データを適切なグループに割り振る（K-means法が代表例）。

どちらの手法の結果も、データ間の「距離」の定義に大きく左右されます。まず距離の話から始めましょう。

① 距離尺度――「似ている」を数値化する

データのベクトル表現

各データ（人・物・観測値）は複数の特徴量をまとめた $p$ 次元ベクトル で表します。 例えば今回の TripAdvisor データでは「レストラン評価」「劇場評価」「ビーチ評価」の3次元ベクトルです。

ユーザ1のベクトル $\mathbf{x}_1 = (7.7,\ 5.8,\ 7.7)$、ユーザ2のベクトル $\mathbf{x}_2 = (10.1,\ 5.9,\ 7.3)$

1-1. ミンコフスキー距離

$p$ 次元のベクトル $\mathbf{x} = (x_1, x_2, \ldots, x_p)$ と $\mathbf{y} = (y_1, y_2, \ldots, y_p)$ の間のミンコフスキー距離は次の式で定義されます。

$$d_m(\mathbf{x}, \mathbf{y}) = \left(\sum_{j=1}^{p} |x_j - y_j|^m\right)^{1/m}$$

式を言葉で分解すると：

  1. 各次元 $j$ で差の絶対値 $|x_j - y_j|$ を計算する
  2. その $m$ 乗を全次元で足し合わせる
  3. 最後に $1/m$ 乗（= $m$ 乗根）をとる

パラメータ $m$ の値によって、具体的な距離が変わります。

1-2. ユークリッド距離（$m=2$ の場合）

$m=2$ のとき、ミンコフスキー距離は私たちが日常的に使う「まっすぐな距離」＝ユークリッド距離になります。

$$d_2(\mathbf{x}, \mathbf{y}) = \sqrt{\sum_{j=1}^{p} (x_j - y_j)^2}$$

💡 具体例：ユーザ1 $\mathbf{x}_1=(7.7, 5.8, 7.7)$ とユーザ2 $\mathbf{x}_2=(10.1, 5.9, 7.3)$ の距離

各次元の差：

  • レストラン：$10.1 - 7.7 = 2.4$
  • 劇場：$5.9 - 5.8 = 0.1$
  • ビーチ：$7.3 - 7.7 = -0.4$

差の二乗和：

$$2.4^2 + 0.1^2 + (-0.4)^2 = 5.76 + 0.01 + 0.16 = 5.93$$

平方根をとる：

$$d_2(\mathbf{x}_1, \mathbf{x}_2) = \sqrt{5.93} \approx 2.4$$

✅ これが距離行列の $(1,2)$ 成分 $2.4$ と一致します。

1-3. マンハッタン距離（$m=1$ の場合）

$m=1$ のとき、各次元の差の絶対値を単純に足し合わせた距離になります。 碁盤の目の街（マンハッタン）を歩く距離のイメージです。

$$d_1(\mathbf{x}, \mathbf{y}) = \sum_{j=1}^{p} |x_j - y_j|$$

💡 具体例：同じユーザ1とユーザ2で計算すると：

$$d_1(\mathbf{x}_1, \mathbf{x}_2) = |2.4| + |0.1| + |-0.4| = 2.4 + 0.1 + 0.4 = 2.9$$

ユークリッド距離は斜め移動、マンハッタン距離は縦横移動のイメージです。

1-4. マハラノビス距離

ユークリッド距離は「各特徴量が同じスケールで独立」と仮定しています。 しかし実際のデータでは特徴量間に相関があったり、スケールが大きく異なることがあります。 マハラノビス距離は、共分散行列 $\Sigma$ を使ってそれを補正した距離です。

$$d_M(\mathbf{x}, \mathbf{y}) = \sqrt{(\mathbf{x} - \mathbf{y})^T \Sigma^{-1} (\mathbf{x} - \mathbf{y})}$$

式の意味：

  • $(\mathbf{x} - \mathbf{y})$：差のベクトル（縦ベクトル）
  • $(\mathbf{x} - \mathbf{y})^T$：差のベクトルの転置（横ベクトル）
  • $\Sigma^{-1}$：共分散行列の逆行列。特徴量間の相関・スケール差を吸収する役割を果たす
  • $\Sigma = I$（単位行列）のとき、マハラノビス距離はユークリッド距離に一致する

💡 直感的なイメージ：身長（cm単位）と体重（kg単位）を使う場合、身長の差 10cm と体重の差 10kg は「同じ 10」でも意味が違います。共分散行列でスケールを揃えることで、フェアな距離計算ができます。

共分散行列 $\Sigma$ は、$n$ 個のサンプル $\mathbf{x}_1, \ldots, \mathbf{x}_n$ から次のように推定します（経験（標本）分散共分散行列）：

$$\hat{\Sigma} = \frac{1}{n} \sum_{i=1}^{n} (\mathbf{x}_i - \bar{\mathbf{x}})(\mathbf{x}_i - \bar{\mathbf{x}})^T, \quad \bar{\mathbf{x}} = \frac{1}{n}\sum_{i=1}^{n}\mathbf{x}_i$$

② 階層的クラスタリング

アルゴリズムの流れ（アグロメラティブ法）

階層的クラスタリングの代表的な手順（凝集型：下から上へ積み上げていく方法）は以下のとおりです。

  1. 初期状態：$n$ 個のサンプルを、それぞれ1つのクラスターとして扱う（$n$ 個のクラスター）
  2. 最近傍クラスターを探す：すべてのクラスターのペアの中で、距離が最小のペアを見つける
  3. クラスターを併合：選んだペアを1つのクラスターにまとめる（クラスター数が1つ減る）
  4. 繰り返す：クラスターが1つになるまで 2〜3 を繰り返す

この過程を樹状図（デンドログラム）で描くと、どのクラスターがどの高さ（距離）で結合されたかが一目で分かります。

クラスター間距離の測り方（リンケージ）

個々のサンプル間の距離は定義できても、クラスター同士の距離はどう測るかで複数の方法があります。 以下では $C_1, C_2$ という 2 つのクラスターの間の距離 $d(C_1, C_2)$ を説明します。

① 最近隣法（単リンク法 / 最短距離法）

$$d(C_1, C_2) = \min_{\mathbf{x} \in C_1,\, \mathbf{y} \in C_2} d(\mathbf{x}, \mathbf{y})$$

2つのクラスターから1点ずつ選ぶすべての組合せで距離を計算し、その中で最も近い（小さい）距離をクラスター間距離とします。

💡 直感：2つのグループで「一番仲良しのペアの距離」でグループ間の近さを測るイメージ。
特性：鎖状の細長いクラスターができやすい（チェーン効果）。

② 最遠隣法（最長距離法）

$$d(C_1, C_2) = \max_{\mathbf{x} \in C_1,\, \mathbf{y} \in C_2} d(\mathbf{x}, \mathbf{y})$$

最も遠い（大きい）距離をクラスター間距離とします。

💡 直感：「2つのグループで最も疎遠なペアの距離」でグループ間の距離を測る。コンパクトで球状のクラスターが得られやすい。

③ 重心法（セントロイド法）

$$d_{\text{centroid}}(C_1, C_2) = d(\bar{\mathbf{x}}_1, \bar{\mathbf{x}}_2), \quad \bar{\mathbf{x}}_k = \frac{1}{|C_k|}\sum_{\mathbf{x} \in C_k} \mathbf{x}$$

各クラスターの重心（平均ベクトル）を求め、重心間の距離をクラスター間距離とします。

💡 直感：クラスターを「重心=代表点」1点で表し、代表点間の距離を使う。

④ 群平均法（グループ平均法）

$$d(C_1, C_2) = \frac{1}{|C_1|\,|C_2|} \sum_{\mathbf{x} \in C_1} \sum_{\mathbf{y} \in C_2} d(\mathbf{x}, \mathbf{y})$$

すべてのペア組合せの距離の平均をクラスター間距離とします。

💡 直感：「全員同士の距離を総当たりして平均をとる」公平な方法。最近隣法と最遠隣法の中間的な性質がある。

⑤ ウォード法

少し複雑ですが、考え方は「合併するとクラスター内のばらつきがどれだけ増えるか」を最小にする手法です。

クラスター $C$ 内のばらつき（クラスター内二乗和）を次のように定義します。

$$W(C) = \sum_{\mathbf{x} \in C} \|\mathbf{x} - \bar{\mathbf{x}}\|^2$$

ここで $\bar{\mathbf{x}} = \frac{1}{|C|}\sum_{\mathbf{x} \in C}\mathbf{x}$ はクラスター重心です。

$C_1$ と $C_2$ を合併したとき増えるばらつきを「クラスター間距離」として定義します。

$$d_{\text{Ward}}(C_1, C_2) = W(C_1 \cup C_2) - W(C_1) - W(C_2)$$

展開すると以下の公式と同値です。

$$d_{\text{Ward}}(C_1, C_2) = \sum_{\mathbf{z} \in C_1 \cup C_2}\|\mathbf{z} - \bar{\mathbf{z}}\|^2 - \sum_{\mathbf{x} \in C_1}\|\mathbf{x} - \bar{\mathbf{x}}_1\|^2 - \sum_{\mathbf{y} \in C_2}\|\mathbf{y} - \bar{\mathbf{y}}_2\|^2$$

ここで $\bar{\mathbf{z}}$ は $C_1 \cup C_2$ 全体の重心です。

💡 重心の加重平均：合併後の重心は次の式で求まります（逐一全データを足さなくてもよい）。

$$\bar{\mathbf{z}} = \frac{|C_1|\,\bar{\mathbf{x}}_1 + |C_2|\,\bar{\mathbf{x}}_2}{|C_1| + |C_2|}$$

💡 直感：「合併してもばらつきがあまり増えないペアを選ぶ」→ コンパクトで自然なクラスターが得られやすい。実務でよく使われる手法。

③ K-means法（非階層的クラスタリング）

考え方

K-means 法は「K 個のクラスターに分けたい」と最初から決める方法です。 目標は、各クラスターの中心からサンプルへの距離の二乗和（目的関数 $J$）を最小にするクラスター割当を見つけることです。

目的関数

$$J(\{C_k\}) = \sum_{k=1}^{K} \sum_{\mathbf{x}_i \in C_k} \|\mathbf{x}_i - \bar{\boldsymbol{\mu}}_k\|^2$$

式を言葉で解読すると：

  • $\bar{\boldsymbol{\mu}}_k$：クラスター $k$ に属するサンプルの平均ベクトル（重心）
  • $\|\mathbf{x}_i - \bar{\boldsymbol{\mu}}_k\|^2$：サンプル $\mathbf{x}_i$ と重心 $\bar{\boldsymbol{\mu}}_k$ の距離の二乗
  • $\sum_{\mathbf{x}_i \in C_k}$：クラスター $k$ 内のすべてのサンプルについて足す
  • $\sum_{k=1}^{K}$：全 $K$ クラスターについて足す

つまり $J$ は「クラスター内のばらつき（散らばり）の合計」であり、これが小さいほど密なクラスターが得られます。

アルゴリズムの手順

この最適化問題を厳密に解くのは計算量が大きいため、以下の 2 ステップを繰り返す反復アルゴリズムを使います。

  1. クラスター中心の更新：現在の割当のもとで、各クラスターの重心（平均）を計算する。 $$\bar{\boldsymbol{\mu}}_k \leftarrow \frac{1}{|C_k|} \sum_{\mathbf{x} \in C_k} \mathbf{x}$$
  2. クラスター割当の更新：各サンプル $\mathbf{x}_i$ を、距離が最も近い中心のクラスターに再割当する。 $$\text{クラスター}(\mathbf{x}_i) \leftarrow \underset{k \in \{1,\ldots,K\}}{\arg\min}\ \|\mathbf{x}_i - \bar{\boldsymbol{\mu}}_k\|^2$$

割当が変化しなくなったら（収束したら）終了です。目的関数 $J$ はステップごとに単調減少することが保証されています。

⚠️ 初期値依存性：K-means は最初のクラスター中心の選び方によって解が異なる局所最適解に収束します。対策として：

  • 異なる初期値で複数回実行し、$J$ が最小の結果を採用する
  • K-means++：最初の重心を1つランダムに選び、以降「既存の重心から遠いほど選ばれやすい」ように確率的に初期重心を選ぶ方法

④ 混合分布と EM アルゴリズムによるクラスタリング

データが「$K$ 種類の確率分布の混合から生成された」とモデル化する方法です。 たとえば「2つのグループがそれぞれガウス分布に従う」と仮定するのが混合ガウス分布（GMM）です。

混合分布の式

$$f(\mathbf{x};\{\pi_k, \theta_k\}_{k=1}^{K}) = \sum_{k=1}^{K} \pi_k f_k(\mathbf{x}; \theta_k)$$

各記号の意味：

  • $f_k(\mathbf{x}; \theta_k)$：$k$ 番目のクラスターの確率密度関数（パラメータ $\theta_k$ をもつ）
  • $\pi_k$：$k$ 番目のクラスターの混合比（重み）。$\pi_k \ge 0$ かつ $\sum_{k=1}^{K}\pi_k = 1$

あるサンプル $\mathbf{x}$ がクラスター $k$ に属する確率は $\pi_k f_k(\mathbf{x};\theta_k)$ に比例します。 パラメータ $\pi_k, \theta_k$ の推定にはEM アルゴリズム（29章参照）が用いられます。

💡 K-means との違い：K-means では各サンプルを「どれか1つのクラスターに必ず割当」（ハード割当）しますが、混合分布では「クラスター $k$ に属する確率が $p_{ik}$」というソフト割当が可能です。

例題・練習問題

以下の問題はすべて自作問題です。解答は各問の下に折りたたまずに記載しています。まず自分で解いてから確認してください。

問 1　距離の計算（ウォームアップ）

2次元平面上に3点 $P_1=(1,3)$、$P_2=(4,7)$、$P_3=(1,7)$ がある。

  1. $P_1$ と $P_2$ のユークリッド距離 $d_2(P_1, P_2)$ を求めよ。
  2. $P_1$ と $P_2$ のマンハッタン距離 $d_1(P_1, P_2)$ を求めよ。
  3. 3点 $P_1, P_2, P_3$ の間のユークリッド距離をすべて求め、この3点が作る三角形の形（直角三角形・鋭角三角形・鈍角三角形）を判定せよ。

問 1　解答

（1）ユークリッド距離 $d_2(P_1, P_2)$

定義式に代入する。

$$d_2(P_1, P_2) = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}$$

各座標の差を計算する：

  • $x$ 方向：$4 - 1 = 3$
  • $y$ 方向：$7 - 3 = 4$

二乗して足し合わせ、平方根をとる：

$$d_2(P_1, P_2) = \sqrt{3^2 + 4^2} = \sqrt{9 + 16} = \sqrt{25} = \boldsymbol{5}$$

（2）マンハッタン距離 $d_1(P_1, P_2)$

差の絶対値を単純に足し合わせる：

$$d_1(P_1, P_2) = |4-1| + |7-3| = 3 + 4 = \boldsymbol{7}$$

ユークリッド距離（5）よりマンハッタン距離（7）のほうが大きい。これは「斜め移動」が許可される（ユークリッド）か「縦横移動のみ」（マンハッタン）かの違いによる。

（3）3辺すべてのユークリッド距離と三角形の形

$d_2(P_1, P_2)$ は（1）で求めた。残り2辺を計算する。

$d_2(P_1, P_3)$：

$$d_2(P_1, P_3) = \sqrt{(1-1)^2 + (7-3)^2} = \sqrt{0 + 16} = 4$$

$d_2(P_2, P_3)$：

$$d_2(P_2, P_3) = \sqrt{(4-1)^2 + (7-7)^2} = \sqrt{9 + 0} = 3$$

3辺の長さ：$d_2(P_1,P_2)=5$、$d_2(P_1,P_3)=4$、$d_2(P_2,P_3)=3$

ピタゴラスの定理 $a^2 + b^2 = c^2$ が成立するか確認する：

$$3^2 + 4^2 = 9 + 16 = 25 = 5^2 \quad \checkmark$$

✅ $3$ – $4$ – $5$ の辺の比をもつ 直角三角形（$P_3$ が直角）である。

💡 $P_3 = (1,7)$ は $P_1$ の真上、$P_2$ の真左に位置しているため、$P_3$ での角が直角になる。

問 2　階層的クラスタリング（最近隣法 vs 最遠隣法）

5つのサンプル A, B, C, D, E に対して、ユークリッド距離に基づくサンプル間距離行列が次のように与えられているとする。

A B C D E
A028610
B20648
C86035
D64305
E108550
  1. 最近隣法（単リンク法）を用いて、この5サンプルを階層的にクラスタリングする過程をすべて示せ。各ステップで距離表を更新しながら、どのクラスターが結合されるかを明示すること。
  2. 最遠隣法を用いて、同じ5サンプルをクラスタリングする過程を示せ。
  3. （1）と（2）のデンドログラムの形（特に後半の結合順序）がどのように異なるか、理由とともに説明せよ。

問 2　解答

（1）最近隣法（単リンク法）

非対角成分の最小値を順番に探して結合していく。

◆ イテレーション 1：最初の結合

距離行列の全非対角成分のうち最小値を探すと：

$$d(A,B) = 2 \quad \leftarrow \text{最小}$$

→ A と B を結合して、新クラスター $\{A,B\}$ を作る。

$\{A,B\}$ と残りのサンプル（C, D, E）との最近隣距離を計算する。

最近隣法では $d(\{A,B\}, X) = \min\{d(A,X),\ d(B,X)\}$。

  • $d(\{A,B\},C) = \min\{d(A,C), d(B,C)\} = \min\{8, 6\} = 6$
  • $d(\{A,B\},D) = \min\{d(A,D), d(B,D)\} = \min\{6, 4\} = 4$
  • $d(\{A,B\},E) = \min\{d(A,E), d(B,E)\} = \min\{10, 8\} = 8$

更新後の距離行列（3クラスター）：

{A,B} C D E
{A,B}—64 ★8
C6—3 ★5
D43—5
E855—

◆ イテレーション 2：次の結合

最小値は $d(C,D) = 3$。

→ C と D を結合して $\{C,D\}$ を作る。

$\{C,D\}$ と残りとの最近隣距離：

  • $d(\{C,D\},\{A,B\}) = \min\{d(C,\{A,B\}), d(D,\{A,B\})\} = \min\{6, 4\} = 4$
  • $d(\{C,D\},E) = \min\{d(C,E), d(D,E)\} = \min\{5, 5\} = 5$

更新後の距離行列（3クラスター）：

{A,B} {C,D} E
{A,B}—4 ★8
{C,D}4—5
E85—

◆ イテレーション 3：次の結合

最小値は $d(\{A,B\},\{C,D\}) = 4$。

→ $\{A,B\}$ と $\{C,D\}$ を結合して $\{A,B,C,D\}$ を作る。

$\{A,B,C,D\}$ と E の最近隣距離：

$$d(\{A,B,C,D\}, E) = \min\{d(A,E), d(B,E), d(C,E), d(D,E)\} = \min\{10, 8, 5, 5\} = 5$$

◆ イテレーション 4：最終結合

→ $\{A,B,C,D\}$ と $E$ を結合して全体の1クラスター $\{A,B,C,D,E\}$ になる。

✅ 最近隣法の結合順序：

  1. $\{A,B\}$（距離2）
  2. $\{C,D\}$（距離3）
  3. $\{A,B\} + \{C,D\} \to \{A,B,C,D\}$（距離4）
  4. $\{A,B,C,D\} + E \to$ 全体（距離5）

（2）最遠隣法（最長距離法）

最遠隣法では $d(\{A,B\}, X) = \max\{d(A,X),\ d(B,X)\}$。

◆ イテレーション 1：最初の結合は最近隣法と同じ。非対角最小値 $d(A,B)=2$ で $\{A,B\}$ を作る。

最遠隣距離の更新：

  • $d(\{A,B\},C) = \max\{8, 6\} = 8$
  • $d(\{A,B\},D) = \max\{6, 4\} = 6$
  • $d(\{A,B\},E) = \max\{10, 8\} = 10$

更新後（最遠隣）の距離行列：

{A,B} C D E
{A,B}—8610
C8—3 ★5
D63—5
E1055—

◆ イテレーション 2：最小値は $d(C,D)=3$（個体間距離そのものなので最近・最遠で同値）。$\{C,D\}$ を作る。

最遠隣距離の更新：

  • $d(\{C,D\},\{A,B\}) = \max\{d(C,\{A,B\}), d(D,\{A,B\})\} = \max\{8, 6\} = 8$
  • $d(\{C,D\},E) = \max\{d(C,E), d(D,E)\} = \max\{5, 5\} = 5$

更新後（最遠隣）の距離行列：

{A,B} {C,D} E
{A,B}—810
{C,D}8—5 ★
E105—

◆ イテレーション 3：最小値は $d(\{C,D\},E)=5$。

→ $\{C,D\}$ と $E$ を結合して $\{C,D,E\}$ を作る。

$\{C,D,E\}$ と $\{A,B\}$ の最遠隣距離：

$$d(\{C,D,E\},\{A,B\}) = \max\{d(C,A), d(C,B), d(D,A), d(D,B), d(E,A), d(E,B)\}$$ $$= \max\{8, 6, 6, 4, 10, 8\} = 10$$

◆ イテレーション 4：→ $\{A,B\}$ と $\{C,D,E\}$ を結合して全体 $\{A,B,C,D,E\}$ になる。

✅ 最遠隣法の結合順序：

  1. $\{A,B\}$（距離2）
  2. $\{C,D\}$（距離3）
  3. $\{C,D\} + E \to \{C,D,E\}$（距離5）
  4. $\{A,B\} + \{C,D,E\} \to$ 全体（距離10）

（3）デンドログラムの違いと理由

最近隣法：イテレーション3で $\{A,B\}$ と $\{C,D\}$ が「距離4」で先に結合し、その後 $E$ が「距離5」で加わった。

最遠隣法：イテレーション3で $\{C,D\}$ と $E$ が「距離5」で先に結合し、最後に $\{A,B\}$ が「距離10」で加わった。

違いの理由：$\{A,B\}$ と $E$ の間には $d(A,E)=10$ という大きな距離がある。
最近隣法は「最も近いペアさえ近ければ結合する」ため、$\{C,D\}$ の中に $B$ に近い $D$（$d(B,D)=4$）がいることで $\{A,B\}$ と $\{C,D\}$ が先に引き寄せられる。
最遠隣法は「最も遠いペアの距離」で評価するため、$A$ と $E$ の遠さ（10）が足を引っ張り $\{A,B\}$-$\{C,D,E\}$ の結合が最後になる。

問 3　K-means 法（$K=2$）

2次元空間上の4つのデータ点 $\mathbf{x}_1 = (1,1)$、$\mathbf{x}_2 = (1,3)$、$\mathbf{x}_3 = (5,4)$、$\mathbf{x}_4 = (7,4)$ に対して、$K=2$ の K-means 法を適用する。

初期クラスター中心を $\boldsymbol{\mu}_1^{(0)} = (0, 2)$、$\boldsymbol{\mu}_2^{(0)} = (6, 5)$ とする。

  1. 各データ点と 2 つの初期中心とのユークリッド距離をすべて計算し、各データを $C_1$ または $C_2$ に割当よ。
  2. 割当結果から新しいクラスター中心 $\bar{\boldsymbol{\mu}}_1$、$\bar{\boldsymbol{\mu}}_2$ を計算せよ。
  3. 新しい中心をもとに再度クラスター割当を行い、割当が変化するかどうかを確認せよ。変化しない場合は「収束した」と判断できる。

問 3　解答

（1）初期中心との距離計算とクラスター割当

ユークリッド距離の定義 $d(\boldsymbol{\mu}, \mathbf{x}) = \sqrt{(\mu_1 - x_1)^2 + (\mu_2 - x_2)^2}$ を使う。

$\mathbf{x}_1 = (1,1)$

$$d(\boldsymbol{\mu}_1^{(0)}, \mathbf{x}_1) = \sqrt{(0-1)^2 + (2-1)^2} = \sqrt{1 + 1} = \sqrt{2} \approx 1.41$$ $$d(\boldsymbol{\mu}_2^{(0)}, \mathbf{x}_1) = \sqrt{(6-1)^2 + (5-1)^2} = \sqrt{25 + 16} = \sqrt{41} \approx 6.40$$

$1.41 < 6.40$ なので $\mathbf{x}_1 \to C_1$

$\mathbf{x}_2 = (1,3)$

$$d(\boldsymbol{\mu}_1^{(0)}, \mathbf{x}_2) = \sqrt{(0-1)^2 + (2-3)^2} = \sqrt{1 + 1} = \sqrt{2} \approx 1.41$$ $$d(\boldsymbol{\mu}_2^{(0)}, \mathbf{x}_2) = \sqrt{(6-1)^2 + (5-3)^2} = \sqrt{25 + 4} = \sqrt{29} \approx 5.39$$

$1.41 < 5.39$ なので $\mathbf{x}_2 \to C_1$

$\mathbf{x}_3 = (5,4)$

$$d(\boldsymbol{\mu}_1^{(0)}, \mathbf{x}_3) = \sqrt{(0-5)^2 + (2-4)^2} = \sqrt{25 + 4} = \sqrt{29} \approx 5.39$$ $$d(\boldsymbol{\mu}_2^{(0)}, \mathbf{x}_3) = \sqrt{(6-5)^2 + (5-4)^2} = \sqrt{1 + 1} = \sqrt{2} \approx 1.41$$

$1.41 < 5.39$ なので $\mathbf{x}_3 \to C_2$

$\mathbf{x}_4 = (7,4)$

$$d(\boldsymbol{\mu}_1^{(0)}, \mathbf{x}_4) = \sqrt{(0-7)^2 + (2-4)^2} = \sqrt{49 + 4} = \sqrt{53} \approx 7.28$$ $$d(\boldsymbol{\mu}_2^{(0)}, \mathbf{x}_4) = \sqrt{(6-7)^2 + (5-4)^2} = \sqrt{1 + 1} = \sqrt{2} \approx 1.41$$

$1.41 < 7.28$ なので $\mathbf{x}_4 \to C_2$

✅ 割当結果（1回目）：

  • $C_1 = \{\mathbf{x}_1, \mathbf{x}_2\} = \{(1,1),\ (1,3)\}$
  • $C_2 = \{\mathbf{x}_3, \mathbf{x}_4\} = \{(5,4),\ (7,4)\}$

（2）クラスター中心の更新

各クラスターに属するデータ点の座標を次元ごとに平均する。

$C_1 = \{\mathbf{x}_1, \mathbf{x}_2\}$ の新しい重心 $\bar{\boldsymbol{\mu}}_1$：

$$\bar{\boldsymbol{\mu}}_1 = \frac{1}{2}(\mathbf{x}_1 + \mathbf{x}_2) = \frac{1}{2}\big((1,1) + (1,3)\big) = \frac{1}{2}(2, 4) = (1, 2)$$

$C_2 = \{\mathbf{x}_3, \mathbf{x}_4\}$ の新しい重心 $\bar{\boldsymbol{\mu}}_2$：

$$\bar{\boldsymbol{\mu}}_2 = \frac{1}{2}(\mathbf{x}_3 + \mathbf{x}_4) = \frac{1}{2}\big((5,4) + (7,4)\big) = \frac{1}{2}(12, 8) = (6, 4)$$

✅ 更新後のクラスター中心：

  • $\bar{\boldsymbol{\mu}}_1 = (1, 2)$
  • $\bar{\boldsymbol{\mu}}_2 = (6, 4)$

（3）再割当と収束確認

新しい中心 $\bar{\boldsymbol{\mu}}_1=(1,2)$、$\bar{\boldsymbol{\mu}}_2=(6,4)$ に対して、各データ点との距離を計算する。

$\mathbf{x}_1 = (1,1)$：

$$d(\bar{\boldsymbol{\mu}}_1, \mathbf{x}_1) = \sqrt{(1-1)^2+(2-1)^2} = \sqrt{0+1} = 1$$ $$d(\bar{\boldsymbol{\mu}}_2, \mathbf{x}_1) = \sqrt{(6-1)^2+(4-1)^2} = \sqrt{25+9} = \sqrt{34} \approx 5.83$$

$1 < 5.83$ → $C_1$（変化なし）

$\mathbf{x}_2 = (1,3)$：

$$d(\bar{\boldsymbol{\mu}}_1, \mathbf{x}_2) = \sqrt{(1-1)^2+(2-3)^2} = \sqrt{0+1} = 1$$ $$d(\bar{\boldsymbol{\mu}}_2, \mathbf{x}_2) = \sqrt{(6-1)^2+(4-3)^2} = \sqrt{25+1} = \sqrt{26} \approx 5.10$$

$1 < 5.10$ → $C_1$（変化なし）

$\mathbf{x}_3 = (5,4)$：

$$d(\bar{\boldsymbol{\mu}}_1, \mathbf{x}_3) = \sqrt{(1-5)^2+(2-4)^2} = \sqrt{16+4} = \sqrt{20} \approx 4.47$$ $$d(\bar{\boldsymbol{\mu}}_2, \mathbf{x}_3) = \sqrt{(6-5)^2+(4-4)^2} = \sqrt{1+0} = 1$$

$1 < 4.47$ → $C_2$（変化なし）

$\mathbf{x}_4 = (7,4)$：

$$d(\bar{\boldsymbol{\mu}}_1, \mathbf{x}_4) = \sqrt{(1-7)^2+(2-4)^2} = \sqrt{36+4} = \sqrt{40} \approx 6.32$$ $$d(\bar{\boldsymbol{\mu}}_2, \mathbf{x}_4) = \sqrt{(6-7)^2+(4-4)^2} = \sqrt{1+0} = 1$$

$1 < 6.32$ → $C_2$（変化なし）

✅ すべてのデータ点の割当が変化しなかったため、アルゴリズムは収束した。

最終クラスター中心：

  • $\bar{\boldsymbol{\mu}}_1 = (1, 2)$：左下グループ（$\mathbf{x}_1, \mathbf{x}_2$）
  • $\bar{\boldsymbol{\mu}}_2 = (6, 4)$：右上グループ（$\mathbf{x}_3, \mathbf{x}_4$）

💡 初期中心 $(0,2)$ と $(6,5)$ は最終重心 $(1,2)$ と $(6,4)$ に非常に近かったため、1回の反復で収束した。

まとめと注意点

手法 計算量 特徴
階層的クラスタリング$O(n^2)$デンドログラムで可視化可。Kを事前決定不要
K-means法$O(nK)$大規模データに適用可。K を事前決定が必要
混合分布（GMM）—ソフト割当・確率的解釈が可能。EM法で推定
  • クラスター分析の結果は距離尺度（ユークリッド, マハラノビス, etc.）とリンケージ手法（最近隣, 最遠隣, ウォード, etc.）の両方に大きく依存する。
  • K-means は初期値依存性があるため、複数回実行して目的関数 $J$ 最小の結果を採用すること。
  • サンプルサイズが大きい（$n$ が大きい）場合、$O(n^2)$ の階層的手法は計算が重くなるので K-means が有利。
  • ウォード法は球状のコンパクトなクラスターを見つけやすく、実務では特によく使われる。
  • 混合分布によるクラスタリングの詳細は 29章（EMアルゴリズム）で扱う。
