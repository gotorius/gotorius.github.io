ノンパラメトリック法――分布を仮定しない検定

これまでの仮説検定（第10〜12章）では、母集団が正規分布やポアソン分布などの 特定の分布に従うと仮定した上で検定統計量を作りました。 しかし現実のデータでは、「その分布に本当に従っているか分からない」「サンプルが少なすぎて確認できない」 という場面が多くあります。

そこで登場するのがノンパラメトリック法（non-parametric method）です。 データの順位（大きさの順番）を使うことで、分布の形を仮定せずに検定を行えます。

1. パラメトリック法 vs ノンパラメトリック法

パラメトリック法とは

「母集団が $N(\mu, \sigma^2)$ に従う」「ポアソン分布に従う」などの分布の型を仮定し、 そのパラメータ（$\mu, \sigma^2$ など）を使って検定する方法です。 正規分布を仮定すれば、第11章の $t$ 検定が使えます。

ノンパラメトリック法とは

母集団分布の型を仮定しない検定です。 基本的なアイデアは「数値の大小関係（順位）だけに注目する」こと。 34点が68点より小さい、という順番の情報だけ使うので、分布の形に左右されません。

いつノンパラを使うか？

  • サンプルサイズが小さく、正規分布かどうか確認しにくいとき
  • データが順序尺度（「大・中・小」など）で、数値の大きさに意味がないとき
  • 外れ値が大きく、平均が信頼できないとき（順位は外れ値に強い）

デメリット

ノンパラメトリック法は数値の大きさではなく順位を使うため、 正規分布の仮定が正しいときは検出力がパラメトリック法より低い（帰無仮説が誤りでも棄却しにくい）ことがあります。

2. ウィルコクソンの順位和検定

場面の設定

群A（$m$ 人）と群B（$n$ 人）について「2群の分布は同じか」を検定します。 例えば「2種類の薬の効果に差があるか」などです。

手順

  1. A群とB群の全データを合わせて小さい順に順位を付ける（1位、2位、…、$m+n$ 位）
  2. A群に割り当てられた順位の合計 $W_A$ を検定統計量（順位和）とする
  3. 帰無仮説が正しければ「順位は A・B にランダムに割り振られる」と考えられる
  4. ${}_{m+n}C_m$ 通りの組合せから $W_A \leq w_A$ となる確率（片側P値）を求め、有意水準と比較する

タイ（同じ値）の処理

5位と6位に同じ値がある場合は、それぞれに $\dfrac{5+6}{2} = 5.5$ を割り当てます（順位の平均を分配）。

大標本近似（正規分布近似）

各群のサンプルサイズ $m, n$ が大きくタイがない場合、帰無仮説のもとで $W_A$ は近似的に正規分布に従います：

$$E[W_A] = \frac{m(m+n+1)}{2}, \quad V[W_A] = \frac{mn(m+n+1)}{12}$$ $$Z = \frac{W_A - E[W_A]}{\sqrt{V[W_A]}} \xrightarrow{d} N(0,1)$$

直感的な理解

帰無仮説が正しく「2群の分布が同じ」なら、$m+n$ 個の順位は A と B に均等にランダム割り振りされます。 したがって A 群の順位和の期待値は全順位の和 $\dfrac{(m+n)(m+n+1)}{2}$ の $\dfrac{m}{m+n}$ 倍 $= \dfrac{m(m+n+1)}{2}$ になります。

3. 並べ替え検定（Permutation Test）

順位和検定が「順位」を使うのに対し、並べ替え検定は元の数値を使います。 帰無仮説のもと「$m+n$ 人からA群として $m$ 人が無作為に選ばれた」という発想で P 値を計算します。

手順

  1. 実際の A 群の平均 $\bar{X}_A$ を計算する
  2. $m+n$ 人の全データから A 群として $m$ 人を選ぶすべての組合せ（${}_{m+n}C_m$ 通り）を列挙し、それぞれの平均を計算する
  3. 「$\bar{X}_A$ 以下になる確率」 = 片側 P 値

順位和検定と並べ替え検定の違い

どちらが優れているというわけではなく、検定統計量の選び方が異なるだけです。 同じデータでも P 値が異なることがあります。 並べ替え検定は数値の大きさも使うため、外れ値に弱い側面もあります。

4. 符号付き順位検定（Wilcoxon Signed-Rank Test）

場面の設定

「補習前後の点数差」「薬を変えた前後の血圧差」など、対応のある2つの測定値の差 $D_i$ を扱います。 「$D$ の中央値が 0 か」（効果がないか）を検定します。

⚠ 前提条件：差 $D$ の分布が0 を中心に対称であること

手順

  1. $D_i = 0$ のデータを除き、残りの $n$ 個を使う（$n$ を更新）
  2. $|D_i|$（絶対値）の小さい順に順位を付ける。タイは平均値を分配する
  3. 各順位に $D_i$ の符号を付けて符号付き順位を作る（例：$D_i > 0$ なら $+k$、$D_i < 0$ なら $-k$）
  4. 正値の合計 $T_+ = \sum_{\{i: D_i > 0\}} \text{(順位)}$ を検定統計量とする
  5. 帰無仮説のもと「$+/-$ の符号は $n$ 個の順位にランダムに割り振られる」$\Rightarrow$ $2^n$ 通り全列挙で P 値を求める

大標本近似

$$E[T_+] = \frac{n(n+1)}{4}, \quad V[T_+] = \frac{n(n+1)(2n+1)}{24}$$ $$Z = \frac{T_+ - E[T_+]}{\sqrt{V[T_+]}} \xrightarrow{d} N(0,1)$$

直感的理解

「差がゼロ（効果なし）」なら、符号 $+/-$ は $1/2$ ずつランダムに付くはずです。 $T_+$ の期待値は $1 + 2 + \cdots + n = n(n+1)/2$ の半分 $= n(n+1)/4$ になります。

5. 符号検定（Sign Test）

符号付き順位検定よりシンプルで前提条件が少ない検定です。 差 $D_i$ の符号（正か負か）だけを使い、順位は無視します。

手順と分布

$D_i = 0$ のデータを除いた $n$ 個について、$D_i > 0$ の個数を $T_+$ とします。

帰無仮説（$D$ の中央値 $= 0$）のもとでは、$D_i > 0$ と $D_i < 0$ の確率がそれぞれ $1/2$：

$$T_+ \sim \mathrm{Bin}(n, 0.5)$$

符号付き順位検定と符号検定の比較

比較項目符号付き順位検定符号検定
使う情報差の符号＋絶対値の大きさ差の符号のみ
前提条件$D$ の分布が対称なし（より一般的）
検出力高い（情報をより多く使う）低い（情報が少ない）

6. クラスカル・ウォリス検定（Kruskal-Wallis Test）

3群以上の分布に差があるかを検定する（一元配置分散分析のノンパラ版）。

  • 帰無仮説 $H_0$：すべての群の分布は同じ
  • 対立仮説 $H_1$：少なくとも1群の分布は他と異なる
  • 前提条件：各群の分布の形は同じ（中央値の位置だけが異なる可能性がある）

検定統計量 $H$

$k$ 群のサンプルサイズを $n_1, n_2, \ldots, n_k$、合計 $N = \sum n_g$、 各群の順位和を $R_g$、順位の平均を $\bar{R}_g = R_g / n_g$、 全順位の中央値を $\bar{N} = (N+1)/2$ とするとき：

$$H = \frac{12}{N(N+1)} \sum_{g=1}^k n_g \left(\bar{R}_g - \bar{N}\right)^2$$

これは次のように書き直せる（計算に便利）：

$$H = \frac{12}{N(N+1)} \sum_{g=1}^k \frac{R_g^2}{n_g} - 3(N+1)$$

各群のサンプルサイズが大きいとき、帰無仮説のもとで $H$ は自由度 $k-1$ のカイ二乗分布に近似的に従います：

$$H \xrightarrow{d} \chi^2(k-1)$$

直感的理解

「群間に差がない」なら各群の順位の平均は全体の中央値に近いはずです。 $H$ は各群の順位平均と全体中央値の「ずれの大きさ」を表しているので、 $H$ が大きいほど群間に差があると判断されます。

7. 順位相関係数

$n$ 組のデータ $(x_i, y_i)$（$i=1,\ldots,n$）について、 数値そのものではなく順位を使った相関係数です。

スピアマンの順位相関係数 $r_s$

$x_i, y_i$ それぞれに順位を付けた場合のピアソン相関係数です。 タイがない場合は簡単な公式になります：

$$r_s = 1 - \frac{6\displaystyle\sum_{i=1}^n d_i^2}{n(n^2-1)}, \quad d_i = (\text{$x_i$ の順位}) - (\text{$y_i$ の順位})$$

ケンドールの順位相関係数 $\tau_K$

2組のデータ $(x_i, y_i)$ と $(x_j, y_j)$（$i < j$）について、 $(x_i - x_j)$ と $(y_i - y_j)$ が同符号（一致）の組数を $P$、 異符号（不一致）の組数を $N$ とするとき：

$$\tau_K = \frac{P - N}{\displaystyle\binom{n}{2}} = \frac{P - N}{n(n-1)/2}$$

2つの順位相関の比較

  • どちらも $-1$ から $1$ の値をとる
  • 同じデータでも値は異なる（例：$r_s \approx 0.86$, $\tau_K \approx 0.71$）
  • スピアマンは習慣的によく使われ、ケンドールは確率的な解釈が明確

8. 公式まとめ

検定統計量帰無仮説のもとでの分布（大標本）
ウィルコクソン順位和 $W_A$ $\mu = \frac{m(m+n+1)}{2}$, $\sigma^2 = \frac{mn(m+n+1)}{12}$
符号付き順位 $T_+$ $\mu = \frac{n(n+1)}{4}$, $\sigma^2 = \frac{n(n+1)(2n+1)}{24}$
符号検定 $T_+$ $\mathrm{Bin}(n, 0.5)$
クラスカル・ウォリス $H$ $\chi^2(k-1)$

9. 例題・演習

問13.1：血圧降下薬の効果比較（各種ノンパラ検定）

問題

新しい血圧降下薬（A薬）と従来薬（B薬）の効果を比較する。 帰無仮説を「2種類の薬の効果は等しい」、片側対立仮説を「A薬のほうが効果が高い（血圧をより下げる）」とする。

  1. 血圧がほぼ等しい高血圧患者6人をランダムに3人ずつに分け、 それぞれA薬・B薬を投与した。投与後の血圧（mmHg）が次のとおりであった。 ウィルコクソンの順位和検定を用いて片側P値を求めよ。
    A薬：135, 127, 131　　B薬：132, 144, 138
  2. 別の患者データで同じ仮説に対し順位和検定を行ったところ、片側P値が3%未満になった。 このとき、最低でも何人の患者がいたか（A群・B群の人数は等しいとする）。
  3. 別の3人の患者にA薬とB薬を時間を置いて投与した（対応あり）。 投与後血圧の差（B薬 $-$ A薬）が $-3, 17, 7$ であった。 符号付き順位検定を用いて片側P値を求めよ（A薬のほうが血圧を下げると言えるか）。
  4. 別のデータで符号付き順位検定を行ったとき、片側P値が5%未満になった。 最低何人の患者が必要か。
解答

(1) 順位和検定（6人：各3人）

6人全体を合わせて小さい順に並べると：127(A), 131(A), 132(B), 135(A), 138(B), 144(B)。

順位：A群 → 1位, 2位, 4位（順位和 $W_A = 7$）、B群 → 3位, 5位, 6位（$W_B = 14$）。

「A薬のほうが効果が高い（血圧が低い）」という対立仮説 → A群の順位が小さいほど支持される。 帰無仮説のもとで ${}_{6}C_3 = 20$ 通りの等確率な組合せがある。 $W_A \leq 7$ となるのは:

  • $\{1,2,3\} \to W_A=6$（1通り）
  • $\{1,2,4\} \to W_A=7$（1通り）
$$\text{片側P値} = \frac{2}{20} = \mathbf{0.10}$$

有意水準5%では棄却できません。

(2) 片側P値 < 3% となる最低人数

A群・B群が $k$ 人ずつ（計 $2k$ 人）のとき、P値の最小値は $1/{{}_{2k}C_k}$ です。

  • $k=3$：$1/{}_{6}C_3 = 1/20 = 5\%$→ 3%未満にならない
  • $k=4$：$1/{}_{8}C_4 = 1/70 \approx 1.4\%$ → 3%未満 ✅

しかし問題では「P値が3%未満になった」とあるので、最小でもそれが達成できる最低人数は $k=3$（各群3人）では最小P値が5%で3%未満にならないため $k=4$（各群4人）では ${}_{8}C_4=70$ ではなく 等しくない場合も考えます。A群3人・B群4人（7人）の場合 ${}_{7}C_3 = 35$、最小P値 $= 1/35 \approx 0.029 < 0.03$ ✅

最低7人（A群3人、B群4人）。

(3) 符号付き順位検定（差 $-3, 17, 7$）

$|D|$ を小さい順に並べると $|-3|=3, |7|=7, |17|=17$ → 順位 1, 2, 3。

符号付き順位：$D_1=-3 \to -1$、$D_3=7 \to +2$、$D_2=17 \to +3$。

実測 $T_+ = 2 + 3 = 5$（D>0の順位の和）。

「A薬が効果大＝差が負」→ $T_+$ が小さいほど対立仮説を支持 → 片側検定で $T_+ \leq 5$ の確率を求める。

$2^3 = 8$ 通りの符号割り当てのうち $T_+ \leq 5$ となるのは $T_+ \in \{0,1,2,3,4,5\}$：

  • $T_+=0: \{-1,-2,-3\}$（1通り）
  • $T_+=1: \{+1,-2,-3\}$（1通り）
  • $T_+=2: \{-1,+2,-3\}$（1通り）
  • $T_+=3: \{+1,+2,-3\},\;\{-1,-2,+3\}$（2通り）
  • $T_+=4: \{-1,+2,+3\}$は7→棄却。$T_+=1+3=4: \{+1,-2,+3\}$（1通り）
  • $T_+=5: \{+1,+2,-3\}=3,\; \{-1,-2,+3\}$計算済み。$T_+=2+3=5$は今回の実測。

整理すると $T_+ \leq 5$ は全8通りのうち：0,1,2,3,3,4,5,6（$T_+=0,1,2,3,4,5,6$が各1,1,1,2,1,1,1通り）

$T_+ \leq 5$ の場合の数 $= 1+1+1+2+1+1 = 7$。

$$\text{片側P値} = \frac{7}{8} \neq 0.25$$

対立仮説「A薬が効果大＝差(B-A)が負 → $T_+$ が小さい」の場合、$T_+=5$ は比較的大きい値であり、P値は大きくなります。

実際には「差が正 → B薬有利」という対立仮説で $T_+ \geq 5$ を見ると：$T_+=5,6$の場合の数$=1+1=2$。

$$\text{（B薬有利の} T_+ \geq 5\text{の）片側P値} = \frac{2}{8} = \mathbf{0.25}$$

A薬有利（差が負、$T_+$ が小さい）の片側P値 $= 7/8 = 0.875$。有意水準5%はもちろん棄却できません。

(4) 片側P値 < 5% となる最低人数

$n$ 人のとき最小P値は $1/2^n$。

  • $n=3$: $1/8 = 12.5\%$ → 5%未満にならない
  • $n=4$: $1/16 = 6.25\%$ → 5%未満にならない
  • $n=5$: $1/32 = 3.125\%$ → 5%未満 ✅

最低5人。

問13.2：学力補習の効果（大標本・符号付き順位検定）

問題

40人の学生に補習を行い、補習前後の点数の差 $D$ を求めた。 5人が差0点で、残り35人の正順位和が $T_+ = 420$ であった。 補習の効果があるか、有意水準5%の片側検定を符号付き順位検定（正規分布近似）で行え。

解答

差0の5人を除くと $n = 35$。

$$E[T_+] = \frac{35 \times 36}{4} = 315, \quad V[T_+] = \frac{35 \times 36 \times 71}{24} = 3727.5$$
$$Z = \frac{T_+ - E[T_+]}{\sqrt{V[T_+]}} = \frac{420 - 315}{\sqrt{3727.5}} = \frac{105}{61.05} \approx 1.72$$

片側検定の棄却域：$z_{0.05} = 1.645$、$Z = 1.72 > 1.645$。

帰無仮説を棄却できるので、有意水準5%で補習には効果があったといえます。

問13.3：4群のクラスカル・ウォリス検定

問題

4種類の肥料（A〜D）を使った植物の成長量（cm）を各5回測定した。 全 $N=20$ 個のデータに順位を付けたところ、各群の順位和が $R_A = 75, R_B = 70, R_C = 35, R_D = 30$ であった。 帰無仮説「4群の分布は同じ」を有意水準5%でクラスカル・ウォリス検定で考察せよ （$\chi^2_{0.05}(3) = 7.81$ を用いよ）。

解答

$N=20$、各群 $n_g=5$。順位平均 $\bar{R}_g = R_g/5$：

$$\bar{R}_A = 15,\; \bar{R}_B = 14,\; \bar{R}_C = 7,\; \bar{R}_D = 6$$

全体の順位中央値 $\bar{N} = (20+1)/2 = 10.5$。

$$H = \frac{12}{N(N+1)}\sum_{g} n_g(\bar{R}_g - \bar{N})^2$$ $$= \frac{12}{20 \times 21} \times 5 \times \left[(15-10.5)^2 + (14-10.5)^2 + (7-10.5)^2 + (6-10.5)^2\right]$$ $$= \frac{12}{420} \times 5 \times [20.25 + 12.25 + 12.25 + 20.25]$$ $$= \frac{12}{420} \times 5 \times 65 = \frac{3900}{420} \approx 9.29$$

$H \approx 9.29 > 7.81 = \chi^2_{0.05}(3)$ なので、帰無仮説を棄却します。

有意水準5%で4群の分布に差があるといえます。

10. まとめ・ポイント

  • ノンパラメトリック法の核心は「順位」を使うこと。数値の大小を順番だけに変換することで、分布の仮定なしに推論できます。
  • ウィルコクソンの順位和検定は2群の比較。帰無仮説のもと「${}_{m+n}C_m$ 通りの等確率な順位割り当て」から正確P値を求めます。
  • 符号付き順位検定は対応のある2測定の差を扱います。$D$ の分布の対称性が前提。大標本では正規近似で $Z$ 検定に帰着。
  • 符号検定は差の符号だけを使う最もシンプルな方法。前提条件なしに使えますが検出力は低め。$T_+ \sim \mathrm{Bin}(n, 0.5)$。
  • クラスカル・ウォリス検定は3群以上の比較（一元配置分散分析のノンパラ版）。大標本では $H \sim \chi^2(k-1)$。
  • スピアマン $r_s$ とケンドール $\tau_K$：どちらも順位相関だが計算法と意味が異なります。同じデータで異なる値になる。
  • パラメトリック法との使い分け：正規分布が信頼できるならパラメトリック法が検出力で有利。分布が不明・小標本・順序尺度ならノンパラが安全。
