概要

第16章では重回帰分析を扱う。ガウス・マルコフモデルによる定式化、 最小二乗推定量の性質、決定係数、F検定・t検定による変数の有意性評価、 正則化手法（Ridge回帰・Lasso・Elastic-Net）、およびモデル選択規準を学ぶ。

キーワード：ガウス・マルコフモデル、最小二乗法、決定係数、F検定、正則化

重要概念

ガウス・マルコフモデル

$n$ 個の観測に対し、$p$ 個の説明変数をもつ線形モデル：

$$Y = X\beta + \varepsilon, \quad \varepsilon \sim N_n(0, \sigma^2 I_n)$$
  • $Y$：$n \times 1$ の応答変数ベクトル
  • $X$：$n \times (p+1)$ の計画行列（第1列が1ベクトル → 切片）
  • $\beta = (\beta_0, \beta_1, \ldots, \beta_p)^T$：回帰係数
  • $\varepsilon$：誤差項（独立正規分布）

最小二乗推定量（OLS）

残差平方和 $\|Y - X\beta\|^2$ を最小化する $\hat{\beta}$：

$$\hat{\beta} = (X^T X)^{-1} X^T Y$$

正規方程式：$X^T X \hat{\beta} = X^T Y$

射影行列と予測値

$$\hat{Y} = X\hat{\beta} = P_X Y, \quad P_X = X(X^T X)^{-1}X^T$$

$P_X$ は $X$ の列空間への直交射影行列。残差 $e = Y - \hat{Y} = (I - P_X)Y$。

変動の分解

$$\underbrace{\|Y - \bar{Y}\mathbf{1}\|^2}_{S_T\text{（総変動）}} = \underbrace{\|\hat{Y} - \bar{Y}\mathbf{1}\|^2}_{S_R\text{（回帰変動）}} + \underbrace{\|Y - \hat{Y}\|^2}_{S_e\text{（残差変動）}}$$

決定係数

$$R^2 = \frac{S_R}{S_T} = 1 - \frac{S_e}{S_T}$$

$0 \leq R^2 \leq 1$。変数を追加すると必ず増加するため、自由度調整済み決定係数を用いる：

$$R^{*2} = 1 - \frac{S_e / (n - p - 1)}{S_T / (n - 1)}$$

OLS推定量の性質

  • 不偏性：$E[\hat{\beta}] = \beta$
  • 最小分散性（ガウス・マルコフの定理）：線形不偏推定量の中で分散最小
  • 分散共分散行列：$\text{Cov}[\hat{\beta}] = \sigma^2 (X^T X)^{-1}$
  • $\sigma^2$ の不偏推定量：$\hat{\sigma}^2 = S_e / (n - p - 1)$

F検定

帰無仮説 $H_0$：一部の回帰係数が0 を検定する。

検定対象帰無仮説F統計量
$\beta_k$ の有意性$H_0: \beta_k = 0$$\frac{(S_e^{(-k)} - S_e)/1}{S_e/(n-p-1)} \sim F(1, n-p-1)$
切片以外全体$H_0: \beta_1 = \cdots = \beta_p = 0$$\frac{S_R/p}{S_e/(n-p-1)} \sim F(p, n-p-1)$

個別の $\beta_k$ に対するF検定は $t$ 検定（$t^2 = F$）と等価。

正則化（ペナルティ付き回帰）

手法ペナルティ特徴
Ridge回帰$\lambda \sum \beta_j^2$（L2）係数を縮小、変数選択なし
Lasso$\lambda \sum |\beta_j|$（L1）係数をスパースに → 変数選択
Elastic-Net$\alpha\lambda\sum|\beta_j| + \frac{(1-\alpha)\lambda}{2}\sum\beta_j^2$L1とL2の混合
Fused LassoL1 + 隣接係数差のL1隣接するパラメータの融合

モデル選択：Mallows' $C_p$ 規準

$$C_p = \frac{S_e^{(p')}}{\hat{\sigma}^2} - n + 2(p'+1)$$

$p'$ は候補モデルの変数の数、$\hat{\sigma}^2$ はフルモデルの残差分散。$C_p \approx p'+1$ となるモデルが望ましい。

公式・定理

ガウス・マルコフの定理

ガウス・マルコフモデルのもとで、OLS推定量 $\hat{\beta}$ はBLUE（Best Linear Unbiased Estimator）、すなわちすべての線形不偏推定量の中で分散が最小。

分散分析表（ANOVA Table）

要因平方和自由度平均平方
回帰$S_R$$p$$S_R / p$
残差$S_e$$n - p - 1$$S_e / (n-p-1)$
全体$S_T$$n - 1$

Lasso の解パス

$\lambda$ を大きくすると多くの係数が0に → 変数が脱落。$\lambda$ を小さくするとOLS解に近づく。

例題・演習

問16.1：アミトリプチリン中毒データ

応答変数 TOT（トータル TCAD 血漿レベル）に対し、5つの説明変数（AMT, GEN, PR, DIAP, QRS）を用いた重回帰。

(1) モデル3の選択：

  • 3つの候補モデルのうち $C_p$ が最小のモデル3を選択
  • モデル3の $R^{*2} = 0.477$

(2) 有意な変数：

  • GEN（性別）：$p < 0.05$ で有意
  • AMT（服用量）：$p < 0.05$ で有意

問16.2：airquality データ

オゾン濃度を Solar.R, Wind, Temp, Month, Day で回帰。

(1) 非有意な変数：Day は有意でない

(2) AIC によるモデル選択：

  • 3つの候補モデルのうちモデル1（Day を除いたモデル）が AIC 最小

問16.3：AquaticToxicity データ（正則化）

9つの説明変数に対し Lasso を適用。

(1) $\lambda = 1$ のときの係数推定：

多くの係数が0になり、スパースな解が得られる。

(2) 解パス：$\lambda$ の変化に伴う各係数の推移を描画し、変数のスパース性を確認。

メモ

  • $R^2$ は変数追加で必ず増加 → モデル比較には $R^{*2}$ や AIC / $C_p$ を使う
  • 多重共線性がある場合、$(X^TX)^{-1}$ が不安定 → Ridge 回帰で対処
  • Lasso は変数選択と推定を同時に行える利点がある
  • F検定の $F$ 値と $t$ 検定の $t$ 値には $t^2 = F$ の関係がある（自由度1の場合）
  • 切片 $\beta_0$ はデータの中心化によって影響を受ける。F検定の「全変数」では切片を除いた仮説を検定
  • Elastic-Net は Lasso（$\alpha=1$）と Ridge（$\alpha=0$）の中間
