確率過程の基礎――「時間とともに変化する確率」を捉える

これまでの章では、ある固定した時点での確率変数やその分布を扱ってきました。しかし現実の世界では、株価・気温・地震の発生回数のように、値が時間とともにランダムに変化するものを扱う必要があります。このような「時間つきの確率変数の集まり」を確率過程（stochastic process）といいます。

本章で学ぶ4つのテーマ

  • ① 独立定常増分過程：確率過程の基本的な性質（増分の独立性・定常性）
  • ② ブラウン運動：連続パスを持つ確率過程。株価・為替のモデルに使われる
  • ③ ポアソン過程：地震・事故などの「稀なイベントの発生回数」のモデル
  • ④ 複合ポアソン過程：イベントの回数と1回あたりの大きさを同時にモデル化する拡張

ブラウン運動・ポアソン過程はどちらも「独立定常増分過程」として統一的に理解できます。まずその基本を押さえてから個々の過程に入りましょう。

1. 確率過程と独立定常増分過程

確率過程とは

時間 $t \geq 0$ に対して確率変数 $X_t$ が割り当てられているとき、その族 $$X = (X_t)_{t \geq 0}$$ を確率過程（stochastic process）といいます。

イメージ

  • $t$ が固定された一瞬 → $X_t$ は1つの確率変数（通常の確率論）
  • $t$ を動かす → 確率変数が時系列でつながって「軌道（パス）」を描く

時間が連続（$t \in [0,\infty)$）の場合を連続時間確率過程、離散（$t = 0,1,2,\ldots$）の場合を時系列といいます。

独立定常増分過程

応用上特に重要な確率過程は、次の2つの性質をもつものです。

定義：独立定常増分過程

確率過程 $X = (X_t)_{t \geq 0}$ が以下の2つを満たすとき、独立定常増分過程といいます。

（1）独立増分性：$0 = t_0 < t_1 < \cdots < t_n$ を任意にとると、

$$X_{t_0},\quad X_{t_1}-X_{t_0},\quad X_{t_2}-X_{t_1},\quad \ldots,\quad X_{t_n}-X_{t_{n-1}}$$

がすべて互いに独立。

（2）定常増分性：$X_{t+h} - X_t$ の分布が $t$ によらず $h$ だけで決まる（増分のスタート時点がどこでも関係ない）。

直感的なイメージ

独立増分性は「過去の変化と未来の変化は無関係」、定常増分性は「10分間の変化の分布は、それが朝9時から9時10分でも、午後2時から2時10分でも同じ」という意味です。

2. ブラウン運動（Brownian Motion）

定義

$B_0 = 0$ を初期値とし、以下の3つの性質を持つ確率過程 $B = (B_t)_{t \geq 0}$ をブラウン運動（Brownian motion）といいます。

ブラウン運動の定義

（1）独立定常増分過程である。

（2）各 $t > s \geq 0$ に対して、増分 $B_t - B_s$ は正規分布に従う：

$$B_t - B_s \;\sim\; N\!\left(\mu(t-s),\; \sigma^2(t-s)\right)$$

$\mu$：ドリフト（傾きの平均）、$\sigma^2$：拡散係数（ばらつきの大きさ）

（3）パス（軌道）が連続：時間が少し変化すると $B_t$ の値もわずかしか変わらない（ジャンプしない）。

特に $\mu = 0$, $\sigma^2 = 1$ の場合を標準ブラウン運動（またはウィーナー過程）といいます。

ブラウン運動の性質

定義(2)から直接導ける性質：

$$E[B_t] = \mu t, \qquad \mathrm{Var}[B_t] = \sigma^2 t$$

時間に比例して平均が増え、ばらつきも広がっていきます。

標準ブラウン運動との関係

標準ブラウン運動 $W = (W_t)_{t \geq 0}$ があれば、 $$B_t = \mu t + \sigma W_t$$ はドリフト $\mu$・拡散係数 $\sigma^2$ のブラウン運動になります。$B_{t+h} - B_t = \mu h + \sigma(W_{t+h}-W_t) \sim N(\mu h, \sigma^2 h)$ が確認できます。

3. ブラウン運動のパラメータ推定

設定

ブラウン運動の実現値を、時間間隔 $\Delta > 0$ ごとに測定してデータ $B_0, B_\Delta, B_{2\Delta}, \ldots, B_{n\Delta}$ を得たとします。ここから $\mu$ と $\sigma^2$ を推定します。

増分 $Z_k := B_{k\Delta} - B_{(k-1)\Delta}$（$k = 1, \ldots, n$）とおくと、独立定常増分性から $$Z_1, Z_2, \ldots, Z_n \quad \text{は独立に} \quad Z_k \sim N(\mu\Delta,\; \sigma^2\Delta)$$ に従います。平均 $\mu\Delta$・分散 $\sigma^2\Delta$ の正規分布から独立に $n$ 回サンプリングした構造と同じです。

最尤推定量

$$\hat{\mu} = \frac{\bar{Z}}{\Delta} = \frac{B_{n\Delta}}{n\Delta}$$ $$\hat{\sigma}^2 = \frac{1}{n\Delta}\sum_{k=1}^n(Z_k - \bar{Z})^2$$

$\hat{\mu}$ はスタートから終点までの全体の変化量 $B_{n\Delta}$ を総時間 $n\Delta$ で割るだけで、観測の細かさに関係なく始点と終点だけで決まります。

モーメント法による確認

1次モーメント（$E[Z_k] = \mu\Delta$ と標本平均を等置）：$\bar{Z} = \hat{\mu}\Delta \Rightarrow \hat{\mu} = \bar{Z}/\Delta$

2次モーメント（$E[Z_k^2] = \sigma^2\Delta + (\mu\Delta)^2$）：

$$\frac{1}{n}\sum_{k=1}^n Z_k^2 = \hat{\sigma}^2\Delta + (\hat{\mu}\Delta)^2 \quad \Rightarrow \quad \hat{\sigma}^2 = \frac{S_Z^2 - \hat{\mu}^2\Delta^2}{\Delta}$$

（$S_Z^2 = \frac{1}{n}\sum Z_k^2$）

高頻度観測（High-Frequency Observations）の重要結論

$\hat{\mu}$ と $\hat{\sigma}^2$ の精度の違い

  • $\hat{\sigma}^2$（分散の推定）：観測間隔 $\Delta$ を小さくして高頻度で観測すると推定精度が向上します。サンプルサイズが増えるためです。
  • $\hat{\mu}$（ドリフトの推定）：始点と終点のみで決まるため、観測間隔を細かくしても精度は改善しません。精度は総観測時間 $T = n\Delta$ にのみ依存します。

直感：ドリフト（全体的なトレンド）は「長期間眺める」ことで初めて推測できます。日中の細かい動きを観察しても、年間のトレンドはわかりません。

4. ポアソン過程（Poisson Process）

定義

$N_0 = 0$ を初期値とし（最初はイベントが0件）、以下の2つを満たす確率過程 $N = (N_t)_{t \geq 0}$ を強度 $\lambda$ のポアソン過程といいます。

ポアソン過程の定義

（1）独立定常増分過程である。

（2）任意の $t > s \geq 0$ に対して、増分はポアソン分布に従う：

$$N_t - N_s \;\sim\; \mathrm{Poi}\!\left(\lambda(t-s)\right)$$

平均と分散

$$E[N_t] = \lambda t, \qquad \mathrm{Var}[N_t] = \lambda t$$

$\lambda$ は単位時間あたりの平均発生件数（強度）を表します。

$N_t$ は「時刻 $t$ までに発生したイベントの累積件数」を表します。パスは非減少の右連続階段状になります（イベントが起きるたびに1ずつ増える）。

適用例

稀なイベントで、かつ過去の発生が未来に影響を与えない（独立）ような現象に使われます：地震発生回数、コールセンターへの着信、ウェブサイトへのアクセス件数など。

発生間隔と指数分布

$k$ 番目のイベントの発生間隔を $W_k = T_k - T_{k-1}$（$T_0 = 0$）とおきます。

定理：$N$ が強度 $\lambda$ のポアソン過程 $\Longleftrightarrow$ $W_1, W_2, W_3, \ldots$ が独立に指数分布 $\mathrm{Exp}(\lambda)$ に従う。

$$W_k \;\sim\; \mathrm{Exp}(\lambda), \quad f_W(w) = \lambda e^{-\lambda w} \quad (w > 0), \quad E[W_k] = \frac{1}{\lambda}$$

無記憶性との関係

指数分布は無記憶性（$P(W > s+t \mid W > s) = P(W > t)$）をもつ唯一の分布です。 「前回のイベントからどれだけ時間が経っていても、次が起きるまでの待ち時間の分布は変わらない」という性質が、ポアソン過程の独立増分性と整合しています。

5. ポアソン過程のパラメータ推定

強度 $\lambda$ を推定するデータの形式には2種類ありますが、どちらも同じ形の推定量になります。

形式 (I)：発生時刻 $T_1 < T_2 < \cdots < T_n$ を観測した場合

対数尤度（$W_k \sim \mathrm{Exp}(\lambda)$ より）：

$$\ell(\lambda) = \sum_{k=1}^n \log(\lambda e^{-\lambda W_k}) = n\log\lambda - \lambda\sum_{k=1}^n W_k$$

最尤方程式 $d\ell/d\lambda = 0$：

$$\frac{n}{\lambda} - \sum_{k=1}^n W_k = 0 \quad \Rightarrow \quad \hat{\lambda} = \frac{n}{\sum_{k=1}^n W_k} = \frac{n}{T_n}$$

形式 (II)：区間カウント $M_k = N_{k\Delta} - N_{(k-1)\Delta}$ を観測した場合

$M_k \sim \mathrm{Poi}(\lambda\Delta)$（独立）なので、対数尤度を $\lambda$ で微分して：

$$\hat{\lambda} = \frac{\sum_{k=1}^n M_k}{n\Delta} = \frac{N_{n\Delta}}{n\Delta}$$

共通の結論

$$\hat{\lambda} = \frac{\text{イベントの総件数}}{\text{観測時間}}$$

データ形式によらず常にこの直感的な形になります。「1時間に平均何件起きたか」をそのまま推定量にすればよいです。

6. 複合ポアソン過程

定義

ポアソン過程 $N_t$（強度 $\lambda$）の各イベントに「大きさ」$U_k$ を付与した累積和 $$X_t = \sum_{k=1}^{N_t} U_k$$ を複合ポアソン過程（compound Poisson process）といいます。$U_1, U_2, \ldots$ は互いに独立同分布で、$N_t$ とも独立です。

具体例：地震の総損害額

  • $N_t$：時刻 $t$ までの地震の発生件数（ポアソン過程）
  • $U_k$：$k$ 回目の地震による損害額
  • $X_t = \sum_{k=1}^{N_t} U_k$：時刻 $t$ までの総損害額

$U_k = 1$ のとき $X_t = N_t$（通常のポアソン過程）。

期待値と分散

$E[U_k] = \nu$、$E[U_k^2] = \nu^2 + \sigma_U^2$（$\mathrm{Var}[U_k] = \sigma_U^2$）とおきます。

計算のコツ：$N_t$ を既知（定数 $m$）として固定して計算し、後で $N_t$ の期待値をとる（条件付き期待値のタワー性）。

期待値：$N_t = m$ と固定すると $X_t = U_1+\cdots+U_m$ なので $E[X_t \mid N_t = m] = m\nu$。

$$E[X_t] = E\!\left[E[X_t \mid N_t]\right] = E[N_t \cdot \nu] = \nu\lambda t$$

分散：$\mathrm{Var}[X_t \mid N_t = m] = m\sigma_U^2$、$E[N_t^2] = \lambda t + (\lambda t)^2$ を使って：

$$\mathrm{Var}[X_t] = (\sigma_U^2 + \nu^2)\lambda t$$

公式まとめ（$E[U_k]=\nu$, $E[U_k^2]=\nu^2+\sigma_U^2$）

$$E[X_t] = \lambda t \cdot E[U_k], \qquad \mathrm{Var}[X_t] = \lambda t \cdot E[U_k^2]$$

7. 公式まとめ

過程増分の分布$E$, $\mathrm{Var}$パスの形
ブラウン運動 $B_t-B_s \sim N(\mu(t-s), \sigma^2(t-s))$ $\mu t$, $\sigma^2 t$ 連続
ポアソン過程 $N_t-N_s \sim \mathrm{Poi}(\lambda(t-s))$ $\lambda t$, $\lambda t$ 右連続階段状
複合ポアソン — $\lambda t \nu$, $\lambda t E[U^2]$ 右連続階段状

推定量まとめ

過程パラメータ最尤推定量
ブラウン運動 $\mu$ $\hat{\mu} = B_{n\Delta}/(n\Delta)$（始点・終点のみ）
ブラウン運動 $\sigma^2$ $\hat{\sigma}^2 = \frac{1}{n\Delta}\sum(Z_k - \bar{Z})^2$（高頻度で改善）
ポアソン過程 $\lambda$ $\hat{\lambda} = \text{総件数}/\text{観測時間}$

8. 例題・演習

問15.1：ブラウン運動のパラメータ推定

問題

ある株価が $S_t = S_0 + \sigma W_t$（$S_0 = 100$、$W_t$：標準ブラウン運動）でモデル化されている（ドリフト $\mu = 0$）。時間単位を「日」とし、毎日の終値を観測した。

  1. $n=200$ 日間の観測（$\Delta = 1$ 日）で、増分の2乗平均が $V_1 = \frac{1}{200}\sum_{k=1}^{200}(S_{k} - S_{k-1})^2 = 9.00$ であった。モーメント法により $\hat{\sigma}$ を求めよ。
  2. 同じ期間を2時間ごと（$\Delta = 1/12$ 日）に高頻度で観測した結果、$V_2 = \frac{1}{2400}\sum_{k=1}^{2400}(Z_k)^2 = 0.750$ であった。モーメント法により $\hat{\sigma}$ を求め、(1) の結果と比較せよ。
  3. もし株価がドリフト $\mu \neq 0$ のブラウン運動に従うとき、$\hat{\mu}$ の推定精度を改善するには「観測間隔を細かくする」「観測期間を延ばす」のどちらが有効か説明せよ。
解答

(1) $\Delta = 1$ 日の推定

増分 $Z_k = S_k - S_{k-1} \sim N(0, \sigma^2)$（$\Delta=1$, $\mu=0$）。2次モーメント $E[Z_k^2] = \sigma^2\Delta = \sigma^2$（$\Delta=1$）なので：

$$\hat{\sigma}^2 = V_1 = 9.00 \quad\Rightarrow\quad \hat{\sigma} = 3.00$$

(2) 高頻度（$\Delta = 1/12$ 日）の推定

$\Delta = 1/12$ のとき $E[Z_k^2] = \sigma^2\Delta = \sigma^2/12$ なので：

$$V_2 = \hat{\sigma}^2 \cdot \Delta \quad\Rightarrow\quad \hat{\sigma}^2 = \frac{V_2}{\Delta} = 0.750 \times 12 = 9.00 \quad\Rightarrow\quad \hat{\sigma} = 3.00$$

推定値は(1)と同じです。ただし観測数が200→2400（12倍）になったため、(2)の推定量の分散は小さく（精度が高く）なっています。

(3) $\hat{\mu}$ の精度改善

$\hat{\mu} = B_{n\Delta}/(n\Delta)$ はスタートと終点のみで決まります。観測間隔を細かくしても $T = n\Delta$ が変わらなければ精度は改善しません。

「観測期間を延ばす」ことが有効です。ドリフト（全体的なトレンド）は長期間観察することで初めて推測できます。

問15.2：ポアソン過程の強度推定

問題

あるコールセンターへの着信件数がポアソン過程 $N_t$（強度 $\lambda$）に従うとモデル化する。

  1. 8時間の営業時間中にちょうど24件の着信があり、最後の着信時刻が $T_{24} = 7.85$ 時間であった。最尤推定量 $\hat{\lambda}$ を求めよ。
  2. 別の日に、1時間ごとの着信件数（各時間帯で 3, 4, 2, 5, 3, 4, 2, 1 件）が記録された。最尤推定量 $\hat{\lambda}$ を求めよ。
  3. 2つの推定値を比較し、ポアソン強度推定量の一般的な形を述べよ。
解答

(1) 発生時刻データ

$$\hat{\lambda} = \frac{n}{T_n} = \frac{24}{7.85} \approx 3.06 \text{ 件/時間}$$

(2) 区間カウントデータ

合計 $= 3+4+2+5+3+4+2+1 = 24$ 件、観測時間 $= 8$ 時間：

$$\hat{\lambda} = \frac{24}{8} = 3.00 \text{ 件/時間}$$

(3) 比較と一般的な形

(1)は3.06、(2)は3.00で近い値です（(1)で $T_{24}=7.85 < 8$ のため$わずかな差がある）。どちらの形式でも：

$$\hat{\lambda} = \frac{\text{イベントの総件数}}{\text{観測時間}}$$

問15.3：複合ポアソン過程の期待値・分散

問題

ある保険会社が自動車事故の保険金支払いを複合ポアソン過程でモデル化する。

  • $N_t$：強度 $\lambda = 50$（件/年）のポアソン過程
  • $U_k$：$k$ 件目の保険金支払額（万円）、$E[U_k] = 30$, $E[U_k^2] = 1300$
  • $X_t = \sum_{k=1}^{N_t} U_k$：$t$ 年間の総支払額（万円）
  1. $E[X_t]$ と $\mathrm{Var}[X_t]$ を $t$ の関数として求めよ。
  2. $t = 2$（2年間）での平均と標準偏差を求めよ。
  3. タワー性 $E[X_t] = E[E[X_t \mid N_t]]$ を使って $E[X_t]$ を丁寧に導出せよ。
解答

(1) 期待値と分散の公式

$$E[X_t] = \lambda t \cdot E[U_k] = 50t \times 30 = 1500t \text{（万円）}$$ $$\mathrm{Var}[X_t] = \lambda t \cdot E[U_k^2] = 50t \times 1300 = 65000t \text{（万円}^2\text{）}$$

(2) $t = 2$ 年間

$$E[X_2] = 1500 \times 2 = \mathbf{3000} \text{ 万円}$$ $$\mathrm{SD}[X_2] = \sqrt{65000 \times 2} = \sqrt{130000} \approx 360.6 \text{ 万円}$$

(3) タワー性を使った導出

ステップ1：$N_t = m$ と固定 → $X_t = U_1 + \cdots + U_m$（独立和）

$$E[X_t \mid N_t = m] = m \cdot E[U_k] = 30m$$

つまり $E[X_t \mid N_t] = 30 N_t$（$N_t$ の関数）。

ステップ2：$N_t$ に関して期待値をとる（$E[N_t] = 50t$）：

$$E[X_t] = E[E[X_t \mid N_t]] = E[30 N_t] = 30 \cdot E[N_t] = 30 \times 50t = 1500t \quad \checkmark$$

9. まとめ・ポイント

  • 独立定常増分過程：「過去と未来の変化が無関係」「どの時点から測っても同じ増分分布」という2条件でブラウン運動とポアソン過程が統一的に理解できます。
  • ブラウン運動：増分が正規分布 $N(\mu(t-s), \sigma^2(t-s))$、パスが連続。$E[B_t]=\mu t$、$\mathrm{Var}[B_t]=\sigma^2 t$。
  • ポアソン過程：増分がポアソン分布 $\mathrm{Poi}(\lambda(t-s))$、パスが右連続階段状。発生間隔は $\mathrm{Exp}(\lambda)$（独立）。
  • 推定の非対称性（ブラウン運動）：$\hat{\sigma}^2$ は高頻度観測で精度向上、$\hat{\mu}$ は観測時間の長さのみに依存します。
  • ポアソン強度の推定：$\hat{\lambda} = \text{総件数}/\text{観測時間}$。データ形式によらず同じ形。
  • 複合ポアソン過程：$X_t = \sum_{k=1}^{N_t} U_k$。$E[X_t] = \lambda t E[U]$、$\mathrm{Var}[X_t] = \lambda t E[U^2]$。タワー性で計算します。
