はじめに：「yes/no」のデータを回帰する

これまで学んできた回帰分析は、応答変数 $y$ が連続値（身長・気温・売上など）の場合を想定していました。 しかし現実には、「病気か否か（0/1）」「高血圧か否か（0/1）」「コンサートに行く/行かない（0/1）」 といった2値データ（0か1か）や、 「1試合で決めたゴールの数（0, 1, 2, ...）」のような計数データを分析したいことがよくあります。

普通の回帰では何がまずい？

$y$ が0か1しかとらないのに、普通の線形回帰 $\hat{y} = \beta_0 + \beta_1 x$ を使うと 予測値が1より大きくなったり、0より小さくなったりします。 「確率が120%」「確率が-5%」は意味をなしません。 また、誤差が正規分布に従うという仮定も成り立ちません。

そこで、質的回帰（離散応答回帰）と呼ばれるモデルが使われます。代表的なものが：

  • ロジスティック回帰モデル：2値応答（0/1）を分析
  • プロビットモデル：ロジスティック回帰の代替（正規分布ベース）
  • ポアソン回帰モデル：計数データを分析
  • 一般化線形モデル（GLM）：上記を統一的に扱うフレームワーク

キーワード：ロジスティック回帰、オッズ比、プロビットモデル、ポアソン回帰、一般化線形モデル（GLM）

ロジスティック回帰

「オッズ」って何？

「競馬の3倍オッズ」という表現を聞いたことがあるかもしれません。 確率 $\pi$ で何かが起こる場合、オッズ（odds）とは「起こる確率」と「起こらない確率」の比です：

$$\text{オッズ} = \frac{\pi}{1-\pi}$$

たとえば確率 $\pi = 0.75$（75%）なら、オッズ $= 0.75/0.25 = 3$（3対1で起こる）。 オッズの取り得る値は $[0, \infty)$ で、$\pi = 0.5$ のとき オッズ $= 1$（1対1）。

さらにオッズの対数をとったものをロジット（logit）といいます。 ロジットは $(-\infty, +\infty)$ のすべての実数値をとれるので、右辺に線形式を当てはめられます：

$$\underbrace{\log \frac{\pi}{1 - \pi}}_{\text{ロジット}} = \beta_0 + \beta_1 x_1 + \cdots + \beta_p x_p$$

これがロジスティック回帰モデルです。左辺は $(-\infty, +\infty)$ の値をとるので、右辺の線形式と整合します。

確率への変換（ロジスティック関数）

右辺の値 $z = \beta_0 + \beta_1 x_1 + \cdots$ から確率 $\pi$ を求めるには、逆変換をします：

$$\hat{\pi} = \frac{e^z}{1 + e^z} = \frac{1}{1 + e^{-z}}$$

この関数をロジスティック関数（またはシグモイド関数）といいます。 入力がどんな値でも、出力は必ず $0$ と $1$ の間に収まります。

係数の解釈：オッズ比

係数 $\beta_j$ の解釈は少しユニークです。 $x_j$ を1増やすと、ロジットが $\beta_j$ 増える → オッズは $e^{\beta_j}$ 倍になります。 この $e^{\beta_j}$ をオッズ比といいます。

具体例：肥満と高血圧

「肥満」の係数が $\hat{\beta} = 0.695$ だったとします。

  • オッズ比 $= e^{0.695} \approx 2.0$
  • 意味：肥満の人は肥満でない人に比べて、高血圧になるオッズが約2倍

オッズ比が1より大きければ「リスクを高める」、1より小さければ「リスクを下げる」と解釈します。

母数の推定：最尤法

ロジスティック回帰では、最小二乗法は使えません。代わりに最尤法（likelihood method）を使います。 「このパラメータのもとでは、観測データが得られる確率が最も高くなるように推定する」方法です。 対数尤度関数は：

$$\ell(\beta) = \sum_{i=1}^{n} \left[ y_i \cdot \log \hat{\pi}_i + (1-y_i) \cdot \log(1-\hat{\pi}_i) \right]$$

この式を解析的に解く公式はないため、コンピュータの数値計算で最大化します。

プロビットモデル

正規分布を使ったロジスティック回帰の代替

プロビットモデルもロジスティック回帰と同様に2値応答を扱いますが、 ロジット関数のかわりに標準正規分布の累積分布関数 $\Phi(z)$ をリンク関数として使います：

$$\pi = P(Y=1) = \Phi(\beta_0 + \beta_1 x_1 + \cdots + \beta_p x_p)$$

$\Phi(z)$ は「標準正規分布で $z$ 以下の確率」です。$z$ が大きくなるほど $\Phi(z) \to 1$、小さくなるほど $\Phi(z) \to 0$ となり、 確率が0〜1の範囲に収まります。

潜在変数（latent variable）による解釈

プロビットモデルには直感的な「潜在変数」解釈があります：

潜在変数のイメージ

たとえば「高血圧か否か（0/1）」を判定する背後に、直接観測できない 「潜在的な血圧リスクスコア $Y^*$」があると考えます。

  • $Y^* \sim N(\beta_0 + \beta_1 x_1 + \cdots, \sigma^2)$（正規分布に従う）
  • $Y^*$ が閾値 $\tau$ を超えると $Y = 1$（高血圧）、超えなければ $Y = 0$

この確率が $P(Y^* > \tau) = \Phi(\beta_0 + \beta_1 x_1 + \cdots)$ となり、プロビットモデルが導かれます。

限界効果（marginal effect）

ロジスティック回帰では係数が「オッズ比の対数」として解釈できましたが、 プロビットモデルの係数 $\beta_j$ を直接「確率の変化量」と解釈できません。 代わりに限界効果を計算します：

$$\frac{\partial \pi}{\partial x_j} = \phi(\beta_0 + \beta_1 x_1 + \cdots + \beta_p x_p) \cdot \beta_j$$

$\phi(z)$ は標準正規分布の密度関数（確率密度）です。 「$x_j$ が1増えると確率 $\pi$ はどのくらい変化するか」の近似値を与えます。 この値は $x_1, \ldots, x_p$ の値によって変わる点に注意。

ロジスティック回帰との比較

比較項目ロジスティック回帰プロビットモデル
リンク関数ロジット $\log(\pi/(1-\pi))$正規 CDF の逆関数 $\Phi^{-1}(\pi)$
係数の解釈オッズ比：直感的限界効果：変化量
推定結果ほぼ同じ（大きく変わらない）ほぼ同じ（大きく変わらない）
理論的背景オッズのモデリング正規潜在変数モデル

ポアソン回帰

「数を数える」データとは

サッカーの1試合の得点数、1時間の交通事故件数、1年間の論文発表数… こういった非負の整数値（0, 1, 2, 3, ...）のデータを計数データといいます。 計数データには自然な確率分布としてポアソン分布がよく使われます。

ポアソン分布では平均（期待値） $\tau$ が重要なパラメータです。 問題は「$\tau$ は絶対に正（0より大きい）でなければならない」ことです。 線形回帰で $\tau = \beta_0 + \beta_1 x_1 + \cdots$ とすると、負の値が出てしまう可能性があります。

ポアソン回帰モデル

解決策は、$\tau$ の対数をとることです。対数は必ず値が区切られているので正の実数をとる $\tau$ と一対一対応します：

$$\log \tau = \beta_0 + \beta_1 x_1 + \cdots + \beta_p x_p$$

ゆえに $\tau = e^{\beta_0 + \beta_1 x_1 + \cdots}$ は必ず正の値になります。対数線形モデル（log-linear model）とも呼ばれます。

係数の解釈

$x_j$ を1増やすと、対数期待値が $\beta_j$ 増える → 期待値 $\tau$ は $e^{\beta_j}$ 倍になります。

サッカーの例

スプリント回数の係数 $\hat{\beta} = 0.0196$ なら：

  • スプリント1回増 → 得点の期待値 $e^{0.0196} \approx 1.02$ 倍
  • スプリント10回増 → 得点の期待値 $e^{0.196} \approx 1.22$ 倍（約22%増）

一般化線形モデル（GLM）

バラバラなモデルを「統一」する考え方

線形回帰・ロジスティック回帰・ポアソン回帰は、それぞれ別々のモデルのように見えますが、 実は一般化線形モデル（GLM: Generalized Linear Model）という一つの枠組みで統一的に表せます。

GLMは次の3つの要素からなります：

  1. 確率分布：$Y$ が従う分布（正規分布・ベルヌーイ分布・ポアソン分布など、指数型分布族）
  2. 線形予測子：$\eta = \beta_0 + \beta_1 x_1 + \cdots + \beta_p x_p$（説明変数の線形結合）
  3. リンク関数 $g$：$\eta = g(\mu)$（平均 $\mu = E[Y]$ と線形予測子をつなぐ関数）

指数型分布族とは

多くの有名な分布（正規・ベルヌーイ・ポアソン・ガンマなど）は、次の形に書けます：

$$f(y;\theta,\phi) = \exp\left\{\frac{y\theta - b(\theta)}{a(\phi)} - c(y,\phi)\right\}$$

$\theta$ は正準母数（その分布の「形」を決めるパラメータ）、$\phi$ は散らばりを表す母数です。この形に書ける分布の集まりを指数型分布族といいます。

各モデルの対応

応答の分布正準リンク関数モデル名典型的な応用
正規分布恒等 $g(\mu)=\mu$線形回帰連続値の予測
ベルヌーイ分布ロジット $\log\frac{\pi}{1-\pi}$ロジスティック回帰2値分類（0/1）
ベルヌーイ分布プロビット $\Phi^{-1}(\pi)$プロビットモデル2値分類（代替）
ポアソン分布対数 $\log\tau$ポアソン回帰計数データ

全てのモデルで、パラメータの推定は最尤法を使います。線形回帰以外は解析解がないため、 ニュートン・ラフソン法などの数値最適化アルゴリズムで求めます。

公式まとめ

ロジスティック回帰の対数尤度

$n$ 個の独立なベルヌーイ観測 $(y_1, ..., y_n)$ に対して：

$$\ell(\beta) = \sum_{i=1}^{n}\left[y_i \log\hat{\pi}_i + (1-y_i)\log(1-\hat{\pi}_i)\right] = y^T X\beta - \sum_{i=1}^{n}\log(1 + e^{x_i^T\beta})$$

生起確率の予測値

$$\hat{\pi}_i = \frac{\exp(x_i^T\hat{\beta})}{1 + \exp(x_i^T\hat{\beta})} = \frac{1}{1+e^{-x_i^T\hat{\beta}}}$$

ポアソン回帰の対数尤度

$$\ell(\beta) = \sum_{i=1}^n \left[y_i x_i^T\beta - e^{x_i^T\beta} - \log y_i!\right]$$

オッズ比の95%信頼区間

係数 $\hat{\beta}_j$ の標準誤差を SE とすると：

$$\text{オッズ比のCI} = \left[e^{\hat{\beta}_j - 1.96\,\text{SE}},\; e^{\hat{\beta}_j + 1.96\,\text{SE}}\right]$$

この区間に1が含まれなければ、$x_j$ の効果が有意（有意水準5%）です。 1が含まれれば「オッズ比が1（効果なし）と統計的に区別できない」ということです。

例題・演習

問18.1：がん患者の寛解確率（ロジスティック回帰）

問題

がん患者27人のLI値（腫瘍増殖指数：大きいほど活発）と寛解（治癒）のデータにロジスティック回帰を適用し、次の推定式を得た：

$$\log\frac{\hat{\pi}}{1-\hat{\pi}} = -3.77714 + 0.14486 \times \text{LI}$$
  1. 寛解確率が0.5となるLI値を求めよ。また LI = 30 のとき寛解確率を求めよ。
  2. LI値が2増えるとオッズはどのくらい変化するか。
解答

(1) 寛解確率が0.5となるLI値：

確率 $\pi = 0.5$ のとき、オッズ $= 0.5/0.5 = 1$ なので $\log \text{オッズ} = 0$。よって：

$$0 = -3.77714 + 0.14486 \times \text{LI} \implies \text{LI} = \frac{3.77714}{0.14486} \approx 26.1$$

LI ≈ 26 のとき寛解確率がちょうど50%です。

LI = 30 のとき：

$$\log\frac{\pi}{1-\pi} = -3.77714 + 0.14486 \times 30 = 0.5687$$
$$\hat{\pi} = \frac{e^{0.5687}}{1+e^{0.5687}} = \frac{1.766}{2.766} \approx 0.638$$

LI = 30 では寛解確率は約 64% です。

(2) LI値が2増えるときのオッズ変化：

$x$ が1増えるとオッズは $e^{0.14486} \approx 1.156$ 倍になるので、 2増えると：

$$(e^{0.14486})^2 = e^{0.28972} \approx 1.336$$

LI値が2上がるとオッズは約 1.34倍（34%増）になります。

問18.2：高血圧のロジスティック回帰

問題

40歳以上男性433人について以下の説明変数で高血圧（$y=1$）を予測するロジスティック回帰を行った：

  • $x_1 = 1$（喫煙あり）、$x_1 = 0$（喫煙なし）
  • $x_2 = 1$（肥満あり）、$x_2 = 0$（肥満なし）
  • $x_3 = 1$（いびきあり）、$x_3 = 0$（いびきなし）

推定式：

$$\log\frac{\hat{\pi}}{1-\hat{\pi}} = -2.378 - 0.068x_1 + 0.695x_2 + 0.872x_3$$
  1. 喫煙・肥満・いびきすべて「あり」の男性の高血圧確率を求めよ。
  2. 肥満のオッズ比を求め、その意味を説明せよ。
  3. 肥満のオッズ比の95%信頼区間を求めよ（SEは0.285）。
解答

(1) $x_1=x_2=x_3=1$ のとき：

$$\log\frac{\pi}{1-\pi} = -2.378 - 0.068 + 0.695 + 0.872 = -0.879$$
$$\hat{\pi} = \frac{e^{-0.879}}{1+e^{-0.879}} = \frac{0.415}{1.415} \approx 0.293 \approx 29\%$$

(2) 肥満のオッズ比：

$$e^{\hat{\beta}_2} = e^{0.695} \approx 2.00$$

意味：他の条件（喫煙・いびき）が同じであれば、肥満の男性は肥満でない男性に比べて高血圧のオッズが約2倍。

(3) 肥満オッズ比の95%信頼区間：

まず係数の95%信頼区間：

$$\hat{\beta}_2 \pm 1.96 \times \text{SE} = 0.695 \pm 1.96 \times 0.285 = [0.136,\; 1.254]$$

指数変換してオッズ比の信頼区間：

$$[e^{0.136},\; e^{1.254}] = [1.15,\; 3.51]$$

この区間に1が含まれていないので、肥満と高血圧の関係は統計的に有意(5%水準)。

問18.3：プロビットモデル

問題

問18.2と同じデータにプロビットモデルを適用した。推定式：

$$\hat{\pi} = \Phi(-1.373 - 0.039x_1 + 0.400x_2 + 0.465x_3)$$
  1. 喫煙・肥満・いびきすべて「あり」の高血圧確率を求めよ。
  2. 喫煙・いびきなしの男性において、肥満の限界効果を求めよ。
解答

(1) $x_1=x_2=x_3=1$ のとき：

$$z = -1.373 - 0.039 + 0.400 + 0.465 = -0.547$$
$$\hat{\pi} = \Phi(-0.547) \approx 0.292 \approx 29\%$$

ロジスティック回帰とほぼ同じ結果（29%）。2つのモデルの推定確率はほとんど変わらない。

(2) $x_1=0, x_2=0, x_3=0$ での肥満（$x_2$）の限界効果：

まず $x_2=0$ でのzの値：

$$z = -1.373 - 0.039 \times 0 + 0.400 \times 0 + 0.465 \times 0 = -1.373$$

標準正規密度関数の値（表より）：$\phi(-1.373) \approx 0.155$

$$\frac{\partial \hat{\pi}}{\partial x_2} = \phi(-1.373) \times 0.400 = 0.155 \times 0.400 \approx 0.062$$

意味：この条件（喫煙・いびきなし）の男性で、肥満になると高血圧確率が約6.2ポイント増加する。

問18.4：Jリーグのポアソン回帰

問題

横浜F・マリノスの試合別得点数（計数データ）をスプリント回数・走行距離で回帰した：

$$\log\hat{\tau} = 6.340 - 0.0813 \times \text{Dist} + 0.0196 \times \text{Sprint}$$

（Dist: 走行距離 [km]、Sprint: スプリント回数。Sprint のp値=0.016、Dist のp値=0.101）

  1. 各係数の解釈を述べよ。
  2. 有意な変数はどれか。
解答

(1) 係数の解釈：

  • 走行距離が1km増えると、得点の期待値は $e^{-0.0813} \approx 0.922$ 倍（約7.8%減少）
  • スプリント回数が1回増えると、得点の期待値は $e^{0.0196} \approx 1.020$ 倍（約2.0%増加）

走行距離が増えると得点が減るという結果は直感に反しますが、 「長い距離を走る試合 = 自チームがボールを保持できず走り回らされている試合」という解釈も可能です。

(2) 有意性：

  • Sprint（スプリント回数）：$p = 0.016 < 0.05$ → 有意（5%水準で有意）
  • Dist（走行距離）：$p = 0.101 > 0.05$ → 有意でない（5%水準では棄却されない）

まとめ・ポイント

3つのモデルの使い分け

応答の型使うモデル具体例
2値（0か1か）ロジスティック回帰 / プロビット病気の有無、合否、購買の有無
計数（0,1,2,...）ポアソン回帰事故件数、得点数、来店客数
連続値線形回帰身長、気温、売上
  • ロジスティック回帰の係数はオッズ比の対数→ $e^{\hat{\beta}_j}$ がオッズ比
  • オッズ比の信頼区間が1を含まなければ有意
  • プロビットモデルはロジスティック回帰とほぼ同じ結果を出す（違いは裾の形）
  • ポアソン回帰の係数は対数期待値の変化量→ $e^{\hat{\beta}_j}$ 倍の効果
  • GLMは「分布 + 線形予測子 + リンク関数」の3要素で定義され、最尤法で推定する
