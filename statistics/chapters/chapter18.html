概要

第18章では質的回帰（離散応答に対する回帰モデル）を扱う。 応答が2値（0/1）や計数値をとる場合に適用される、ロジスティック回帰モデル、 プロビットモデル、ポアソン回帰モデルを学び、これらを統一的に扱う 一般化線形モデル（GLM）の枠組みを理解する。

キーワード：ロジスティック回帰モデル、プロビットモデル、ポアソン回帰モデル、一般化線形モデル

重要概念

離散応答に対する回帰モデル

線形回帰分析は $(-\infty, \infty)$ の実数値をとる応答に正規性を仮定するが、応答が2値や計数値の場合には不適切。一般化線形モデルの枠組みが必要。

ロジスティック回帰モデル

応答 $Y \in \{0,1\}$ がベルヌーイ分布に従う場合、生起確率 $\pi = E[Y]$ に対して：

$$\log \frac{\pi}{1 - \pi} = \beta_0 + \beta_1 x_1 + \cdots + \beta_p x_p$$
  • ロジット変換：$\pi \mapsto \log\frac{\pi}{1-\pi}$（リンク関数）
  • ロジスティック変換：$z \mapsto \frac{1}{1+e^{-z}}$（逆リンク関数）
  • オッズ：$\frac{\pi}{1-\pi} = e^{\beta_0}(e^{\beta_1})^{x_1}\cdots(e^{\beta_p})^{x_p}$
  • $x_j$ を1増やすとオッズは $e^{\beta_j}$ 倍される
  • 母数推定は最尤法（対数尤度を数値的に最大化）

プロビットモデル

標準正規分布の累積分布関数 $\Phi(z)$ を用いて：

$$\pi = \Phi(\beta_0 + \beta_1 x_1 + \cdots + \beta_p x_p)$$
  • 潜在変数解釈：観測できない $Y^* \sim N(\mu, \sigma^2)$ が閾値 $\tau$ を超えると $Y=1$
  • 限界効果：$\frac{\partial \pi}{\partial x_j} = \phi(\beta_0 + \beta_1 x_1 + \cdots + \beta_p x_p)\beta_j$（$\phi$ は標準正規の密度関数）
  • ロジスティック回帰と異なり、推定量の直接的なオッズ比解釈ができないため限界効果で評価

ポアソン回帰モデル

応答 $Y$ が計数値（ポアソン分布）の場合：

$$\log \tau = \beta_0 + \beta_1 x_1 + \cdots + \beta_p x_p$$
  • 対数線形モデル（log-linear model）とも呼ばれる
  • $x_j$ を1増やすと期待値は $e^{\beta_j}$ 倍
  • ポアソン分布は稀な事象の生起回数のモデル

一般化線形モデル（GLM）

指数型分布族に属する分布を統一的に扱う枠組み：

$$f(y;\theta,\phi) = \exp\left\{\frac{y\theta - b(\theta)}{a(\phi)} - c(y,\phi)\right\}$$
  • $\theta$：正準母数（canonical parameter）, $E[Y] = b'(\theta)$, $V[Y] = b''(\theta)a(\phi)$
  • $\phi$：散らばり母数（dispersion parameter）
  • リンク関数 $g$：$g(\pi) = \beta_0 + \beta_1 x_1 + \cdots + \beta_p x_p$
分布正準母数 $\theta$リンク関数モデル名
正規分布$\mu$恒等関数線形回帰
ベルヌーイ分布$\log\frac{\pi}{1-\pi}$ロジット関数ロジスティック回帰
ベルヌーイ分布-プロビット関数プロビットモデル
ポアソン分布$\log\tau$対数関数ポアソン回帰

公式・定理

ロジスティック回帰の対数尤度

$$\ell(\beta) = y^T X\beta - \sum_{i=1}^{n}\log(1 + e^{x_i^T\beta})$$

生起確率の予測値

$$\hat{\pi}_i = \frac{\exp(x_i^T\hat{\beta})}{1 + \exp(x_i^T\hat{\beta})}$$

ポアソン回帰の対数尤度

$$\ell(\beta) = y^T X\beta - \sum_{i=1}^{n}(e^{x_i^T\beta} + \log y_i!)$$

オッズ比の信頼区間

$\hat{\beta}_j$ の近似的な95%信頼区間を $[\hat{\beta}_j \pm 1.96 \times \text{SE}]$ とすれば、対応するオッズ比の95%信頼区間は $[e^{\hat{\beta}_j - 1.96\text{SE}},\; e^{\hat{\beta}_j + 1.96\text{SE}}]$

例題・演習

問18.1：がん患者の寛解確率（ロジスティック回帰）

がん患者27人のLI値と寛解のデータにロジスティック回帰を適用。

推定式：$\log\frac{\hat{\pi}}{1-\hat{\pi}} = -3.77714 + 0.14486 \times \text{LI}$

(1) 寛解確率が0.5となるLI値：

$\pi=0.5$ のとき左辺＝0 なので $\text{LI} = 3.77714 / 0.14486 \approx 26.07$。すなわち LI ≈ 26 で寛解確率 0.5。

LI = 30 のとき：$\log\frac{\pi}{1-\pi} = -3.777 + 0.145 \times 30 = 0.569$ → $\hat{\pi} = \frac{e^{0.569}}{1+e^{0.569}} \approx 0.638$

(2) LI値が2増えるときのオッズ変化：

オッズは $(e^{0.14486})^2 = (1.1559)^2 \approx 1.34$ 倍になる。

問18.2：高血圧のロジスティック回帰

40歳以上男性433人の喫煙・肥満・いびきと高血圧の関連を分析。

推定式：$\log\frac{\hat{\pi}}{1-\hat{\pi}} = -2.378 - 0.068x_1 + 0.695x_2 + 0.872x_3$

(1) 全て「あり」の高血圧確率：

$\log\frac{\pi}{1-\pi} = -0.878$ → $\hat{\pi} = \frac{e^{-0.878}}{1+e^{-0.878}} \approx 0.29$

(2) 肥満のオッズ比：

$e^{0.695} \approx 2.00$（肥満の男性は肥満でない男性に比べて高血圧のオッズが約2倍）

(3) 肥満オッズ比の95%信頼区間：

$\hat{\beta}_2 = 0.695$, SE $= 0.285$ → $[0.695 \pm 1.96 \times 0.285] = [0.136, 1.254]$

オッズ比CI：$[e^{0.136}, e^{1.254}] = [1.14, 3.49]$（1を含まないので有意）

問18.3：プロビットモデル

問18.2と同じデータにプロビットモデルを適用。

(1) 全て「あり」の高血圧確率：

$\hat{\pi} = \Phi(-1.373 - 0.039 + 0.400 + 0.465) = \Phi(-0.547) \approx 0.29$（ロジスティック回帰とほぼ同じ結果）

(2) 肥満の限界効果：

喫煙・いびきなしの男性（$x_1=x_3=0$）について：

$\frac{\partial\pi}{\partial x_2}\bigg|_{x_2=0} = \phi(-1.373) \times 0.400 = 0.155 \times 0.400 \approx 0.062$

問18.4：Jリーグのポアソン回帰

横浜F・マリノスの得点数をスプリント回数・走行距離で回帰。

推定式：$\log\hat{\tau} = 6.340 - 0.0813 \times \text{Dist} + 0.0196 \times \text{Sprint}$

(1) モデルの解釈：

  • 走行距離1km増 → 得点数 $e^{-0.0813} \approx 0.92$ 倍
  • スプリント1回増 → 得点数 $e^{0.0196} \approx 1.02$ 倍

(2) 有意性：Sprint は有意（$p=0.016$）、Dist は非有意（$p=0.101$）。走行距離→得点減の解釈に統計的根拠は乏しいが、スプリント回数は有意に得点を説明。

メモ

  • ロジスティック回帰とプロビットモデルは2値応答に対する2大モデル → 多くの場合ほぼ同じ結果（π=0.5付近で差が出やすい）
  • ロジスティック回帰は係数がそのままオッズ比の対数 → 解釈が容易
  • プロビットモデルは潜在変数の解釈が可能 → 限界効果で評価
  • GLMの3要素：確率分布（指数型分布族）、線形予測子、リンク関数
  • GLMの母数推定はいずれも最尤法（陽に解けず数値的に解く）
  • オッズ比の信頼区間は回帰係数の信頼区間を指数変換して求める
